Looking at this problem:

We have segments. We want a closed star-shaped (with respect to origin) polyline using parts of existing roads, maximizing total length. Star-shaped means each ray from origin hits the belt exactly once.

So for each angle θ ∈ [0, 2π), exactly one point of the belt is at that angle. The belt is described by radius r(θ) for each angle. We want to maximize total arc length = ∫ sqrt(r² + r'²) dθ. But we use parts of segments.

A segment, parametrized by angle, gives a function r(θ) over some angular range (the angular interval the segment spans as seen from origin). For each angle we pick which segment covers it. The length contributed by a segment over an angular sub-interval is just the Euclidean length of that part of the segment.

So this is: we have angular intervals [a_i, b_i] each from a segment. We want to cover the full circle [0,2π) exactly, choosing for each angle which segment's piece, maximizing total length. This is like an interval covering / scheduling around a circle where we partition the circle into arcs, each arc assigned to one segment that spans it, and the length is the segment-length over that arc.

Key: a segment as seen from origin spans an angular interval. As angle increases, the point on segment moves monotonically. The length of segment piece from angle α to β equals the chord length between those two points.

We need to partition the full circle into consecutive arcs, each assigned to one segment covering that arc, maximizing sum of chord pieces. This is a shortest-path / DP on "breakpoints". Breakpoints are candidate angles (endpoints of segments). At each breakpoint we can switch segments. Between consecutive breakpoints, the assigned segment's contribution is its chord length over that angular range.

So: collect all candidate angles (endpoints of segments, normalized to [0,2π)). These define elementary arcs. For each elementary arc, find which segments cover it fully. ```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n; cin>>n;
    vector<array<double,4>> seg(n);
    vector<double> angs;
    for(auto&s:seg){cin>>s[0]>>s[1]>>s[2]>>s[3];
        for(int k=0;k<2;k++){double x=s[2*k],y=s[2*k+1];
            if(x==0&&y==0) continue;
            double a=atan2(y,x); if(a<0)a+=2*M_PI; angs.push_back(a);}}
    angs.push_back(0); angs.push_back(2*M_PI);
    sort(angs.begin(),angs.end());
    angs.erase(unique(angs.begin(),angs.end(),[](double a,double b){return fabs(a-b)<1e-12;}),angs.end());
    double total=0;
    bool covered=true;
    for(size_t i=0;i+1<angs.size();i++){
        double a1=angs[i],a2=angs[i+1];
        double mid=(a1+a2)/2;
        // direction of mid ray
        double dx=cos(mid),dy=sin(mid);
        double best=-1;
        for(auto&s:seg){
            // intersect ray param t*(dx,dy), t>0 with segment, and need segment covers both a1,a2
            // compute restriction of segment to angular interval [a1,a2]
            // find points on segment at angle a1 and a2
            // segment P=(s0,s1), Q=(s2,s3)
            double px=s[0],py=s[1],qx=s[2],qy=s[3];
            auto ptAt=[&](double ang,double&rx,double&ry)->bool{
                double cx=cos(ang),cy=sin(ang);
                // solve P + u*(Q-P) such that point at angle ang: cross of (point, dir)=0 and same side
                double ex=qx-px,ey=qy-py;
                double denom=cx*ey-cy*ex;
                if(fabs(denom)<1e-15) return false;
                double u=(cx*(-py)-cy*(-px))/(-denom);
                // line: point = P+u*E. angle condition: (P+uE) parallel to (cx,cy), t>0
                // (Px+u*Ex)*cy-(Py+u*Ey)*cx=0
                u=(py*cx-px*cy)/(ex*cy-ey*cx);
                if(u<-1e-9||u>1+1e-9) return false;
                rx=px+u*ex; ry=py+u*ey;
                double t=rx*cx+ry*cy;
                if(t<1e-9) return false;
                return true;
            };
            double r1x,r1y,r2x,r2y;
            if(!ptAt(a1,r1x,r1y))continue;
            if(!ptAt(a2,r2x,r2y))continue;
            double len=hypot(r1x-r2x,r1y-r2y);
            best=max(best,len);
        }
        if(best<0){covered=false;}
        else total+=best;
    }
    if(!covered) printf("0\n");
    else printf("%.6f\n",total);
    return 0;
}
