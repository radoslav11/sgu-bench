Looking at this problem:

Extremal permutations are "alternating permutations" (zigzag). The count of alternating permutations of n elements: for n≥2 it's 2*A(n) where A(n) is the number of "up-down" alternating permutations (zigzag numbers / tangent and secant numbers). Actually the total number of alternating permutations (both starting up and down) equals 2*E(n) for n≥2, where E(n) is the Euler zigzag number for one direction.

The Euler zigzag numbers satisfy the recurrence using the "boustrophedon transform" / triangle:
- T(0,0)=1
- Each row computed from previous row, accumulating.

Let me use the standard entringer/zigzag triangle. The number of alternating permutations starting with up = E_n (zigzag). For n=1, answer is 1. For n≥2 answer is 2*E_n.

The boustrophedon (Seidel) triangle: 
- a[0][0]=1
- For each row k: go alternately left-to-right or right-to-left accumulating sums.
- Zigzag number E_k = a[k][0] (leftmost when going right-to-left).

Let me recall Seidel-Entringer-Arnold. The Euler numbers via Seidel triangle where E_n = T_{n,n} or first entry. 

Standard: T(0,0)=1. For n>=1, fill row n: if n even, fill left to right T(n,0)=0... Let me just use known: the count of alternating permutations is A_n where A_0=A_1=1, and 2*A_n = sum_{k} C(n-1,k) A_k A_{n-1-k}... 

Simpler: use Entringer numbers. E(n,k). E(0,0)=1, E(n,0)=0 for n>0, E(n,k)=E(n,k-1)+E(n-1,n-k). Then zigzag number = E(n,n). Total alternating perms for n≥2 = 2*E(n,n).

Let me verify: n=3, E(3,3) should be 2 → 2*2=4. ✓

All arithmetic mod m.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    long long n, m;
    cin >> n >> m;
    if(m==1){ cout<<0<<endl; return 0;}
    if(n==1){ cout<<(1%m)<<endl; return 0;}
    // Entringer numbers
    // E(0,0)=1, E(n,0)=0 (n>0), E(n,k)=E(n,k-1)+E(n-1,n-k)
    // zigzag = E(n,n)
    int N = (int)n;
    vector<long long> prev(N+1,0), cur(N+1,0);
    prev[0]=1%m;
    for(int i=1;i<=N;i++){
        cur.assign(N+1,0);
        cur[0]=0;
        for(int k=1;k<=i;k++){
            cur[k]=(cur[k-1]+prev[i-k])%m;
        }
        prev=cur;
    }
    long long ans = (2*prev[N])%m;
    cout<<ans<<endl;
    return 0;
}
```
