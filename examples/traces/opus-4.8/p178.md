Breaking k links creates k single links plus k+1 segments. With k broken links, max chain = (k+1)*2^k + k... let me think.

Each broken link gives a single link as change. With k broken links, you have k pieces of size 1, and the remaining forms segments. The classic answer: with k breaks, you can handle up to (k+1)*2^k links.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    unsigned long long n;
    cin>>n;
    for(unsigned long long k=0;;k++){
        // max with k breaks: (k+1)*2^k
        unsigned long long cap;
        // careful overflow
        if(k>=60){cout<<k;return 0;}
        cap=(k+1)*(1ULL<<k);
        if(cap>=n){cout<<k;return 0;}
    }
}
```
