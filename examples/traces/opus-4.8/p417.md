Mass = ∫∫ ln(x²+y²) over disc.

Mean value property: for a harmonic function, the average over a disc equals the value at center... but ln(x²+y²) = 2·ln|z| is harmonic where defined, and since origin is outside the disc, it's harmonic throughout. 

So average over disc = value at center: ln(x0²+y0²).

Mass = π·r²·ln(x0²+y0²).

Check: π·4·ln(25) = π·4·3.2188758... = 40.4495865... ✓

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    double x0,y0,r;
    cin>>x0>>y0>>r;
    double mass=acos(-1.0)*r*r*log(x0*x0+y0*y0);
    printf("%.15f\n",mass);
}
```
