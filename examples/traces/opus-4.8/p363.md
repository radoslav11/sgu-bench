Looking at this problem:

- Each window has a queue. When a person arrives, first any people who finish at or before the arrival time leave their queues.
- The person picks the queue with least number of people (lowest count), tie broken by lowest window number.
- The person's finish time = (current time the window becomes free for them) + ti. Actually they start service after the people ahead finish.

Let me track for each window: a "finish time" = the moment when the last person in that queue leaves, and a count of people currently in queue.

When person arrives at time k:
1. Remove finished people: for each window, while the front person's leave time ≤ k... but actually we need to track count properly. We can store for each window the time when it becomes free (finishTime) and the number of people still in queue (those who haven't left by time k).

Let me think simpler. For each window, maintain a list (or just the leave times of people in queue). At arrival time k, count for each window how many people have leave time > k (still in queue). The leave time of a newly added person to window w = max(k, lastLeaveTime[w]) + ti, where lastLeaveTime is the leave time of the last person assigned to that window (if still relevant).

Actually the new person leaves at: max(k, previous person's leave time in that window) + ti. The previous person's leave time matters even if equal to k. If previous leaves exactly at k, that person has left (count excludes them), and the new person starts at k.

So for each window store `lastLeave` (leave time of most recently added person). Count of people in queue = number with leaveTime > k. To get count, store all leave times per window, or track count by removing those ≤ k.

Let me store a vector of leave times per window, and a pointer/count.

```cpp
for each window: vector<int> leaves; 
on arrival k:
  for each window: count = number of leaves entries > k
  pick window with min count, lowest index
  start = max(k, lastLeave[w])  // lastLeave = leaves.back() or 0
  leave = start + ti
  push leave to that window
```

Let me verify with example: windows finish times start 0.
P1 k=1,t=2: counts both 0, pick w1, start=max(1,0)=1, leave=3. Output 1 3. ✓
P2 k=3,t=4: w1 leaves={3}, >3? no, count0. w2 count0. pick w1, lastLeave=3, start=max(3,3)=3, leave=7. Output 1 7. ✓
P3 k=4,t=1: w1 leaves={3,7}, >4: just 7→count1. w2 count0. pick w2, start=max(4,0)=4, leave=5. Output 2 5. ✓
P4 k=5,t=2: w1: {3,7}>5→count1. w2:{5}>5→count0. pick w2, lastLeave=5,start=max(5,5)=5,leave=7. Output 2 7. ✓
P5 k=6,t=1: w1:{3,7}>6→count1(7). w2:{5,7}>6→count1(7). tie pick w1, lastLeave=7, start=max(6,7)=7,leave=8. Output 1 8. ✓

All match.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n,m;
    scanf("%d %d",&n,&m);
    vector<vector<int>> leaves(m+1);
    vector<int> lastLeave(m+1,0);
    for(int i=0;i<n;i++){
        int k,t;
        scanf("%d %d",&k,&t);
        int best=-1, bestCount=INT_MAX;
        for(int w=1;w<=m;w++){
            int cnt=0;
            for(int x:leaves[w]) if(x>k) cnt++;
            if(cnt<bestCount){bestCount=cnt;best=w;}
        }
        int start=max(k,lastLeave[best]);
        int lv=start+t;
        leaves[best].push_back(lv);
        lastLeave[best]=lv;
        printf("%d %d\n",best,lv);
    }
    return 0;
}
```
