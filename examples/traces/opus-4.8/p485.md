Looking at this problem:

- Sort all 3N numbers ascending.
- The N smallest go to B (they're subtracted, so we want them small, leaving the largest pool for A and C).
- For B, sorted descending as b': b'_i = (i-th largest of B).
- For the top 2N numbers (call them y, sorted ascending), pairing consecutive elements maximizes the sum of products Σ c_i·a_i (rank-paired partition). Each consecutive pair (y_{2i-1}, y_{2i}) forms one rank; choosing the smaller (y_{2i-1}) as the C value minimizes the subtraction term c_i·b'_i.

So the answer is:
Σ_{i=1}^N y_{2i-1}·(y_{2i} − b'_i)

int main(){
    int T, N;
    scanf("%d %d", &T, &N);
    while(T--){
        int M = 3*N;
        vector<long long> X(M);
        for(int i=0;i<M;i++) scanf("%lld", &X[i]);
        sort(X.begin(), X.end());
        // B = smallest N: X[0..N-1], b'_i = X[N-i] (i from 1..N), the i-th largest of B
        // y = X[N..3N-1] (top 2N), sorted ascending
        long long ans = 0;
