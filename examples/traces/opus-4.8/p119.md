Looking at this:

For all X,Y: N | A0·X+B0·Y ⟹ N | A·X+B·Y.

A pair (A,B) works iff (A,B) ≡ k·(A0,B0) mod N for some k... let me verify the condition properly.

The condition N | A0·X+B0·Y for all such X,Y implies N | A·X+B·Y. 

Take X=1,Y=0: not necessarily divisible. We need: whenever A0·X+B0·Y≡0, then A·X+B·Y≡0.

The set of (X,Y) with A0·X+B0·Y≡0 mod N forms a subgroup. A·X+B·Y must vanish on that subgroup. Simplest sufficient: (A,B) = k·(A0,B0) mod N. Let me check sample: A0=1,B0=2,N=3. k=0→(0,0), k=1→(1,2), k=2→(2,1). That gives exactly the sample output! So answer is all k·(A0,B0) mod N for k=0..N-1, deduplicated.

But is this complete? Need to confirm only multiples work. For sample it matches. Generally the condition is (A,B) is a linear combination preserving the kernel — multiples of (A0,B0) modulo N. Let me just generate all k·(A0,B0) mod N, dedupe, sort.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long N,A0,B0;
    cin>>N>>A0>>B0;
    A0%=N; B0%=N;
    set<pair<long long,long long>> s;
    for(long long k=0;k<N;k++){
        s.insert({(A0*k)%N,(B0*k)%N});
    }
    cout<<s.size()<<"\n";
    for(auto&p:s) cout<<p.first<<" "<<p.second<<"\n";
    return 0;
}
```
