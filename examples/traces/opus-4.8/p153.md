If N==0, the previous player took the last match and loses, so current player wins (the one to move at N=0... but you can't move). Let me define win[n] = true if player to move wins.

win[0]: no matches, previous player took last → previous loses, current wins? Player to move can't move. The player who took last loses, so previous player loses, current player wins. So win[0]=true.

For n≥1: player can move to n-1 or n-pi (if ≥0). If any reachable state has win=false, current wins.
