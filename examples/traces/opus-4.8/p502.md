Looking at this problem:

- n up to 10^17 means up to 18 digits.
- We need to permute digits to form a number divisible by 17, no leading zero.

Brute force over permutations is too much (18!). But we can use DP over digit counts and remainder.

State: counts of each digit used so far (multiset), position, remainder mod 17. Too big.

Better: DP where state = (mask of which counts remain)... but with up to 18 digits and duplicates.

Let me think: count of each digit 0-9. We build the number digit by digit. State = (remaining counts of each digit, remainder). The remaining counts space is product of (count_d+1). For 18 digits this could be large but typically manageable. Actually worst case all same... product small. Worst case spread: e.g. digits making product big. With 18 digits across 10 types, product of (c+1) could be up to ~ when distributed. Max product with sum 18 over counts: maximize product of (c_i+1) → roughly distribute, gives manageable size (like 2^... bounded). Combined with 17 remainders and position. Let's just do memoized DFS with map.

Let me do DFS: at each step pick a digit to place (from available counts), track remainder = (rem*10 + digit) % 17. Memoize on (counts tuple, remainder). Reconstruct path.

To avoid leading zero, first digit cannot be 0 (unless number is single digit "0", but n≥1 so... n could have value... n positive, could n be like just digits forming need). Actually first placed digit can't be zero.

Let me encode counts as a vector, use map<pair<array,int>,bool> for visited(failed) states. We want to find any solution.

```cpp
#include <bits/stdc++.h>
using namespace std;
int cnt[10];
int total;
string result;
set<string> failed;

string key(int rem){
    string s;
    for(int i=0;i<10;i++) s+=('0'+cnt[i]);
    s+=('a'+rem);
    return s;
}

bool dfs(int rem, int placed){
    if(placed==total) return rem==0;
    string k=key(rem);
    if(failed.count(k)) return false;
    for(int d=0;d<10;d++){
        if(cnt[d]==0) continue;
        if(placed==0 && d==0) continue; // no leading zero
        cnt[d]--;
        result.push_back('0'+d);
        if(dfs((rem*10+d)%17, placed+1)) { cnt[d]++; return true;}
        result.pop_back();
        cnt[d]++;
    }
    failed.insert(k);
    return false;
}

int main(){
    string n; cin>>n;
    for(char c:n) cnt[c-'0']++;
    total=n.size();
    // special case: if all zeros? n positive so not all zero unless n=0, but n>=1
    if(dfs(0,0)) cout<<result<<endl;
    else cout<<-1<<endl;
}
```
