Looking at this:

Start with 1. Apply amplifiers. We want to reach N.

Each step: X → 2X-1 (type1) or 2X+1 (type2).

Work backwards from N to 1. Reverse:
- 2X-1 = N → X = (N+1)/2 (N must be odd)
- 2X+1 = N → X = (N-1)/2 (N must be odd)

So N must be odd at each step except we start at 1. If N is even, no solution (since 2X±1 is always odd, and 1 is odd... starting value is 1 which is odd). Actually X starts at 1. All values produced are odd. So if N is even, no solution (unless N=... even can't be reached). 

Wait, N could be reached with zero amplifiers if N=1.

Working backwards: given N (odd, >1), it came from either (N+1)/2 via type1 or (N-1)/2 via type2. To minimize amplifiers... greedy: we want to reach 1 fastest. Going back, we roughly halve. To minimize steps, we want each backward step to reduce as much as possible, but both reduce to ~N/2. Choose the one giving smaller value? That reduces faster but both are about N/2 differing by 1. Let's think: we want to reach exactly 1. 

Let me just BFS/greedy backward, preferring to keep value odd and reach 1. Since both predecessors are (N±1)/2, and we need to continue until 1. Greedy: choose predecessor that's odd if needed... Actually both (N+1)/2 and (N-1)/2 — one is even one is odd (since N odd, N+1 and N-1 are even; (N+1)/2 and (N-1)/2 differ by 1, so one even one odd). We need next value to be reachable — value must eventually be 1, all intermediate values must be odd (except final 1 which is odd). So we must pick the odd predecessor each time. That's deterministic!

If both even... can't happen since they differ by 1. So pick the odd one. Type determines: if (N+1)/2 odd → type1, else (N-1)/2 odd → type2.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long N;
    cin>>N;
    if(N==1){cout<<0<<"\n"<<"\n";return 0;}
    if(N%2==0){cout<<"No solution\n";return 0;}
    vector<int> seq;
    long long x=N;
    while(x>1){
        if(x%2==0){cout<<"No solution\n";return 0;}
        long long a=(x+1)/2; // type1 predecessor: 2a-1=x
        long long b=(x-1)/2; // type2 predecessor: 2b+1=x
        // pick odd predecessor
        if(a%2==1){seq.push_back(1);x=a;}
        else {seq.push_back(2);x=b;}
        if(x==0){cout<<"No solution\n";return 0;}
    }
    reverse(seq.begin(),seq.end());
    cout<<seq.size()<<"\n";
    for(int i=0;i<(int)seq.size();i++){
        cout<<seq[i];
        if(i+1<(int)seq.size())cout<<" ";
    }
    cout<<"\n";
    return 0;
}
```
