Looking at this problem:

We have a regular N-gon with vertices numbered clockwise. We know positions of N1 and N2. The center is equidistant from all vertices. The angle between consecutive vertices around the center is 2π/N.

The angle between vertex N1 and N2 (going clockwise) is `k * (2π/N)` where `k = (N2 - N1) mod N`.

The center lies such that N1 and N2 are on a circle. The chord between them subtends angle `k*2π/N` at center.

Let me find the center. Given two points on a circle separated by angle `theta` (the arc), the center is at distance R from both. The midpoint M of the chord, and center is along perpendicular bisector. Distance from midpoint to center = (chord/2)/tan(theta/2).

Since islands are clockwise numbered, I need correct orientation. Let me set up: center C. Vertex i is at angle `phi_0 - i*(2π/N)` (clockwise means decreasing angle). 

Let me just compute center, then rotate N1 around center by appropriate angle to get each vertex.

The rotation from N1 to N2: going clockwise by k steps. Clockwise rotation by angle `a` is by `-a` in standard math. So to go from N1 position to N2 position, rotate around center by `-k*2π/N`.

Let me find center: from N1, rotating clockwise by `k*2π/N` gives N2. Center C satisfies: `N2 - C = Rot(-k*2π/N) * (N1 - C)`.

Let `a = k*2π/N`, R = rotation matrix for angle `-a`: 
`[cos a, sin a; -sin a, cos a]` (since rotating by -a: cos(-a)=cos a, sin(-a)=-sin a, matrix = [cos -sin; sin cos] with angle -a → [cos a, sin a; -sin a, cos a]).

So `N2 - C = M (N1 - C)`, giving `N2 - M*N1 = C - M*C = (I-M)C`, so `C = (I-M)^{-1} (N2 - M*N1)`.

Then vertex j = C + Rot(-(j-N1)*2π/N)*(N1-C).

Let me handle the case where I-M is singular (k such that a is multiple of 2π — won't happen since k in 1..N-1). When a near 0... k≥1 so fine. But if N1,N2 diametrically opposite, M = -I, I-M = 2I, invertible.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int N,n1,n2;
    cin>>N>>n1>>n2;
    double x1,y1,x2,y2;
    cin>>x1>>y1>>x2>>y2;
    int k = ((n2-n1)%N+N)%N;
    double a = k*2.0*M_PI/N;
    // M = [cos a, sin a; -sin a, cos a]
    double ca=cos(a), sa=sin(a);
    double m00=ca, m01=sa, m10=-sa, m11=ca;
    // (I-M)
    double b00=1-m00, b01=-m01, b10=-m10, b11=1-m11;
    // RHS = N2 - M*N1
    double r0 = x2 - (m00*x1+m01*y1);
    double r1 = y2 - (m10*x1+m11*y1);
    double det = b00*b11-b01*b10;
    double cx = (r0*b11 - b01*r1)/det;
    double cy = (b00*r1 - r0*b10)/det;
    cout<<fixed<<setprecision(6);
    for(int j=1;j<=N;j++){
        int d = ((j-n1)%N+N)%N;
        double ang = -d*2.0*M_PI/N;
        double cc=cos(ang), ss=sin(ang);
        double dx=x1-cx, dy=y1-cy;
        double nx = cx + cc*dx + (-ss)*dy; // rotation by ang: [cos -sin; sin cos]
        double ny = cy + ss*dx + cc*dy;
        cout<<nx<<" "<<ny<<"\n";
    }
}
```

Let me verify with sample: N=4, n1=1,n2=3, k=2, a=π. ca=-1,sa=0. M=[-1,0;0,-1]. I-M=[2,0;0,2]. RHS = N2 - M*N1 = (1,2)-(-1,0)=(2,2). C=(1,1). Vertex 2: d=1, ang=-π/2. cos=0,sin=-1. dx=0,dy=-1. nx=1+0*0-(-1)*(-1)=1-1=0. ny=1+(-1)*0+0*(-1)=1. → (0,1). Correct!
