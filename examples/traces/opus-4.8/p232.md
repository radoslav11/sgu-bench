For each starting index i, the sequence is D[(i + j*K) mod N] for j=0,1,2,...

We want to find the starting index whose sequence (first N digits, but compared as infinite sequence to break ties) is largest, then output first N digits.

We compare sequences for different starting points. Each sequence is periodic with period at most N (since indices cycle). We need to find the maximum among N candidate sequences, comparing digit by digit.

We can compare candidates by generating their sequences. To do this efficiently within memory/time, we use a tournament: keep set of candidate starting indices that are still "tied as maximum", compare them digit by digit.
