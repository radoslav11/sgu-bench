Looking at this problem:

- Grid of n rows, m columns of blocks.
- Pedestrian on streets/avenues (grid points (n+1)x(m+1)).
- Start north-west corner, facing east.
- When moving along an edge, that edge borders up to 2 blocks; each block passed gets b (first time) or b/2 (subsequent).

Let me set coordinates: point (r,c), r=0..n (street index), c=0..m (avenue). North-west corner is (0,0). Facing east means increasing c.

Directions: east = (0,+1), then L/R turns. Let me define direction vectors. East: dr=0,dc=+1. Left turn (90 CCW)... facing east, left turn = north (dr=-1). Right turn from east = south (dr=+1).

When moving from a point along an edge, the edge is between two points. A horizontal edge (moving in c direction) borders block above (row r-1) and below (row r). A vertical edge (moving in r direction) borders block left (col c-1) and right (col c).

Let me track visit count per block. Each move: determine the edge, find adjacent blocks (valid ones), add beauty.

Let me set directions array. Index dir: 0=east(0,+1),1=north(-1,0),2=west(0,-1),3=south(+1,0). L = turn left: east->north->west->south->east, so dir=(dir+1)%4. R = (dir+3)%4.

For a move in direction d from (r,c):
- If horizontal (dc!=0): edge row coordinate r, columns between c and c+dc. The block column index = min(c,c+dc). Blocks: row r-1 (above) and row r (below), column = that min. Valid if 0<=row<n.
- If vertical (dr!=0): edge column c, rows min(r,r+dr). Block row = that, columns c-1 (left), c (right). Valid if 0<=col<m.

Each passed block: satisfaction += first ? b : b/2, increment count.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n,m;
    cin>>n>>m;
    vector<string> g(n);
    for(int i=0;i<n;i++) cin>>g[i];
    string path; cin>>path;
    vector<vector<int>> cnt(n, vector<int>(m,0));
    int dr[4]={0,-1,0,1};
    int dc[4]={1,0,-1,0};
    int dir=0;
    int r=0,c=0;
    long long sat=0;
    auto pass=[&](int br,int bc){
        if(br<0||br>=n||bc<0||bc>=m) return;
        int b=g[br][bc]-'0';
        if(cnt[br][bc]==0) sat+=b;
        else sat+=b/2;
        cnt[br][bc]++;
    };
    for(char ch:path){
        if(ch=='L') dir=(dir+1)%4;
        else if(ch=='R') dir=(dir+3)%4;
        else{
            int nr=r+dr[dir], nc=c+dc[dir];
            if(dr[dir]==0){
                // horizontal move, edge at row r, column min(c,nc)
                int bc=min(c,nc);
                pass(r-1,bc);
                pass(r,bc);
            } else {
                int br=min(r,nr);
                pass(br,c-1);
                pass(br,c);
            }
            r=nr;c=nc;
        }
    }
    cout<<sat<<endl;
    return 0;
}
```

Let me verify with sample: grid 123/456/789, path MRMMLM, expected 22.

Start (0,0) dir east.
- M east: horizontal, row0,col0. pass(-1,0)skip, pass(0,0)=1. sat=1. move to (0,1).
- R: dir=(0+3)%4=3 (south).
- M south: vertical, edge col1, row min(0,1)=0. pass(0,0)? cnt was1 -> 1/2=0... pass(0,c-1=0)=1/2=0, pass(0,1)=2. sat=1+0+2=3. move to (1,1).
- M south: vertical col1 row min(1,2)=1. pass(1,0)=4, pass(1,1)=5. sat=3+4+5=12. move to (2,1).
- L: dir=(3+1)%4=0 east.
- M east: horizontal row2 col1. pass(1,1)? cnt1->5/2=2, pass(2,1)=8. sat=12+2+8=22. 

Output 22. Correct!
