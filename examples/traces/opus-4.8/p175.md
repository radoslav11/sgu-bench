Let me trace the recursion.

phi(W) with W = w1..wN, K = N/2.
- First half (positions 1..N-K of output) = phi(wN..wK+1) — the second part reversed
- Second half = phi(wK..w1) — the first part reversed

So letter wq: if q <= K, it's in second part (first K letters w1..wK reversed → wK..w1), offset by (N-K). Within that subproblem of length K, the reversed sequence is wK,wK-1,...,w1, so wq is at index (K - q + 1) in that subproblem.

If q > K, it's in first part of length N-K, sequence wN,wN-1,...,wK+1, so wq at index (N - q + 1), offset 0.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long N,q;
    cin>>N>>q;
    long long offset=0;
    while(N>1){
        long long K=N/2;
        if(q<=K){
            offset += (N-K);
            // subproblem length K, new q = K-q+1
            q = K-q+1;
            N = K;
        } else {
            // subproblem length N-K, new q = N-q+1
            q = N-q+1;
            N = N-K;
        }
    }
    cout << offset+1 << endl;
}
```
