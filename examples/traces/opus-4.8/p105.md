Looking at this:

The k-th term is formed by concatenating 1,2,...,k. A number is divisible by 3 iff sum of digits divisible by 3, which equals sum of 1+2+...+k = k(k+1)/2 mod 3.

Check k(k+1)/2 mod 3 for k=1..: 
- k=1: 1, no
- k=2: 3, yes
- k=3: 6, yes
- k=4: 10, no
- k=5: 15, yes
- k=6: 21, yes
- k=7: 28, no
- pattern period 3: positions k≡2,0 mod 3 → divisible.

So among k=1..N: divisible when k mod 3 == 0 or k mod 3 == 2.
Count = N - (number of k with k mod 3 == 1) = N - floor((N+2)/3).

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long n;
    cin>>n;
    long long c1=(n+2)/3; // k mod3==1
    cout<<(n-c1)<<endl;
}
```
