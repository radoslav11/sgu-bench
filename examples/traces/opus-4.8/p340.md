Let me analyze the structure. The input lines are wrapped in `$...$`. I process the content between dollars.

Key rules:
- Ignore whitespace
- Letters: wrap maximal sequences in `<i>...</i>`
- Binary operators `+-*/`: surround with `&nbsp;`
- `^{...}` or `^c`: `<sup>...</sup>`
- `_{...}` or `_c`: `<sub>...</sub>`

The example for `Bki` shows `B^{k_i}` produced `<i>B</i><sup><i>k</i><sub><i>i</i></sub></sup>`. So recursive processing inside braces.

Let me write a recursive parser. I need to track italic state — maximal sequences of letters should be wrapped. The tricky part: italic spans need to close before operators/sup/sub markers but the `<sup>` content itself starts fresh.

Let me think about italic grouping. A maximal sequence of letters gets one `<i>...</i>`. When I hit a non-letter (operator, paren, digit, ^, _), I close italic. Within `<sup>`/`<sub>`, it's a separate context.

Let me build output processing a string (already whitespace-stripped), with italic state managed.

I'll write a function that processes a substring and returns HTML. Italic state is local to each call (since braces start fresh contexts). Within a call, I track whether italic is open.

```
process(s):
  result, italicOpen=false
  i=0
  while i<len:
    c=s[i]
    if letter: if !italicOpen: result+="<i>"; italicOpen=true; result+=c; i++
    else:
      if italicOpen: result+="</i>"; italicOpen=false
      if operator: result+="&nbsp;"+c+"&nbsp;"
      elif c=='^' or '_': get content (brace group or single char), recurse, wrap in sup/sub
      else (digit/paren): result+=c
      i++
  if italicOpen: result+="</i>"
```

#include <bits/stdc++.h>
using namespace std;

string process(const string& s){
    string res; bool it=false;
    int n=s.size();
    for(int i=0;i<n;){
        char c=s[i];
        if(isalpha((unsigned char)c)){
            if(!it){res+="<i>";it=true;}
            res+=c;i++;
        } else {
            if(it){res+="</i>";it=false;}
            if(c=='+'||c=='-'||c=='*'||c=='/'){
                res+="&nbsp;";res+=c;res+="&nbsp;";i++;
            } else if(c=='^'||c=='_'){
                bool sup=(c=='^'); i++;
                string inner;
                if(i<n&&s[i]=='{'){
                    int depth=1;i++;
                    while(i<n&&depth>0){
                        if(s[i]=='{')depth++;
                        else if(s[i]=='}'){depth--;if(depth==0){i++;break;}}
                        inner+=s[i];i++;
                    }
                } else { if(i<n){inner+=s[i];i++;} }
                res+= sup?"<sup>":"<sub>";
                res+=process(inner);
                res+= sup?"</sup>":"</sub>";
            } else {
                res+=c;i++;
            }
        }
    }
    if(it)res+="</i>";
    return res;
}

int main(){
    string line;
    while(getline(cin,line)){
        // extract between first and last $
        size_t a=line.find('$');
        size_t b=line.rfind('$');
        string content;
        if(a!=string::npos&&b!=string::npos&&b>a){
            content=line.substr(a+1,b-a-1);
        }
        string stripped;
        for(char c:content) if(!isspace((unsigned char)c)) stripped+=c;
        cout<<process(stripped)<<"\n";
    }
}
