Looking at this problem: a synchrograph is a marked graph (Petri net where each place has one input and one output transition). 

Key insight: tokens are non-negative, so a directed circuit with total token count 0 means every arc on it has 0 tokens. Such a circuit is permanently dead — transitions on it can never fire. Deadness propagates forward along arcs (if transition `u` never fires, arc `u->v` dries up, eventually blocking `v` since `v` needs all incoming arcs positive).

So:
1. Find vertices lying on a cycle in the subgraph of 0-token arcs (SCC of size >1, or 0-token self-loop) — these are dead seeds.
2. Propagate deadness forward over all arcs.
3. A vertex is alive iff not dead.
