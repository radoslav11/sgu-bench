A bracket is NOT an emoticon (i.e., it's a "real" bracket) if:
- Opening bracket: nearest bracket to right is closing, with only spaces/letters between (and not immediately followed - well "()" is not part of emoticon... wait, re-read).

Let me re-read. "()" are NOT parts of emoticons. So we count brackets that ARE part of emoticons. Total brackets minus the "real" bracket pairs... but need careful logic.

A bracket is part of emoticon if it does NOT satisfy the conditions. Let me just compute for each bracket whether it's a "real" (non-emoticon) bracket per rules, then count those that are emoticons.

For opening bracket at i: find nearest bracket to right. If it's closing and between them only spaces/letters → real bracket. Note "()" : between them nothing, that's allowed (only spaces/letters, empty is ok)... but statement says "()" are not parts of emoticons. So that's consistent (they're real brackets).

For closing bracket at i: find nearest bracket to left. If it's opening and between only spaces/letters → real bracket.

Count = total brackets - real brackets.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    string s;
    getline(cin, s);
    int n = s.size();
    auto isBracket = [&](char c){ return c=='('||c==')'; };
    auto isLetterSpace = [&](char c){ return c==' '||(c>='a'&&c<='z')||(c>='A'&&c<='Z'); };
    
    int total = 0, real = 0;
    for(int i=0;i<n;i++){
        if(s[i]=='('){
            total++;
            // nearest bracket to right
            int j=i+1; bool ok=true;
            while(j<n && !isBracket(s[j])){
                if(!isLetterSpace(s[j])){ ok=false; break; }
                j++;
            }
            if(ok && j<n && s[j]==')') real++;
        } else if(s[i]==')'){
            total++;
            int j=i-1; bool ok=true;
            while(j>=0 && !isBracket(s[j])){
                if(!isLetterSpace(s[j])){ ok=false; break; }
                j--;
            }
            if(ok && j>=0 && s[j]=='(') real++;
        }
    }
    cout << (total-real) << endl;
    return 0;
}
```
