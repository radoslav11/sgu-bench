Looking at this problem:

- N up to 2,000,000, Q up to 10.
- We need to simulate erasing, but a naive linked list with counting Q steps each time is O(N*Q) = 20,000,000 which is fine.

Let me carefully understand the process with the example N=5, Q=3.

We start moving clockwise from 1. Numbers 1,2,3,4,5 clockwise.

Step: count Q-th integer in current direction and erase it. Then move to next integer clockwise. Check parity to set direction.

Let me simulate N=5,Q=3:
- Start at 1, direction clockwise. Count 3rd integer in clockwise direction. Counting from current position... need to determine if current counts as 1.

The expected order is 3,1,4,5,2.

Counting 3rd clockwise starting: positions 1,2,3 -> erase 3. That means starting at 1, count 1→1, 2→2, 3→3. So we count starting at current position, the Q-th is erased. Current was 1 (position). Counting 1,2,3 gives integer 3. Erase 3.

Then move to next integer clockwise from 3 → that's 4. 4 is even → direction counter-clockwise.

Now count Q-th (3rd) in counter-clockwise from current(4): 4(1),2(2),1(3) [3 erased, skip]. Erase 1. Matches (1 erased second).

Move to next clockwise from 1 → 2. 2 even → counter-clockwise.

Count 3rd CCW from 2: 2(1),5(2)[1 erased],4(3). Erase 4. Matches.

Move next clockwise from 4 →5. 5 odd→clockwise. Count 3rd CW from 5: 5(1),2(2),5(3)? Remaining are 2,5. 5(1),2(2),5(3). Erase 5. Matches.

Move next clockwise from 5→2. Count 3rd CW: only 2 left, erase 2. Matches. Last erased = 2. 

So counting starts AT current position (current counts as 1). Let me implement with a doubly linked list.

Let me verify N=5,Q=2 expected answer 3. Start at 1 CW, count 2: 1(1),2(2) erase 2. Move next CW from 2→3, odd→CW. Count 2 CW from 3: 3(1),4(2) erase 4. Move next CW→5 odd CW. count2: 5(1),1(2) erase1. next CW from1→3, odd CW. count2:3(1),5(2) erase5. next from5→3 count2 only 3: 3,3 erase3. Last=3. Correct!

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int N,Q;
    scanf("%d %d",&N,&Q);
    vector<int> nxt(N+1), prv(N+1);
    for(int i=1;i<=N;i++){ nxt[i]= (i==N)?1:i+1; prv[i]=(i==1)?N:i-1; }
    int cur=1; // position
    int dir=1; // 1 clockwise, 0 ccw
    int remaining=N;
    int last=0;
    while(remaining>0){
        // count Q-th in dir starting at cur (cur is 1st)
        int p=cur;
        int steps=(Q-1)%remaining;
        for(int s=0;s<steps;s++){
            p = dir? nxt[p] : prv[p];
        }
        last=p;
        // erase p
        int pn=nxt[p], pp=prv[p];
        nxt[pp]=pn; prv[pn]=pp;
        remaining--;
        if(remaining==0) break;
        // move to next integer clockwise from p
        cur = pn; // clockwise next
        // set direction based on parity of cur
        if(cur%2==1) dir=1; else dir=0;
    }
    printf("%d\n",last);
    return 0;
}
```
