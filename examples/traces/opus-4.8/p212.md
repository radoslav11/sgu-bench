Looking at this problem, I need to find a **blocking flow** in a layered graph (which is exactly what Dinic's algorithm computes in each phase).

The graph is already layered by levels 1 to L. Source is the node with level 1, sink is the node with level L. Edges go from level i to level i+1. So the entire graph is a DAG with edges only between consecutive levels — exactly the structure of Dinic's level graph.

I just need to compute a blocking flow on this layered network using Dinic's blocking flow procedure (DFS with current-arc optimization), starting from source to sink.

A blocking flow means: no augmenting path exists using only forward edges (no residual capacity path). Dinic's blocking flow phase computes exactly this.

Let me implement Dinic's blocking flow (single phase, since graph is already leveled by node levels).

I'll store edges, run repeated DFS augmentations using current-arc pointers until no more path from source.

Let me code it. Each original edge i has capacity c, flow = c - residual. Output flow = c - cap[edge].

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Edge { int to; long long cap; };
vector<Edge> edges;
vector<vector<int>> g;
vector<int> it; // current arc

long long dfs(int u, int sink, long long f) {
    if (u == sink) return f;
    for (int &i = it[u]; i < (int)g[u].size(); i++) {
        int id = g[u][i];
        if (edges[id].cap > 0) {
            long long d = dfs(edges[id].to, sink, min(f, edges[id].cap));
            if (d > 0) { edges[id].cap -= d; return d; }
        }
    }
    return 0;
}
```

Since this is a layered DAG, only forward edges; no back edges needed for blocking flow correctness in a DAG. Blocking flow just needs no path with residual remains.
