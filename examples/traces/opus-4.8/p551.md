Each person writes solutions at times t1, 2*t1, ... and at the moment they finish a solution, they check total ready count. If < n they continue.

Let me simulate: track count of solutions each has written, and decide whether to write next.

A person writes solution k (finishing at k*t). They start writing it if, when they finished solution k-1 (at time (k-1)*t), the total ready count was < n. For k=1, they always start (begin at time 0... actually they start at time 0). Wait, the first solution: they start at time 0, no prior finish. So they always write the first one.

Let me simulate step by step over events.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    long long n,t1,t2;
    cin>>n>>t1>>t2;
    // count[i] = solutions finished by person i
    // each person decides to start next based on ready count at their last finish time
    long long c1=0,c2=0; // finished solutions
    long long last1=0,last2=0; // time of last finish
    bool active1=true,active2=true; // currently writing (will produce next)
    // person starts writing solution number c+1, finishing at last + t
    // Initially both start writing first solution at time 0.
    // We process finishes in time order.
    long long totalFinished=0;
    long long lastTime=0;
    // simulate
    while(active1||active2){
        // next finish time
        long long n1 = active1? last1+t1 : LLONG_MAX;
        long long n2 = active2? last2+t2 : LLONG_MAX;
        long long t=min(n1,n2);
        // all finishes at time t happen
        if(active1 && n1==t){c1++;last1=t;totalFinished++;}
        if(active2 && n2==t){c2++;last2=t;totalFinished++;}
        lastTime=t;
        // now ready count = totalFinished (all finished by time t)
        // decide whether each continues: if just finished and total<n, continue
        if(active1 && last1==t){
            active1 = (totalFinished < n);
        }
        if(active2 && last2==t){
            active2 = (totalFinished < n);
        }
    }
    cout<<totalFinished<<" "<<lastTime<<"\n";
}
```
