Looking at this problem:

- We have a convex polygon, lamp at origin.
- Each segment blocks light: it casts a shadow on the polygon walls.
- Each segment, viewed from origin, covers an angular interval. The light rays going through angles covered by the segment hit the polygon wall in the shadowed region.
- We need total length of polygon perimeter that is shaded.

Each segment spans an angular range [a1, a2] (the angles of its two endpoints from origin, taking the shorter arc since segment doesn't contain origin). The union of these angular intervals determines which directions are shaded. For each shaded direction range, we find where rays at those angles hit the polygon boundary, and sum the lengths.

So: collect angular intervals (each less than π since segment is a chord not through origin). Union them. Then for the union of angular intervals, compute the length of polygon perimeter covered by rays in those angle ranges.

The polygon boundary: each vertex has an angle. A ray at angle θ hits the boundary at a point. The length of boundary covered between angle θ1 and θ2 corresponds to the actual polygon edge lengths intersected.

Approach: For an angular interval [a, b], the boundary points are: point where ray at angle a hits, then polygon vertices with angles in (a,b), then point where ray at angle b hits. Sum the segment lengths along these.

Let me precompute vertex angles. For each query interval, find intersection points with polygon edges.

Let me implement: function point-on-boundary at angle θ → find the edge whose angular span contains θ, intersect ray with that edge.

Since vertices given CCW, their angles are monotonic CCW (since origin inside convex polygon). So I can binary search by angle.

Let me sort vertices by angle (they're already CCW so increasing angle, but may wrap). I'll build arrays and for a given angle find the edge.

For computing length covered: integrate over union of angular intervals. For each interval, walk through edges.

To keep it simple given constraints, I'll merge intervals then for each merged interval find boundary length by computing intersection points at both ends and summing edges between, using vertex angles via binary search.

```cpp
#include <bits/stdc++.h>
using namespace std;
```
```cpp
int N,M;
vector<double> vx,vy,vang;
// find intersection of ray from origin at angle th with the edge that covers th
struct PT{double x,y;};
PT hit(double th){
    double dx=cos(th),dy=sin(th);
    // binary search edge: vertices sorted by angle in vang (increasing), wrap
    // find i such that angle in [vang[i],vang[i+1])
    int n=N;
    // vang sorted ascending; handle wrap by lower_bound
    int idx = upper_bound(vang.begin(),vang.end(),th)-vang.begin();
    // edge between idx-1 and idx (mod n)
    int a=(idx-1+n)%n, b=idx%n;
    double ax=vx[a],ay=vy[a],bx=vx[b],by=vy[b];
    // ray: t*(dx,dy); edge: (ax,ay)+s*(bx-ax,by-ay)
    double ex=bx-ax,ey=by-ay;
    double denom=dx*ey-dy*ex;
    double s=(dx*(ay)-dy*(ax))*1.0; // not used
    // solve t*dx = ax+s*ex ; t*dy=ay+s*ey
    // t = (ax*ey-ay*ex)/(dx*ey-dy*ex)
    double t=(ax*ey-ay*ex)/denom;
    return {t*dx,t*dy};
}
```

Now length of boundary in [a,b] (a<b, both in angle range matching vang): start hit(a), then vertices with angle in (a,b), then hit(b).

```cpp
int main(){
    scanf("%d %d",&N,&M);
    vx.resize(N);vy.resize(N);vang.resize(N);
    vector<pair<double,int>> tmp(N);
    vector<double> rx(N),ry(N);
    for(int i=0;i<N;i++){double x,y;scanf("%lf %lf",&x,&y);rx[i]=x;ry[i]=y;tmp[i]={atan2(y,x),i};}
    sort(tmp.begin(),tmp.end());
    for(int i=0;i<N;i++){int id=tmp[i].second;vx[i]=rx[id];vy[i]=ry[id];vang[i]=tmp[i].first;}
    vector<pair<double,double>> iv;
    for(int j=0;j<M;j++){
        double x1,y1,x2,y2;scanf("%lf %lf %lf %lf",&x1,&y1,&x2,&y2);
        double a1=atan2(y1,x1),a2=atan2(y2,x2);
        double lo=min(a1,a2),hi=max(a1,a2);
        if(hi-lo>M_PI){ // wrap, split
            iv.push_back({-M_PI,lo});
            iv.push_back({hi,M_PI});
        } else iv.push_back({lo,hi});
    }
    sort(iv.begin(),iv.end());
    double total=0;
    double cl=-1e18,cr=-1e18;
    auto proc=[&](double a,double b){
        if(b-a<1e-15)return;
        PT p=hit(a);
        double px=p.x,py=p.y;
        int idx=upper_bound(vang.begin(),vang.end(),a)-vang.begin();
        while(idx<N && vang[idx]<b){
            total+=hypot(vx[idx]-px,vy[idx]-py);px=vx[idx];py=vy[idx];idx++;
        }
        PT q=hit(b);
        total+=hypot(q.x-px,q.y-py);
    };
    for(auto&p:iv){
        if(p.first>cr){if(cr>-1e17)proc(cl,cr);cl=p.first;cr=p.second;}
        else cr=max(cr,p.second);
    }
    if(cr>-1e17)proc(cl,cr);
    printf("%.12f\n",total);
}
```
