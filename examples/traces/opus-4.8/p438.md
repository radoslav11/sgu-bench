Looking at this problem:

- Tourists start at bank y=0, need to reach y=W.
- They jump up to D meters. Each pile holds at most C tourists at a time.
- One jump = one second.
- Min time for all M tourists to cross.

This is a time-expanded max-flow problem. We split each pile into in/out nodes with capacity C (vertex capacity), and create time layers. We binary search / increment time T until max flow ≥ M.

Setup: Source connects to bank-start nodes (can jump from bank directly). Each time step, tourists at a pile at time t can move to reachable piles at time t+1, or to opposite bank if within D.

Approach: time-expanded graph. For time T (number of seconds), node = (pile, time). 
- Source -> piles reachable from start bank (y ≤ D) at time 1 (after first jump). Actually source represents bank at time 0.
- Each pile node at time t has capacity C: split into in/out.
- Edge from pile a (out) at time t to pile b (in) at time t+1 if dist ≤ D.
- A pile can also stay? Actually staying isn't a jump but capacity persists... A tourist at pile can stay (waiting). So edge from (pile,t) to (pile,t+1) with capacity C (staying), but each jump is a second. Waiting also costs a second but they stay on same pile. We allow waiting.
- Sink: pile reaches opposite bank if W - y ≤ D, edge to sink at any time.

Let me incrementally add time layers and compute max flow until ≥ M or until no improvement possible (cap at some max time, e.g. 2*N+M+... ; if after many layers no progress, IMPOSSIBLE).

Max time bound: if reachable at all, with capacities it could take up to ~ M + 2N seconds. Use limit like 4000 to be safe given small constraints. If flow can't reach M within reasonable bound, IMPOSSIBLE.

I'll build graph dynamically. Nodes: S=0, T=1. Pile p at time t: in = base + (t*N+p)*2, out = +1. Each new time layer we add nodes and edges, then run maxflow on residual (incremental). Add to flow accumulator. Stop when totalFlow >= M.

```cpp
#include <bits/stdc++.h>
using namespace std;
struct Dinic{
  struct E{int to,cap,rev;};
  vector<vector<E>> g; vector<int> level,iter;
  Dinic(int n=0){g.resize(n);}
  void resize(int n){g.resize(n);}
  void add(int a,int b,int c){g[a].push_back({b,c,(int)g[b].size()});g[b].push_back({a,0,(int)g[a].size()-1});}
  bool bfs(int s,int t){level.assign(g.size(),-1);queue<int>q;level[s]=0;q.push(s);while(!q.empty()){int v=q.front();q.pop();for(auto&e:g[v])if(e.cap>0&&level[e.to]<0){level[e.to]=level[v]+1;q.push(e.to);}}return level[t]>=0;}
  int dfs(int v,int t,int f){if(v==t)return f;for(int&i=iter[v];i<(int)g[v].size();i++){E&e=g[v][i];if(e.cap>0&&level[v]<level[e.to]){int d=dfs(e.to,t,min(f,e.cap));if(d>0){e.cap-=d;g[e.to][e.rev].cap+=d;return d;}}}return 0;}
  int maxflow(int s,int t){int fl=0;while(bfs(s,t)){iter.assign(g.size(),0);int f;while((f=dfs(s,t,INT_MAX))>0)fl+=f;}return fl;}
};
```

int main(){
  int N,M,D,W; cin>>N>>M>>D>>W;
  vector<int> X(N),Y(N),C(N);
  for(int i=0;i<N;i++)cin>>X[i]>>Y[i]>>C[i];
  if(W<=D){cout<<1<<endl;return 0;}
  int MAXT=300; // time limit search
  int total=2+MAXT*N*2;
  Dinic d(total);
  int S=0,T=1;
  auto in=[&](int t,int p){return 2+(t*N+p)*2;};
  auto out=[&](int t,int p){return 2+(t*N+p)*2+1;};
  auto sq=[](int x){return x*x;};
  int flow=0;
  for(int t=1;t<MAXT;t++){
    // add layer t
    for(int p=0;p<N;p++){
      d.add(in(t,p),out(t,p),C[p]);
      // from bank at time t-1 -> reach at t
      if(t==1){ if(Y[p]<=D) d.add(S,in(t,p),INT_MAX); }
      // from previous piles
      if(t>1){ for(int q=0;q<N;q++){ if(sq(X[p]-X[q])+sq(Y[p]-Y[q])<=D*D) d.add(out(t-1,q),in(t,p),INT_MAX);} }
      // to sink
      if(W-Y[p]<=D) d.add(out(t,p),T,INT_MAX);
    }
    flow+=d.maxflow(S,T);
    if(flow>=M){cout<<t<<endl;return 0;}
  }
  cout<<"IMPOSSIBLE"<<endl;
}
