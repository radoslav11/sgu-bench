Looking at this problem, I need to find the minimum time for a messenger to travel from 0 to B using horses from stables.

Key observations:
1. The messenger starts at position 0, needs to reach position B
2. At each stable, they can switch to any horse available there
3. A horse with speed v and max distance d can travel up to d units, taking time proportional to distance/speed
4. The messenger must always be on a horse (no walking)

Let me think about the DP approach:
- Sort stables by position
- State: dp[i] = minimum time to reach stable i while being "at" stable i and having just arrived (with some horse that still has remaining range, or switched to a new horse)

Wait, I need to track what horse the messenger is currently riding when at each stable, since the remaining distance of the horse matters for future travel.

Let me redefine:
- For each stable i, and each horse h available in stables ≤ i, track: minimum time to be at stable i riding horse h (with full range of horse h remaining, since you'd only pick up a horse at its stable).

Actually, the messenger picks up a horse at some stable j, then rides it continuously until either switching at a later stable or reaching B. The horse can travel at most d_h distance.

So: dp[i][h] = minimum time to arrive at stable i, having just picked up horse h at stable i.

Transition: From stable i with horse h (speed v, max distance d), the messenger can ride to:
- Any stable j where X_j - X_i ≤ d (horse can reach it)
- Point B if B - X_i ≤ d

Time spent riding from stable i to stable j = (X_j - X_i) / v

At stable j, the messenger can pick up any horse from stable j.

Let me implement this:
1. Sort stables by position
2. For each stable i, dp[i][h] for horse h in stable i = min time to arrive at stable i + pick horse h (arriving time is already computed)
3. Let `best[i]` = minimum time to arrive at stable i over all choices

Actually let's simplify: `arrive[i]` = minimum time to reach stable i. Then from stable i, for each horse h in stable i, we can go to stable j if X_j - X_i <= d_h, contributing arrive[i] + (X_j - X_i)/v_h to arrive[j].

But we also need to consider: the messenger might pass through a stable without stopping (or stop to change horses). Since changing is free and instantaneous, they'd always want to consider changing.

So: `arrive[i]` = min time to be at stable i. From there, try all horses in stable i to reach further stables or B.

This gives O(N^2 * horses_per_stable) which with N=5000 and total horses=10^5 could be up to 5*10^8... might be tight.

But average horses per stable = 20, so it's more like N^2 * 20 = 5*10^8 still large. However with 0.25s limit, need to be careful.

Let me just implement it and see. For each stable i, for each horse in stable i, for each stable j reachable, update arrive[j].

Also, stables at position 0 can be used as starting points.

The messenger starts at position 0. If there's a stable at position 0, they can use those horses. If not, they can't move (answer is -1).

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    long long B, N;
    cin >> B >> N;
    
    vector<long long> X(N);
    vector<vector<pair<long long,long long>>> horses(N);
    
    for(int i=0;i<N;i++){
        long long mi; cin>>X[i]>>mi;
        horses[i].resize(mi);
        for(int j=0;j<mi;j++) cin>>horses[i][j].first>>horses[i][j].second;
    }
    
    // Sort stables by position
    vector<int> idx(N); iota(idx.begin(),idx.end(),0);
    sort(idx.begin(),idx.end(),[&](int a,int b){return X[a]<X[b];});
    
    vector<double> dp(N, 1e18);
    double ans = 1e18;
    
    for(int ii=0;ii<N;ii++){
        int i=idx[ii];
        if(X[i]==0) dp[ii]=0;
        if(dp[ii]>=1e17) continue;
        for(auto [v,d]:horses[i]){
            if(X[i]+d>=B) ans=min(ans,dp[ii]+(double)(B-X[i])/v);
            for(int jj=ii+1;jj<N;jj++){
                int j=idx[jj];
                if(X[j]-X[i]>d) break;
                dp[jj]=min(dp[jj],dp[ii]+(double)(X[j]-X[i])/v);
            }
        }
    }
    
    if(ans>=1e17) cout<<-1<<endl;
    else cout<<fixed<<setprecision(8)<<ans<<endl;
}
```
