Looking at this problem, I need to find the minimum number of rays from origin (0,0) that hit all wolves but no sheep.

Key observations:
1. Each animal (wolf/sheep) corresponds to an angular interval as seen from the origin - the range of angles a ray from origin can take to intersect the segment.
2. We need to cover all wolf intervals with rays, such that no ray hits any sheep interval.
3. A ray hits a segment if the ray's angle is within the segment's angular interval.

So I need to find minimum number of "valid" angles (not in any sheep interval) that cover all wolf angular intervals.

This is essentially: find minimum number of points from the complement of sheep-intervals that cover all wolf-intervals.

**Approach:**
1. For each segment, compute its angular interval [lo, hi] (in angles from origin).
2. Wolf intervals must be "stabbed" by rays; sheep intervals must not be stabbed.
3. Find minimum number of rays (angles not in any sheep interval) that stab all wolf intervals.

The greedy approach: Sort wolf intervals by right endpoint, greedily pick the rightmost valid point in each interval.

**Angular interval computation:**
- For segment endpoints P1, P2, compute angles θ1=atan2(y1,x1), θ2=atan2(y2,x2).
- The interval is [min(θ1,θ2), max(θ1,θ2)] if the angular span < π, otherwise it wraps around.
- Actually need to be careful: the ray hits the segment if the angle is "between" the two endpoints angularly. Since Y >= 1, all points are in upper half-plane (y >= 1 > 0), so angles are in (0, π). No wraparound issues!

Since Y1, Y2 >= 1, all segments are strictly above x-axis. So all angles are in (0°, 180°) strictly. No wraparound.

**Algorithm:**
1. Compute angular intervals for all wolves and sheep.
2. Merge sheep intervals into a set of forbidden angular regions.
3. Use greedy interval stabbing: sort wolf intervals by right endpoint, for each unstabbed wolf interval (leftmost right endpoint), try to place a ray at its right endpoint if not forbidden by sheep; otherwise find the latest valid point within this interval.

For step 3: For each wolf interval [lo_w, hi_w], if it's not yet stabbed, we want to place a ray as far right as possible within [lo_w, hi_w] but not in any sheep interval.

To find the rightmost valid point in [lo_w, hi_w]: start from hi_w, check if it's in a sheep interval; if yes, move left to the start of that sheep interval minus epsilon, repeat.

I'll represent sheep intervals merged, then for a query "rightmost valid point in [a,b]", binary search.

Let me implement this with sorted merged sheep intervals.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N, M;
    cin >> N >> M;
    
    auto getInterval = [](double x1,double y1,double x2,double y2) -> pair<double,double> {
        double a1 = atan2(y1,x1), a2 = atan2(y2,x2);
        if(a1 > a2) swap(a1,a2);
        return {a1, a2};
    };
    
    vector<pair<double,double>> wolves(N), sheep(M);
    for(int i=0;i<N;i++){
        double x1,y1,x2,y2; cin>>x1>>y1>>x2>>y2;
        wolves[i] = getInterval(x1,y1,x2,y2);
    }
    for(int i=0;i<M;i++){
        double x1,y1,x2,y2; cin>>x1>>y1>>x2>>y2;
        sheep[i] = getInterval(x1,y1,x2,y2);
    }
    
    // Merge sheep intervals
    sort(sheep.begin(),sheep.end());
    vector<pair<double,double>> merged;
    for(auto& s: sheep){
        if(!merged.empty() && s.first <= merged.back().second)
            merged.back().second = max(merged.back().second, s.second);
        else merged.push_back(s);
    }
    
    // Greedy: sort wolves by right endpoint
    sort(wolves.begin(),wolves.end(),[](auto&a,auto&b){return a.second<b.second;});
    
    int shots = 0;
    double last_shot = -1e18;
    for(auto& w: wolves){
        if(last_shot >= w.first) continue; // already stabbed
        // Find rightmost valid point in [w.first, w.second]
        double pos = w.second;
        // Check sheep intervals overlapping [w.first, w.second]
        bool found = false;
        double cur = w.second;
        while(cur >= w.first){
            // find sheep interval containing cur
            auto it = upper_bound(merged.begin(),merged.end(),make_pair(cur,1e18));
            if(it != merged.begin()){
                --it;
                if(it->first <= cur && cur <= it->second){
                    cur = it->first - 1e-9;
                    continue;
                }
            }
            found = true; pos = cur; break;
        }
        if(!found) { cout << "No solution\n"; return 0; }
        last_shot = pos; shots++;
    }
    cout << shots << "\n";
}
```
