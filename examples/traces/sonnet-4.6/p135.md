The maximum number of regions created by N lines is given by the formula: R(N) = N*(N+1)/2 + 1

For N=0: 1, N=1: 2, N=2: 4, etc.

Note that N can be up to 65535, so N*(N+1)/2 can be up to about 65535*65536/2 ≈ 2.1 billion, which exceeds int range. Use long long.

```cpp
#include<iostream>
using namespace std;
int main(){
    long long n;
    cin>>n;
    cout<<n*(n+1)/2+1<<endl;
    return 0;
}
```
