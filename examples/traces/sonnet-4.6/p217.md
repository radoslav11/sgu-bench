Looking at this problem, I need to find the volume of intersection of two perpendicular cylinders whose axes intersect.

Let me set up coordinates. Let cylinder 1 have axis along the z-axis with radius R1, and cylinder 2 have axis along the x-axis with radius R2.

Cylinder 1: x² + y² ≤ R1²
Cylinder 2: y² + z² ≤ R2²

The intersection is the set of points satisfying both.

I need to compute the volume:
∫∫∫ [x² + y² ≤ R1²] [y² + z² ≤ R2²] dx dy dz

For fixed y, x ranges over [-√(R1²-y²), √(R1²-y²)] (if |y| ≤ R1), and z ranges over [-√(R2²-y²), √(R2²-y²)] (if |y| ≤ R2).

So the volume is:
∫_{y=-min(R1,R2)}^{min(R1,R2)} 2√(R1²-y²) · 2√(R2²-y²) dy

= 4 ∫_{-min(R1,R2)}^{min(R1,R2)} √(R1²-y²) · √(R2²-y²) dy

= 8 ∫_{0}^{min(R1,R2)} √((R1²-y²)(R2²-y²)) dy

Let me verify with R1=R2=R:
V = 8 ∫_0^R √((R²-y²)²) dy = 8 ∫_0^R (R²-y²) dy = 8[R²y - y³/3]_0^R = 8(R³ - R³/3) = 8·(2R³/3) = 16R³/3

For R=1: 16/3 ≈ 5.3333. ✓

Now I need to compute:
8 ∫_0^{min(R1,R2)} √((R1²-y²)(R2²-y²)) dy

This integral can be expressed in terms of elliptic integrals. Let's assume R1 ≤ R2 (WLOG, since they're symmetric... wait, not symmetric - if we swap we get same formula with R1,R2 swapped, but the integral limit changes).

Actually the formula is symmetric: 8 ∫_0^{min(R1,R2)} √((R1²-t²)(R2²-t²)) dt.

Let m = min(R1,R2), M = max(R1,R2).

Substitute t = m·sin(θ), dt = m·cos(θ)dθ, t: 0→m, θ: 0→π/2:
= 8 ∫_0^{π/2} √((m²-m²sin²θ)(M²-m²sin²θ)) · m·cos(θ) dθ
= 8 ∫_0^{π/2} m·cos(θ) · √(m²cos²θ · (M²-m²sin²θ)) dθ
= 8 ∫_0^{π/2} m²·cos²(θ) · √(M²-m²sin²θ) dθ
= 8m²M ∫_0^{π/2} cos²(θ) · √(1-(m/M)²sin²θ) dθ

This involves elliptic integrals. Let k = m/M, then:
= 8m²M ∫_0^{π/2} cos²(θ)·√(1-k²sin²θ) dθ

This equals 8m²M · [(E(k)(1-k²) + k²·... )] — complex formula.

I'll just use numerical integration (Gaussian quadrature or Simpson's rule with many steps).

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    double R1, R2;
    cin >> R1 >> R2;
    
    double lim = min(R1, R2);
    
    // Numerical integration using Simpson's rule
    int N = 1000000;
    double h = lim / N;
    double sum = 0.0;
    
    auto f = [&](double y) {
        return sqrt((R1*R1 - y*y) * (R2*R2 - y*y));
    };
    
    sum = f(0) + f(lim);
    for(int i = 1; i < N; i++){
        double y = i * h;
        sum += (i % 2 == 0 ? 2.0 : 4.0) * f(y);
    }
    sum *= h / 3.0;
    sum *= 8.0;
    
    cout << fixed << setprecision(4) << sum << endl;
    return 0;
}
```
