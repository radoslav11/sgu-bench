Looking at this problem, I need to calculate the total surface area of a 3D city model.

Let me think about how to compute the surface area:

1. **Top faces**: Each cell with height h > 0 contributes 1 to the top surface area (the top face of the tallest cube).

2. **Bottom faces**: Each cell with height h > 0 contributes 1 to the bottom surface area (the bottom face touching the ground).

3. **Side faces**: For each pair of adjacent cells, the exposed side area is |h1 - h2| (the difference in heights). For boundary cells, the exposed side area is h (the full height faces outward).

Let me verify with the examples:

**Example 1**: 3x3 grid
```
111
212
111
```

Top faces: 9 cells all non-zero → 9
Bottom faces: 9 → 9

Side faces:
- Horizontal adjacencies (between columns): each row has 2 pairs
  - Row 0: |1-1|=0, |1-1|=0
  - Row 1: |2-1|=1, |1-2|=1
  - Row 2: same as row 0: 0, 0
  Total horizontal internal: 2
- Vertical adjacencies (between rows): each column has 2 pairs
  - Col 0: |1-2|=1, |2-1|=1
  - Col 1: |1-1|=0, |1-1|=0
  - Col 2: same as col 0: 1+1=2
  Total vertical internal: 4
- Boundary faces:
  - Top row (i=0): 1+1+1 = 3
  - Bottom row (i=2): 1+1+1 = 3
  - Left col (j=0): 1+2+1 = 4
  - Right col (j=2): 1+2+1 = 4
  Total boundary: 14

Total = 9 + 9 + 2 + 4 + 14 = 38 ✓

Great, the approach works. Let me code this:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n, m;
    cin >> n >> m;
    
    vector<vector<int>> h(n, vector<int>(m));
    for(int i = 0; i < n; i++){
        string s;
        cin >> s;
        for(int j = 0; j < m; j++){
            h[i][j] = s[j] - '0';
        }
    }
    
    long long area = 0;
    
    for(int i = 0; i < n; i++){
        for(int j = 0; j < m; j++){
            if(h[i][j] > 0){
                area += 2; // top and bottom
            }
            // Internal horizontal (right neighbor)
            if(j+1 < m){
                area += abs(h[i][j] - h[i][j+1]);
            }
            // Internal vertical (down neighbor)
            if(i+1 < n){
                area += abs(h[i][j] - h[i+1][j]);
            }
            // Boundary
            if(i == 0) area += h[i][j];
            if(i == n-1) area += h[i][j];
            if(j == 0) area += h[i][j];
            if(j == m-1) area += h[i][j];
        }
    }
    
    cout << area << endl;
    return 0;
}
```
