Looking at this problem, I need to parse text and reformat it according to rules about hyphens and dashes.

Let me understand the rules:
1. Hyphen: `-` between two words (no spaces around it), like `co-coach`
2. Short dash: `--` with spaces on both sides, like ` -- `
3. Long dash: `---` with spaces on both sides, like ` --- `

The input text has words, spaces, and `-` characters. I need to reformat it.

First, let me tokenize the input into meaningful tokens:
- Words (sequences of letters)
- Sequences of `-` (1, 2, or 3 dashes)

Then, I need to figure out what each `-` sequence is and format appropriately.

Rules:
- If `-` is between two words (hyphen): `word-word` (no spaces)
- If `--` or `---` is a dash: ` -- ` or ` --- ` (spaces around)
- Between words with no dash: single space
- `-` alone (not between two words) is invalid

Let me tokenize: extract words and dash-sequences, ignoring spaces.

Tokens will be either WORD or DASH(count).

Then process the token sequence:
- A DASH(1) must be between two WORDs → hyphen
- A DASH(2) must NOT be between two words in a hyphen context → it's a short dash `--`
- A DASH(3) must NOT be between two words → long dash `---`
- Any other dash count → error

Wait, but `--` could appear in any position. Let me think about what's valid:
- DASH(1): must be flanked by words on both sides → `word-word`
- DASH(2): `--` → must have spaces around it (so cannot be directly adjacent to a word... but in our tokenized form, we just output ` -- `)
- DASH(3): `---` → similarly ` --- `
- DASH(4+): error

Also, `-` at start or end (not between words) is error.

So after tokenizing into words and dash sequences:
- DASH(1): previous and next token must both be WORDs → output without spaces: `word-word`
- DASH(2): output ` -- ` (with spaces)
- DASH(3): output ` --- ` (with spaces)
- Other dash counts: error

For words between each other (no dash), output single space.

- Process tokens left to right

#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n; cin.ignore();
    while(n--) {
        string line; getline(cin, line);
        // Remove surrounding quotes
        string text = line.substr(1, line.size()-2);
        // Tokenize
        vector<pair<int,string>> tokens; // type: 0=word, 1=dash
        int i = 0;
        while(i < text.size()) {
            if(isalpha(text[i])) {
                string w; while(i<text.size()&&isalpha(text[i])) w+=text[i++];
                tokens.push_back({0,w});
            } else if(text[i]=='-') {
                string d; while(i<text.size()&&text[i]=='-') d+=text[i++];
                tokens.push_back({1,d});
            } else i++; // space
        }
        // Validate and build output
        bool err = false;
        string out = "";
        for(int j = 0; j < tokens.size(); j++) {
            auto [type, val] = tokens[j];
            if(type == 1) { // dash
                int len = val.size();
                if(len == 1) {
                    // must be between two words
                    if(j==0||j==tokens.size()-1||tokens[j-1].first!=0||tokens[j+1].first!=0) { err=true; break; }
                    out += "-"; // will be concatenated with words
                } else if(len==2||len==3) {
                    // short/long dash
                    if(out.size()>0&&out.back()!=' ') out+=" ";
                    out += val;
                    if(j+1<tokens.size()) out+=" ";
                } else { err=true; break; }
            } else {
                if(j>0&&tokens[j-1].first==1&&tokens[j-1].second.size()==1) {
                    out += val; // hyphen case, no space
                } else {
                    if(out.size()>0&&out.back()!=' ') out+=" ";
                    out += val;
                }
            }
        }
        if(err) cout << "error\n";
        else cout << "\"" << out << "\"\n";
    }
}
