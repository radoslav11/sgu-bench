Looking at this problem, I need to find how many friends paid 1 ruble more than others across all M pubs.

Let me think through the approach:

For each pub with bill S and N friends:
- Each friend pays at least floor(S/N) rubles
- S mod N friends pay one extra ruble (floor(S/N) + 1)

So across all pubs, each friend pays some base amount plus potentially extra rubles. The total extra rubles distributed across all pubs is sum of (S_i mod N) for all i.

Total sum paid = sum of all S_i
Each person pays either floor(total/N) or floor(total/N) + 1.
X people pay floor(total/N) + 1, so:
X * (floor(total/N) + 1) + (N - X) * floor(total/N) = total
X = total mod N

Wait, but is it always achievable that exactly X = total mod N people pay 1 more? The problem says "the maximum difference for any of two friends between total sums they paid should be minimal" and "Obviously the value will not be more than 1 rub."

So the answer is simply (sum of all S_i) mod N.

Let me verify with examples:

Example 1: N=3, M=2, bills: 200, 100. Total = 300. 300 mod 3 = 0. Output: 0. ✓

Example 2: N=7, M=3, bills: 1, 4, 5. Total = 10. 10 mod 7 = 3. Output: 3. ✓

Great, the answer is just total_sum mod N.

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int N, M;
    cin >> N >> M;
    long long total = 0;
    for(int i = 0; i < M; i++){
        int s;
        cin >> s;
        total += s;
    }
    cout << total % N << endl;
    return 0;
}
```
