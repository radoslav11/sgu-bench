**Key observations:**
- For N players, each player plays N-1 games total
- If N is even: N-1 tours (each tour has N/2 matches, all players play each tour)
- If N is odd: N tours (each tour has (N-1)/2 matches, one player sits out each tour)

**Standard round-robin construction:**

**For even N:**
Fix player N-1 (0-indexed), rotate others around a circle.
- In round r (0-indexed), player N-1 plays against player r.
- The other pairs: player (r-1+k) mod (N-1) plays player (r-1-k) mod (N-1) for k=1..N/2-1.

**For odd N:**
Similar but player i sits out in round i (0-indexed), and the pairing is among remaining N-1 players using a circle.

using namespace std;

int main(){

    int T = M - 1;
    
    // schedule[i][j] = tour for match between players i and j (1-indexed, 1..M)
    vector<vector<int>> sched(M+1, vector<int>(M+1, 0));
    
    for(int r = 1; r <= T; r++){
        // M plays r
        sched[M][r] = r;
        sched[r][M] = r;
