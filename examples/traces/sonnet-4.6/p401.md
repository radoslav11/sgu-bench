Looking at this problem, I need to find how Dubrovsky crosses N rivers to maximize distance from start point, given total time t.

**Key observations:**
- For river i, width wi, flow vi, swim speed u
- Time to cross river i: ti = wi/u (he must swim perpendicular to maximize eastward progress... wait, no)
- Actually, he can choose the angle he swims. His velocity vector has magnitude u in any direction he chooses (relative to water). The river adds vi northward.
- To cross river i (west to east), he needs net eastward component = wi/ti, so his eastward speed component = wi/ti, and his northward speed component from himself = sqrt(u² - (wi/ti)²). Plus river adds vi northward.
- Constraint: ti ≥ wi/u (can't swim faster than u eastward)
- Total northward displacement from river i: ti * (sqrt(u² - (wi/ti)²) + vi)
- Total time: sum of ti = t (use all time to maximize displacement)
- Wait, does he need to use all t seconds? The problem says "by the time of sunset" - he wants maximum distance, so using all time optimally makes sense.

**Feasibility:** Each river requires minimum time wi/u. If sum(wi/u) > t, output -1.

**Optimization:**
Total north displacement = sum_i [ti * sqrt(u² - (wi/ti)²) + ti*vi]
= sum_i [sqrt(u²*ti² - wi²) + vi*ti]

Total east displacement = sum(wi) (fixed, must cross all rivers)

Maximize north displacement (since east is fixed, maximizing total distance from origin means maximizing north displacement to maximize sqrt(east² + north²)).

Wait, the start is on the left bank of the westernmost river. After crossing all rivers, east displacement = sum(wi). We want to maximize total distance = sqrt((sum wi)² + (north)²). So we maximize north.

North = sum_i [sqrt(u²*ti² - wi²) + vi*ti]

Subject to sum(ti) = t (use all time), ti ≥ wi/u.

Use Lagrange multipliers. For each i:
d/dti [sqrt(u²ti² - wi²) + vi*ti] = u²*ti/sqrt(u²ti² - wi²) + vi = λ

So u²*ti/sqrt(u²ti² - wi²) = λ - vi

Let xi = sqrt(u²ti² - wi²) (northward self-component * ti):
u²ti/xi = λ - vi
xi = u²ti/(λ-vi)
Also xi² = u²ti² - wi², so u⁴ti²/(λ-vi)² = u²ti² - wi²
wi² = u²ti²[1 - u²/(λ-vi)²] = u²ti²[(λ-vi)²-u²]/(λ-vi)²
ti = wi*(λ-vi)/[u*sqrt((λ-vi)²-u²)]

Binary search on λ. For ti to be valid, λ-vi > u, so λ > vi+u for all i.

Sum of ti = t. Binary search λ from max(vi)+u (exclusive) to some large value.

As λ→∞, ti→wi/u (minimum). As λ decreases toward max(vi+u), some ti→∞.

So binary search: find λ such that sum(ti(λ)) = t.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int N,u,t;
    cin>>N>>u>>t;
    vector<double> w(N),v(N);
    double minT=0;
    for(int i=0;i<N;i++){cin>>w[i]>>v[i]; minT+=w[i]/u;}
    if(minT>t+1e-9){cout<<-1;return 0;}
    double sumW=0; for(auto x:w) sumW+=x;
    auto calcT=[&](double lam)->vector<double>{
        vector<double> ti(N);
        for(int i=0;i<N;i++){
            double d=lam-v[i];
            if(d<=u){ti[i]=1e18;}
            else ti[i]=w[i]*d/(u*sqrt(d*d-u*u));
        }
        return ti;
    };
    double vmax=*max_element(v.begin(),v.end());
    double lo=vmax+u+1e-12, hi=1e12;
    for(int iter=0;iter<200;iter++){
        double mid=(lo+hi)/2;
        auto ti=calcT(mid);
        double s=0; for(auto x:ti) s+=x;
        if(s>t) lo=mid; else hi=mid;
    }
    auto ti=calcT((lo+hi)/2);
    double north=0;
    for(int i=0;i<N;i++) north+=sqrt(u*u*ti[i]*ti[i]-w[i]*w[i])+v[i]*ti[i];
    double dist=sqrt(sumW*sumW+north*north);
    cout<<fixed<<setprecision(10)<<dist<<"\n";
    for(int i=0;i<N;i++) cout<<fixed<<setprecision(10)<<ti[i]<<" \n"[i==N-1];
}
```
