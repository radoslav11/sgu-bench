Looking at this problem, I need to find for each city i (2 to N), the shortest path from capital (city 1) to city i when the last edge on the tree path T to city i is removed.

Let me understand the structure:
- T is a shortest path tree rooted at city 1
- For each city i, the "last road" on the tree path is the edge connecting i to its parent in T
- When that edge is removed, I need to find the new shortest path from 1 to i

**Approach:**
1. Build the shortest path tree T from the input
2. For each node i, compute dist[i] = distance from root to i using T edges
3. For each non-tree edge (u,v,l), it potentially offers an alternative path. Specifically, if a tree edge (parent[i], i) is removed, we need the best alternative.

**Key insight:** For a non-tree edge (u, v, l) connecting two nodes, it can "bypass" the last edge of any node in the subtree of i (where i is the deeper node). Specifically, if we're trying to reach node x (and its last tree edge is (parent[x], x)), we can potentially go: dist[u] + l + (dist from v to x not using last edge of x). But this gets complex.

Actually, let me think differently. For each node v, we want the minimum over all alternative paths. An alternative path to v must use at least one non-tree edge. The path would be: go along tree to some node u, take a non-tree edge (u,w,l), then follow tree path from w toward v.

For each non-tree edge (u, w, l) with weight l:
- This edge can help reach any node x that is on the path from w toward leaves, provided the "last edge" being removed is an ancestor edge of x that is below the LCA(u,w) in some sense... 

Actually, when the last edge of x (the edge from parent[x] to x) is removed, an alternative path via non-tree edge (u,w,l) works if:
- u is NOT in the subtree of x (or w is not in subtree of x), meaning we can reach u without using the removed edge
- w is in the subtree of x OR x is ancestor of w... 

Let me think again: the path is `1 -> ... -> u -> (non-tree edge) -> w -> ... -> x` where the tree path from w to x doesn't use edge (parent[x], x). So x must be in the subtree of w (meaning w is an ancestor of x or w=x). And the path from 1 to u must not use edge (parent[x], x), which means u is NOT in the subtree of x.

So for non-tree edge (u, w, l), and for any node x in subtree(w): cost = dist[u] + l + dist[x] - dist[w], provided u is not in subtree(x).

If u is not in subtree(w), then for any x in subtree(w), u is not in subtree(x), so the condition is satisfied.
If u is in subtree(w), say u is in subtree(c) where c is a child of w on path to u, then for nodes x in subtree(w) but NOT in subtree(c), u is not in subtree(x). For nodes x in subtree(c) including u itself... u is in subtree(x) only if x is an ancestor of u, but x is in subtree(c) so x must be between c and u on tree path.

This is getting complex. Let me use an Euler tour + segment tree approach where for each non-tree edge, I propagate the potential improvement to subtrees.

For each non-tree edge (u, w, l): it offers cost `dist[u] + l - dist[w] + dist[x]` for x in subtree(w) where u not in subtree(x). Since dist[x] is fixed, I want to minimize `dist[u] + l - dist[w]` over applicable edges for each x.

The "u not in subtree(x)" condition: if u is not in subtree(w), all x in subtree(w) are fine. If u IS in subtree(w), then x must not be an ancestor of u within subtree(w).

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll INF=1e18;
int par[100005];ll dist_[100005];int dep[100005];
vector<pair<int,ll>>tree[100005];
int in_[100005],out_[100005],timer_=0;
int anc[100005][17];
int n,m;
void dfs(int u,int p,int d,ll dd){dep[u]=d;dist_[u]=dd;in_[u]=timer_++;anc[u][0]=p;for(int i=1;i<17;i++)anc[u][i]=anc[anc[u][i-1]][i-1];for(auto[v,w]:tree[u])if(v!=p){par[v]=u;dfs(v,u,d+1,dd+w);}out_[u]=timer_-1;}
int lca(int u,int v){if(dep[u]<dep[v])swap(u,v);int diff=dep[u]-dep[v];for(int i=0;i<17;i++)if((diff>>i)&1)u=anc[u][i];if(u==v)return u;for(int i=16;i>=0;i--)if(anc[u][i]!=anc[v][i]){u=anc[u][i];v=anc[v][i];}return anc[u][0];}
ll seg[400005];
void upd(int node,int l,int r,int ql,int qr,ll val){if(ql>qr||r<ql||l>qr)return;if(ql<=l&&r<=qr){seg[node]=min(seg[node],val);return;}int mid=(l+r)/2;upd(2*node,l,mid,ql,qr,val);upd(2*node+1,mid+1,r,ql,qr,val);}
ll query(int node,int l,int r,int pos){ll res=seg[node];if(l==r)return res;int mid=(l+r)/2;if(pos<=mid)return min(res,query(2*node,l,mid,pos));else return min(res,query(2*node+1,mid+1,r,pos));}
int main(){
    scanf("%d%d",&n,&m);
    vector<tuple<int,int,ll,int>>edges(m);
    for(auto&[a,b,l,t]:edges){scanf("%d%d%lld%d",&a,&b,&l,&t);if(t)tree[a].push_back({b,l}),tree[b].push_back({a,l});}
    fill(seg,seg+4*(n+1),INF);
    dfs(1,1,0,0);
    for(auto&[a,b,l,t]:edges){
        if(!t){
            // non-tree edge (a,b,l)
            // For node x in subtree(b) not ancestor of a: val=dist_[a]+l-dist_[b]
            // For node x in subtree(a) not ancestor of b: val=dist_[b]+l-dist_[a]
            // handle both directions
            auto handle=[&](int u,int w){
                ll val=dist_[u]+l-dist_[w];
                // add val to subtree(w) minus ancestors of u within subtree(w)
                if(in_[w]>in_[u]||out_[w]<in_[u]){
                    // u not in subtree(w), update whole subtree(w)
                    upd(1,0,n-1,in_[w],out_[w],val);
                } else {
                    // u is in subtree(w), find child of w on path to u
                    int c=u;for(int i=16;i>=0;i--)if(dep[anc[c][i]]>dep[w])c=anc[c][i];
                    // update subtree(w) minus subtree(c)
                    upd(1,0,n-1,in_[w],in_[c]-1,val);
                    upd(1,0,n-1,out_[c]+1,out_[w],val);
                }
            };
            handle(a,b);handle(b,a);
        }
    }
    for(int i=2;i<=n;i++){
        ll q=query(1,0,n-1,in_[i]);
        ll ans=(q>=INF)?-1:q+dist_[i];
        printf("%lld",ans);if(i<n)printf(" ");
    }
    printf("\n");
}
