Looking at this problem, I need to find which pizza huts can be closed. A pizza hut A can be closed if at least K other pizza huts' pizza boys can reach it in 10 minutes.

Pizza boy from hut B (with speed w_B) can reach hut A if the Manhattan distance from B to A is ≤ w_B.

So: |x_A - x_B| + |y_A - y_B| ≤ w_B

This means: hut B can reach hut A if dist(A, B) ≤ w_B.

For each hut A, I need to count how many other huts B satisfy: Manhattan distance(A, B) ≤ w_B.

N = 10^5, so O(N^2) might be 10^10 which is too slow... but wait, 10^5 * 10^5 = 10^10. With simple operations and 1.5 seconds... probably too slow.

Let me think of a smarter approach. 

For hut B at (x_B, y_B) with speed w_B: it can reach all points (x, y) such that |x - x_B| + |y - y_B| ≤ w_B.

Using the rotation trick: let u = x + y, v = x - y. Then Manhattan distance becomes Chebyshev distance: max(|u_A - u_B|, |v_A - v_B|) ≤ w_B.

So hut B covers a square in (u,v) space: [u_B - w_B, u_B + w_B] × [v_B - w_B, v_B + w_B].

For each hut A (with coordinates u_A, v_A in transformed space), I need to count how many huts B (B ≠ A) have u_A ∈ [u_B - w_B, u_B + w_B] AND v_A ∈ [v_B - w_B, v_B + w_B].

This is equivalent to: for point (u_A, v_A), count rectangles containing it, where rectangle B is [u_B - w_B, u_B + w_B] × [v_B - w_B, v_B + w_B].

This is a 2D stabbing query problem. For N = 10^5, I can use sweep line + segment tree or BIT.

Approach:
- For each query point (u_A, v_A), count rectangles [l_B, r_B] × [b_B, t_B] containing it (where l_B = u_B - w_B, r_B = u_B + w_B, b_B = v_B - w_B, t_B = v_B + w_B).

Sweep line on u: 
- Events: for each rectangle B, add it at u = l_B, remove at u = r_B + 1.
- For each query point A, at u = u_A, count how many active rectangles have b_B ≤ v_A ≤ t_B.

Sort events by u coordinate. For each query point, it's a 1D stabbing query on v.

I'll coordinate compress v values. Use a BIT where I add +1 at b_B when rectangle starts, -1 at t_B+1, and query prefix sum at v_A.

Since coordinates can be large, I need to compress. All values are integers.

Let me implement this:

1. Compute u_i = x_i + y_i, v_i = x_i - y_i for all huts.
2. For each hut B, rectangle is [u_B - w_B, u_B + w_B] × [v_B - w_B, v_B + w_B].
3. Coordinate compress v values: collect all v_B - w_B, v_B + w_B, v_A values.
4. Sweep line on u: events sorted by u.
5. For each query point A at u_A, query BIT for stabbing count at v_A.
6. Count[A] = result - (does rectangle A contain point A, i.e., always true since dist(A,A)=0 ≤ w_A).

So I subtract 1 from count[A] since we count B ≠ A.

```cpp
#include <bits/stdc++.h>
using namespace std;
// BIT
struct BIT { vector<int> t; int n; void init(int n){this->n=n;t.assign(n+1,0);} void upd(int i,int v){for(i++;i<=n;i+=i&-i)t[i]+=v;} int qry(int i){int s=0;for(i++;i>0;i-=i&-i)s+=t[i];return s;} };
int main(){
    int N,K; scanf("%d%d",&N,&K);
    vector<long long> X(N),Y(N),W(N),U(N),V(N);
    for(int i=0;i<N;i++){scanf("%lld%lld%lld",&X[i],&Y[i],&W[i]);U[i]=X[i]+Y[i];V[i]=X[i]-Y[i];}
    // collect v coords
    vector<long long> vc;
    for(int i=0;i<N;i++){vc.push_back(V[i]-W[i]);vc.push_back(V[i]+W[i]);vc.push_back(V[i]);}
    sort(vc.begin(),vc.end());vc.erase(unique(vc.begin(),vc.end()),vc.end());
    int M=vc.size();
    auto ci=[&](long long x){return (int)(lower_bound(vc.begin(),vc.end(),x)-vc.begin());};
    // events: (u_val, type, data)
    // type: 0=rect_start(add), 1=query, 2=rect_end(remove)
    vector<tuple<long long,int,int>> evs;
    for(int i=0;i<N;i++){evs.push_back({U[i]-W[i],0,i});evs.push_back({U[i]+W[i],2,i});evs.push_back({U[i],1,i});}
    sort(evs.begin(),evs.end());
    BIT bit; bit.init(M);
    vector<int> cnt(N,0);
    for(auto&[u,tp,i]:evs){
        if(tp==0){bit.upd(ci(V[i]-W[i]),1);if(ci(V[i]+W[i])+1<M)bit.upd(ci(V[i]+W[i])+1,-1);}
        else if(tp==2){bit.upd(ci(V[i]-W[i]),-1);if(ci(V[i]+W[i])+1<M)bit.upd(ci(V[i]+W[i])+1,1);}
        else{cnt[i]=bit.qry(ci(V[i]))-1;}
    }
    vector<int> res;
    for(int i=0;i<N;i++)if(cnt[i]>=K)res.push_back(i+1);
    printf("%d\n",(int)res.size());
    for(int i=0;i<(int)res.size();i++)printf("%d%c",res[i]," \n"[i+1==(int)res.size()]);
    if(res.empty())printf("\n");
}
```
