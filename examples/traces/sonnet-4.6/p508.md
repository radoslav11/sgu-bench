Looking at this problem, I need to find the posterior probability distribution of k (number of black balls initially) given that we observed l1 black and l2 white balls drawn from n total balls.

**Prior:** P(k) = 1/(n+1) for k = 0, 1, ..., n (uniform)

**Likelihood:** P(l1 black, l2 white | k) = C(k, l1) * C(n-k, l2) / C(n, l1+l2)

This is the hypergeometric distribution.

**Posterior:** P(k | observations) ∝ P(k) * P(observations | k) ∝ C(k, l1) * C(n-k, l2)

Note: C(k, l1) = 0 if k < l1, and C(n-k, l2) = 0 if n-k < l2, i.e., k > n-l2.

So valid k range is [l1, n-l2].

I need to compute these values carefully. Since n ≤ 50, I can work with exact values or use logarithms to avoid overflow.

Let me use logarithms of the weights: log_weight(k) = log(C(k, l1)) + log(C(n-k, l2))

Then normalize to get probabilities.

For the segment [a, b] with minimum length such that sum of probabilities ≥ p/100, and among all such segments choose the one with smallest a.

Let me compute log of binomial coefficients carefully, then exponentiate relative to the max to get normalized probabilities.

Steps:
1. Compute log_w[k] for k = 0..n
2. Find max log_w among valid k
3. Compute w[k] = exp(log_w[k] - max_log_w) for valid k, 0 for invalid
4. Normalize to get prob[k]
5. Find minimum window [a,b] such that sum prob[a..b] >= p/100

For the sliding window approach: try all possible window sizes from 0 to n, find first size where some window achieves >= p/100, pick the one with smallest a.

Actually, iterate over all possible pairs (a, b) with minimum b-a first, then pick smallest a.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, l1, l2, p;
    cin >> n >> l1 >> l2 >> p;
    
    double target = p / 100.0;
    
    // log binomial coefficient
    auto logC = [&](int nn, int kk) -> double {
        if (kk < 0 || kk > nn) return -1e18;
        if (kk == 0 || kk == nn) return 0.0;
        double res = 0;
        for (int i = 0; i < kk; i++) {
            res += log((double)(nn - i)) - log((double)(i + 1));
        }
        return res;
    };
    
    vector<double> logw(n + 1, -1e18);
    for (int k = 0; k <= n; k++) {
        double lc1 = logC(k, l1);
        double lc2 = logC(n - k, l2);
        if (lc1 > -1e17 && lc2 > -1e17) {
            logw[k] = lc1 + lc2;
        }
    }
    
    double maxlw = *max_element(logw.begin(), logw.end());
    
    vector<double> w(n + 1, 0.0);
    double total = 0;
    for (int k = 0; k <= n; k++) {
        if (logw[k] > -1e17) {
            w[k] = exp(logw[k] - maxlw);
        }
        total += w[k];
    }
    
    vector<double> prob(n + 1);
    for (int k = 0; k <= n; k++) {
        prob[k] = w[k] / total;
    }
    
    // prefix sums
    vector<double> prefix(n + 2, 0.0);
    for (int k = 0; k <= n; k++) {
        prefix[k + 1] = prefix[k] + prob[k];
    }
    
    int best_a = 0, best_b = n;
    for (int len = 0; len <= n; len++) {
        bool found = false;
        for (int a = 0; a + len <= n; a++) {
            int b = a + len;
            double s = prefix[b + 1] - prefix[a];
            if (s >= target - 1e-9) {
                best_a = a;
                best_b = b;
                found = true;
                break;
            }
        }
        if (found) break;
    }
    
    cout << best_a << " " << best_b << endl;
    return 0;
}
```
