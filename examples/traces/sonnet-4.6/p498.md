Looking at this problem, I need to find P(p < q) given observations.

By Bayesian analysis:
- Prior for p and q are uniform on [0,1]
- After observing m1 heads in n1 tosses, posterior for p is Beta(m1+1, n1-m1+1)
- After observing m2 heads in n2 tosses, posterior for q is Beta(m2+1, n2-m2+1)

So p ~ Beta(a, b) where a = m1+1, b = n1-m1+1
And q ~ Beta(c, d) where c = m2+1, d = n2-m2+1

I need P(p < q) where p and q are independent Beta random variables.

P(p < q) = ∫∫_{p<q} f_p(p) f_q(q) dp dq

The PDF of Beta(a,b) is x^(a-1)(1-x)^(b-1) / B(a,b)

So P(p < q) = (1/B(a,b)) * (1/B(c,d)) * ∫_0^1 ∫_0^q p^(a-1)(1-p)^(b-1) q^(c-1)(1-q)^(d-1) dp dq

Let me use the formula for P(X < Y) where X~Beta(a,b) and Y~Beta(c,d).

There's a known formula involving the regularized incomplete beta function, but let me think of a direct numerical integration approach.

Given the constraints: a = m1+1 ≤ 51, b = n1-m1+1 ≤ 1001, similarly for c,d.

I can compute P(p < q) = ∫_0^1 F_p(q) * f_q(q) dq

where F_p(q) = I_q(a, b) (regularized incomplete beta function), and f_q(q) is Beta(c,d) PDF.

Numerical integration with sufficient points should work. Given precision requirement of 1e-4, I can use Gaussian quadrature or Simpson's rule with enough points.

Let me use numerical integration with ~10000 points using Simpson's rule.

Actually, let me think about a combinatorial approach. 

P(p < q) = ∫_0^1 ∫_0^q C_p * p^a-1*(1-p)^b-1 * C_q * q^c-1*(1-q)^d-1 dp dq

= C_p * C_q * ∫_0^1 q^(c-1)*(1-q)^(d-1) * [∫_0^q p^(a-1)*(1-p)^(b-1) dp] dq

I'll use numerical integration. Simpson's rule with N=10000 subdivisions should be fine within time limit.

With T up to some reasonable number and each integration taking O(N) time, should be okay.

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int T;
    cin >> T;
    while(T--){
        int n1,m1,n2,m2;
        cin >> n1 >> m1 >> n2 >> m2;
        
        double a = m1+1, b = n1-m1+1;
        double c = m2+1, d = n2-m2+1;
        
        // Use log for beta function normalization
        // lgamma(a)+lgamma(b)-lgamma(a+b)
        double logBab = lgamma(a)+lgamma(b)-lgamma(a+b);
        double logBcd = lgamma(c)+lgamma(d)-lgamma(c+d);
        
        // Integrate using Simpson's rule
        int N = 10000;
        double h = 1.0/N;
        
        // F_p(q) = integral_0^q p^(a-1)*(1-p)^(b-1)/B(a,b) dp
        // We accumulate this as we go
        
        double ans = 0.0;
        // Precompute cumulative integral of f_p up to each q point
        // Use trapezoidal for inner, Simpson for outer
        
        double cumP = 0.0;
        double prev_fp = 0.0; // f_p at p=0
        double prev_fq = 0.0; // f_q at q=0
        
        for(int i = 1; i <= N; i++){
            double q = i * h;
            double fp_q = exp((a-1)*log(q) + (b-1)*log(1-q) - logBab);
            double fq_q = exp((c-1)*log(q) + (d-1)*log(1-q) - logBcd);
            cumP += (prev_fp + fp_q) * h / 2.0;
            ans += cumP * fq_q * h;
            prev_fp = fp_q;
            prev_fq = fq_q;
        }
        
        printf("%.10f\n", ans);
    }
}
```
