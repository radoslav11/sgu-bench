The classic geometric probability problem. Two people arrive uniformly in [X, Y]. They meet if |a - b| <= Z minutes, where a and b are in [0, T] minutes (T = (Y-X)*60).

The probability is 1 - ((T-Z)/T)^2 if Z < T, else 1.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int X, Y;
    double Z;
    cin >> X >> Y >> Z;
    
    double T = (Y - X) * 60.0;
    
    double prob;
    if (Z >= T) {
        prob = 1.0;
    } else {
        double ratio = (T - Z) / T;
        prob = 1.0 - ratio * ratio;
    }
    
    printf("%.7f\n", prob);
    return 0;
}
```
