The expected number of prizes given out equals the expected number of distinct boxes chosen among M picks from N boxes.

For each box i, the probability that box i is chosen at least once = 1 - (1 - 1/N)^M = 1 - ((N-1)/N)^M.

By linearity of expectation, the expected number of distinct boxes chosen = N * (1 - ((N-1)/N)^M).

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long N, M;
    cin >> N >> M;
    
    double prob_not_chosen = pow((double)(N - 1) / N, M);
    double expected = N * (1.0 - prob_not_chosen);
    
    cout << fixed << setprecision(9) << expected << endl;
    
    return 0;
}
```
