Looking at this problem, I need to find polygon vertices given the midpoints of edges.

If the polygon has vertices V1, V2, ..., VN, and midpoints M1, M2, ..., MN where Mi is the midpoint of edge (Vi, Vi+1), then:
- Mi = (Vi + Vi+1) / 2
- So Vi + Vi+1 = 2*Mi

This gives us:
- V1 + V2 = 2*M1
- V2 + V3 = 2*M2
- V3 + V4 = 2*M3
- ...
- VN + V1 = 2*MN

From these equations, if we know V1, we can determine all other vertices:
- V2 = 2*M1 - V1
- V3 = 2*M2 - V2 = 2*M2 - 2*M1 + V1
- V4 = 2*M3 - V3 = 2*M3 - 2*M2 + 2*M1 - V1
- ...

In general: Vi = (-1)^(i-1) * V1 + some combination of midpoints.

For even i: Vi = -V1 + (linear combination of M's)
For odd i: Vi = +V1 + (linear combination of M's)

The last constraint VN + V1 = 2*MN gives us an equation to solve for V1.

Let me define:
- Vi = (-1)^(i-1) * V1 + Ci (where Ci is known from midpoints)

C1 = (0,0)
C2 = 2*M1
C3 = 2*M2 - 2*M1
C4 = 2*M3 - 2*M2 + 2*M1
...
Ci = 2*(M_{i-1} - M_{i-2} + M_{i-3} - ... )

The constraint from edge N: VN + V1 = 2*MN:
- (-1)^(N-1)*V1 + CN + V1 = 2*MN

If N is odd: (-1)^(N-1) = 1, so 2*V1 + CN = 2*MN → V1 = MN - CN/2
If N is even: (-1)^(N-1) = -1, so 0*V1 + CN = 2*MN → CN must equal 2*MN (check), V1 can be anything... wait, but then V1 is free? No, for even N, we get a constraint that must be satisfied, and if satisfied, V1 is arbitrary?

Wait, for even N: if 0*V1 = 2*MN - CN, this either has no solution or infinitely many. But the polygon isn't unique then... Let me re-read.

Actually for even N, CN = 2*(M1 - M2 + M3 - ... - M_{N-1}). The constraint becomes CN = 2*MN. If this isn't satisfied, output NO. If satisfied, V1 can be anything - but then there are infinitely many solutions. We just pick any V1, say V1 = (0, 0).

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<double> mx(n), my(n);
    for (int i = 0; i < n; i++) cin >> mx[i] >> my[i];
    
    // Ci: offset for Vi, Vi = (-1)^(i) * V1 + C[i] (0-indexed: V[0]=V1)
    // V[0] = V1, sign = +1
    // V[i] = (-1)^i * V1 + C[i]
    vector<double> cx(n), cy(n);
    cx[0] = cy[0] = 0;
    for (int i = 1; i < n; i++) {
        cx[i] = 2*mx[i-1] - cx[i-1];
        cy[i] = 2*my[i-1] - cy[i-1];
    }
    
    // Constraint: V[n-1] + V[0] = 2*M[n-1]
    // (-1)^(n-1)*V1 + C[n-1] + V1 = 2*M[n-1]
    double v1x, v1y;
    int sign = (n % 2 == 1) ? 1 : -1; // (-1)^(n-1)
    double coeff = sign + 1.0;
    double rhsx = 2*mx[n-1] - cx[n-1];
    double rhsy = 2*my[n-1] - cy[n-1];
    
    if (abs(coeff) < 1e-9) {
        if (abs(rhsx) > 1e-6 || abs(rhsy) > 1e-6) { cout << "NO\n"; return 0; }
        v1x = 0; v1y = 0;
    } else {
        v1x = rhsx / coeff;
        v1y = rhsy / coeff;
    }
    
    cout << "YES\n";
    cout << fixed << setprecision(3);
    for (int i = 0; i < n; i++) {
        double s = (i % 2 == 0) ? 1.0 : -1.0;
        cout << s*v1x + cx[i] << " " << s*v1y + cy[i] << "\n";
    }
    return 0;
}
```
