Looking at this problem, I need to find the shortest path that:
1. Starts at a cabin on the outer surface
2. Ends at a cabin on the outer surface
3. Mines all passageways (edges) in the 3D grid graph

This is essentially finding the shortest path (Eulerian path) that traverses all edges of the 3D grid graph.

For an Eulerian path to exist, the graph needs 0 or 2 vertices of odd degree. If there are 0 odd-degree vertices, it's an Eulerian circuit. If there are 2, it's an Eulerian path between those two vertices.

If there are more than 2 odd-degree vertices, we need to add extra edges (repeat traversals) to make it Eulerian. The minimum number of extra edges needed is (number of odd-degree vertices / 2 - 1) if we need a path, or (number of odd-degree vertices / 2) for a circuit.

Wait, let me think more carefully. The minimum number of additional edge traversals needed equals the minimum number of edges to add to make all vertices have even degree (or exactly 2 have odd degree). The number of odd-degree vertices must be even. If there are `k` odd-degree vertices, we need to add `k/2 - 1` edges (for a path between two of the odd-degree vertices) or `k/2` edges (for a circuit).

The total path length = (number of edges) + (extra edges added).

For an X×Y×Z grid:
- Number of edges = (X-1)*Y*Z + X*(Y-1)*Z + X*Y*(Z-1)

Now I need to count odd-degree vertices. A vertex at position (x,y,z) has degree = (number of neighbors). Interior vs boundary affects degree.

The degree of vertex (i,j,k) where i∈[1,X], j∈[1,Y], k∈[1,Z]:
- degree = (i>1) + (i<X) + (j>1) + (j<Y) + (k>1) + (k<Z)

A vertex has odd degree when the sum of its "boundary indicators" is odd.

Number of odd-degree vertices: count (i,j,k) where degree is odd.

The degree = (number of free directions). Let a = 2 if interior in x, 1 if boundary in x (with X>1), 0 if X=1. Actually let me define:
- dx = degree contribution from x: (i>1)+(i<X). For interior: 2, for boundary (when X>1): 1, for X=1: 0.

Similarly dy, dz. Total degree = dx+dy+dz. We need dx+dy+dz to be odd.

For a path, we need number_of_odd_vertices ≤ 2 (and since it must be even, 0 or 2).
- If 0 odd vertices: answer = edges (Eulerian circuit, but both endpoints must be on outer surface — circuit can start/end anywhere on outer surface if we choose appropriately)
- If 2 odd vertices: answer = edges (Eulerian path)
- If k > 2 odd vertices: answer = edges + k/2 - 1

We need to check if start/end can be on outer surface. The problem says start and end must be on the outer surface. If odd-degree vertices are all on the outer surface (which they typically are in grids), this works out.

Let me verify with examples:
- 2×1×2: edges = 1*1*2 + 2*0*2 + 2*1*1 = 2+0+2 = 4. Odd degree vertices? With Y=1, dy=0. dx: boundary when X=2 → 1. dz: boundary when Z=2 → 1. Interior: dx=2,dz=2. Corner: 1+0+1=2 (even). All corners even. Answer = 4. ✓

- 2×2×2: edges = 1*2*2+2*1*2+2*2*1 = 4+4+4=12. All vertices are corners: degree=3 (odd). 8 odd vertices. Extra = 8/2-1=3. Total=12+3=15. ✓

#include <bits/stdc++.h>
using namespace std;

int main(){
    long long X, Y, Z;
    int cas = 1;
    while(cin >> X >> Y >> Z){
        long long edges = (X-1)*Y*Z + X*(Y-1)*Z + X*Y*(Z-1);
        
        // Count odd degree vertices
        // For each axis, positions: 2 boundary (degree 1 each) if size>1, else 0; interior (size-2) positions with degree 2
        // degree = dx+dy+dz, odd when sum is odd
        // Let's count by parity
        // px = count of positions with odd contribution in x: boundary count if X>1 else 0
        long long bx = (X>1)?2:0, ix = (X>1)?X-2:X; // boundary, interior counts
        long long by = (Y>1)?2:0, iy = (Y>1)?Y-2:Y;
        long long bz = (Z>1)?2:0, iz = (Z>1)?Z-2:Z;
        // odd parity position: odd number of odd contributions
        // odd_x means boundary in x (contribution 1, odd)
        // even_x means interior in x (contribution 2, even) or X=1 (contribution 0, even)
        long long ox=bx, ex=ix; if(X==1){ex=1;ox=0;}
        long long oy=by, ey=iy; if(Y==1){ey=1;oy=0;}
        long long oz=bz, ez=iz; if(Z==1){ez=1;oz=0;}
        
        long long odd_count = ox*oy*ez + ox*ey*oz + ex*oy*oz + ox*oy*oz;
        // odd+odd+even, odd+even+odd, even+odd+odd, odd+odd+odd
        // Wait: odd count = number with odd total = sum of products where odd number of factors are "odd"
        odd_count = ox*oy*ez + ox*ey*oz + ex*oy*oz + ox*oy*oz;
        
        long long extra = (odd_count <= 2) ? 0 : odd_count/2 - 1;
        cout << "Case #" << cas++ << ": " << edges + extra << "\n";
    }
}
