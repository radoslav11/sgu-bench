The classic "find the farthest node from each node in a tree" problem. The standard approach:

1. Root the tree, do a DFS to find the two farthest descendants for each node (the longest and second longest paths going down).
2. Do another DFS/BFS top-down to propagate the answer considering paths going "up" through the parent.

For each node, the answer is the maximum of:
- The farthest descendant path (going down)
- The path going up through the parent

Let me implement this carefully.

**Step 1:** Root the tree at node 1. For each node, compute `down[v]` = farthest distance to a descendant.

**Step 2:** For each node, we also need the second-farthest descendant (in case the farthest goes through a particular child, we need an alternative for that child).

**Step 3:** Propagate `up[v]` = farthest distance going upward from v (through parent).

For a node `v` with parent `p`:
- `up[v] = max(up[p], best_down_from_p_not_through_v) + edge(p,v)`

To find "best_down_from_p not through v": store top 2 down distances from p with their child.

#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int n;
vector<pair<int,ll>> adj[10001];
ll down1[10001], down2[10001]; // best and second best downward distances
int child1[10001]; // which child gives down1
ll up_dist[10001]; // best distance going upward

void dfs1(int u, int par) {
    down1[u] = down2[u] = 0;
    child1[u] = -1;
    for (auto [v, w] : adj[u]) {
        if (v == par) continue;
        dfs1(v, u);
        ll cand = down1[v] + w;
        if (cand >= down1[u]) {
            down2[u] = down1[u];
            down1[u] = cand;
            child1[u] = v;
        } else if (cand > down2[u]) {
            down2[u] = cand;
        }
    }
}

void dfs2(int u, int par) {
    for (auto [v, w] : adj[u]) {
        if (v == par) continue;
        // up_dist[v] = max(up_dist[u], best down from u not through v) + w
        ll best_down_not_v = (child1[u] == v) ? down2[u] : down1[u];
        up_dist[v] = max(up_dist[u], best_down_not_v) + w;
        dfs2(v, u);
    }
}

int main(){

        cout << max(down1[i], up_dist[i]) << "\n";

    return 0;
}
