Looking at this problem, I need to find the minimal K such that every necklace of 2N-1 beads with K black beads is "beautiful".

A necklace is beautiful if we can choose two black beads such that one of the two arcs strictly between them contains exactly N beads.

The total beads = 2N-1. If we pick two black beads, the two arcs between them have lengths that sum to 2N-1 (total) minus 2 (the beads themselves) = 2N-3. Wait, let me reconsider.

Actually, "parts strictly between them" - between two beads on a necklace (circular), there are two paths. The beads on each path (not including the chosen beads) sum to 2N-1-2 = 2N-3. For one part to contain exactly N beads, we need N beads on one side and N-3 on the other (since N + (N-3) = 2N-3). Wait: 2N-3 total between them, one side has N, other has 2N-3-N = N-3.

So we need two black beads such that there are exactly N beads strictly between them on one arc.

A necklace is NOT beautiful if for every pair of black beads, neither arc between them has exactly N beads.

We want the minimal K where every necklace is beautiful. Equivalently, find the maximum number of black beads we can place such that the necklace is NOT beautiful, then K = that + 1.

So I need to find: max number of black beads in a necklace of 2N-1 beads such that no two black beads have exactly N beads between them (on either arc).

If bead i and bead j are black, the condition "N beads strictly between them on one arc" means |i-j| = N+1 (mod 2N-1), i.e., their positions differ by N+1 or N-2 (= 2N-1-(N+1)). Wait: N+1 + (N-2) = 2N-3. Hmm, that's 2N-3, but total is 2N-1, so arcs including endpoints: one arc length N+1 means N-1 beads between... Let me recount.

Positions 0..2N-2. Two beads at positions i and j. Strictly between them on the shorter/longer arc. Clockwise from i to j (not including i,j): j-i-1 beads (if j>i). For this to equal N: j-i-1=N, so j-i=N+1.

So two black beads at distance N+1 (or 2N-1-(N+1)=N-2, since it's circular) make the necklace beautiful.

NOT beautiful: no two black beads at circular distance N+1 (equivalently, no two at distance N-2 since (N+1)+(N-2)=2N-3≠2N-1... wait N+1 and N-2 aren't complementary mod 2N-1 unless N+1+N-2=2N-1, which gives 2N-1=2N-1. Yes! So distance N+1 and distance N-2 are the same pair).

So necklace is NOT beautiful iff no two black beads are at distance exactly N+1 (mod 2N-1).

This is an independent set problem on a circulant graph C(2N-1, {N+1}).

In C(2N-1, {N+1}): each node connects to node+N+1 and node-(N+1) mod 2N-1. Since gcd(N+1, 2N-1): 2N-1 = 2*(N+1) - 3, gcd(N+1,2N-1)=gcd(N+1,3).

The graph C(2N-1, {N+1}) decomposes into gcd(N+1,2N-1) cycles each of length (2N-1)/gcd. Let g=gcd(N+1,3), cycle length = (2N-1)/g.

Max independent set in a cycle of length L is floor(L/2). So max independent set = g * floor((2N-1)/(2g)) = floor((2N-1)/2) when g=1, or when g=3: 3*floor((2N-1)/6).

So max non-beautiful = floor((2N-1)/2) if gcd(N+1,3)=1, else 3*floor((2N-1)/6).

Answer K = max_non_beautiful + 1.

Given input 2N-1, let M=2N-1, N=(M+1)/2, N+1=(M+3)/2.
