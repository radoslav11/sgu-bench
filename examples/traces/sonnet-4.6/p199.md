Actually, X hates Y if (Si ≤ Sj AND Bi ≥ Bj) OR (Si ≥ Sj AND Bi ≤ Bj). This means X and Y hate each other if they are NOT in a situation where X strictly dominates Y or Y strictly dominates X. Actually wait - they hate each other unless one strictly dominates the other in BOTH dimensions.

So two people DON'T hate each other only if one strictly dominates the other in BOTH S and B (Si < Sj AND Bi < Bj, or Si > Sj AND Bi > Bj).

So we need a subset where every pair has one strictly dominating the other in both dimensions. This means we need a chain in the strict partial order where Si < Sj AND Bi < Bj. This is the Longest Increasing Subsequence (LIS) problem in 2D!

We want to find the largest subset where we can order them as p1, p2, ..., pk with S strictly increasing and B strictly increasing.

So: sort by S (ascending), then for equal S sort by B descending (so that equal S values can't both be in the LIS since B would be decreasing among them), then find LIS on B values.

Wait, for equal S, we need Si < Sj strictly, so no two people with equal S can be in the same chain. Similarly for equal B. So sort by S ascending, ties broken by B descending. Then find LIS with strict increase on B.

Let me implement this:
1. Sort people by S ascending, ties by B descending.
2. Find LIS (strictly increasing) on B values.
3. Reconstruct the path.

For LIS reconstruction with indices, I'll use patience sorting approach with binary search.

using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int n;
    cin >> n;
    
    vector<long long> S(n), B(n);
    for(int i = 0; i < n; i++) cin >> S[i] >> B[i];
    
    vector<int> idx(n);
    iota(idx.begin(), idx.end(), 0);
    sort(idx.begin(), idx.end(), [&](int a, int b){
        if(S[a] != S[b]) return S[a] < S[b];
        return B[a] > B[b];
    });
    
    // LIS on B[idx[i]] strictly increasing
    vector<long long> tails; // tails[i] = smallest tail of IS of length i+1
    vector<int> tailIdx; // actual index in idx array
    vector<int> dp(n), parent(n, -1);
    vector<int> tailPos; // which idx position holds the tail
    
    // Standard LIS with reconstruction
    vector<int> lis_tails_pos; // positions in idx
    vector<long long> lis_tails_val;
    
    for(int i = 0; i < n; i++){
        int p = idx[i];
        long long bval = B[p];
        // find first tail >= bval (strict increase, so find first >= bval)
        int pos = (int)(lower_bound(lis_tails_val.begin(), lis_tails_val.end(), bval) - lis_tails_val.begin());
        dp[i] = pos; // length of LIS ending here = pos+1
        parent[i] = (pos > 0) ? lis_tails_pos[pos-1] : -1;
        if(pos == (int)lis_tails_val.size()){
            lis_tails_val.push_back(bval);
            lis_tails_pos.push_back(i);
        } else {
            lis_tails_val[pos] = bval;
            lis_tails_pos[pos] = i;
        }
    }
    
    int lisLen = lis_tails_val.size();
    // reconstruct
    vector<int> result;
    int cur = lis_tails_pos[lisLen-1];
    while(cur != -1){
        result.push_back(idx[cur]+1);
        cur = parent[cur];
    }
    reverse(result.begin(), result.end());
    
    cout << lisLen << "\n";
    for(int x : result) cout << x << " ";
    cout << "\n";
}
