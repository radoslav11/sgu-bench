Looking at this problem, I need to:
1. Read a convex polygon with N vertices (given in counter-clockwise order)
2. Read M explosion points
3. Count how many explosions are inside (or on the boundary of) the convex polygon
4. Output "YES" if count >= K, else "NO"

For efficiently checking if a point is inside a convex polygon, I can use the binary search approach:
- Pick the first vertex as a "pivot"
- The polygon is divided into triangles by connecting the pivot to all other edges
- For a query point P, binary search to find which "sector" it falls in, then check if it's inside that triangle

The convex polygon point-in-polygon test with binary search works in O(log N) per query.

**Algorithm:**
1. Let `p[0]` be the pivot vertex
2. For a point `q`, check:
   - Is `q` on the correct side of the line from `p[0]` to `p[1]`? (cross product `p[1]-p[0]` × `q-p[0]` >= 0)
   - Is `q` on the correct side of the line from `p[0]` to `p[N-1]`? (cross product `p[N-1]-p[0]` × `q-p[0]` <= 0)
   - Binary search for the "wedge" index `i` such that `q` is between rays `p[0]->p[i]` and `p[0]->p[i+1]`
   - Check if `q` is inside the triangle `p[0], p[i], p[i+1]`

Using `long long` for cross products to avoid overflow (coordinates up to 1e9, so cross products up to ~2e18 which fits in long long).

Let me implement this carefully:

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef pair<ll,ll> pll;

ll cross(pll O, pll A, pll B) {
    return (A.first - O.first) * (ll)(B.second - O.second) - (A.second - O.second) * (ll)(B.first - O.first);
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int N, M, K;
    cin >> N >> M >> K;
    
    vector<pll> poly(N);
    for(int i = 0; i < N; i++) cin >> poly[i].first >> poly[i].second;
    
    int cnt = 0;
    
    for(int j = 0; j < M; j++){
        pll q;
        cin >> q.first >> q.second;
        
        // Check point in convex polygon
        // pivot = poly[0]
        // Check if q is in the "fan"
        ll c1 = cross(poly[0], poly[1], q);
        if(c1 < 0) continue; // outside first edge ray
        
        ll cn = cross(poly[0], poly[N-1], q);
        if(cn > 0) continue; // outside last edge ray
        
        // Binary search for sector
        int lo = 1, hi = N - 1;
        while(hi - lo > 1){
            int mid = (lo + hi) / 2;
            ll c = cross(poly[0], poly[mid], q);
            if(c >= 0) lo = mid;
            else hi = mid;
        }
        
        // Check if q is inside triangle poly[0], poly[lo], poly[lo+1]
        ll c = cross(poly[lo], poly[lo+1], q);
        if(c >= 0) cnt++;
    }
    
    cout << (cnt >= K ? "YES" : "NO") << endl;
    return 0;
}
```
