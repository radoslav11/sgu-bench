For 2×3 constraint (rows i, i+1, all 3 cols): sum must be 2. Since each row has exactly 1 blue per 3 columns in the tiled pattern, 2 rows give 2 blues. ✓ Always satisfied if each row has exactly 1 blue per period-3.

For 3×2 constraint (3 rows, cols j, j+1): must have exactly 2 blues. With 3 rows and 2 columns, we need exactly 2 blues. Each row contributes 0 or 1 blue. So exactly 2 of the 3 rows have their blue in column j or j+1 (mod 3).

So I need: for each pair of consecutive columns (mod 3), exactly 2 of the 3 rows have their blue in those columns. This means no column is "alone" - each column pair covers 2 rows, so each column is covered by... Each column appears in 2 pairs. Total coverage = 6 = 2×3. Each column has 1 blue row, appearing in 2 pairs → contributes 2. Sum = 3×2 = 6 ✓. The constraint is that no pair covers 3 or 1. Since one blue per row and one per column (permutation matrix in 3×3), this is automatically satisfied! Every valid permutation works.
