This is a classic scheduling problem. We want to minimize the makespan where each job has a processing time T_i and a delivery time L_i.

The completion time for leaflet i if it's printed at position k in the sequence is:
- Print starts after all previous jobs finish
- Delivery completes at: (sum of T_j for j before i) + T_i + L_i

We want to minimize the maximum over all leaflets of (prefix_sum_of_T + T_i + L_i).

This is the classic problem where the optimal strategy is to sort by L_i in decreasing order (longest delivery time first). This is similar to the "minimize makespan with tails" problem, solved by Johnson's rule / SPT for delivery times.

Let me verify with the example:
- N=2, T=[2,1], L=[2,1]
- Sort by L descending: leaflet 1 (T=2,L=2) first, then leaflet 2 (T=1,L=1)
- Leaflet 1: starts at 0, printed at 2, delivered at 4
- Leaflet 2: starts at 2, printed at 3, delivered at 4
- Answer: 4 ✓

If we did the other order:
- Leaflet 2 (T=1,L=1) first: printed at 1, delivered at 2
- Leaflet 1 (T=2,L=2): printed at 3, delivered at 5
- Answer: 5

So sorting by L descending gives 4, which matches.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> T(n), L(n);
    for (int i = 0; i < n; i++) cin >> T[i];
    for (int i = 0; i < n; i++) cin >> L[i];
    
    // Sort by L descending
    vector<int> idx(n);
    iota(idx.begin(), idx.end(), 0);
    sort(idx.begin(), idx.end(), [&](int a, int b) {
        return L[a] > L[b];
    });
    
    int ans = 0;
    int time_so_far = 0;
    for (int i = 0; i < n; i++) {
        int j = idx[i];
        time_so_far += T[j];
        ans = max(ans, time_so_far + L[j]);
    }
    
    cout << ans << endl;
    return 0;
}
```
