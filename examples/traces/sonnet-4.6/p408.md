   - D = largest Euclidean distance between any pair of points
4. Maximize M × D

Key observations:
- To maximize M (max segments at a point), we can connect all N segments to A0. That gives M = N.
- To maximize D (max distance between two points), we want two points as far apart as possible.

The farthest point from any other: we can put one segment from A0 going in -x direction length 1 to reach (-1, 0), and the chain goes to (1 + m, 0). Distance = 2 + m.

So M = k, D = 2 + m, N = k + m, score = k(2 + m) = k(2 + N - k).

Maximize over k: f(k) = k(2 + N - k) = k(N+2) - k²
f'(k) = N+2 - 2k = 0 → k = (N+2)/2

#include <bits/stdc++.h>
using namespace std;
int main(){
    int N;
    cin>>N;
    if(N==0){cout<<fixed<<setprecision(3)<<0.0;return 0;}
    double best=0;
    for(int k=1;k<=N;k++){
        int m=N-k;
        double M=k;
        double D=2.0+m;
        best=max(best,M*D);
    }
    cout<<fixed<<setprecision(3)<<best<<endl;
}
