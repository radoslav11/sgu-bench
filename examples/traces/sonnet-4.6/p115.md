The year 2001 starts on Monday (January 1, 2001 was a Monday).

Let me figure out the day of week for any date in 2001.

January 1, 2001 = Monday = day 1.

Steps:
1. Check if date N/M is valid in 2001 (2001 is not a leap year)
2. Calculate day of year
3. Compute day of week

Days in each month of 2001 (not leap year):
Jan=31, Feb=28, Mar=31, Apr=30, May=31, Jun=30, Jul=31, Aug=31, Sep=30, Oct=31, Nov=30, Dec=31

Let me verify with the sample: Oct 21.
Days before October: 31+28+31+30+31+30+31+31+30 = 273
Day of year = 273 + 21 = 294
Jan 1 is day 1 = Monday (1)
Day 294: (294 - 1) % 7 = 293 % 7 = 293/7=41*7=287, 293-287=6, so Monday + 6 = day 7 (Sunday)
Output: 7 ✓

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    int n, m;
    cin >> n >> m;
    
    int days_in_month[] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    
    if(m < 1 || m > 12 || n < 1 || n > days_in_month[m]){
        cout << "Impossible" << endl;
        return 0;
    }
    
    int day_of_year = 0;
    for(int i = 1; i < m; i++){
        day_of_year += days_in_month[i];
    }
    day_of_year += n;
    
    // Jan 1, 2001 is Monday = day 1
    // day_of_year=1 -> Monday=1
    int dow = (day_of_year - 1) % 7 + 1;
    
    cout << dow << endl;
    
    return 0;
}
```
