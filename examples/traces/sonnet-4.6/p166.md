Let me implement each command carefully:

- **Letter/space**: Insert character at cursor position. If cursor is beyond end of line, pad with spaces first. Then advance cursor right by 1.
- **L**: Move cursor left (min 0)
- **R**: Move cursor right (no upper bound restriction from moving right alone)
- **U**: Move cursor up one line (if not first line)
- **D**: Move cursor down one line (if not last line)
- **N**: Split current line at cursor. Text from cursor onwards goes to new line. Cursor goes to position 0 of new line.
- **E**: Move cursor to position after end of current line
- **H**: Move cursor to position 0
- **X**: Delete character at cursor. If cursor is beyond end, pad with spaces and attach next line (if exists). If at last line and beyond end, nothing happens.
- **B**: Delete character before cursor. If cursor is beyond end of line, just move left. If cursor is at 0 and first line, nothing. If cursor is at 0 and not first line, attach current line to previous line.
- **Q**: Quit
