Looking at this problem, I need to find the Nash equilibrium strategies for two businessmen competing to get N approvals first.

Let me define the state as (i, j) where businessman 1 has i approvals remaining and businessman 2 has j approvals remaining.

Each day, each businessman chooses a bribe amount b1 and b2. The probability of getting an approval is p1 = 1 - 0.99^((1-F1)*b1) for businessman 1, and similarly for businessman 2.

Let W(i,j) = expected profit for businessman 1 in state (i,j) where i is remaining approvals for B1, j for B2.

At Nash equilibrium, both players maximize their own expected payoff simultaneously.

Let me think about this as a zero-sum-like game. Let V1(i,j) be the expected net payoff for player 1 in state (i,j).

On each day, player 1 pays b1 and gets approval with prob p1, player 2 pays b2 and gets approval with prob p2.

State transitions: (i,j) -> (i-1,j), (i,j-1), (i-1,j-1), (i,j) with probabilities p1(1-p2), p2(1-p1), p1*p2, (1-p1)(1-p2).

Boundary: W(0,j>0) = V (player 1 wins), W(i>0,0) = 0 (player 1 loses), W(0,0) = V/2.

The probability of winning P(i,j) is what we need.

For the Nash equilibrium: player 1 chooses b1 to maximize their payoff, player 2 chooses b2 to maximize their payoff.

Given the complexity, let me think about the probability of winning (ignoring costs for now to find equilibrium mixed strategies in continuous bribe space).

Actually this is a continuous game. The optimal bribe satisfies first-order conditions. Let me denote by q = 0.99^(1-F), so p = 1 - q^b.

At Nash equilibrium for state (i,j), the derivative of payoff w.r.t. b1 = 0:
-1 + dp1/db1 * [(1-p2)(V(i-1,j) - V(i,j)) + p2*(V(i-1,j-1) - V(i,j-1))] = 0

This is quite complex. Let me implement a numerical approach: for each state (i,j), solve for Nash equilibrium numerically using best-response iteration.

For state (i,j), given p2, player 1 maximizes over b1:
-b1 + p1*(something) + constant → dp/db * marginal_value = 1

Here p1 = 1 - α^b1 where α = 0.99^(1-F1), so dp1/db1 = -ln(α)*α^b1 = ln(1/α)*(1-p1)

So: ln(1/α1)*(1-p1)*MV1 = 1, where MV1 = marginal value of getting approval for player 1.

I'll use iterative best-response to find Nash equilibrium. For each state, iterate until convergence.

Let me implement this with value functions for both players.

#include <bits/stdc++.h>
using double_t = double;
const int MAXN = 11;
double V1[MAXN][MAXN], V2[MAXN][MAXN]; // expected payoff
double P1[MAXN][MAXN]; // prob player 1 wins
int N; double F1, F2, V;

int main(){
    scanf("%d %lf %lf %lf",&N,&F1,&F2,&V);
    double a1 = log(1.0/pow(0.99,1-F1));
    double a2 = log(1.0/pow(0.99,1-F2));
    // boundary
    for(int j=1;j<=N;j++){V1[0][j]=V;V2[0][j]=0;P1[0][j]=1;}
    for(int i=1;i<=N;i++){V1[i][0]=0;V2[i][0]=V;P1[i][0]=0;}
    V1[0][0]=V/2;V2[0][0]=V/2;P1[0][0]=0.5;
    for(int s=1;s<=2*N;s++) for(int i=1;i<=N;i++){int j=s-i;if(j<1||j>N)continue;
        double p1=0.5,p2=0.5;
        for(int it=0;it<10000;it++){
            double mv1=(1-p2)*(V1[i-1][j]-V1[i][j])+p2*(V1[i-1][j-1]-V1[i][j-1]);
            double mv2=(1-p1)*(V2[i][j-1]-V2[i][j])+p1*(V2[i-1][j-1]-V2[i-1][j]);
            double np1=mv1>0?1-1.0/(a1*mv1):0; np1=max(0.0,min(0.99,np1));
            double np2=mv2>0?1-1.0/(a2*mv2):0; np2=max(0.0,min(0.99,np2));
            p1=np1;p2=np2;
        }
        double b1=p1>0?log(1-p1)/log(pow(0.99,1-F1)):0;
        double b2=p2>0?log(1-p2)/log(pow(0.99,1-F2)):0;
        V1[i][j]=-b1+p1*(1-p2)*V1[i-1][j]+p2*(1-p1)*V1[i][j-1]+p1*p2*V1[i-1][j-1]+(1-p1)*(1-p2)*V1[i][j];
        V1[i][j]/=(p1+p2-p1*p2);
        V2[i][j]=-b2+p1*(1-p2)*V2[i-1][j]+p2*(1-p1)*V2[i][j-1]+p1*p2*V2[i-1][j-1]+(1-p1)*(1-p2)*V2[i][j];
        V2[i][j]/=(p1+p2-p1*p2);
        P1[i][j]=p1*(1-p2)*P1[i-1][j]+p2*(1-p1)*P1[i][j-1]+p1*p2*P1[i-1][j-1];
        P1[i][j]/=(p1+p2-p1*p2);
    }
    printf("%.9f %.9f\n",P1[N][N],1-P1[N][N]);
}
