Looking at this problem, I need to:

1. Project N segments onto the X-axis
2. For each point on the X-axis projection, determine which segment is "nearest to OX" (has minimum Y value at that X coordinate) among all segments whose projection covers that X point
3. Sum up the lengths of projections for each color

The approach:
- For each x position in the projection range, find which segments cover that x, compute their y-value at that x, and take the one with minimum y. That segment's color "owns" that x point.
- Since coordinates go up to 32000, I can sweep through x values with small steps.

But we need precision 0.01, so I should use a sweep with fine granularity or handle analytically.

Let me think analytically: The "ownership" of x-intervals can change at segment endpoints and at intersections of segment y-values (projected to x-axis). Since no two segments share more than one point, they can intersect at most once.

Critical x-values: all segment endpoint x-coordinates, plus x-coordinates where two segments have equal y-values (i.e., they cross when projected, or their y-values are equal).

Between consecutive critical x-values, the ordering of segments by y-value is constant, so the minimum-y segment is constant.

Algorithm:
1. Collect all critical x-values
2. Between consecutive critical x-values, evaluate at the midpoint to determine which segment has minimum y
3. Add (x_right - x_left) to the color of that segment

For finding where two segments have equal y at the same x: given segment i with y = f_i(x) and segment j with y = f_j(x), find x where f_i(x) = f_j(x) and both segments are active (x is within both projections).

Let me implement this:

For segment i: x ranges from min(xi1,xi2) to max(xi1,xi2), and y(x) = yi1 + (yi2-yi1)*(x-xi1)/(xi2-xi1) (if xi1 != xi2), else it's vertical.

Vertical segments: they exist only at one x point (measure zero), so we can skip them for length calculation.

For two segments i and j both non-vertical, solve f_i(x) = f_j(x) within overlapping x-range.

using namespace std;

int main(){

    vector<char> col(N);

        cin>>x1[i]>>y1[i]>>x2[i]>>y2[i]>>col[i];
        if(x1[i]>x2[i]){swap(x1[i],x2[i]);swap(y1[i],y2[i]);}
    }
    
    set<double> crits;
    for(int i=0;i<N;i++){
        crits.insert(x1[i]);
        crits.insert(x2[i]);
    }
    for(int i=0;i<N;i++){
        for(int j=i+1;j<N;j++){
            if(x1[i]>=x2[i]||x1[j]>=x2[j]) continue;
            double lo=max(x1[i],x1[j]),hi=min(x2[i],x2[j]);
            if(lo>=hi) continue;
            double ai=y1[i]+(y2[i]-y1[i])/(x2[i]-x1[i])*(lo-x1[i]);
            double bi=y1[i]+(y2[i]-y1[i])/(x2[i]-x1[i])*(hi-x1[i]);
            double aj=y1[j]+(y2[j]-y1[j])/(x2[j]-x1[j])*(lo-x1[j]);
            double bj=y1[j]+(y2[j]-y1[j])/(x2[j]-x1[j])*(hi-x1[j]);
            double di=ai-aj, df=(bi-bj)-(ai-aj);
            if(abs(df)>1e-12){
                double t=-di/df;
                if(t>0&&t<1) crits.insert(lo+t*(hi-lo));
            }
        }
    }

    sort(cv.begin(),cv.end());
    
    double SR=0,SG=0,SB=0;
    auto getY=[&](int i,double x)->double{
        if(x2[i]==x1[i]) return min(y1[i],y2[i]);
        return y1[i]+(y2[i]-y1[i])/(x2[i]-x1[i])*(x-x1[i]);
    };
    
    for(int k=0;k+1<(int)cv.size();k++){
        double lo=cv[k],hi=cv[k+1];
        if(hi<=lo) continue;
        double mx=(lo+hi)/2;
        double miny=1e18; int best=-1;
        for(int i=0;i<N;i++){
            if(x1[i]>=x2[i]) continue;
            if(mx<x1[i]||mx>x2[i]) continue;
            double yy=getY(i,mx);
            if(yy<miny){miny=yy;best=i;}
        }
        if(best<0) continue;
        double len=hi-lo;
        if(col[best]=='R') SR+=len;
        else if(col[best]=='G') SG+=len;
        else SB+=len;
    }
    printf("R %.2f\nG %.2f\nB %.2f\n",SR,SG,SB);
}
