To find the next lexicographic bracket sequence, I need to find the rightmost position where I can increase a '(' to ')', while still being able to form a valid sequence.

The approach:
1. Find the rightmost '(' that can be changed to ')' such that the remaining suffix can be filled with a valid bracket sequence.
2. To change position i from '(' to ')', we need:
   - The prefix up to i-1 remains valid (non-negative balance)
   - After changing position i to ')', the balance at position i is still >= 0
   - The remaining positions (i+1 to n-1) can form a valid completion (need exactly `balance_after_i` opens to close, followed by closes, i.e., we need `balance_after_i` to equal the number of remaining positions / ... let me think more carefully)

Let me think differently. The next permutation in lex order where '(' < ')':

- Scan from right to find the rightmost position we can "increment" (change '(' to ')')
- At that position, change '(' to ')', then fill the rest with the lexicographically smallest valid completion

For position i to be changeable from '(' to ')':
- Current char must be '('
- The balance just before position i must be > 0 (since we're changing '(' to ')', the balance after position i decreases by 2 from what it was if we kept '(')
  - Actually: balance before i = b, if we change '(' to ')', new balance at i = b - 1. We need b - 1 >= 0, so b >= 1.
- The remaining n-1-i positions must be fillable: we need exactly (b-1) more ')' to close, and then we need (n-1-i) - (b-1) must be even and >= 0, and (n-1-i-(b-1))/2 open-close pairs... Actually, after position i with balance b-1, we need to fill n-1-i positions with a valid sequence starting from balance b-1. The minimum length needed is 2*(b-1) (to close all opens). So we need n-1-i >= 2*(b-1) ... wait, not quite. We need (n-1-i) positions to reduce balance from (b-1) to 0. The minimum positions needed = b-1 (all ')'), but we also need remaining = n-1-i to have same parity as b-1: (n-1-i - (b-1)) must be even and >= 0.

Actually for the smallest valid completion from balance `bal` with `rem` positions: we want lex smallest = as many '(' as possible early. But actually lex smallest means we want '(' first. The lex smallest valid sequence from balance `bal` with `rem` positions:
- Put as many '(' as possible, then close them all. The number of '(' we can add is (rem - bal) / 2, then we have bal + (rem-bal)/2 closes.

So the lex smallest completion: `(((...)))` pattern: `(rem-bal)/2` opens followed by `bal + (rem-bal)/2` closes.

Algorithm:
1. Compute prefix balances.
2. Scan from right to left for the rightmost '(' (position i) where:
   - balance[i] (balance before position i, i.e., number of unmatched '(' in prefix 0..i-1) >= 1
   - (n - 1 - i - (balance[i] - 1)) >= 0 and even
3. Change position i to ')', set new_bal = balance[i] - 1, remaining = n-1-i
4. Fill positions i+1..n-1 with lex smallest: `(remaining - new_bal)/2` '(' then `new_bal + (remaining-new_bal)/2` ')'

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;
    int n = s.size();
    
    // Compute prefix balance: bal[i] = balance before position i
    vector<int> bal(n+1, 0);
    for (int i = 0; i < n; i++) {
        bal[i+1] = bal[i] + (s[i] == '(' ? 1 : -1);
    }
    
    // Find rightmost '(' that can be changed to ')'
    int pos = -1;
    for (int i = n-1; i >= 0; i--) {
        if (s[i] == '(') {
            int b = bal[i]; // balance before position i
            if (b >= 1) {
                int new_bal = b - 1;
                int rem = n - 1 - i;
                if (rem >= new_bal && (rem - new_bal) % 2 == 0) {
                    pos = i;
                    break;
                }
            }
        }
    }
    
    if (pos == -1) {
        cout << "No solution" << endl;
    } else {
        s[pos] = ')';
        int new_bal = bal[pos] - 1;
        int rem = n - 1 - pos;
        int opens = (rem - new_bal) / 2;
        int closes = new_bal + opens;
        for (int i = pos+1; i < n; i++) {
            if (opens > 0) { s[i] = '('; opens--; }
            else { s[i] = ')'; }
        }
        cout << s << endl;
    }
    return 0;
}
```
