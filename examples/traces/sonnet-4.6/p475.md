Key observations:
- n ≤ 10, so we can use bitmask DP for who's on the raft
- m ≤ 1000 riffles
- At each intermediate point, we need to track: the current time elapsed, and who is on the raft

State: `dp[mask]` = minimum time to reach current point with `mask` being the set of people currently on the raft.

At each point, after arriving, we can reorganize who's on the raft (paying switching costs), then proceed.

Let me think more carefully:

At each segment i (from p_{i-1} to p_i):
- We have some set `mask` on the raft, rest walking
- Raft time: D_i (capsize) or d_i (no capsize), depending on total weight vs c_i
- Walking time: max(t_j) for j NOT in mask
- The segment takes max(raft_time, walk_time) to complete
- At the arrival point, people can switch: cost = sum of s_j for those changing

Initial state: everyone on bank (mask = 0... but wait, the raft can't be empty during the trip. Actually, someone must be on the raft to navigate it.

Wait - "you can't leave the raft empty during the trip" - so mask must be non-empty for each segment.

At start (p0): everyone is on bank, so we need to board people first (paying s_j for those getting on), then start.

Let me define `dp[mask]` = minimum time at current point with `mask` on raft.

Initial: At p0, everyone on bank. We transition to some non-empty mask by paying sum(s_j for j in mask). So `dp[mask] = sum s_j for j in mask`.

For each riffle i:
- For each current state `mask`, for each new arrangement `new_mask` (non-empty, except at final step):
  - switching_cost = sum s_j for j in (mask XOR new_mask)
  - raft starts after switching is done
  - raft_time = D_i if sum(w_j for j in new_mask) > c_i else d_i
  - walk_time = max(t_j for j NOT in new_mask), or 0 if everyone on raft
  - segment_time = max(raft_time, walk_time)
  - new_time = dp[mask] + switching_cost + segment_time

At finish (after last riffle): everyone must be on bank. So from the final mask, add sum(s_j for j in mask) to get everyone off.

Wait, at the finish everyone must be on the bank. So after the last riffle, we add disembarkation cost.

Also at the finish, the raft must be there too (can't leave raft behind). Since raft always travels with whoever is on it, it arrives at finish.

Let me re-examine: at each point between riffles, first everyone arrives (raft waits for walkers or vice versa), THEN people switch. The switching happens before the next segment.

So the flow is: at point p_{i-1}, switch people (cost = switching), then traverse segment i.

Initial state at p0: everyone on bank (mask=0). 
- Switch to some non-empty mask: cost = sum s_j for j in mask
- Then traverse riffle 1

After last riffle (at pm): switch to empty mask (everyone off): cost = sum s_j for j in final_mask.

For the final point, the raft must also arrive, so mask must be non-empty for the last segment too.

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n,m; cin>>n>>m;
    vector<int> w(n),t(n),s(n);
    for(int i=0;i<n;i++) cin>>w[i]>>t[i]>>s[i];
    vector<int> c(m),D(m),d(m);
    for(int i=0;i<m;i++) cin>>c[i]>>D[i]>>d[i];
    int full=1<<n;
    const long long INF=1e18;
    vector<long long> dp(full,INF);
    for(int mask=1;mask<full;mask++){
        long long cost=0;
        for(int j=0;j<n;j++) if(mask>>j&1) cost+=s[j];
        dp[mask]=cost;
    }
    for(int i=0;i<m;i++){
        // precompute for each mask: weight, max walk time
        vector<int> wt(full,0); vector<long long> maxwalk(full,0);
        for(int mask=0;mask<full;mask++){
            for(int j=0;j<n;j++) if(mask>>j&1) wt[mask]+=w[j];
        }
        for(int mask=0;mask<full;mask++){
            // walkers are those NOT in mask
            for(int j=0;j<n;j++) if(!(mask>>j&1)) maxwalk[mask]=max(maxwalk[mask],(long long)t[j]);
        }
        // for this riffle, from prev dp, try all new_mask
        vector<long long> ndp(full,INF);
        for(int prev=1;prev<full;prev++){
            if(dp[prev]==INF) continue;
            for(int nm=1;nm<full;nm++){
                long long sw=0;
                for(int j=0;j<n;j++) if(((prev>>j&1)^(nm>>j&1))) sw+=s[j];
                long long raft_t=(wt[nm]>c[i])?D[i]:d[i];
                long long seg=max(raft_t,maxwalk[nm]);
                long long val=dp[prev]+sw+seg;
                if(val<ndp[nm]) ndp[nm]=val;
            }
        }
        dp=ndp;
    }
    long long ans=INF;
    for(int mask=1;mask<full;mask++){
        if(dp[mask]==INF) continue;
        long long cost=0;
        for(int j=0;j<n;j++) if(mask>>j&1) cost+=s[j];
        ans=min(ans,dp[mask]+cost);
    }
    cout<<ans<<endl;
}
```
