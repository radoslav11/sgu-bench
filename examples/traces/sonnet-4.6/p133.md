The problem asks us to find outposts that are strictly contained within another outpost. Outpost i is redundant if there exists j such that Aj < Ai AND Bi < Bj (strictly contained).

Since all Ak are different and all Bk are different, we need to count intervals [Ai, Bi] such that there exists [Aj, Bj] with Aj < Ai and Bj > Bi.

Approach:
1. Sort intervals by Ak (left endpoint) in increasing order.
2. For each interval in this sorted order, we need to check if there's a previous interval (with smaller A) that has a larger B.
3. If we process intervals sorted by A ascending, for interval i to be redundant, there must exist some j before it (in sorted order, so Aj < Ai) with Bj > Bi.
4. So we need to check: among all intervals processed so far (with smaller A), is there any with B > Bi?
5. This is simply: is the maximum B seen so far (among previous intervals) > Bi?

Wait, let me verify with the example:
- Intervals: [0,10], [2,9], [3,8], [1,15], [6,11]
- Sort by A: [0,10], [1,15], [2,9], [3,8], [6,11]
- Process [0,10]: no previous, max_B = 10
- Process [1,15]: max_B=10 < 15, not redundant. max_B = 15
- Process [2,9]: max_B=15 > 9, redundant! max_B stays 15
- Process [3,8]: max_B=15 > 8, redundant! max_B stays 15
- Process [6,11]: max_B=15 > 11, redundant! max_B stays 15

Count = 3. Matches the expected output!

So the algorithm is simply:
1. Sort by A ascending
2. Track maximum B seen so far
3. If current B < max B so far, it's redundant

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int n;
    cin >> n;
    
    vector<pair<long long, long long>> intervals(n);
    for (int i = 0; i < n; i++) {
        cin >> intervals[i].first >> intervals[i].second;
    }
    
    // Sort by A ascending
    sort(intervals.begin(), intervals.end());
    
    int redundant = 0;
    long long maxB = LLONG_MIN;
    
    for (int i = 0; i < n; i++) {
        if (maxB > intervals[i].second) {
            redundant++;
        }
        maxB = max(maxB, intervals[i].second);
    }
    
    cout << redundant << endl;
    
    return 0;
}
```
