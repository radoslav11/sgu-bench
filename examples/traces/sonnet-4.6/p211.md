Looking at this problem, I need to simulate adding powers of 2 to a counter where each register holds 0, 1, or 2.

When I add 2^i, I increment register i by 1. But if register i becomes 3, I need to carry:
- 3 * 2^i = 2^i + 2^(i+1), so set register i to 1 and add 1 to register i+1... wait, 3*2^i = 1*2^i + 1*2^(i+1)? No: 3*2^i = 2*2^i + 2^i = 2^(i+1) + 2^i... that's 3*2^i = 2^(i+1) + 2^i, which means register i stays 1 and carry 1 to i+1. Hmm, but also if register i becomes 3, another option: 3*2^i = 2^(i+1) * 1 + 2^i * 1. 

Wait: 3 = 2 + 1, so 3*2^i = 2^(i+1) + 2^i. So register i becomes 1 and we carry 1 to register i+1.

But what if register i is 2 and we add 1? 2+1=3, which means register i becomes 1 and carry to i+1.

Actually, the carry propagation: if adding 1 to register i:
- If A[i] is 0 or 1: just increment, no carry.
- If A[i] is 2: set A[i] = 0... wait: 2+1=3 = 2^(i+1) + 2^i, so register i = 1, carry 1 to i+1.

Hmm wait: 3*2^i = (2+1)*2^i = 2^(i+1) + 2^i. So register i becomes 1 (not 0), and carry 1 to i+1.

But we need at most 4 register changes. Starting from register i, we propagate carries. The carry chain:
- Add 1 to register i: if it was 0→1 (done, 1 change), if 1→2 (done, 1 change), if 2→1 and carry to i+1.

So each carry step changes one register. In worst case, how many carries can propagate? Since the sum doesn't exceed 2^N - 1, and registers hold 0,1,2... The claim is at most 4 changes.

Let me think: when we add 2^i, carry propagates while register value is 2. Each carry: register goes from 2 to 1 (adds 1 to that register conceptually). But we need to bound this at 4.

Actually, I think the key insight is that with values 0,1,2, a carry can propagate at most a few steps before stopping naturally. Let me just simulate and trust the problem statement that it's at most 4.

Let me simulate: add 1 to register pos, track changes (only record if value actually changes from before).

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N;
    cin >> N;
    int M;
    cin >> M;
    
    vector<int> A(N, 0);
    
    for(int q = 0; q < M; q++){
        int pos;
        cin >> pos;
        
        vector<pair<int,int>> changes;
        
        // Add 1 to register pos
        int carry = 1;
        int cur = pos;
        
        while(carry && cur < N){
            int old_val = A[cur];
            int new_val = A[cur] + carry;
            if(new_val <= 2){
                A[cur] = new_val;
                changes.push_back({cur, new_val});
                carry = 0;
            } else {
                // new_val == 3: register becomes 1, carry 1 to next
                A[cur] = 1;
                changes.push_back({cur, 1});
                carry = 1;
                cur++;
            }
        }
        
        cout << changes.size();
        for(auto& p : changes){
            cout << " " << p.first << " " << p.second;
        }
        cout << "\n";
    }
    
    return 0;
}
```
