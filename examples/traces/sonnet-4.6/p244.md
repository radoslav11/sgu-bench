Looking at this problem, I need to find a triangle ABC where:
- AH is the height from A to BC (length h)
- AD is the angle bisector from A to BC (length b)  
- AM is the median from A to BC (length m)

Key insight: Place BC on the x-axis. Let B = (-c, 0) and C = (d, 0) where c, d > 0. Let A = (x, h) where h is the height length.

For the median: M is midpoint of BC = ((d-c)/2, 0). So AM = sqrt(x - (d-c)/2)² + h²) = m.

For the angle bisector: D divides BC in ratio AB:AC. 
- AB = sqrt((x+c)² + h²), AC = sqrt((x-d)² + h²)
- D = (-c + AB/(AB+AC) * (c+d), 0)
- AD = sqrt((x - Dx)² + h²) = b

Let me set up coordinates differently. Let M be the midpoint of BC. Place M at origin on x-axis. Then B = (-a, 0), C = (a, 0) for some a > 0. A = (p, h).

Median AM = sqrt(p² + h²) = m, so p² = m² - h².

For the angle bisector: AB = sqrt((p+a)² + h²), AC = sqrt((p-a)² + h²).
D divides BC in ratio AB:AC: Dx = (-a * AC + a * AB)/(AB + AC) = a(AB - AC)/(AB + AC).
AD² = (p - Dx)² + h² = b².

Let me denote AB = sqrt((p+a)² + h²) = u, AC = sqrt((p-a)² + h²) = v.

Dx = a(u - v)/(u + v)
AD² = (p - a(u-v)/(u+v))² + h² = b²

So (p - a(u-v)/(u+v))² = b² - h²

We need b² ≥ h² and m² ≥ h² for real solutions.

I'll search for a using binary search or numerical methods. Note: p = sqrt(m² - h²) (taking p ≥ 0 by symmetry, we can handle both signs).

Actually p can be negative. Let me try p = sqrt(m²-h²) first, then -sqrt(m²-h²).

For a given p and h, I need to find a > 0 such that AD = b. Let f(a) = AD(a) - b. 

As a → 0+: u = v = sqrt(p²+h²), D → 0, AD = sqrt(p²+h²) = m. So f(0) = m - b.
As a → ∞: if p ≥ 0, u ≈ p+a, v ≈ a-p (for large a), Dx ≈ a*(2p)/(2a) = p. AD → sqrt((p-p)²+h²) = h. So f(∞) = h - b.

So if h < b < m (or h ≤ b ≤ m with some care), there's a solution with p ≥ 0.

I'll binary search on a.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    double h, b, m;
    // skip "Test #N" lines
    string line;
    while(getline(cin, line)){
        if(line.find("Test") != string::npos) continue;
        istringstream iss(line);
        if(iss >> h >> b >> m) break;
    }
    
    if(h > b || b > m){ cout << "NO\n"; return 0; }
    if(m*m < h*h){ cout << "NO\n"; return 0; }
    
    double p = sqrt(m*m - h*h);
    // f(a) = AD(a), search for a where AD = b
    // A = (p, h), B = (-a, 0), C = (a, 0)
    auto AD = [&](double a, double pp) -> double {
        double u = sqrt((pp+a)*(pp+a)+h*h);
        double v = sqrt((pp-a)*(pp-a)+h*h);
        double Dx = a*(u-v)/(u+v);
        return sqrt((pp-Dx)*(pp-Dx)+h*h);
    };
    
    // try p and -p
    for(int sign : {1, -1}){
        double pp = sign * p;
        double lo = 1e-9, hi = 1e9;
        double flo = AD(lo, pp), fhi = AD(hi, pp);
        if((flo - b)*(fhi - b) > 0) continue;
        for(int i = 0; i < 200; i++){
            double mid = (lo+hi)/2;
            if((AD(lo,pp)-b)*(AD(mid,pp)-b) <= 0) hi = mid;
            else lo = mid;
        }
        double a = (lo+hi)/2;
        if(fabs(AD(a,pp)-b) < 1e-6){
            cout << "YES\n";
            cout << fixed << setprecision(8) << pp << " " << h << "\n";
            cout << fixed << setprecision(8) << -a << " " << 0.0 << "\n";
            cout << fixed << setprecision(8) << a << " " << 0.0 << "\n";
            return 0;
        }
    }
    cout << "NO\n";
}
```
