We have a sequence a1, a2, ..., an where 0 ≤ ai ≤ m-1.

We define a sequence S(j) for j from 0 to m-1. The sequence S(j) is obtained by adding j to each element modulo m: S(j) = [(a1+j) mod m, (a2+j) mod m, ..., (an+j) mod m].

For each j from 0 to m-1, we need to find the smallest cyclic shift of S(j), and output its k-th element.

So for each j (0 to m-1):
1. Compute S(j) = [(ai+j) mod m for i in 1..n]
2. Find the lexicographically smallest cyclic shift of S(j)
3. Output the k-th element (1-indexed) of that shift

To find the smallest cyclic shift, I can use Booth's algorithm which finds the starting position of the lexicographically smallest rotation in O(n).

For each j, the sequence is just the original sequence with each element shifted by j mod m. So instead of recomputing from scratch each time, I can use Booth's algorithm on the transformed sequence.

Let me think of a smarter approach. The sequence S(j) = [(ai+j) mod m]. The cyclic shifts of S(j) are the same cyclic shifts as the original sequence but with all elements shifted by j mod m.

The lexicographic order of cyclic shifts of S(j) is the same as cyclic shifts of the original but with comparison using the shifted values. The starting position of the minimum rotation might change with j.

Actually, I think for this problem, given the constraints are not shown clearly, let me just implement O(n*m) with Booth's algorithm and hope it's fast enough.

Booth's algorithm returns the starting index (0-based) of the lexicographically smallest rotation. Then the k-th element (1-indexed) of that rotation starting at index `start` is `S(j)[(start + k - 1) % n]` = `(a[(start+k-1)%n] + j) % m`.

```cpp
#include <bits/stdc++.h>
using namespace std;

int booth(vector<int>& s) {
    int n = s.size();
    vector<int> f(2*n, -1);
    int k = 0;
    for (int j = 1; j < 2*n; j++) {
        int sj = s[j % n];
        int i = f[j - 1 - k];
        while (i != -1 && sj != s[(k + i + 1) % n]) {
            if (sj < s[(k + i + 1) % n]) k = j - i - 1;
            i = f[i];
        }
        if (sj != s[(k + i + 1) % n]) {
            if (sj < s[k % n]) k = j;
            f[j - k] = -1;
        } else {
            f[j - k] = i + 1;
        }
    }
    return k;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int n, m, k;
    cin >> n >> m >> k;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    vector<int> s(n);
    for (int j = 0; j < m; j++) {
        for (int i = 0; i < n; i++) s[i] = (a[i] + j) % m;
        int start = booth(s);
        int idx = (start + k - 1) % n;
        cout << s[idx] << "\n";
    }
    
    return 0;
}
```
