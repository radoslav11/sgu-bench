Looking at this problem, I need to find how many times each soldier asks a password (i.e., meets another soldier moving in the opposite direction) during time [0, T].

Key observations:
1. Two soldiers meet when they are at the same position AND moving in opposite directions.
2. The circle has perimeter 1000.
3. Soldiers move at constant speeds.

For soldiers i and j to meet, they need opposite velocities signs (one positive, one negative).

Position of soldier i at time t: `(L[i] + V[i] * t) mod 1000`

Two soldiers i and j meet at time t if:
- `(L[i] + V[i]*t) ≡ (L[j] + V[j]*t) (mod 1000)`
- They have opposite velocity directions (one positive, one negative)

The condition: `(L[i] - L[j]) + (V[i] - V[j])*t ≡ 0 (mod 1000)`

Let `dL = L[i] - L[j]`, `dV = V[i] - V[j]`

We need `dL + dV * t ≡ 0 (mod 1000)`, i.e., `dV * t ≡ -dL (mod 1000)`

We need to find all t in [0, T] satisfying this, where t can be a real number.

Actually t is continuous. The equation: `t = (-dL + 1000*k) / dV` for integer k.

For each pair (i,j) with opposite velocity signs, find all meeting times in [0, T].

Meeting times: `t = (-dL + 1000*k) / dV` for integer k, where t ∈ [0, T].

But we need to be careful: t=0 counts (inclusive), t=T also counts (inclusive based on "T minutes from initial moment inclusive").

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N, T;
    cin >> N >> T;
    vector<int> L(N), V(N);
    for(int i=0;i<N;i++) cin>>L[i];
    for(int i=0;i<N;i++) cin>>V[i];
    
    vector<int> B(N,0);
    
    for(int i=0;i<N;i++){
        for(int j=i+1;j<N;j++){
            if((V[i]>0&&V[j]<0)||(V[i]<0&&V[j]>0)){
                int dL = L[i]-L[j];
                int dV = V[i]-V[j];
                // t = (-dL + 1000*k)/dV in [0,T]
                // Find k range
                for(int k=-1100;k<=1100;k++){
                    double num = -dL + 1000.0*k;
                    if(dV==0) continue;
                    double t = num/dV;
                    if(t >= -1e-9 && t <= T+1e-9){
                        B[i]++;
                        B[j]++;
                    }
                }
            }
        }
    }
    
    for(int i=0;i<N;i++){
        if(i) cout<<" ";
        cout<<B[i];
    }
    cout<<endl;
}
```
