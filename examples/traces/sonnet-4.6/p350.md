Key observations:
1. M = N*(N-1)/2, so I need to find N from M.
2. The XOR of all elements in B: each A[i] appears in (N-1) pairs, so XOR of all B = XOR of A[i] each appearing (N-1) times. If (N-1) is odd, XOR of all B = XOR of all A. If (N-1) is even, XOR of all B = 0.

If I know A[0], I can find all other elements since A[0] XOR A[i] is in B for all i. So I need to guess A[0].

- But A[0]=0 might not work. However if A[0]=0, then A[0]^A[i]=A[i], so B contains all A[i] (i>0), plus all pairwise XORs of A[i] for i>0. This is similar to a linear algebra over GF(2) problem.

1. Place all B in multiset S.
2. Assume A[0] = 0.
3. For i from 1 to N-1: pick smallest element of S as A[i], then for all j < i, remove A[j] XOR A[i] from S.

This works if A[0]=0 is valid. Since the problem says any solution works, and if a solution exists with A[0]=x, then XORing everything with x gives a solution with A[0]=0 (all pairwise XORs remain the same since (a^x)^(b^x)=a^b). So we can always assume A[0]=0!

#include <bits/stdc++.h>
using namespace std;
int main(){
    int m; cin>>m;
    int n=1; while(n*(n-1)/2!=m) n++;
    multiset<int> S;
    for(int i=0;i<m;i++){int x;cin>>x;S.insert(x);}
    vector<int> A;
    A.push_back(0);
    while((int)A.size()<n){
        for(auto it=S.begin();it!=S.end();it++){
            int cand=*it;
            multiset<int> tmp=S;
            bool ok=true;
            for(int a:A){
                auto f=tmp.find(a^cand);
                if(f==tmp.end()){ok=false;break;}
                tmp.erase(f);
            }
            if(ok){A.push_back(cand);S=tmp;break;}
        }
    }
    for(int x:A) cout<<x<<" ";
}
