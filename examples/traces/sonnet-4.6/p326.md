Looking at this problem, I need to determine if team 1 can finish with at least as many wins as any other team in the division.

Key observations:
1. Team 1 can win all its remaining games: max possible wins for team 1 = w[1] + r[1]
2. Let W = w[1] + r[1] be the maximum wins team 1 can achieve
3. For each other team i, we need w[i] + (their future wins) ≤ W
4. So team i can win at most W - w[i] more games
5. Games between teams within the division (excluding team 1) need to be distributed such that no team exceeds their allowed wins

This is a max-flow problem similar to the baseball elimination problem.

The approach:
- Team 1 wins all remaining games: W = w[1] + r[1]
- For team i (i ≥ 2): the maximum additional wins allowed = W - w[i]. If this is negative, answer is "NO"
- For games between teams i and j (both ≥ 2), those games must be distributed. The combined wins from game (i,j) is exactly a[i][j], and we need both teams to stay within their caps.
- Also, games between team i (i≥2) and outside division teams: team i can win at most max(0, W - w[i]) total remaining games. Games vs team 1 will be lost by team i (since team 1 wins all). So team i's remaining games excluding vs team 1 = r[i] - a[i][1] games (some inside division, some outside).

Let me model this with max-flow:
- Source → game node (i,j) with capacity a[i][j]
- Game node (i,j) → team i node with capacity a[i][j]
- Game node (i,j) → team j node with capacity a[i][j]
- Team i node → sink with capacity cap[i] = W - w[i]

If max flow = sum of all a[i][j] for i,j ≥ 2, then all intra-division games (excluding team 1) can be scheduled within caps.

But what about games outside division? Team i has r[i] - a[i][1] - sum_j(a[i][j] for j≥2) games outside division, and they can win some of them. The cap[i] limits total additional wins including outside games. The flow through team i node represents wins from intra-division games among teams 2..N. The remaining wins from outside = cap[i] - (flow through team i). This is automatically handled as long as cap[i] is the total allowed.

Wait, but we need: wins from intra-div games (among 2..N) ≤ cap[i], and the remaining can come from outside or vs team 1 (but team 1 wins all, so team i gets 0 from vs team 1 games).

So the flow model handles it: if total flow equals sum of all a[i][j] (i<j, i,j≥2), then YES, else NO.

Let me also check: if W - w[i] < 0 for any i, immediately NO.

```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 1e9;

struct Edge { int to, cap, rev; };
vector<Edge> graph[500];
int level[500], iter[500];

void add_edge(int from, int to, int cap) {
    graph[from].push_back({to, cap, (int)graph[to].size()});
    graph[to].push_back({from, 0, (int)graph[from].size()-1});
}

bool bfs(int s, int t, int V) {
    fill(level, level+V, -1);
    queue<int> q;
    level[s] = 0; q.push(s);
    while (!q.empty()) {
        int v = q.front(); q.pop();
        for (auto& e : graph[v]) if (e.cap > 0 && level[e.to] < 0) { level[e.to] = level[v]+1; q.push(e.to); }
    }
    return level[t] >= 0;
}

int dfs(int v, int t, int f) {
    if (v == t) return f;
    for (int& i = iter[v]; i < (int)graph[v].size(); i++) {
        Edge& e = graph[v][i];
        if (e.cap > 0 && level[v] < level[e.to]) {
            int d = dfs(e.to, t, min(f, e.cap));
            if (d > 0) { e.cap -= d; graph[e.to][e.rev].cap += d; return d; }
        }
    }
    return 0;
}

int max_flow(int s, int t, int V) {
    int flow = 0;
    while (bfs(s, t, V)) { fill(iter, iter+V, 0); int d; while ((d=dfs(s,t,INF))>0) flow+=d; }
    return flow;
}

int main(){
    int N; cin>>N;
    vector<int> w(N), r(N);
    for (int i=0;i<N;i++) cin>>w[i];
    for (int i=0;i<N;i++) cin>>r[i];
    vector<vector<int>> a(N, vector<int>(N));
    for (int i=0;i<N;i++) for (int j=0;j<N;j++) cin>>a[i][j];
    
    int W = w[0] + r[0];
    for (int i=1;i<N;i++) if (w[i] > W) { cout<<"NO\n"; return 0; }
    
    // nodes: source=0, sink=1, teams 2..N+1, game nodes start at N+2
    int S=0, T=1;
    int total_nodes = 2 + (N-1) + (N-1)*(N-2)/2;
    for (int i=0;i<total_nodes;i++) graph[i].clear();
    
    int team_node[20];
    for (int i=1;i<N;i++) team_node[i] = 2+(i-1);
    int gnode = 2+(N-1);
    
    for (int i=1;i<N;i++) add_edge(team_node[i], T, W - w[i]);
    
    int total = 0;
    for (int i=1;i<N;i++) for (int j=i+1;j<N;j++) {
        if (a[i][j] > 0) {
            add_edge(S, gnode, a[i][j]);
            add_edge(gnode, team_node[i], a[i][j]);
            add_edge(gnode, team_node[j], a[i][j]);
            total += a[i][j];
        }
        gnode++;
    }
    
    cout << (max_flow(S, T, total_nodes) == total ? "YES" : "NO") << "\n";
}
```
