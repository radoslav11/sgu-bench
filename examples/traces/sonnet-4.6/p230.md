Looking at this problem:

- N boxes, each containing one coin of value 1..N (a permutation)
- Weight of coin i is i grams
- M constraints: box P is lighter than box Q, meaning coin in box P < coin in box Q
- Find a valid assignment or report "No solution"

This is essentially: assign distinct values 1..N to nodes such that for each edge P→Q, value[P] < value[Q].

This is a topological sort problem. We need to find a permutation assignment respecting the partial order defined by the constraints.

The approach:
1. Build a DAG from the constraints (P must have smaller value than Q)
2. Topological sort the boxes
3. Assign values 1, 2, ..., N in topological order (boxes earlier in topo order get smaller values)

If there's a cycle, output "No solution".

Let me think more carefully. The constraints say box[P] < box[Q] in terms of coin value. We need to assign a permutation of 1..N to the boxes respecting these inequalities.

Topological sort gives us an ordering of boxes. The box that comes first in topological order should get the smallest coin, etc. This works because topological sort respects all the constraints.

Wait, but there might be multiple valid assignments. The problem just asks for any valid one.

So:
1. Build directed graph: P → Q means P must come before Q (smaller value)
2. Topological sort
3. If cycle detected → "No solution"
4. Assign values: position k in topo order gets value k

Let me verify with the example:
- N=3, M=2
- Constraints: box 2 < box 1 (2→1), box 1 < box 3 (1→3)
- Topo order: 2, 1, 3
- Assignment: box 2 gets value 1, box 1 gets value 2, box 3 gets value 3
- Output: box[1]=2, box[2]=1, box[3]=3 → "2 1 3" ✓

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int N, M;
    cin >> N >> M;
    
    vector<vector<int>> adj(N + 1);
    vector<int> indegree(N + 1, 0);
    
    for (int i = 0; i < M; i++) {
        int p, q;
        cin >> p >> q;
        // box p lighter than box q: value[p] < value[q]
        // edge p -> q
        adj[p].push_back(q);
        indegree[q]++;
    }
    
    // Topological sort (Kahn's algorithm)
    queue<int> q;
    for (int i = 1; i <= N; i++) {
        if (indegree[i] == 0) {
            q.push(i);
        }
    }
    
    vector<int> topo;
    while (!q.empty()) {
        int node = q.front();
        q.pop();
        topo.push_back(node);
        for (int next : adj[node]) {
            indegree[next]--;
            if (indegree[next] == 0) {
                q.push(next);
            }
        }
    }
    
    if ((int)topo.size() != N) {
        cout << "No solution" << endl;
        return 0;
    }
    
    // topo[0] gets value 1, topo[1] gets value 2, etc.
    vector<int> ans(N + 1);
    for (int i = 0; i < N; i++) {
        ans[topo[i]] = i + 1;
    }
    
    for (int i = 1; i <= N; i++) {
        cout << ans[i];
        if (i < N) cout << " ";
    }
    cout << endl;
    
    return 0;
}
```
