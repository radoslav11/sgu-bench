Looking at this problem, I need to find the total length of the polygon's walls that are shaded by shelves, where light comes from the origin (0,0).

Each shelf (segment) blocks light in an angular range as seen from the origin. The shaded portion of a wall edge is where the shadow of any shelf falls on it.

**Approach:**
1. For each shelf segment, compute the angular interval [θ_min, θ_max] it subtends as seen from the origin.
2. Union all these angular intervals.
3. For each polygon edge, find the intersection of the shadow arc with the edge's angular span, then compute what portion of the edge is lit.

**Details:**

For each polygon edge (a line segment), and a given angular range [α, β]:
- The shadow from angle α hits the edge at the point where the ray from origin at angle α intersects the edge.
- Similarly for angle β.
- The shaded portion of the edge is the arc of rays between α and β intersecting the edge.

For each edge of the polygon, I need to compute the total arc of angles that hit that edge (rays from origin that intersect the edge), then intersect with the union of shadow intervals.

**Computing ray-edge intersection:** A ray from origin at angle θ hits edge from P to Q if the ray intersects the segment. The edge spans an angular range [θ_P, θ_Q] (in terms of atan2 of endpoints), though this needs care with wraparound.

For an edge, the angular range is the set of angles of rays from origin that intersect it. Since 0 is strictly inside the polygon, each edge subtends an angular interval.

For each shadow interval (from union), intersect with each edge's angular interval, compute the corresponding chord length on the edge.

**Converting angle range on edge to length:** 
Given ray at angle θ from origin hits edge at point t along the edge. I need to map angular intervals to arc lengths on the edge.

The intersection of ray direction (cos θ, sin θ) with edge P + s*(Q-P): 
- The parameter s and the distance r are related. The point on edge is at distance r = (P × Q) / (sin(θ - φ)) type formula... let me use the direct intersection formula.

Ray: (t·cos θ, t·sin θ). Edge: P + s·(Q-P), s∈[0,1].
t·cos θ = Px + s·(Qx-Px), t·sin θ = Py + s·(Qy-Py).

From these: t = (Px·(Qy-Py) - Py·(Qx-Px)) / (cos θ·(Qy-Py) - sin θ·(Qx-Px))

The point on edge as function of θ: x(θ), y(θ). The length element along the edge as θ varies can be computed.

Actually, for a given angular sub-interval [α,β] hitting the edge, the shaded length is simply the distance between the two intersection points (ray at α hits edge at point A, ray at β hits edge at point B), since all rays hit the same straight edge, the shaded part is segment AB, length = |A - B|.

This works because the edge is a straight line, and each angle maps to a unique point on it.

So for each edge, and each shadow sub-interval that intersects the edge's angular range, compute the two boundary rays' intersections with the edge and measure the distance.
