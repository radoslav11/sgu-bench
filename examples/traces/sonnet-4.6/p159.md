Looking at this problem, I need to find all n-digit numbers x in base b such that x² ≡ x (mod b^n), i.e., x(x-1) ≡ 0 (mod b^n).

This means x is an "automorphic" number - x² ends in x in base b.

**Approach: Digit-by-digit construction (lifting)**

I'll build the solution digit by digit. Start with 1-digit solutions, then lift to 2-digit, etc.

A number x is self-replicating if x² ≡ x (mod b^n), i.e., x² - x ≡ 0 (mod b^n).

**Lifting:** If x is a solution mod b^k, I try x + d*b^k for d in 0..b-1 and check if it's a solution mod b^(k+1).

(x + d*b^k)² - (x + d*b^k) = x² - x + 2x*d*b^k - d*b^k + d²*b^(2k)
≡ x² - x + d*b^k*(2x-1) (mod b^(k+1))

So I need: (x² - x)/b^k + d*(2x-1) ≡ 0 (mod b).

But with n up to 2000, I need big number arithmetic.

I'll implement this with Python-style big integers using vectors. Actually, let me use Java BigInteger... no, C++. I'll implement big number mod operations with digit arrays in base b.

Since we're working mod b^k and extending, I'll represent numbers as arrays of digits in base b (little-endian).

For each step k (from 1 to n), I maintain a list of candidate solutions (as digit arrays of length k). For each candidate x of length k, I try extending by digit d (0..b-1) to get x' = x + d*b^k, then check if x'² ≡ x' (mod b^(k+1)).

The key computation: check if (x² - x + d*b^k*(2x-1)) ≡ 0 (mod b^(k+1)).

Since x² ≡ x (mod b^k), we have (x²-x) = q*b^k for some integer q. So condition is: q + d*(2x-1) ≡ 0 (mod b). Here q mod b is computed from (x²-x)/b^k mod b, and (2x-1) mod b.

Also, the n-digit constraint: the leading digit (digit at position n-1) must be nonzero.

I'll work with numbers represented as vectors of digits mod b^n using big integer arithmetic with base-b representation.

Let me implement big number arithmetic (addition, multiplication, mod b^k) using vectors in base b.

```cpp
#include <bits/stdc++.h>
using namespace std;

int B, N;

// Numbers represented as vectors of digits in base B, little-endian, length N
// We work mod B^N

typedef vector<int> BigN;

BigN multiply(const BigN& a, const BigN& b, int len) {
    BigN res(len, 0);
    for (int i = 0; i < len; i++) {
        if (a[i] == 0) continue;
        long long carry = 0;
        for (int j = 0; j < len - i; j++) {
            long long cur = res[i+j] + (long long)a[i]*b[j] + carry;
            res[i+j] = cur % B;
            carry = cur / B;
        }
    }
    return res;
}

BigN subtract(const BigN& a, const BigN& b, int len) {
    BigN res(len);
    int borrow = 0;
    for (int i = 0; i < len; i++) {
        int d = a[i] - b[i] - borrow;
        if (d < 0) { d += B; borrow = 1; } else borrow = 0;
        res[i] = d;
    }
    return res;
}

int main(){
    cin >> B >> N;
    // candidates: list of digit arrays of increasing length
    vector<BigN> cands;
    // Start: 1-digit solutions x in [0,B-1] with x^2 ≡ x (mod B)
    for (int x = 0; x < B; x++) {
        if ((long long)x*x % B == x % B) {
            BigN v(N, 0); v[0] = x;
            cands.push_back(v);
        }
    }
    for (int k = 1; k < N; k++) {
        vector<BigN> next;
        for (auto& x : cands) {
            // compute x^2 - x, check digit k
            BigN x2 = multiply(x, x, k+1);
            BigN diff = subtract(x2, x, k+1);
            // diff[0..k-1] should be 0, diff[k] is the carry
            int q = diff[k];
            int twox1 = (2*x[0] % B - 1 + B) % B; // (2x-1) mod B ... need (2x-1) mod B where x mod B = x[0]
            // find d: q + d*twox1 ≡ 0 (mod B)
            for (int d = 0; d < B; d++) {
                if ((q + (long long)d*twox1) % B == 0) {
                    BigN nx = x; nx[k] = d;
                    next.push_back(nx);
                }
            }
        }
        cands = next;
    }
    // Filter: leading digit (index N-1) != 0
    vector<BigN> res;
    for (auto& x : cands) if (x[N-1] != 0) res.push_back(x);
    cout << res.size() << "\n";
    string alpha = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    for (auto& x : res) {
        string s(N, '0');
        for (int i = 0; i < N; i++) s[N-1-i] = alpha[x[i]];
        cout << s << "\n";
    }
}
```
