Looking at this problem, I need to find a Knight's tour on an N×N chessboard.

Key observations:
- N=1: trivially solved (1 cell)
- N=2,3,4: no solution exists
- N≥5: solution exists

For competitive programming with N up to 250, I need an efficient algorithm. Warnsdorff's heuristic works well: always move to the square with the fewest onward moves.

Let me implement Warnsdorff's heuristic:
1. Start from (0,0)
2. At each step, choose the move that leads to a square with minimum number of accessible neighbors
3. Break ties (e.g., by choosing the one with the smallest next-degree or by some fixed order)

For large boards, Warnsdorff's usually works well. However, it can fail for some starting positions. I'll try different starting positions if needed.

Known cases with no solution: N=2, N=3, N=4. For N=1, output is just "1".

Let me implement this carefully. For tie-breaking in Warnsdorff's, I'll use the accessibility count of the next step (looking 2 steps ahead) or just try multiple starting positions.

Actually, for robustness, let me try all starting positions (or a few) until one works, using Warnsdorff's heuristic.

With N=250, board has 62500 cells. Trying all starting positions would be 62500 × 62500 operations in worst case - too slow.

Instead, I'll use Warnsdorff's from position (0,0) and if it fails, try a few other positions. In practice, Warnsdorff's almost always works from any starting position for N≥5.

For tie-breaking, I'll use the sum of degrees of neighbors of the candidate (Warnsdorff's rule applied one level deeper).

```cpp
#include <bits/stdc++.h>
using namespace std;

int N;
int board[255][255];
int dx[] = {1,2,2,1,-1,-2,-2,-1};
int dy[] = {2,1,-1,-2,-2,-1,1,2};

int degree(int x, int y) {
    int cnt = 0;
    for(int i=0;i<8;i++){
        int nx=x+dx[i], ny=y+dy[i];
        if(nx>=0&&nx<N&&ny>=0&&ny<N&&board[nx][ny]==0) cnt++;
    }
    return cnt;
}

bool solve(int sx, int sy) {
    memset(board,0,sizeof(board));
    int x=sx, y=sy;
    board[x][y]=1;
    for(int step=2;step<=N*N;step++){
        int best=-1, bx=-1, by=-1;
        for(int i=0;i<8;i++){
            int nx=x+dx[i], ny=y+dy[i];
            if(nx>=0&&nx<N&&ny>=0&&ny<N&&board[nx][ny]==0){
                int d=degree(nx,ny);
                if(best==-1||d<best){
                    best=d; bx=nx; by=ny;
                }
            }
        }
        if(bx==-1) return false;
        board[bx][by]=step;
        x=bx; y=by;
    }
    return true;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cin>>N;
    if(N==1){
        cout<<"There is solution:"<<endl;
        cout<<"1"<<endl;
        return 0;
    }
    if(N<=4){
        cout<<"No solution."<<endl;
        return 0;
    }
    // Try multiple starting positions
    bool found=false;
    for(int si=0;si<N&&!found;si++){
        for(int sj=0;sj<N&&!found;sj++){
            if(solve(si,sj)) found=true;
        }
    }
    if(!found){
        cout<<"No solution."<<endl;
    } else {
        cout<<"There is solution:"<<endl;
        for(int i=0;i<N;i++){
            for(int j=0;j<N;j++){
                if(j) cout<<" ";
                cout<<board[i][j];
            }
            cout<<"\n";
        }
    }
    return 0;
}
```
