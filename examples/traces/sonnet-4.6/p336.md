Looking at this problem, I need to track:
1. Which blocks/parties have discreditable information about other blocks/parties
2. Merging blocks together

Let me understand the semantics:
- Initially, party `a` knows something about party `b` for given pairs
- When we merge block `a` and block `b` into new block `c`:
  - Block `c` has all info that `a` had AND all info that `b` had (union of their "knows about" sets)
  - All info that was known about `a` and `b` now concerns `c` (anyone who knew about `a` or `b` now knows about `c`)

Query `1 a b`: Does `a` know anything discreditable about `b`?

So I need to track:
- For each entity: what entities does it "know about" (outgoing edges)
- For each entity: what entities "know about" it (incoming edges)

When merging `a` and `b` into new `c`:
- `c`'s "knows about" = `a`'s "knows about" ∪ `b`'s "knows about"
- For each entity `x` that knew about `a` or `b`, now `x` knows about `c` (instead of `a`/`b`)
- `c`'s "known by" = `a`'s "known by" ∪ `b`'s "known by"

For Query `1 a b`: check if `b` is in `a`'s "knows about" set.

I'll use sets and merge by size (small-to-large) to keep efficiency.

Each entity has:
- `out_set`: set of entities it knows about
- `in_set`: set of entities that know about it

When merging `a` and `b` → new `c` (numbered N+1, N+2, ...):
- `out[c]` = merge of `out[a]` and `out[b]`
- `in[c]` = merge of `in[a]` and `in[b]`
- For each `x` in `in[a]`: remove `a` from `out[x]`, add `c` to `out[x]`
- For each `x` in `in[b]`: remove `b` from `out[x]`, add `c` to `out[x]`
- For each `x` in `out[a]`: remove `a` from `in[x]`, add `c` to `in[x]`
- For each `x` in `out[b]`: remove `b` from `in[x]`, add `c` to `in[x]`

This is O(M log N) with careful implementation using unordered_sets or sets.

Let me use `unordered_set` for O(1) operations.

The total work is bounded by the total number of edge updates. Each edge (u,v) can be updated when u or v gets merged. With small-to-large, each element moves O(log N) times, so total work is O((N+M) log N).

        int a, b;
        scanf("%d %d", &a, &b);
        out[a].insert(b);
        in[b].insert(a);
    }
    
    int Q;
    scanf("%d", &Q);
    int nextId = N + 1;
    
    while(Q--){

        scanf("%d %d %d", &type, &a, &b);
        if(type == 1){
            printf(out[a].count(b) ? "YES\n" : "NO\n");
        } else {
            int c = nextId++;
            // merge out[a] and out[b] into out[c]
            // merge in[a] and in[b] into in[c]
            // update references
            // For each x in in[a]: out[x]: remove a, add c
            for(int x : in[a]) { out[x].erase(a); out[x].insert(c); }

            out[b].clear(); in[b].clear();
