The cost is S operations (opening all links of sacrificed chains). We want to minimize S.

So: choose a subset of chains to sacrifice such that sum_of_their_lengths >= N - (number in subset) - 1, minimize sum_of_their_lengths.

Let me try all subsets or greedily pick smallest chains first.

For each possible subset size m (0,1,2,...), sort chains ascending, try sacrificing m smallest chains. Check if sum(m smallest) >= N - m - 1. Find minimum sum.

Special case: N=1, answer is 0.

int main() {

    if (n == 1) {
        cout << 0 << endl;
        return 0;
    }
    
    sort(L.begin(), L.end());
    
    int ans = INT_MAX;
    // Try sacrificing m chains (the m smallest)
    for (int m = 0; m <= n; m++) {
        int s = 0;
        for (int i = 0; i < m; i++) s += L[i];
        // remaining chains: n - m, need n-m-1 connectors
        // if n-m <= 1, already connected (or 0 chains left but we have connectors)
        int need = max(0, (n - m) - 1);
        if (s >= need) {
            ans = min(ans, s);
        }
    }
    
    cout << ans << endl;
    return 0;
}
