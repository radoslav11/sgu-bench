Looking at this problem, I need to find a minimum cost simple polygon that:
1. Passes along province borders
2. Separates two headquarters in different provinces

The key insight is that the wall is a simple polygon along province borders that separates the two HQ points. This is essentially finding a minimum cut in a planar graph where we need a cycle that separates two faces (the faces containing the two HQs).

In planar graphs, minimum cut between two faces equals minimum cycle separating those faces, which by planar duality equals shortest path between the dual nodes corresponding to those faces.

So the approach is:
1. Build the planar graph from segments
2. Find which face contains each HQ
3. In the dual graph, find shortest path between those two dual nodes
4. The path in the dual graph corresponds to a cycle in the primal graph
