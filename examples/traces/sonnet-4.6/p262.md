Looking at this problem, I need to find a minimal set of pixels such that every pair of symbols differs on at least one pixel in the set.

This is essentially a set cover / hitting set problem: for each pair of symbols, I need at least one pixel that distinguishes them (i.e., has different values in the two symbols). I need to find the minimum number of pixels such that all pairs are "hit".

With N,M ≤ 10, there are up to 100 pixels. With K ≤ 6, there are at most C(6,2) = 15 pairs.

Since the answer is at most... well, let's think about the minimum. I can try BFS/brute force over subsets of pixels. But 2^100 is too large.

Wait, but I only need to find which pixels to include. For each pixel p, it has a "signature" over pairs: for each pair (i,j), does pixel p distinguish them? I need to cover all pairs.

So this is a minimum hitting set / set cover problem where:
- Universe = set of pairs of symbols (at most 15 elements)
- Each pixel covers a subset of pairs (those it distinguishes)

With at most 15 pairs, I can represent coverage as a bitmask. I need to find minimum number of pixels whose union of coverage = all pairs that are distinguishable.

Since there are up to 100 pixels, and I need minimum set cover with universe size 15, I can use dynamic programming on subsets of pairs.

DP approach: dp[mask] = minimum pixels needed to cover exactly the pairs in mask. But I want to cover all pairs (full mask). 

Actually: dp[mask] = minimum pixels to cover at least the pairs in mask. 

Let me do it differently: 
- For each pixel, compute its coverage bitmask.
- Find minimum subset of pixels covering the full mask (all pairs).

With universe of 15, I can do DP: dp[mask] = min pixels to achieve coverage mask.

Number of states = 2^15 = 32768. For each state, I try adding each pixel. This gives 32768 * 100 = ~3M operations. Very fast.

Also need to track which pixels were chosen.

Let me implement this:

1. Read symbols
2. For each pair (i,j), compute which pixels differ
3. For each pixel p, compute its pair-coverage bitmask
4. Run DP to find minimum set of pixels covering all pairs
5. Output the result

For reconstruction, store for each dp state, which pixel was last added and what the previous state was.

Let me also handle the case where some pairs might not be distinguishable by any single pixel... but the problem says symbols are unique, so every pair has at least one distinguishing pixel.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N, M, K;
    cin >> N >> M >> K;
    
    vector<vector<string>> sym(K, vector<string>(N));
    for(int k = 0; k < K; k++){
        for(int i = 0; i < N; i++) cin >> sym[k][i];
    }
    
    int numPairs = K*(K-1)/2;
    // pair index
    vector<pair<int,int>> pairs;
    for(int i = 0; i < K; i++)
        for(int j = i+1; j < K; j++)
            pairs.push_back({i,j});
    
    int P = N*M;
    // For each pixel, compute coverage bitmask
    vector<int> cov(P);
    for(int p = 0; p < P; p++){
        int r = p/M, c = p%M;
        int mask = 0;
        for(int q = 0; q < (int)pairs.size(); q++){
            int a = pairs[q].first, b = pairs[q].second;
            if(sym[a][r][c] != sym[b][r][c]) mask |= (1<<q);
        }
        cov[p] = mask;
    }
    
    int full = (1<<numPairs)-1;
    vector<int> dp(1<<numPairs, INT_MAX);
    vector<int> from_mask(1<<numPairs, -1);
    vector<int> from_pixel(1<<numPairs, -1);
    dp[0] = 0;
    
    for(int mask = 0; mask < (1<<numPairs); mask++){
        if(dp[mask] == INT_MAX) continue;
        for(int p = 0; p < P; p++){
            int nmask = mask | cov[p];
            if(dp[nmask] > dp[mask]+1){
                dp[nmask] = dp[mask]+1;
                from_mask[nmask] = mask;
                from_pixel[nmask] = p;
            }
        }
    }
    
    // Reconstruct
    vector<bool> chosen(P, false);
    int cur = full;
    while(cur != 0){
        int p = from_pixel[cur];
        chosen[p] = true;
        cur = from_mask[cur];
    }
    
    cout << dp[full] << "\n";
    for(int r = 0; r < N; r++){
        for(int c = 0; c < M; c++){
            cout << (chosen[r*M+c] ? '1' : '0');
        }
        cout << "\n";
    }
}
```
