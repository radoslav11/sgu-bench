4. Output accepted applications or "Army"

Let me parse carefully:

- M scores B[1..M]
- For each faculty:
  - Faculty name (line)
  - K and K subject numbers
  - K triplets X_i < Y_i < Z_i (3*K numbers on one line)
  - S specialties, each with name and required sum

Grade calculation: if T < X → 2, X ≤ T < Y → 3, Y ≤ T < Z → 4, Z ≤ T → 5

The subject numbers are 1-indexed.
