Looking at this problem, I need to find the fixed monthly payment x for an annuity loan.

**Math Setup:**

Let me derive the formula. Each month:
- Current debt: s'
- Interest part: a = s' * p / 100
- Principal part: b = x - a = x - s' * p / 100
- New debt: s' - b = s' - x + s' * p/100 = s' * (1 + p/100) - x

Let r = 1 + p/100. After each month, debt multiplies by r and decreases by x.

Starting debt: s
After month 1: s*r - x
After month 2: (s*r - x)*r - x = s*r² - x*r - x
After month m: s*r^m - x*(r^(m-1) + r^(m-2) + ... + 1) = s*r^m - x*(r^m - 1)/(r - 1)

For the debt to be 0 after m months:
s*r^m - x*(r^m - 1)/(r - 1) = 0

So: x = s * r^m * (r - 1) / (r^m - 1)

**Special case when p = 0:**
r = 1, so r^m = 1, and we have division by zero. When p=0, no interest, so x = s/m.

Let me verify with the example: s=100, m=2, p=50
r = 1.5
x = 100 * 1.5² * 0.5 / (1.5² - 1) = 100 * 2.25 * 0.5 / (2.25 - 1) = 100 * 1.125 / 1.25 = 112.5 / 1.25 = 90 ✓

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double s, m, p;
    cin >> s >> m >> p;
    
    double x;
    if (p == 0) {
        x = s / m;
    } else {
        double r = 1.0 + p / 100.0;
        double rm = pow(r, m);
        x = s * rm * (r - 1.0) / (rm - 1.0);
    }
    
    cout << fixed << setprecision(5) << x << endl;
    return 0;
}
```
