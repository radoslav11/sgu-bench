Looking at this problem:

- Start with `b` girls (boys are away initially)
- At each observation, some boys may join (0 to remaining boys) and some girls may leave (0 to current girls)
- We need to find how many boys joined and girls left before each observation

Let me think about this carefully:
- Initial state: `current = b` (all girls, no boys)
- Before observation `i`, let `join_i` boys join and `leave_i` girls leave
- New count = `current + join_i - leave_i`
- Constraints: `join_i >= 0`, `leave_i >= 0`, `join_i <= remaining_boys`, `leave_i <= current_girls_count`

We need to track boys remaining and girls remaining separately.

Let me define:
- `boys_in_group`: boys currently in group
- `girls_in_group`: girls currently in group
- `boys_available`: boys not yet joined = `a - boys_joined_so_far`

Initially: `boys_in_group = 0`, `girls_in_group = b`, `boys_available = a`

At each step:
- We add `j` boys and remove `g` girls
- New total = `boys_in_group + j + girls_in_group - g = target[i]`
- Need: `j >= 0`, `j <= boys_available`, `g >= 0`, `g <= girls_in_group`

So: `j - g = target[i] - boys_in_group - girls_in_group`

Let `delta = target[i] - (boys_in_group + girls_in_group)`

We need `j - g = delta`, with `0 <= j <= boys_available`, `0 <= g <= girls_in_group`.

Strategy: if delta >= 0, we need to add more than we remove. Set `j = delta, g = 0` if `delta <= boys_available`. Otherwise try `j = boys_available, g = boys_available - delta` — but g must be >= 0, so this only works if `boys_available >= delta`. Hmm.

General approach: pick `j` and `g` such that `j - g = delta`.
- If delta >= 0: try `j = delta, g = 0`. Check `j <= boys_available`. If not, we can increase both: `j = boys_available`, `g = boys_available - delta`. Check `g >= 0` (need `boys_available >= delta`) and `g <= girls_in_group`.
- If delta < 0: try `j = 0, g = -delta`. Check `g <= girls_in_group`. If not, increase both: `g = girls_in_group`, `j = girls_in_group + delta`. Check `j >= 0` (need `girls_in_group + delta >= 0`) and `j <= boys_available`.

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int a, b, n;
    cin >> a >> b >> n;
    vector<int> t(n);
    for(int i=0;i<n;i++) cin >> t[i];
    
    int boys_in = 0, girls_in = b, boys_avail = a;
    vector<pair<int,int>> ans;
    bool ok = true;
    
    for(int i=0;i<n;i++){
        int cur = boys_in + girls_in;
        int delta = t[i] - cur;
        int j = 0, g = 0;
        if(delta >= 0){
            j = delta; g = 0;
            if(j > boys_avail){
                // increase g too: j = boys_avail, g = boys_avail - delta (negative, error)
                ok = false; break;
            }
        } else {
            g = -delta; j = 0;
            if(g > girls_in){
                // increase j: g = girls_in, j = girls_in + delta
                j = girls_in + delta; g = girls_in;
                if(j < 0 || j > boys_avail){ ok = false; break; }
            }
        }
        ans.push_back({j, g});
        boys_in += j;
        girls_in -= g;
        boys_avail -= j;
    }
    
    if(!ok) cout << "ERROR\n";
    else for(auto& p : ans) cout << p.first << " " << p.second << "\n";
}
```
