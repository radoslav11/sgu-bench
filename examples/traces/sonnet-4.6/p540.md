Looking at this problem, I need to find a speed v0 in [vmin, vmax] that minimizes the number of traffic lights that need to be switched to "always green" (i.e., traffic lights that are red when the king passes them).

Key observations:
- King starts at position 0 at time 0, drives at speed v0
- Arrives at traffic light i at time t_i = x_i / v0
- Traffic light i has cycle period p_i = r_i + g_i
- The light is GREEN when the king passes if: (t_i - d_i) mod p_i <= g_i (it's green from time d_i to d_i + g_i, then red, cyclically)

So the light is green at time t if: t mod p_i is in [d_i + r_i, d_i + p_i) mod p_i union [0, d_i). Hmm, let me think more carefully.

At time 0, the light just changed from green to red at time d_i (or will change at d_i). Green intervals: [0, d_i), [d_i + r_i, d_i + r_i + g_i), etc.

So green at time t if: (t mod p_i) ∈ [0, d_i) ∪ [d_i + r_i, p_i).

The speed v0 is continuous in [vmin, vmax]. For each traffic light i, t_i = x_i / v0. As v0 varies, t_i varies. The condition "light is green" defines certain intervals of v0.

Since vmin=10, vmax=50, x_i up to 20000, r,g in [10,20], I can discretize v0. The possible "interesting" speeds come from the boundaries of green intervals for each traffic light.

For each traffic light i and each period cycle k, the green condition t_i mod p_i ∈ [0, d_i) ∪ [d_i+r_i, p_i). This gives t_i ∈ union of intervals [k*p_i, k*p_i + d_i) ∪ [k*p_i + d_i + r_i, (k+1)*p_i).

t_i = x_i / v0, so v0 = x_i / t_i. Boundaries occur at t_i = k*p_i + d_i or k*p_i + d_i + r_i.

The range of t_i is [x_i/vmax, x_i/vmin]. 

I'll enumerate all critical speeds (boundaries), sort them, and for each interval between consecutive boundaries, pick a test point and count how many lights are red. I want to minimize red lights, and among equal, maximize v0.

Since there can be many critical speeds, I'll collect all boundary v0 values from each traffic light, add vmin and vmax, sort them, and test midpoints of each interval plus the endpoints themselves.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n, s, vmin, vmax;
    cin >> n >> s >> vmin >> vmax;
    vector<int> x(n), r(n), g(n), d(n);
    for(int i=0;i<n;i++) cin>>x[i]>>r[i]>>g[i]>>d[i];
    
    vector<double> cands;
    cands.push_back(vmin); cands.push_back(vmax);
    for(int i=0;i<n;i++){
        int p = r[i]+g[i];
        double tmin = (double)x[i]/vmax, tmax = (double)x[i]/vmin;
        for(int k=0; k*p <= tmax+p; k++){
            double t1 = k*p + d[i];
            double t2 = k*p + d[i] + r[i];
            if(t1>0 && t1>=tmin-1e-9 && t1<=tmax+1e-9){ double v=x[i]/t1; if(v>=vmin-1e-9&&v<=vmax+1e-9) cands.push_back(v); }
            if(t2>0 && t2>=tmin-1e-9 && t2<=tmax+1e-9){ double v=x[i]/t2; if(v>=vmin-1e-9&&v<=vmax+1e-9) cands.push_back(v); }
        }
    }
    sort(cands.begin(),cands.end());
    cands.erase(unique(cands.begin(),cands.end()),cands.end());
    
    auto countRed=[&](double v)->int{ int cnt=0; for(int i=0;i<n;i++){ double t=x[i]/v; int p=r[i]+g[i]; double tm=fmod(t,p); if(tm<0)tm+=p; if(!(tm<d[i]+1e-9 || tm>=d[i]+r[i]-1e-9)) cnt++; } return cnt; };
    auto getSet=[&](double v)->vector<int>{ vector<int> res; for(int i=0;i<n;i++){ double t=x[i]/v; int p=r[i]+g[i]; double tm=fmod(t,p); if(!(tm<d[i]+1e-9||tm>=d[i]+r[i]-1e-9)) res.push_back(i+1); } return res; };
    
    int best=n+1; double bestv=-1;
    vector<double> toTest=cands;
    for(int i=0;i+1<(int)cands.size();i++) toTest.push_back((cands[i]+cands[i+1])/2);
    sort(toTest.begin(),toTest.end());
    for(double v:toTest){ if(v<vmin-1e-9||v>vmax+1e-9)continue; int c=countRed(v); if(c<best||(c==best&&v>bestv)){best=c;bestv=v;} }
    
    cout<<fixed<<setprecision(10)<<bestv<<"\n";
    auto res=getSet(bestv);
    cout<<res.size()<<"\n";
    for(int x:res) cout<<x<<" "; if(!res.empty())cout<<"\n";
}
```
