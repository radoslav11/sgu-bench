Looking at this problem, I need to arrange numbers 0 to 2^(N+M)-1 in a 2^N × 2^M matrix such that adjacent cells (including cyclic adjacency) differ in exactly one bit.

This is essentially placing a Gray code on a toroidal grid where rows and columns both form Gray code sequences.

Key insight: If I use a Gray code for rows (of length 2^N) and a Gray code for columns (of length 2^M), then I can combine them. The value at position (row, col) should be `gray(row) * 2^M + gray(col)` - but wait, that won't work because adjacent rows would differ in one bit in the row part and zero bits in the column part, which is fine. Adjacent columns would differ in one bit in the column part. But the numbers need to be from 0 to 2^(N+M)-1.

Actually, let me think more carefully. The standard Gray code for rows gives row indices where adjacent rows differ in 1 bit. Similarly for columns. 

If I set `matrix[r][c] = (gray_code(r) << M) | gray_code(c)`, then:
- Adjacent rows (same col): values differ by `(gray(r) XOR gray(r+1)) << M`, which is a single bit in the upper N bits. ✓
- Adjacent cols (same row): values differ by `gray(c) XOR gray(c+1)`, which is a single bit in the lower M bits. ✓

The cyclic property also holds because standard reflected Gray code is cyclic (first and last elements differ in exactly 1 bit).

Also, all values 0 to 2^(N+M)-1 appear exactly once because gray codes are permutations.

The standard Gray code for k is: `k ^ (k >> 1)`.

Let me verify with the sample:
- N=1, M=1: 2^1=2 rows, 2^1=2 columns
- gray(0)=0, gray(1)=1
- matrix[0][0] = (0<<1)|0 = 0
- matrix[0][1] = (0<<1)|1 = 1
- matrix[1][0] = (1<<1)|0 = 2
- matrix[1][1] = (1<<1)|1 = 3

So output would be:
```

2 3
```

Both are valid! The problem likely accepts any valid arrangement. Let me just use my formula.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int N, M;
    cin >> N >> M;
    
    int rows = 1 << N;
    int cols = 1 << M;
    
    for (int r = 0; r < rows; r++) {
        int gray_r = r ^ (r >> 1);
        for (int c = 0; c < cols; c++) {
            int gray_c = c ^ (c >> 1);
            int val = (gray_r << M) | gray_c;
            if (c > 0) cout << ' ';
            cout << val;
        }
        cout << '\n';
    }
    
    return 0;
}
```
