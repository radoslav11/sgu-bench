Looking at this problem, I need to simulate a robot that:
1. Starts at position (is, js)
2. Places a bomb, moves to next building
3. The bomb explodes when robot leaves
4. Next building priority: (i+1,j), (i,j-1), (i-1,j), (i,j+1) → D, L, U, R
5. If no neighbors exist (all destroyed), robot explodes with current building

The robot destroys the building it's leaving. So when it moves from (i,j) to the next, (i,j) is destroyed.

The final building (where robot is stuck) also gets destroyed (robot explodes with it).

Let me trace through the example:
- 3x3 grid, start at (1,2)
- At (1,2): neighbors in order D=(2,2), L=(1,1), U=(0,2 invalid), R=(1,3). First non-exploded non-invalid: (2,2). Move D. (1,2) explodes.
- At (2,2): neighbors D=(3,2), L=(2,1), U=(1,2 exploded), R=(2,3). First: (3,2). Move D. (2,2) explodes.
- At (3,2): neighbors D=(4,2 invalid), L=(3,1), U=(2,2 exploded), R=(3,3). First: (3,1). Move L. (3,2) explodes.
- At (3,1): neighbors D=(4,1 invalid), L=(3,0 invalid), U=(2,1), R=(3,2 exploded). First: (2,1). Move U. (3,1) explodes.
- At (2,1): neighbors D=(3,1 exploded), L=(2,0 invalid), U=(1,1), R=(2,2 exploded). First: (1,1). Move U. (2,1) explodes.
- At (1,1): neighbors D=(2,1 exploded), L=(1,0 invalid), U=(0,1 invalid), R=(1,2 exploded). No valid move. Robot explodes here.

Output: DDLUU ✓

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N, M, is, js;
    cin >> N >> M >> is >> js;
    
    // Convert to 0-indexed
    int r = is - 1, c = js - 1;
    
    bool destroyed[10][10] = {};
    
    // Directions: D, L, U, R
    int dr[] = {1, 0, -1, 0};
    int dc[] = {0, -1, 0, 1};
    char dname[] = {'D', 'L', 'U', 'R'};
    
    string result = "";
    
    while(true){
        // Find next building
        int nr = -1, nc = -1;
        char move = ' ';
        for(int d = 0; d < 4; d++){
            int tr = r + dr[d];
            int tc = c + dc[d];
            if(tr >= 0 && tr < N && tc >= 0 && tc < M && !destroyed[tr][tc]){
                nr = tr; nc = tc;
                move = dname[d];
                break;
            }
        }
        
        if(nr == -1){
            // Robot explodes with current building
            destroyed[r][c] = true;
            break;
        }
        
        // Move
        destroyed[r][c] = true;
        result += move;
        r = nr;
        c = nc;
    }
    
    cout << result << endl;
    
    return 0;
}
```
