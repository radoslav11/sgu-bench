Looking at this problem, I need to find a partition of N letters into K groups (buttons) such that no two words in the vocabulary map to the same pair of buttons.

A word "xy" maps to buttons (button[x], button[y]). Two words conflict if they map to the same ordered pair of buttons.

So the constraint is: for any two distinct words (x1,y1) and (x2,y2) in vocabulary, it must NOT be the case that button[x1]==button[x2] AND button[y1]==button[y2].

I need to find minimum K and a valid assignment of letters to buttons (a partition into K groups).

Since N ≤ 13, I can think of this as a graph coloring problem where we need to assign colors (buttons) to letters such that no conflicts arise.

Two letters a and b MUST be on different buttons if: there exist words (a,c) and (b,c) for some c (same second letter but different first), OR words (c,a) and (c,b) for some c (same first letter but different second). Wait, more precisely:

If letters i and j are in the same group, then for any words (i,x) and (j,x) - these would both map to (same_button, button[x]), causing a conflict. Similarly for words (x,i) and (x,j).

So: i and j cannot be in the same group if:
1. There exist words (i,x) and (j,x) for some x (they share a "right neighbor")
2. There exist words (x,i) and (x,j) for some x (they share a "left neighbor")

This gives us a graph where edges mean "must be on different buttons". The minimum number of buttons is the chromatic number of this graph.

With N ≤ 13, I can try to find the chromatic number using bitmask DP.

Let me build the conflict graph: edge between i and j if they conflict as described above.

Then find chromatic number of graph with ≤ 13 nodes. I'll use the inclusion-exclusion / DP approach.

The chromatic number can be found by trying k=1,2,... and checking if k colors suffice. For k colors, I need to cover all vertices with k independent sets.

With N=13, I can enumerate all independent sets (subsets with no edges between them) and use DP with bitmasks to find minimum number of independent sets covering all vertices.

DP: dp[mask] = minimum colors needed to color exactly the vertices in mask. 
- dp[0] = 0
- dp[mask] = min over all independent subsets S of mask: dp[mask ^ S] + 1

Precompute all independent sets. Then DP over all 2^13 = 8192 masks.

Let me code this up.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N, M;
    cin >> N >> M;
    
    vector<pair<int,int>> words(M);
    for(int i=0;i<M;i++){
        string s; cin>>s;
        words[i] = {s[0]-'a', s[1]-'a'};
    }
    
    // Build conflict graph
    vector<int> adj(N, 0);
    for(int i=0;i<N;i++)
        for(int j=i+1;j<N;j++){
            bool conflict = false;
            for(auto& w: words){
                // check if i,j share right neighbor
                // words (i,x) and (j,x)
                // words (x,i) and (x,j)
            }
            for(auto& w1: words){
                for(auto& w2: words){
                    if(w1==w2) continue;
                    if((w1.first==i && w2.first==j && w1.second==w2.second) ||
                       (w1.second==i && w2.second==j && w1.first==w2.first))
                        conflict=true;
                }
            }
            if(conflict){
                adj[i]|=(1<<j);
                adj[j]|=(1<<i);
            }
        }
    
    int full = (1<<N)-1;
    // Precompute independent sets
    vector<bool> indep(1<<N, true);
    for(int mask=1;mask<=(1<<N)-1;mask++){
        int v = __builtin_ctz(mask);
        if((adj[v]&mask)!=0) indep[mask]=false;
        else indep[mask]=indep[mask^(1<<v)];
    }
    
    vector<int> dp(1<<N, INT_MAX);
    vector<int> par(1<<N, -1);
    dp[0]=0;
    for(int mask=1;mask<=(1<<N)-1;mask++){
        for(int sub=mask;sub>0;sub=(sub-1)&mask){
            if(indep[sub] && dp[mask^sub]!=INT_MAX && dp[mask^sub]+1<dp[mask]){
                dp[mask]=dp[mask^sub]+1;
                par[mask]=sub;
            }
        }
    }
    
    cout << dp[full] << "\n";
    int mask=full;
    while(mask){
        int s=par[mask];
        for(int i=0;i<N;i++) if(s>>i&1) cout<<(char)('a'+i);
        cout<<"\n";
        mask^=s;
    }
}
```
