Looking at this problem, I need to validate email addresses according to the given grammar.

Let me parse the grammar:
- `<letter>` = a-z or A-Z
- `<symbol>` = letter | digit | `_` | `-`
- `<word>` = one or more symbols
- `<prefix>` = word | prefix.word (so: one or more words separated by dots)
- `<domain>` = exactly 2 or 3 letters
- `<suffix>` = prefix.domain (prefix followed by dot followed by domain)
- `<address>` = prefix@suffix

So an address is: `prefix@prefix.domain`

Where:
- prefix = one or more "words" (sequences of symbols) separated by dots
- domain = exactly 2 or 3 letters

Let me think about this more carefully:

`<suffix>` = `<prefix>.<domain>`

And `<address>` = `<prefix>@<suffix>` = `<prefix>@<prefix>.<domain>`

So the structure is:
1. Split on `@` - must be exactly one `@`
2. Left part = prefix (one or more words separated by dots, each word is one or more symbols)
3. Right part = suffix = prefix.domain
   - Right part must contain at least one `.`
   - The last `.`-separated component is the domain (2 or 3 letters)
   - Everything before the last dot is a prefix

Let me code this step by step:

1. Find exactly one `@`
2. Validate left side as prefix
3. Validate right side as suffix (which is prefix.domain)

For validating prefix:
- Split by `.`
- Each part must be a valid word (non-empty, all symbols)

For validating suffix:
- Must contain at least one `.`
- Last component is domain (2 or 3 letters)
- All components before the last dot form a prefix

A symbol is: letter, digit, `_`, `-`

Let me code this:

```cpp
#include <bits/stdc++.h>
using namespace std;

bool isLetter(char c) { return isalpha(c); }
bool isSymbol(char c) { return isalpha(c) || isdigit(c) || c == '_' || c == '-'; }

bool isWord(const string& s) {
    if (s.empty()) return false;
    for (char c : s) if (!isSymbol(c)) return false;
    return true;
}

bool isPrefix(const string& s) {
    // one or more words separated by dots
    stringstream ss(s);
    string token;
    bool first = true;
    while (getline(ss, token, '.')) {
        if (!isWord(token)) return false;
        first = false;
    }
    // check if s ends with dot (empty last token)
    if (s.empty() || s.back() == '.') return false;
    // also check if starts with dot
    if (s.front() == '.') return false;
    return !first; // at least one token
}

bool isDomain(const string& s) {
    if (s.size() != 2 && s.size() != 3) return false;
    for (char c : s) if (!isLetter(c)) return false;
    return true;
}

bool isSuffix(const string& s) {
    // prefix.domain
    // find last dot
    size_t pos = s.rfind('.');
    if (pos == string::npos) return false;
    string pre = s.substr(0, pos);
    string dom = s.substr(pos + 1);
    return isPrefix(pre) && isDomain(dom);
}

int main() {
    int n;
    cin >> n;
    cin.ignore();
    while (n--) {
        string line;
        getline(cin, line);
        // find @
        size_t at = line.find('@');
        if (at == string::npos || line.find('@', at+1) != string::npos) {
            cout << "NO\n";
            continue;
        }
        string pre = line.substr(0, at);
        string suf = line.substr(at + 1);
        if (isPrefix(pre) && isSuffix(suf)) cout << "YES\n";
        else cout << "NO\n";
    }
}
```

Let me trace through examples:
- `abc@abc`: pre=`abc` (valid prefix), suf=`abc` (no dot → invalid suffix) → NO ✓
- `abc@abc.abc`: pre=`abc`, suf=`abc.abc` → prefix=`abc`, domain=`abc` (3 letters) → YES ✓  
- `_@-.ru`: pre=`_` (valid), suf=`-.ru` → prefix=`-` (valid word), domain=`ru` (2 letters) → YES ✓
