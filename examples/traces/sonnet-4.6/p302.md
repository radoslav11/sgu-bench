Looking at this problem, I need to:

1. Parse a string containing text and tags `<UP>`, `</UP>`, `<DOWN>`, `</DOWN>`
2. Track the current "active" tag using a stack
3. For each letter outside tags, apply the transformation based on the innermost active tag

Let me trace through the example:
- `Thi` - no tag, output as-is: `Thi`
- `<UP>` - push UP
- `sIs` - inside UP, capitalize: `SIS`
- `<DOWN>` - push DOWN
- `EaSY` - inside DOWN (innermost), lowercase: `easy`
- `</DOWN>` - pop DOWN
- `Pr` - inside UP, capitalize: `PR`
- `<UP>` - push UP
- `O` - inside UP, capitalize: `O`
- `</UP>` - pop UP
- `ble` - inside UP, capitalize: `BLE`
- `</UP>` - pop UP
- `m` - no tag, as-is: `m`

Result: `ThiSISeasyPROBLEm` ✓

Now let me implement this:

1. Parse the string character by character
2. When I see `<`, read until `>` to get the tag
3. If it's `<UP>` push 'U', if `<DOWN>` push 'D', if `</UP>` or `</DOWN>` pop
4. For regular characters, apply transformation based on top of stack

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);
    
    stack<char> st; // 'U' for UP, 'D' for DOWN
    
    int i = 0;
    int n = s.size();
    string result = "";
    
    while (i < n) {
        if (s[i] == '<') {
            // Read tag
            int j = i + 1;
            while (j < n && s[j] != '>') j++;
            string tag = s.substr(i+1, j-i-1); // content between < and >
            i = j + 1;
            
            if (tag == "UP") {
                st.push('U');
            } else if (tag == "DOWN") {
                st.push('D');
            } else if (tag == "/UP" || tag == "/DOWN") {
                if (!st.empty()) st.pop();
            }
        } else {
            char c = s[i];
            if (isalpha(c)) {
                if (!st.empty()) {
                    if (st.top() == 'U') {
                        c = toupper(c);
                    } else {
                        c = tolower(c);
                    }
                }
            }
            result += c;
            i++;
        }
    }
    
    cout << result << endl;
    
    return 0;
}
```
