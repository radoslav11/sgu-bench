Let me think about this. The Cantor function satisfies a self-similar structure. The Cantor set is built by removing middle thirds. The Cantor function:
- On [1/3, 2/3], f(x) = 1/2
- On [0, 1/3], it's a scaled copy: f(x) = f(3x)/2
- On [2/3, 1], it's a scaled copy: f(x) = 1/2 + f(3x-2)/2

So I_n = ∫₀¹ (f(x))^n dx

Split into three intervals:
- [0, 1/3]: substitute u = 3x, f(x) = f(u)/2, dx = du/3
  Contribution: (1/3) ∫₀¹ (f(u)/2)^n du = (1/3)(1/2^n) I_n
- [1/3, 2/3]: f(x) = 1/2, length = 1/3
  Contribution: (1/3)(1/2)^n
- [2/3, 1]: substitute u = 3x-2, f(x) = 1/2 + f(u)/2, dx = du/3
  Contribution: (1/3) ∫₀¹ (1/2 + f(u)/2)^n du

For the third part, expand (1/2 + f(u)/2)^n = Σ_{k=0}^{n} C(n,k) (1/2)^(n-k) (f(u)/2)^k
Integral = (1/3) Σ_{k=0}^{n} C(n,k) (1/2)^(n-k) (1/2)^k I_k = (1/3) Σ_{k=0}^{n} C(n,k) (1/2)^n I_k

So:
I_n = (1/3)(1/2^n) I_n + (1/3)(1/2)^n + (1/3)(1/2)^n Σ_{k=0}^{n} C(n,k) I_k

I_n = (1/(3·2^n)) I_n + (1/(3·2^n)) + (1/(3·2^n)) Σ_{k=0}^{n} C(n,k) I_k
