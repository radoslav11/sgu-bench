Looking at this problem, I need to simulate the "What? Where? When?" game.

Key mechanics:
- 13 sectors, initially all have envelopes
- Each round: spin arrow (uniform over 1-13), then rotate counter-clockwise until hitting a sector with an envelope
- Sector 13's envelope = random internet question
- Game ends when a team reaches 6 points

Let me think about the state space:
- Which envelopes are still on the table (subset of sectors 1-13)
- Current score (club_points, viewer_points)

The state is: set of remaining envelopes × score. But with 13 sectors, 2^13 = 8192 possible envelope subsets. Score states where neither team has won: club_pts in 0-5, viewer_pts in 0-5 = 36 states. Total: 8192 × 36 manageable.

Wait, but when the game ends (6 points), we need to track final scores. Also, when all envelopes are removed and game isn't over... but with at most 12 questions (sectors 1-12) plus sector 13 (internet), the game can last at most 12 rounds before all 12 regular envelopes are used, plus sector 13 can be used once.

Actually sector 13's envelope is also removed when chosen. So we have 13 envelopes total and game ends when one team gets 6 points (which happens within at most 11 rounds).

The arrow rotation: uniform over 1-13, then counter-clockwise until hitting an envelope. Counter-clockwise means decreasing sector numbers (sectors numbered counter-clockwise), so from sector k, we go k, k-1, k-2, ..., 1, 13, 12, ...

Let me simulate with DP. State = (bitmask of remaining envelopes, club_score, viewer_score). 

For each state, compute the probability of reaching each final score.

Actually I'll track probability distributions. Let dp[mask][c][v] = probability of being in this state. Start: dp[(1<<13)-1][0][0] = 1.

For each state, when arrow points to sector s (1-indexed, probability 1/13 each), find the first sector with an envelope going counter-clockwise from s.

Counter-clockwise: sectors are numbered 1..13 counter-clockwise, so going counter-clockwise from s means s, s-1, s-2, ..., 1, 13, 12, ...

For sector 13 (index 12 in 0-based), the internet question is chosen uniformly among N internet questions, giving probability = average of all internet question probabilities for club answering correctly.

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int N;
    cin >> N;
    double p[13]; // p[0..11] for sectors 1..12, p[12] for internet avg
    for(int i=0;i<12;i++) cin >> p[i];
    double sum=0;
    for(int i=0;i<N;i++){double x; cin>>x; sum+=x;}
    p[12] = (N>0) ? sum/N : 0.5;
    
    // dp[mask][c][v] = probability
    // mask: bit i set means sector (i+1) still has envelope
    map<tuple<int,int,int>,double> dp;
    dp[{(1<<13)-1, 0, 0}] = 1.0;
    
    double ans[12]={};
    
    // BFS/iterative
    // Process states in order
    vector<tuple<int,int,int>> states;
    states.push_back({(1<<13)-1,0,0});
    map<tuple<int,int,int>,bool> visited;
    
    queue<tuple<int,int,int>> q;
    q.push({(1<<13)-1,0,0});
    visited[{(1<<13)-1,0,0}]=true;
    
    while(!q.empty()){
        auto [mask,c,v] = q.front(); q.pop();
        double prob = dp[{mask,c,v}];
        if(prob < 1e-15) continue;
        
        for(int s=1;s<=13;s++){
            // find envelope counter-clockwise from s
            int cur=s;
            while(!(mask & (1<<(cur-1)))){
                cur--; if(cur==0) cur=13;
            }
            int sector=cur; // 1-indexed
            int newmask = mask ^ (1<<(sector-1));
            double pc = p[sector-1];
            // club answers: c+1
            // viewer answers: v+1
            for(int win=0;win<2;win++){
                double wp = win ? pc : (1-pc);
                int nc = c + (win?1:0);
                int nv = v + (win?0:1);
                double np = prob/13.0 * wp;
                if(nc==6||nv==6){
                    // game over
                    int idx;
                    if(nc==6) idx=nv; else idx=6+(6-nc);
                    ans[idx]+=np;
                } else {
                    auto key=make_tuple(newmask,nc,nv);
                    dp[key]+=np;
                    if(!visited[key]){visited[key]=true;q.push(key);}
                }
            }
        }
    }
    
    cout << fixed << setprecision(3);
    for(int i=0;i<12;i++) cout << ans[i] << "\n";
}
```
