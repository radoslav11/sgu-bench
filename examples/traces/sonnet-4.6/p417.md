Looking at this problem, I need to compute the integral of ln(x²+y²) over a disc centered at (x0,y0) with radius r.

The mass is:
$$M = \iint_{(x-x_0)^2+(y-y_0)^2 \leq r^2} \ln(x^2+y^2) \, dx \, dy$$

Let me use polar-like coordinates centered at (x0,y0): let x = x0 + ρcos(θ), y = y0 + ρsin(θ).

Then:
$$M = \int_0^r \int_0^{2\pi} \ln((x_0+\rho\cos\theta)^2+(y_0+\rho\sin\theta)^2) \, \rho \, d\theta \, d\rho$$

For the inner integral over θ:
$$I(\rho) = \int_0^{2\pi} \ln((x_0+\rho\cos\theta)^2+(y_0+\rho\sin\theta)^2) \, d\theta$$

Let R² = x0²+y0² (distance from origin to center squared), and let d = √(R²).

We have: $(x_0+\rho\cos\theta)^2+(y_0+\rho\sin\theta)^2 = R^2 + \rho^2 + 2\rho(x_0\cos\theta+y_0\sin\theta)$

This is of the form $a + b\cos(\theta - \phi)$ where $a = R^2+\rho^2$ and $b = 2\rho\sqrt{x_0^2+y_0^2} = 2\rho d$.

Using the known formula: $\int_0^{2\pi} \ln(a + b\cos\theta) d\theta = 2\pi \ln\frac{a + \sqrt{a^2-b^2}}{2}$ when $a > |b|$.

Here $a = R^2+\rho^2$, $b = 2\rho d$, and $a^2 - b^2 = (R^2+\rho^2)^2 - 4\rho^2 d^2 = (R^2+\rho^2)^2 - 4\rho^2 R^2 = (R^2-\rho^2)^2$.

So $\sqrt{a^2-b^2} = |R^2-\rho^2|$.

Since origin is strictly outside the disc: $d > r$, so for $\rho \leq r$, we have $R = d > r \geq \rho$, meaning $R^2 > \rho^2$.

Thus: $I(\rho) = 2\pi \ln\frac{R^2+\rho^2 + R^2-\rho^2}{2} = 2\pi \ln(R^2) = 4\pi \ln(R)$

Wait, that's constant! So:

$$M = \int_0^r 4\pi \ln(R) \cdot \rho \, d\rho = 4\pi \ln(R) \cdot \frac{r^2}{2} = 2\pi r^2 \ln(R)$$

where $R = \sqrt{x_0^2+y_0^2}$.

Let me verify with the sample: x0=3, y0=4, r=2.
R = 5, ln(5) ≈ 1.6094379...
M = 2π * 4 * ln(5) = 8π * 1.6094379... = 8 * 3.14159265... * 1.6094379...
= 25.13274... * 1.6094379... ≈ 40.4496...

That matches the sample output of 40.449586576894895!

So the formula is simply: $M = 2\pi r^2 \ln(\sqrt{x_0^2+y_0^2})= \pi r^2 \ln(x_0^2+y_0^2)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double x0, y0, r;
    cin >> x0 >> y0 >> r;
    
    double R2 = x0*x0 + y0*y0;
    const double PI = acos(-1.0);
    double mass = PI * r * r * log(R2);
    
    printf("%.15f\n", mass);
    return 0;
}
```
