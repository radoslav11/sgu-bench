Looking at this problem, I need to find a circle orthogonal to all given circles.

**Key insight:** A circle with center (a, b) and radius r is orthogonal to circle (xi, yi, ri) if and only if:
`(a - xi)² + (b - yi)² = r² + ri²`

This is the orthogonality condition (sum of squares of radii equals square of distance between centers).

Expanding: `a² - 2a·xi + xi² + b² - 2b·yi + yi² = r² + ri²`

Let me define `w = a² + b² - r²` (the "power" related term).

Then: `w - 2a·xi - 2b·yi + xi² + yi² = ri²`
=> `-2a·xi - 2b·yi + w = ri² - xi² - yi²`

This is a linear equation in (a, b, w)!

So for each circle i, we get:
`-2xi·a - 2yi·b + w = ri² - xi² - yi²`

This is a system of linear equations. We have 3 unknowns: a, b, w.

Once we find a, b, w, we compute r² = a² + b² - w. If r² > 0, we have a valid circle.

**Cases:**
- n = 1: infinitely many solutions → output -2
- n = 2: may be underdetermined (1D family) → -2 or unique
- n ≥ 3: solve the 3×3 system (using first 3 equations), then verify all remaining

For n < 3, we might have -2 unless the equations constrain uniquely.

Let me think about the general approach: collect all unique equations (up to 3), solve the linear system, check consistency with all others.

If the system is underdetermined → -2
If inconsistent → -1
If unique solution → check r² > 0, output or -1

I'll use Gaussian elimination with 3 unknowns.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n;
    scanf("%d",&n);
    vector<long long> X(n),Y(n),R(n);
    for(int i=0;i<n;i++) scanf("%lld%lld%lld",&X[i],&Y[i],&R[i]);
    
    // Equation: -2xi*a - 2yi*b + w = ri^2 - xi^2 - yi^2
    // Augmented matrix rows: [coeff_a, coeff_b, coeff_w, rhs]
    // Use double
    vector<array<double,4>> eqs;
    for(int i=0;i<n;i++){
        double a=-2.0*X[i], b=-2.0*Y[i], c=1.0;
        double rhs=(double)R[i]*R[i]-(double)X[i]*X[i]-(double)Y[i]*Y[i];
        eqs.push_back({a,b,c,rhs});
    }
    
    // Gaussian elimination on 3 unknowns
    // collect up to 3 independent equations
    vector<array<double,4>> mat;
    for(auto& eq:eqs){
        if(mat.size()==3) break;
        mat.push_back(eq);
    }
    while((int)mat.size()<3) mat.push_back({0,0,0,0});
    
    // Gauss
    int rank=0;
    int pivot_col[3]={-1,-1,-1};
    for(int col=0;col<3&&rank<3;col++){
        int best=rank;
        for(int r=rank+1;r<3;r++) if(fabs(mat[r][col])>fabs(mat[best][col])) best=r;
        if(fabs(mat[best][col])<1e-9) continue;
        swap(mat[rank],mat[best]);
        pivot_col[rank]=col;
        double div=mat[rank][col];
        for(int j=0;j<4;j++) mat[rank][j]/=div;
        for(int r=0;r<3;r++) if(r!=rank){
            double f=mat[r][col];
            for(int j=0;j<4;j++) mat[r][j]-=f*mat[rank][j];
        }
        rank++;
    }
    
    if(rank<3){ printf("-2\n"); return 0; }
    
    double sol[3];
    sol[0]=mat[0][3]; sol[1]=mat[1][3]; sol[2]=mat[2][3];
    double a=sol[0],b=sol[1],w=sol[2];
    double r2=a*a+b*b-w;
    if(r2<=1e-9){ printf("-1\n"); return 0; }
    
    // Verify all equations
    for(int i=0;i<n;i++){
        double lhs=-2.0*X[i]*a-2.0*Y[i]*b+w;
        double rhs=(double)R[i]*R[i]-(double)X[i]*X[i]-(double)Y[i]*Y[i];
        if(fabs(lhs-rhs)>1e-3*(1+fabs(rhs))){ printf("-1\n"); return 0; }
    }
    printf("%.9f %.9f %.9f\n",a,b,sqrt(r2));
}
```
