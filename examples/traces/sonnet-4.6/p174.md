Looking at this problem, I need to find which wall, when added, first creates a cycle in a planar graph.

The walls are line segments. Two walls can only cross at endpoints. This means I'm building a planar graph where:
- Vertices are the endpoints of walls
- Edges are the walls

I need to find the first wall (by order of input) that completes a cycle.

This is essentially: process walls in order, and find the first wall that connects two vertices that are already connected (i.e., adding this edge creates a cycle).

This is a classic Union-Find problem! When I add wall i connecting point A to point B:
- If A and B are already in the same connected component → this wall creates a cycle → answer is i
- Otherwise, merge their components

The key challenge is coordinate compression since coordinates can be up to 10^9.

I need to:
1. Map each unique endpoint to an integer ID
2. Use Union-Find to detect when two endpoints are already connected

Let me implement this:

1. Read all walls, storing their two endpoints
2. Collect all unique points and assign IDs
3. Process walls in order 1..M, using Union-Find
4. First wall where both endpoints have same root → output its 1-based index
5. If none found, output 0

Wait, but walls can only cross at endpoints, and the problem says "walls surround some part". A cycle in the graph of segments means a region is enclosed. The first wall to complete a cycle is the answer.

Since the graph is built from line segments that only intersect at endpoints, the graph structure is exactly what I described. A cycle in this graph corresponds to a surrounded area.

So the answer is simply: which wall first forms a cycle when added to the incrementally-built graph?

This is Union-Find on endpoints.

```cpp
#include <bits/stdc++.h>
using namespace std;

struct DSU {
    vector<int> parent, rank_;
    DSU(int n) : parent(n), rank_(n, 0) {
        iota(parent.begin(), parent.end(), 0);
    }
    int find(int x) {
        while (parent[x] != x) {
            parent[x] = parent[parent[x]];
            x = parent[x];
        }
        return x;
    }
    bool unite(int a, int b) {
        a = find(a); b = find(b);
        if (a == b) return false;
        if (rank_[a] < rank_[b]) swap(a, b);
        parent[b] = a;
        if (rank_[a] == rank_[b]) rank_[a]++;
        return true;
    }
};

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int M;
    cin >> M;
    
    vector<pair<long long,long long>> pts1(M), pts2(M);
    vector<pair<long long,long long>> all_pts;
    
    for(int i = 0; i < M; i++){
        long long x1,y1,x2,y2;
        cin >> x1 >> y1 >> x2 >> y2;
        pts1[i] = {x1,y1};
        pts2[i] = {x2,y2};
        all_pts.push_back({x1,y1});
        all_pts.push_back({x2,y2});
    }
    
    sort(all_pts.begin(), all_pts.end());
    all_pts.erase(unique(all_pts.begin(), all_pts.end()), all_pts.end());
    
    auto getId = [&](pair<long long,long long> p) -> int {
        return (int)(lower_bound(all_pts.begin(), all_pts.end(), p) - all_pts.begin());
    };
    
    DSU dsu(all_pts.size());
    
    for(int i = 0; i < M; i++){
        int a = getId(pts1[i]);
        int b = getId(pts2[i]);
        if(!dsu.unite(a, b)){
            cout << i+1 << endl;
            return 0;
        }
    }
    
    cout << 0 << endl;
    return 0;
}
```
