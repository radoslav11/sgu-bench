Looking at this problem, I need to find a permutation of digits of n that is divisible by 17, with no leading zeros.

n can be up to 10^17, so it has at most 18 digits.

Since there are at most 18 digits, I can try to use dynamic programming with bitmask over the digits.

Let me think about the approach:
- Number of digits ≤ 18
- Bitmask DP: dp[mask][remainder] = can we form a number using exactly the digits in mask with that remainder mod 17?
- We also need to track no leading zeros.

State: dp[mask][rem] = true/false, whether we can arrange the digits indicated by mask to form a number with remainder `rem` mod 17.

We also need to reconstruct the answer.

For leading zeros: we need to ensure the first digit is not zero. We can handle this by only starting with non-zero digits.

DP approach:
- dp[mask][rem] = true if we can arrange the digits in `mask` to give remainder `rem`
- Transition: for each mask and each bit not in mask, extend by adding that digit

Wait, we need to build the number digit by digit from left to right.

Let dp[mask][rem] = true if the digits corresponding to bits set in mask, arranged in some order (without leading zeros), give remainder `rem` mod 17.

Transition: dp[mask | (1<<i)][( rem * 10 + digit[i]) % 17] = true, for each i not in mask.

Base case: dp[0][0] = true (empty number, remainder 0).

But we need to handle leading zeros: when mask == 0, we can only pick non-zero digits as the first digit.

Actually wait - if mask == 0, we're placing the first digit, so it can't be 0.

Let me re-define: dp[mask][rem] means: we've placed the digits in `mask` so far, and the current number formed has remainder `rem` mod 17. The number is valid (no leading zero).

Special case: mask = 0 means we haven't placed anything yet.

Transitions from dp[mask][rem]:
- If mask == 0: pick digit[i] where digit[i] != 0, new state: dp[1<<i][digit[i] % 17]
- If mask != 0: pick any digit[i] not in mask, new state: dp[mask|(1<<i)][(rem*10 + digit[i]) % 17]

Answer: dp[(1<<n)-1][0] == true.

For reconstruction, store the choice made.

Let me code this up.

n has at most 18 digits, so mask has at most 2^18 = 262144 states, times 17 remainders = ~4.5M states. This is fine.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    string s;
    cin >> s;
    int n = s.size();
    vector<int> d(n);
    for(int i=0;i<n;i++) d[i]=s[i]-'0';
    
    int full = 1<<n;
    // dp[mask][rem] = true/false
    // prev[mask][rem] = which digit index was last added
    vector<vector<int>> prev(full, vector<int>(17, -1));
    vector<vector<bool>> dp(full, vector<bool>(17, false));
    
    // Base: place first digit
    for(int i=0;i<n;i++){
        if(d[i] != 0){
            int mask = 1<<i;
            int rem = d[i] % 17;
            if(!dp[mask][rem]){
                dp[mask][rem] = true;
                prev[mask][rem] = i; // came from empty, placed digit i
            }
        }
    }
    
    for(int mask=1; mask<full; mask++){
        for(int rem=0; rem<17; rem++){
            if(!dp[mask][rem]) continue;
            for(int i=0;i<n;i++){
                if(mask & (1<<i)) continue;
                int nmask = mask|(1<<i);
                int nrem = (rem*10 + d[i])%17;
                if(!dp[nmask][nrem]){
                    dp[nmask][nrem] = true;
                    prev[nmask][nrem] = i;
                }
            }
        }
    }
    
    int full1 = full-1;
    if(!dp[full1][0]){
        cout << -1 << endl;
        return 0;
    }
    
    // Reconstruct
    string ans = "";
    int mask = full1, rem = 0;
    while(mask != 0){
        int i = prev[mask][rem];
        ans += (char)('0'+d[i]);
        int pmask = mask ^ (1<<i);
        // find previous rem
        int prem;
        if(pmask == 0) prem = 0;
        else {
            // rem = (prem*10 + d[i])%17 => prem = (rem - d[i]) * inv(10) mod 17
            // inv(10) mod 17: 10*12=120=7*17+1, so inv=12
            prem = ((rem - d[i]%17 + 17)*12)%17;
        }
        mask = pmask;
        rem = prem;
    }
    reverse(ans.begin(), ans.end());
    cout << ans << endl;
}
```
