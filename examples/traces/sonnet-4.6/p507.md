Looking at this problem, I need to find for each internal node the minimum absolute difference between any two leaf values in its subtree.

**Approach:**
- Use Euler tour / DFS order to get leaves in order
- For each internal node, find the range of leaves in its subtree
- Sort the leaves by value within each subtree range, then find min adjacent difference

But sorting per node is expensive. Better approach:

**Key insight:** Use DFS. When we finish processing a subtree, we have a sorted list of leaf values. We can merge sorted lists (small-to-large merging) and track the minimum difference.

**Small-to-large merging:**
- For each node, maintain a sorted multiset of leaf values in its subtree
- When merging children into parent, merge smaller sets into larger
- When inserting an element into a set, check neighbors to update min difference
- Each node stores the minimum difference in its subtree

Let me think about this more carefully:
- Each node v has a `min_diff[v]` = minimum absolute difference among leaves in subtree of v
- When merging child c into parent v's set, for each element x being moved from smaller set to larger set, check neighbors in the larger set to potentially update the answer

The answer for node v = min over all children c of `min_diff[c]`, combined with cross-differences found during merging.

**Implementation:**
- Use DSU on tree or small-to-large merging with sets
- For each internal node, answer = min diff found

Let me implement small-to-large:

For each node, maintain a pointer to a `std::set<int>`. When merging, take the largest child's set, then insert elements from all smaller sets into it, updating a running minimum difference for the node.

The answer for node v is the minimum over:
1. answers of all children
2. any new cross-differences found when merging

Actually, we propagate answers upward. Let `ans[v]` = min diff in subtree of v. When merging children into v's set, `ans[v] = min(ans[children])`, then as we insert each element from smaller sets into the big set, we check neighbors and update `ans[v]`.

For leaves: they contribute their value to parent's set, no answer needed.

Let me code this up:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n, m;
    scanf("%d %d", &n, &m);
    
    vector<vector<int>> children(n+1);
    for(int i = 2; i <= n; i++){
        int p; scanf("%d", &p);
        children[p].push_back(i);
    }
    
    int internal = n - m;
    vector<long long> ans(n+1, (long long)2e18);
    // leaf nodes: n-m+1 .. n
    vector<int> leafval(n+1, 0);
    for(int i = n-m+1; i <= n; i++){
        int v; scanf("%d", &v);
        leafval[i] = v;
    }
    
    // small-to-large merging
    // each node owns a set
    vector<set<int>*> S(n+1, nullptr);
    
    // iterative post-order DFS
    vector<int> order;
    vector<int> stk = {1};
    while(!stk.empty()){
        int v = stk.back(); stk.pop_back();
        order.push_back(v);
        for(int c : children[v]) stk.push_back(c);
    }
    reverse(order.begin(), order.end());
    
    const long long INF = 2147483647LL;
    
    for(int v : order){
        if(v > internal){ // leaf
            S[v] = new set<int>();
            S[v]->insert(leafval[v]);
            ans[v] = INF;
        } else {
            // find largest child set
            int big = -1;
            for(int c : children[v]){
                if(big == -1 || S[c]->size() > S[big]->size()) big = c;
            }
            ans[v] = INF;
            for(int c : children[v]) ans[v] = min(ans[v], ans[c]);
            
            S[v] = S[big];
            for(int c : children[v]){
                if(c == big) continue;
                for(int x : *S[c]){
                    auto it = S[v]->insert(x).first;
                    auto next = std::next(it);
                    if(next != S[v]->end()) ans[v] = min(ans[v], (long long)(*next - x));
                    if(it != S[v]->begin()){
                        auto prev = std::prev(it);
                        ans[v] = min(ans[v], (long long)(x - *prev));
                    }
                }
                delete S[c];
                S[c] = nullptr;
            }
        }
    }
    
    for(int v = 1; v <= internal; v++){
        printf("%lld", ans[v] == (long long)2e18 ? INF : ans[v]);
        if(v < internal) printf(" ");
    }
    printf("\n");
    
    delete S[1];
    return 0;
}
```
