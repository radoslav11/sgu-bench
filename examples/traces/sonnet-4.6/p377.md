Looking at this problem, I need to count the number of ways to place students in an N×M grid such that every 2×2 square contains exactly 2 students.

Let me think about what configurations satisfy this constraint for every 2×2 square having exactly 2 students.

For a 2×2 square, exactly 2 of the 4 cells must be occupied. The possible patterns for a 2×2 square with exactly 2 students:
- The 6 ways to choose 2 from 4: but which ones are globally consistent?

Let me think column by column. For a 2×M grid (N=2), let me enumerate valid configurations.

For N=2, M=2: answer is 6. That matches all C(4,2)=6 patterns where each 2×2 has exactly 2.

For larger grids, the constraint propagates. Let me think about what patterns are globally valid.

Key observation: Consider two consecutive rows i and i+1. For every pair of consecutive columns j and j+1, the 2×2 block must have exactly 2 cells filled.

Let me think about valid column patterns. Each column is a binary vector of length N. 

For two adjacent columns (col j and col j+1), and for each consecutive row pair (i, i+1), the four cells (i,j), (i+1,j), (i,j+1), (i+1,j+1) must have exactly 2 filled.

This means: for each row pair, (a + b + c + d = 2) where a,b are the two cells in col j and c,d in col j+1.

This is a complex constraint. Let me think of valid global patterns.

After research/thinking: the valid configurations are essentially "striped" patterns. For every 2×2 block to have exactly 2, the valid tilings are:
1. Horizontal stripes: alternate rows all-filled and all-empty
2. Vertical stripes: alternate columns all-filled and all-empty  
3. Checkerboard patterns: two variants

Wait, for N=2, M=2, we get 6. Horizontal stripes (2 ways: rows {1,3,5...} or {2,4,6...} for 2 rows = top or bottom row filled), vertical stripes (2 ways), checkerboard (2 ways) = 6. That matches!

For general N×M, the count should be:
- Number of ways rows can be striped: the rows alternate between full and empty. Number of valid row-stripe patterns = 2 (start with full or empty row). But wait, for N rows, we need every consecutive row pair to have each 2×2 = 2, meaning each row pair sums to 2 per column-pair. If all columns in a row are the same (all 1 or all 0), then rows alternate 1,0,1,0... or 0,1,0,... giving 2 patterns.
- Similarly for column stripes: 2 patterns.
- Checkerboard: 2 patterns.

Total = 6 for any N,M ≥ 2? But that seems too simple.

Wait, let me check N=2, M=3. With horizontal stripes: row 0 all filled or row 1 all filled = 2. Vertical stripes: columns 0,2 filled or columns 1 filled... For M=3 vertical stripes: col 0 and 2 filled vs col 1 filled = 2 patterns (but col 1 gives only 1 column filled out of 3). Let me verify: columns alternate 1,0,1 or 0,1,0. For each 2×2: cols j,j+1 have patterns (1,0) or (0,1), both rows filled with either → each 2×2 has 2 cells. Yes! So 2 vertical stripe patterns. Checkerboard: 2 patterns. Total = 6.

So the answer is always 6? Let me think again about N=2,M=2 = 6. What about non-stripe patterns for larger grids?

Actually, I believe for any N,M ≥ 2, the answer is always 6. The valid global configurations are exactly: 2 horizontal striped + 2 vertical striped + 2 checkerboard = 6.

The answer is always **6**.
