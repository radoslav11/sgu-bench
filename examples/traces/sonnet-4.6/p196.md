The matrix A^T * A is an M×M matrix. The element (i,j) of A^T * A is the dot product of column i of A^T (which is row i of A... wait, let me think carefully).

A is N×M. A^T is M×N. A^T * A is M×M.

The (i,j) element of A^T * A = sum over k of A^T[i][k] * A[k][j] = sum over k of A[k][i] * A[k][j].

This equals the number of vertices incident to both edge i and edge j.

- If i == j: both endpoints are incident, so the diagonal element is 2.
- If i != j: the value is 1 if edges i and j share a vertex, 0 otherwise.

Sum of all elements = sum of diagonal + sum of off-diagonal.

Sum of diagonal = 2M (each edge contributes 2).

Sum of off-diagonal = number of ordered pairs (i,j) with i≠j where edges i and j share a vertex.

For each vertex v with degree d(v), the number of ordered pairs of edges sharing v is d(v) * (d(v) - 1).

So total sum = 2M + sum over all vertices v of d(v)*(d(v)-1).

Let me verify with the sample:
- 4 vertices, 4 edges: 1-2, 1-3, 2-3, 2-4
- Degrees: d(1)=2, d(2)=3, d(3)=2, d(4)=1
- Sum = 2*4 + (2*1 + 3*2 + 2*1 + 1*0) = 8 + (2 + 6 + 2 + 0) = 8 + 10 = 18 ✓

#include<bits/stdc++.h>
using namespace std;
int main(){
    int n, m;
    scanf("%d %d", &n, &m);
    vector<long long> deg(n+1, 0);
    for(int i = 0; i < m; i++){
        int u, v;
        scanf("%d %d", &u, &v);
        deg[u]++;
        deg[v]++;
    }
    long long ans = 2LL * m;
    for(int i = 1; i <= n; i++){
        ans += deg[i] * (deg[i] - 1);
    }
    printf("%lld\n", ans);
    return 0;
}
