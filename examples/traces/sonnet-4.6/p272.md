Looking at this problem, I need to find a maximal set of vertex-disjoint paths from set A to set B, all of length equal to the minimum distance from A to B.

**Approach:**
1. Run BFS from all B nodes simultaneously (multi-source BFS) to get distances from each node to B.
2. Find the minimum distance `K` = min distance from any A node to B.
3. Build a layered DAG: nodes at distance `d` connect to neighbors at distance `d-1` (getting closer to B), but only nodes reachable within exactly K steps from some A node.
4. More precisely: a node `v` is "useful" if it's reachable from some A node AND `dist[v] <= K`. The path must have length exactly K.
5. Actually, nodes in A must have `dist[a] == K`. Nodes on a valid path from `a ∈ A` must follow edges where dist decreases by 1 each step.
6. Build a flow network:
   - Source → each A node with `dist[a] == K`, capacity 1
   - Each B node → Sink, capacity 1
   - For each edge (u,v): if `dist[v] == dist[u] - 1`, add directed edge u→v with capacity 1 (in terms of vertex-disjoint, split vertices)
   - Vertex capacities = 1 (split each internal node)
7. Run max flow on this network, then extract paths.
