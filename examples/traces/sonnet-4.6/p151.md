Looking at this problem, I need to find coordinates of triangle ABC where:
- |AB| = c
- |AC| = b  
- |AM| = m, where M is the midpoint of BC

Let me place the triangle conveniently. Let me put M at the origin, and B and C symmetric about the origin on the x-axis.

So:
- B = (-a, 0)
- C = (a, 0)
- M = (0, 0) (midpoint of BC)
- A = (x, y)

From |AM| = m: x² + y² = m²
From |AB| = c: (x+a)² + y² = c²
From |AC| = b: (x-a)² + y² = b²

From AB² - AC²: (x+a)² - (x-a)² = c² - b²
4ax = c² - b²
x = (c² - b²) / (4a)

From |AM| = m: y² = m² - x²

From |AB| = c: (x+a)² + y² = c²
x² + 2ax + a² + y² = c²
m² + 2ax + a² = c²
2ax = c² - m² - a²

Also 2ax = (c² - b²)/2
So: (c² - b²)/2 = c² - m² - a²
a² = c² - m² - (c² - b²)/2
a² = c² - m² - c²/2 + b²/2
a² = c²/2 + b²/2 - m²

This is the median formula: m² = (2b² + 2c² - a_side²)/4 where a_side = 2a (full BC side).

Let BC = 2a, so (2a)² = 2b² + 2c² - 4m²
4a² = 2b² + 2c² - 4m²
a² = (b² + c² - 2m²) / 2... wait let me recheck.

Median formula: m_a² = (2b² + 2c² - a²)/4 where a is BC length.

Here BC = 2a (our variable), so:
m² = (2b² + 2c² - (2a)²)/4 = (2b² + 2c² - 4a²)/4

4m² = 2b² + 2c² - 4a²
4a² = 2b² + 2c² - 4m²
a² = (b² + c² - 2m²) / 2... 

Wait: a² = (2b² + 2c² - 4m²)/4 = (b² + c² - 2m²)/2... hmm let me redo:

a² = (b² + c²)/2 - m²

So a² = (b² + c²)/2 - m²

For solution to exist: a² > 0, i.e., (b² + c²)/2 > m²

Also need y² = m² - x² ≥ 0, i.e., |x| ≤ m.

x = (c² - b²)/(4a)

Also need A, B, C to not be collinear (y ≠ 0).

If a² ≤ 0: no solution.
If y² < 0: no solution (but this might be covered by triangle inequality).

Actually if a = 0, then b = c and x = 0, y = m, which gives a degenerate case where B = C = M.

Let me just code this up:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double c, b, m;
    cin >> c >> b >> m;
    
    // a^2 = (b^2 + c^2)/2 - m^2, where BC/2 = a
    double a2 = (b*b + c*c)/2.0 - m*m;
    
    if (a2 <= 0) {
        cout << "Mission impossible" << endl;
        return 0;
    }
    
    double a = sqrt(a2);
    
    // x = (c^2 - b^2) / (4a)
    double x = (c*c - b*b) / (4.0 * a);
    
    double y2 = m*m - x*x;
    
    if (y2 < 0) {
        cout << "Mission impossible" << endl;
        return 0;
    }
    
    double y = sqrt(y2);
    
    if (y < 1e-9) {
        cout << "Mission impossible" << endl;
        return 0;
    }
    
    // A = (x, y), B = (-a, 0), C = (a, 0)
    cout << fixed << setprecision(5);
    cout << x << " " << y << "\n";
    cout << -a << " " << 0.0 << "\n";
    cout << a << " " << 0.0 << "\n";
    
    return 0;
}
```
