Let me analyze:
- A "spot" is a finite, maximally 4-connected set of white cells (a hole in the figure bounded completely by black cells)
- The figure must fit exactly in H×W (touches all 4 borders)

Key observations:
1. Start with a simple rectangle of black cells - that has 0 spots
2. Each "hole" we carve adds spots. But we need to be careful about 8-connectivity of black cells and finiteness of white regions

Let me think about a constructive approach:
- Fill the H×W rectangle with black cells (`.`)
- Each interior cell we make white could create spots
- A single interior white cell surrounded by black cells = 1 spot
- We need exactly N spots

A simple approach: use a filled rectangle and poke holes.

For a filled rectangle, adding isolated white cells in the interior creates spots. Each isolated white cell = 1 spot. But we need enough interior cells.

The maximum number of spots in an H×W grid with the constraint that black cells remain 8-connected... For a filled rectangle, interior cells form a (H-2)×(W-2) grid. We can place white cells in a checkerboard pattern - every other interior cell can be white (isolated) without breaking 8-connectivity of black cells.

Number of cells in checkerboard of (H-2)×(W-2) interior = ceil((H-2)*(W-2)/2).

If N exceeds this, output "Impossible".

Actually isolated white cells (not 4-adjacent to each other) each form their own spot. The black cells around them remain 8-connected as long as the black border is intact.

Max spots = number of "isolated" positions = ceil((H-2)*(W-2)/2) if H>2 and W>2, else 0.

If N > max_spots or (H<=2 or W<=2) and N>0: Impossible.

Wait, if H=2 or W=2, no interior cells exist, so 0 spots max. Since N≥1, impossible for H≤2 or W≤2... unless H=1 or W=1 too. Actually with H=1 or W=1, still no interior.

Let me implement this:

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int T; cin>>T;
    for(int t=0;t<T;t++){
        int H,W,N; cin>>H>>W>>N;
        if(t>0) cout<<"\n";
        // interior is (H-2)x(W-2), need H>=3, W>=3
        int ih=H-2, iw=W-2;
        // max spots via checkerboard
        int maxN=0;
        if(ih>0&&iw>0) maxN=(ih*iw+1)/2;
        if(N>maxN){cout<<"Impossible\n";continue;}
        // build grid all black
        vector<string> g(H,string(W,'.'));
        // place N white cells in checkerboard of interior
        int cnt=0;
        for(int i=1;i<=ih&&cnt<N;i++){
            for(int j=1;j<=iw&&cnt<N;j++){
                if((i+j)%2==0){
                    g[i][j]='#';
                    cnt++;
                }
            }
        }
        for(int i=0;i<H;i++) cout<<g[i]<<"\n";
    }
}
```
