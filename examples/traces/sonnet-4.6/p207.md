Looking at this problem, I need to distribute M coins among N robbers where robber i should ideally get Xi/Y * M coins.

The ideal amount for robber i is `Xi * M / Y`. Since we need integers, each robber gets either `floor(Xi * M / Y)` or `ceil(Xi * M / Y)`.

The total unfairness is:
Sum of |Xi/Y - Ki/M|

Let me think about minimizing this. 

For robber i, if they get Ki coins, the unfairness is |Xi/Y - Ki/M| = |Xi*M - Ki*Y| / (Y*M).

Since Y*M is a constant denominator, we want to minimize Sum |Xi*M - Ki*Y|.

The ideal (real-valued) Ki is Xi*M/Y. The floor is `floor(Xi*M/Y)`.

Let fi = floor(Xi*M/Y). Then Sum(fi) ≤ M. Let S = Sum(fi). We need to distribute M - S extra coins (each robber gets 0 or 1 extra).

For robber i:
- If Ki = fi: deviation = Xi*M - fi*Y (this is Xi*M mod Y, which is ≥ 0)
- If Ki = fi + 1: deviation = fi*Y + Y - Xi*M = Y - (Xi*M mod Y)

The change in total unfairness when giving robber i an extra coin: we go from deviation `ri` (where ri = Xi*M mod Y) to deviation `Y - ri`. The improvement is `ri - (Y - ri) = 2*ri - Y`.

To minimize total unfairness, we should give extra coins to robbers where `2*ri - Y` is largest (i.e., ri is largest, closest to Y).

If ri = 0, giving an extra coin increases unfairness. If ri = Y/2 (exact), it doesn't matter. We prefer to give to those with larger ri.

So the algorithm:
1. Compute fi = floor(Xi*M/Y) and ri = (Xi*M) % Y for each robber.
2. S = sum of fi. Need to give out R = M - S extra coins.
3. Sort robbers by ri descending, give extra coins to top R robbers.

Let me verify with the example:
N=3, M=10, Y=4, X = [1,1,2]

For robber 0: ideal = 1*10/4 = 2.5, fi=2, ri=10%4=2
For robber 1: ideal = 1*10/4 = 2.5, fi=2, ri=10%4=2
For robber 2: ideal = 2*10/4 = 5.0, fi=5, ri=20%4=0

S = 2+2+5 = 9, R = 10-9 = 1.

Robbers sorted by ri: robbers 0 and 1 have ri=2, robber 2 has ri=0.
Give 1 extra to one of robbers 0 or 1.

Result: robber 0 gets 3, robber 1 gets 2, robber 2 gets 5 (or robber 0 gets 2, robber 1 gets 3, robber 2 gets 5).

Expected output: `2 3 5` — matches!

Now I need to output them in original order.

```cpp
#include <bits/stdc++.h>
using std::cin; using std::cout;

int main(){
    int N, M, Y;
    cin >> N >> M >> Y;
    
    std::vector<int> X(N);
    for(int i = 0; i < N; i++) cin >> X[i];
    
    std::vector<long long> f(N), r(N);
    long long S = 0;
    for(int i = 0; i < N; i++){
        long long val = (long long)X[i] * M;
        f[i] = val / Y;
        r[i] = val % Y;
        S += f[i];
    }
    
    long long R = M - S; // extra coins to distribute
    
    // Sort indices by r descending
    std::vector<int> idx(N);
    std::iota(idx.begin(), idx.end(), 0);
    std::sort(idx.begin(), idx.end(), [&](int a, int b){
        return r[a] > r[b];
    });
    
    std::vector<long long> K(N);
    for(int i = 0; i < N; i++) K[i] = f[i];
    for(int i = 0; i < R; i++) K[idx[i]]++;
    
    for(int i = 0; i < N; i++){
        if(i) cout << ' ';
        cout << K[i];
    }
    cout << '\n';
    
    return 0;
}
```
