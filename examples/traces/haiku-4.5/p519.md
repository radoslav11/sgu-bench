Looking at this problem, I need to calculate the total surface area of a 3D city model represented by a grid of heights.

Let me think about what contributes to the surface area:

1. **Top surfaces**: Each cell with height h contributes h to the top (just the top face of the topmost cube)
2. **Bottom surfaces**: Each cell with height h contributes h to the bottom
3. **Side surfaces**: For each cell, I need to count exposed sides

For the sides, I need to consider:
- For each cell at position (i,j) with height h, it has 4 sides (up, down, left, right in terms of grid directions)
- Each side facing a neighbor cell with height h' will expose |h - h'| area
- If there's no neighbor (edge of grid), the entire height h is exposed

Let me think more carefully:
- For each cell (i,j) with height h[i][j]:
  - Top surface: 1 unit (the 1×1 top of the building)
  - Bottom surface: 1 unit
  - For each of 4 directions, if neighbor has height h_neighbor:
    - Exposed height = max(0, h[i][j] - h_neighbor)

Actually, let me reconsider. Each cube in a building at (i,j) with height h occupies a 1×1×1 volume. 

For side surfaces:
- Look at each pair of adjacent cells
- If cell 1 has height h1 and cell 2 has height h2, the difference |h1 - h2| creates exposed wall area

For each cell (i,j) with height h[i][j]:
- Top: 1 (if h > 0)
- Bottom: 1 (if h > 0)
- North side: max(0, h[i][j] - h[i-1][j])
- South side: max(0, h[i][j] - h[i+1][j])
- West side: max(0, h[i][j] - h[i][j-1])
- East side: max(0, h[i][j] - h[i][j+1])

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
  int n, m;
  cin >> n >> m;
  
  vector<string> grid(n);
  for(int i = 0; i < n; i++){
    cin >> grid[i];
  }
  
  long long surface = 0;
  
  for(int i = 0; i < n; i++){
    for(int j = 0; j < m; j++){
      int h = grid[i][j] - '0';
      
      if(h == 0) continue;
      
      // Top and bottom
      surface += 2;
      
      // North
      int h_north = (i > 0) ? (grid[i-1][j] - '0') : 0;
      surface += max(0, h - h_north);
      
      // South
      int h_south = (i < n-1) ? (grid[i+1][j] - '0') : 0;
      surface += max(0, h - h_south);
      
      // West
      int h_west = (j > 0) ? (grid[i][j-1] - '0') : 0;
      surface += max(0, h - h_west);
      
      // East
      int h_east = (j < m-1) ? (grid[i][j+1] - '0') : 0;
      surface += max(0, h - h_east);
    }
  }
  
  cout << surface << endl;
  
  return 0;
}
```
