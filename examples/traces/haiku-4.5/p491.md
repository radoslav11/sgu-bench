Looking at this problem, I need to find all pairs (A, B) where:
1. 1 ≤ A < B ≤ N
2. There exist integers x, y ≥ 1 such that A·x + B·y = N

Let me think about the approach:

For each pair (A, B) with A < B, I need to check if there's a solution to A·x + B·y = N with x, y ≥ 1.

I can rearrange this as: A·x = N - B·y

For a given B, I need to find if there exists a y ≥ 1 such that:
- N - B·y > 0 (so that A·x can be positive)
- (N - B·y) is divisible by some A where 1 ≤ A < B

Alternatively, for each pair (A, B), I can iterate through possible values of x and y:
- For x from 1 to N/A, check if (N - A·x) is divisible by B and (N - A·x)/B ≥ 1

Let me implement this approach:

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    int N;
    cin >> N;
    
    int count = 0;
    
    // Iterate through all pairs (A, B) with A < B
    for(int A = 1; A < N; A++){
        for(int B = A + 1; B <= N; B++){
            // Check if there exist x, y >= 1 such that A*x + B*y = N
            bool found = false;
            
            // Try all possible values of x
            for(int x = 1; x * A < N; x++){
                int remainder = N - A * x;
                // Check if remainder is divisible by B and y >= 1
                if(remainder > 0 && remainder % B == 0){
                    int y = remainder / B;
                    if(y >= 1){
                        found = true;
                        break;
                    }
                }
            }
            
            if(found){
                count++;
            }
        }
    }
    
    cout << count << endl;
    
    return 0;
}
```
