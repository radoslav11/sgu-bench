Looking at this problem, I need to:

1. Determine if a number is a "nearly prime" - which means it can be expressed as the product of exactly two prime numbers (P1 * P2)
2. These two primes don't have to be distinct (so 4 = 2*2 would count)

My approach:
- For each number, find its smallest prime factor
- Divide by that prime factor
- Check if the result is also prime
- If both conditions are met, it's a nearly prime

Let me trace through the example:
- 6 = 2 * 3, where both 2 and 3 are primes → "Yes"

Other examples to consider:
- 4 = 2 * 2 → "Yes" (both are primes)
- 8 = 2 * 4 = 2 * 2 * 2 → "No" (would need to be exactly 2 primes)
- 9 = 3 * 3 → "Yes"
- 15 = 3 * 5 → "Yes"
- 12 = 2 * 6 = 2 * 2 * 3 → "No"

Algorithm:
1. Find the smallest prime factor of the number
2. Divide the number by this factor
3. Check if the quotient is prime
4. If yes, it's a nearly prime

```cpp
#include <iostream>
using namespace std;

bool isPrime(long long n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (long long i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

long long getSmallestPrimeFactor(long long n) {
    if (n % 2 == 0) return 2;
    for (long long i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return i;
    }
    return n;
}

bool isNearlyPrime(long long n) {
    if (n < 4) return false;
    
    long long p1 = getSmallestPrimeFactor(n);
    long long p2 = n / p1;
    
    return isPrime(p2);
}

int main() {
    int N;
    cin >> N;
    
    for (int i = 0; i < N; i++) {
        long long num;
        cin >> num;
        
        if (isNearlyPrime(num)) {
            cout << "Yes\n";
        } else {
            cout << "No\n";
        }
    }
    
    return 0;
}
```

The solution works as follows:

1. **isPrime()**: Checks if a number is prime by testing divisibility up to sqrt(n)
2. **getSmallestPrimeFactor()**: Finds the smallest prime factor of a number
3. **isNearlyPrime()**: 
   - Gets the smallest prime factor (p1)
   - Calculates p2 = n / p1
   - Returns true if p2 is also prime
   
For example with 6:
- Smallest prime factor = 2
- 6 / 2 = 3
- 3 is prime → "Yes"

For example with 12:
- Smallest prime factor = 2
- 12 / 2 = 6
- 6 is not prime → "No"
