Looking at this problem, I need to understand synchrographs and determine which vertices are "alive".

Key observations:
1. A vertex is **active** if all incoming arcs have positive marks
2. A vertex is **potentially alive** if there exists some sequence of fires that leads to it firing
3. A vertex is **alive** if it can fire after ANY valid sequence of fires (i.e., it's always reachable no matter what sequence of valid moves we make)

For a vertex to be "alive", it must be reachable from any reachable state. This means:
- We need to explore all possible game states reachable from the initial state
- A vertex is alive if it appears as an active vertex in ALL reachable states (or at least, in all states that can be reached after any arbitrary sequence of valid moves from the initial state)

Actually, rethinking: a vertex is alive if from ANY valid sequence of fires, that vertex will eventually be able to fire. This is equivalent to saying: in the worst case (adversary chooses fires), the vertex can still eventually fire.

This is a game theory problem. A vertex is alive if:
- From every reachable state, either the vertex is active, or there exists a move that leads to a state where the vertex is still alive

Let me use BFS/DFS to explore all reachable states, then for each state, check which vertices can potentially fire. A vertex is alive if from every reachable state, we can reach a state where that vertex fires.

A simpler approach: Use BFS to find all reachable states. For each vertex v, it's alive if in every reachable state, either:
1. v is currently active, OR
2. There exists at least one active vertex that can be fired, and from all resulting states, v is still alive

Actually, the key insight: a vertex is alive iff it's active in at least one state reachable from every reachable state (i.e., no matter what moves the adversary makes, we can always eventually reach a state where v is active).

Let me implement a solution that:
1. Explores all reachable states using BFS
2. For each vertex, checks if it can be forced to be active from any reachable state

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, M;
    cin >> N >> M;
    
    vector<tuple<int,int,long long>> edges;
    vector<vector<pair<int,int>>> incoming(N+1), outgoing(N+1);
    
    for(int i = 0; i < M; i++){
        int u, v;
        long long w;
        cin >> u >> v >> w;
        edges.push_back({u, v, w});
        incoming[v].push_back({u, i});
        outgoing[u].push_back({v, i});
    }
    
    // BFS to find all reachable states
    map<vector<long long>, int> visited;
    queue<vector<long long>> q;
    set<int> reachable_active;
    
    vector<long long> initial(M);
    for(int i = 0; i < M; i++){
        initial[i] = get<2>(edges[i]);
    }
    
    q.push(initial);
    visited[initial] = 1;
    
    int states_explored = 0;
    while(!q.empty() && states_explored < 100000){
        states_explored++;
        vector<long long> state = q.front();
        q.pop();
        
        // Find active vertices in this state
        vector<int> active;
        for(int v = 1; v <= N; v++){
            bool is_active = true;
            for(auto [u, edge_id] : incoming[v]){
                if(state[edge_id] == 0){
                    is_active = false;
                    break;
                }
            }
            if(is_active && !incoming[v].empty()){
                active.push_back(v);
                reachable_active.insert(v);
            }
        }
        
        // Try firing each active vertex
        for(int v : active){
            vector<long long> new_state = state;
            for(auto [u, edge_id] : incoming[v]){
                new_state[edge_id]--;
            }
            for(auto [w, edge_id] : outgoing[v]){
                new_state[edge_id]++;
            }
            
            if(visited.find(new_state) == visited.end()){
                visited[new_state] = 1;
                q.push(new_state);
            }
        }
    }
    
    for(int v = 1; v <= N; v++){
        if(reachable_active.count(v)){
            cout << "1\n";
        } else {
            cout << "0\n";
        }
    }
    
    return 0;
}
```
