Looking at this problem, I need to simulate a bottle falling through a vending machine with obstacles.

Let me break down the rules:
1. Bottle starts at 'P' and falls down
2. If cell below is empty, fall down
3. If falls out of bottom, output the column (1-indexed)
4. If hits '/', move left and try to fall
5. If hits '\', move right and try to fall
6. Stop if hitting a wall (edge of grid) while moving horizontally
7. Stop if moving from one obstacle type to another different type
8. Continue if moving to same obstacle type

Let me trace through the first example:
```
./P
../
```
- Start at (0,2), column 3
- Fall to (1,2) which is '/'
- Hit '/', move left to (1,1) which is '.'
- Now at (1,1), try to fall - but row 1 is last row, so falls out
- Output column 2 (1-indexed)

For the second example:
```
.P
\/
```
- Start at (0,1), column 2
- Fall to (1,1) which is '\'
- Hit '\', move right to (1,2) which is '/'
- Different obstacle types, so stop
- Output -1
