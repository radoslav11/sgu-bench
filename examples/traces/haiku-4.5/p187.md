Looking at this problem, I need to:

1. Start with a sequence of numbers 1 to N in order
2. Apply M operations, where each operation reverses a subsequence from position Pi to Qi (inclusive, 1-indexed)
3. Output the final sequence

Let me trace through the examples:

**Test #1:**
- Initial: [1, 2, 3, 4, 5]
- Operation 1: reverse positions 1-3 → [3, 2, 1, 4, 5]
- Operation 2: reverse positions 4-5 → [3, 2, 1, 5, 4]
- Output: 3 2 1 5 4 ✓

**Test #2:**
- Initial: [1, 2, 3, 4, 5]
- Operation 1: reverse positions 1-4 → [4, 3, 2, 1, 5]
- Operation 2: reverse positions 2-5 → [4, 5, 1, 2, 3]
- Output: 4 5 1 2 3 ✓

The solution is straightforward:
1. Initialize an array with numbers 1 to N
2. For each operation, reverse the subsequence from Pi to Qi (converting to 0-indexed: from Pi-1 to Qi-1)
3. Output the final array

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
  ios_base::sync_with_stdio(false);
  cin.tie(nullptr);
  
  int N, M;
  cin >> N >> M;
  
  vector<int> seq(N);
  for(int i = 0; i < N; i++){
    seq[i] = i + 1;
  }
  
  for(int i = 0; i < M; i++){
    int P, Q;
    cin >> P >> Q;
    // Convert to 0-indexed and reverse from P-1 to Q-1 (inclusive)
    reverse(seq.begin() + P - 1, seq.begin() + Q);
  }
  
  for(int i = 0; i < N; i++){
    if(i > 0) cout << " ";
    cout << seq[i];
  }
  cout << "\n";
  
  return 0;
}
```

The key points:
- `reverse(seq.begin() + P - 1, seq.begin() + Q)` reverses the range [P-1, Q) which corresponds to positions P to Q in 1-indexed notation (since the end iterator in C++ is exclusive)
- The initial sequence is [1, 2, 3, ..., N]
- We apply each reversal operation in order
- Finally output the resulting sequence with spaces between numbers
