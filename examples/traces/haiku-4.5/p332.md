Looking at this problem, I need to find the largest circle that can fit inside a convex polygon. This circle is the inscribed circle (incircle) of the polygon, and its radius is the maximum distance from any interior point to the nearest edge.

For a convex polygon, the largest inscribed circle will be tangent to at least one edge (or possibly multiple edges). The radius of the largest inscribed circle equals the minimum distance from some interior point to the boundary of the polygon.

The key insight is that the center of the largest inscribed circle will be at a point where the distance to the nearest edge is maximized. For a convex polygon, this is often achieved by finding the point that maximizes the minimum distance to all edges.

A practical approach:
1. For each edge of the polygon, calculate the distance from that edge
2. The radius of the inscribed circle is the minimum distance from an interior point to any edge
3. We can use ternary search or binary search on the radius, or we can use a geometric approach

Actually, the simplest approach is:
- The inradius of a convex polygon can be found using: `radius = Area / semi-perimeter`

But we need to be careful - that formula works for the incircle. Let me verify with the example:
- Square with vertices (0,0), (1,0), (1,1), (0,1)
- Area = 1
- Perimeter = 4, so semi-perimeter = 2
- Radius = 1/2 = 0.5 ✓

This matches! So the formula is: `radius = 2 * Area / Perimeter`

Let me implement this:

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<pair<double, double>> vertices(n);
    for(int i = 0; i < n; i++){
        cin >> vertices[i].first >> vertices[i].second;
    }
    
    // Calculate area using shoelace formula
    double area = 0;
    for(int i = 0; i < n; i++){
        int j = (i + 1) % n;
        area += vertices[i].first * vertices[j].second;
        area -= vertices[j].first * vertices[i].second;
    }
    area = abs(area) / 2.0;
    
    // Calculate perimeter
    double perimeter = 0;
    for(int i = 0; i < n; i++){
        int j = (i + 1) % n;
        double dx = vertices[j].first - vertices[i].first;
        double dy = vertices[j].second - vertices[i].second;
        perimeter += sqrt(dx * dx + dy * dy);
    }
    
    // Radius = Area / semi-perimeter = 2 * Area / Perimeter
    double radius = 2.0 * area / perimeter;
    
    cout << fixed << setprecision(10) << radius << endl;
    
    return 0;
}
```

The solution uses:
1. **Shoelace formula** to calculate the area of the polygon
2. **Distance formula** to calculate the perimeter
3. **Formula for inradius**: radius = Area / semi-perimeter = 2 * Area / Perimeter

This works because for a convex polygon, the largest circle that fits inside is tangent to all edges (or at least, its center is equidistant from all edges in the optimal configuration), and the relationship between area, perimeter, and inradius is: Area = inradius × semi-perimeter.
