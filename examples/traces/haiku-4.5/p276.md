Looking at this problem, I need to determine how many cups of tea Andrew owes based on how late he is.

Let me break down the rules:
- If he arrives on time or early (P ≤ S): 0 cups
- If he's late by at least 1 second (P > S): 1 cup
- If he's late by at least 5 minutes (300 seconds): 2 cups
- If he's late by at least 15 minutes (900 seconds): 3 cups
- If he's late by at least 30 minutes (1800 seconds): 4 cups

Let me verify with the examples:
1. S=10, P=10: delay = 0, answer = 0 ✓
2. S=10, P=11: delay = 1, answer = 1 ✓
3. S=0, P=300: delay = 300, which is ≥ 300 but < 900, answer = 2 ✓

The algorithm:
1. Calculate the delay: `delay = P - S`
2. If delay ≤ 0: return 0
3. If delay < 300: return 1
4. If delay < 900: return 2
5. If delay < 1800: return 3
6. Otherwise: return 4

```cpp
#include <iostream>
using namespace std;

int main() {
    int S, P;
    cin >> S >> P;
    
    int delay = P - S;
    
    int cups = 0;
    if (delay > 0) {
        cups = 1;
        if (delay >= 300) {
            cups = 2;
        }
        if (delay >= 900) {
            cups = 3;
        }
        if (delay >= 1800) {
            cups = 4;
        }
    }
    
    cout << cups << endl;
    
    return 0;
}
```
