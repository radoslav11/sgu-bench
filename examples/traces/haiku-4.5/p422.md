Key observations:
1. Vasya types characters one by one, and can check the screen at any point
2. When he checks and finds a typo, he backtracks to the first error and retypes from there
3. Each keystroke (including backspace) takes 1 unit of time
4. Looking at screen takes t units of time
5. We want to minimize expected total time

Let me think about the strategy:
- We need to decide at which positions to look at the screen
- If we look at position i, we spend t time units looking
- If there's an error in positions 1 to i, we need to backspace back to the first error and retype
- If there's no error, we continue

Let's define `dp[i]` = minimum expected time to correctly type first i characters.

For each position i, we can look at the screen after typing character i. We need to consider:
- The probability that all characters from position j+1 to i are correct (no typo)

Let me think differently. At each checkpoint position, we check the screen:
- If all characters since last checkpoint are correct: cost = 1 (looking at screen)
- If there's an error at position k (first error since last check at position j): 
  - We backspace (i-k) times to get back to position k-1
  - Cost = t (looking) + (i-k) (backspacing) + expected time to retype from k onwards
  - We need to retype those characters again

Let `dp[i]` = minimum expected time to have i correct characters on screen.

For transition from i to j (j > i):
- We type characters i+1 to j: cost = j - i (time units)
- We look at screen: cost = t
- Expected cost if there's a typo at position k (i < k ≤ j):
  - Probability of first typo at k: `p[k] * ∏(1-p[l])` for i < l < k
  - Cost: backspace (j-k) times, then retry from position k
  - This leads to: `dp[k-1] + expected_time_from_k`

The recurrence:
`dp[j] = min over i { dp[i] + (j-i) + t + expected_cost_of_errors }`

For the segment from i to j:
- Prob(all correct) = ∏(1-p[l]) for i < l ≤ j
- If all correct: we just typed them, time = j-i, then check = t
- If error at position k: we backspace and go back to dp[k-1], time = (j-k)

        for(int j = i+1; j <= n; j++){
            // Type characters from i+1 to j, then look at screen
            double cost = (j - i) + t; // typing time + looking time
            
            // Calculate expected additional cost due to errors
            double prob_no_error = 1.0;
            double expected_error_cost = 0.0;

            // If no error in this segment
            expected_error_cost += prob_no_error * 0; // no additional cost

        }
    }

#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    double t;
    cin >> n >> t;
    
    vector<double> p(n+1);
    for(int i = 1; i <= n; i++){
        cin >> p[i];
    }

    dp[0] = 0;
    
    for(int i = 0; i < n; i++){
        for(int j = i+1; j <= n; j++){
            double cost = (j - i) + t;
            double prob_ok = 1.0;
            double error_cost = 0.0;
            
            for(int k = i+1; k <= j; k++){
                double prob_error_at_k = p[k] * prob_ok;
                error_cost += prob_error_at_k * (j - k + dp[k-1]);
                prob_ok *= (1.0 - p[k]);
            }
            
            dp[j] = min(dp[j], dp[i] + cost + error_cost);
        }
    }
    
    cout << fixed << setprecision(12) << dp[n] << endl;
