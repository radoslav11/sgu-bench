Looking at this problem, I need to understand what we're computing:

1. We have an n-dimensional parallelepiped defined by n vectors v1, v2, ..., vn
2. There's a document at every integer point inside or on the border
3. Documents in k-dimensional facets (but not in any (k-1)-dimensional facet) have value 2^k
4. We need to sum all document values modulo p

Key insight: A point is in exactly a k-dimensional facet if exactly k of the n parameters are non-zero when expressing the point in the basis of vectors v1,...,vn.

If we parametrize points as: p = t1*v1 + t2*v2 + ... + tn*vn where 0 ≤ ti ≤ 1 for integer coordinates.

Actually, for integer points: 0 ≤ ti < |vi| essentially, but more precisely, we can think of it as choosing integer coefficients.

Let me reconsider: The integer points in the parallelepiped can be written as:
p = t1*v1 + t2*v2 + ... + tn*vn where 0 ≤ ti ≤ 1 (in continuous form)

For integer points, if we denote by L the lattice volume (which equals |det(v1,...,vn)|), there are exactly L integer points.

A point lies in a k-dimensional facet if it can be expressed using exactly k of the vectors. The number of k-dimensional facets is C(n,k), and each contains points that are combinations of k vectors.

For a k-dimensional facet formed by k specific vectors from our set, the number of integer points in that facet (but not in any lower-dimensional facet of that k-face) can be computed.

The total contribution is: sum over all k of (2^k * count of points in exactly k-dimensional facets)

By inclusion-exclusion or direct counting:
- Points on the boundary of exactly k dimensions contribute 2^k each
- The volume (number of integer points) can be computed using the determinant

The key formula: For a parallelepiped defined by vectors, the number of lattice points is |det(v1, v2, ..., vn)|.

For the k-dimensional contribution: We need to count points where exactly k coordinates are non-zero in the parameter space [0,1]^n.

This is a classic inclusion-exclusion problem. The answer involves computing:
- For each subset S of size k: count integer points where only dimensions in S are active
- This is |det of k vectors in S| for points strictly inside product [0,1]^k

Let me implement this approach:

```cpp
#include<bits/stdc++.h>
using namespace long long;

typedef long long ll;

ll mod;

ll modpow(ll base, ll exp, ll p) {
    ll result = 1;
    base %= p;
    while(exp > 0) {
        if(exp % 2 == 1) result = (result * base) % p;
        base = (base * base) % p;
        exp /= 2;
    }
    return result;
}

ll modinv(ll a, ll p) {
    return modpow(a, p-2, p);
}

ll determinant(vector<vector<ll>>& matrix, int size) {
    ll det = 1;
    vector<vector<ll>> m = matrix;
    
    for(int i = 0; i < size; i++) {
        int pivot = i;
        for(int j = i+1; j < size; j++) {
            if(abs(m[j][i]) > abs(m[pivot][i])) pivot = j;
        }
        if(m[pivot][i] == 0) return 0;
        
        if(pivot != i) {
            swap(m[i], m[pivot]);
            det = (mod - det) % mod;
        }
        
        det = (det * m[i][i]) % mod;
        ll inv = modinv(m[i][i], mod);
        
        for(int j = i; j < size; j++) {
            m[i][j] = (m[i][j] * inv) % mod;
        }
        
        for(int j = i+1; j < size; j++) {
            ll factor = m[j][i];
            for(int k = i; k < size; k++) {
                m[j][k] = (m[j][k] - (factor * m[i][k]) % mod + mod) % mod;
            }
        }
    }
    return det;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n >> mod;
    
    vector<vector<ll>> v(n, vector<ll>(n));
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j++) {
            cin >> v[i][j];
            v[i][j] %= mod;
        }
    }
    
    ll answer = 0;
    
    for(int mask = 1; mask < (1 << n); mask++) {
        int k = __builtin_popcount(mask);
        vector<vector<ll>> sub;
        int idx = 0;
        for(int i = 0; i < n; i++) {
            if(mask & (1 << i)) {
                sub.push_back(v[i]);
            }
        }
        
        ll det = determinant(sub, k);
        ll contrib = (modpow(2, k, mod) * det) % mod;
        answer = (answer + contrib) % mod;
    }
    
    cout << answer << endl;
    
    return 0;
}
```
