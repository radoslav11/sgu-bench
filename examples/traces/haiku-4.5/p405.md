Looking at this problem, I need to:

1. Read n participants and m matches
2. For each match:
   - Read the actual result (a, b)
   - Read n predictions from participants
   - Score each prediction based on:
     - +2 if they guessed the winner/tie correctly (sign of a-b matches sign of prediction_a - prediction_b)
     - +3 if they guessed the goal difference correctly (a-b == prediction_a - prediction_b)
     - +1 if they guessed first team's goals (a == prediction_a)
     - +1 if they guessed second team's goals (b == prediction_b)
3. Output total scores for each participant

Let me trace through the first example:
- 1 participant, 2 matches
- Match 1: actual (3,2), prediction (2,1)
  - Winner: actual 3>2 (first wins), prediction 2>1 (first wins) ✓ +2
  - Difference: actual 3-2=1, prediction 2-1=1 ✓ +3
  - First team goals: 3 vs 2 ✗ 0
  - Second team goals: 2 vs 1 ✗ 0
  - Subtotal: 5
- Match 2: actual (0,2), prediction (0,0)
  - Winner: actual 0<2 (second wins), prediction 0=0 (tie) ✗ 0
  - Difference: actual 0-2=-2, prediction 0-0=0 ✗ 0
  - First team goals: 0 vs 0 ✓ +1
  - Second team goals: 2 vs 0 ✗ 0
  - Subtotal: 1
- Total: 5 + 1 = 6 ✓

Let me verify the scoring logic:
- "guesses the winner (or a tie)" means the outcome (winner or tie) is correct
- "guesses the difference" means a-b == pred_a - pred_b
- "guesses the number of goals scored by the first team" means a == pred_a
- "guesses the number of goals scored by the second team" means b == pred_b

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<int> scores(n, 0);
    
    for (int match = 0; match < m; match++) {
        int a, b;
        cin >> a >> b;
        
        for (int participant = 0; participant < n; participant++) {
            int pred_a, pred_b;
            cin >> pred_a >> pred_b;
            
            // Check winner/tie
            int actual_result = (a > b) ? 1 : (a < b) ? -1 : 0;
            int pred_result = (pred_a > pred_b) ? 1 : (pred_a < pred_b) ? -1 : 0;
            
            if (actual_result == pred_result) {
                scores[participant] += 2;
            }
            
            // Check difference
            if (a - b == pred_a - pred_b) {
                scores[participant] += 3;
            }
            
            // Check first team goals
            if (a == pred_a) {
                scores[participant] += 1;
            }
            
            // Check second team goals
            if (b == pred_b) {
                scores[participant] += 1;
            }
        }
    }
    
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << scores[i];
    }
    cout << endl;
    
    return 0;
}
```
