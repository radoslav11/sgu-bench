Looking at this problem, I need to understand what's being asked:

1. We have a 3D grid of dimensions X × Y × Z
2. Each cell is a cabin, and adjacent cabins (distance 1 apart) are connected by passageways
3. We need to find a path that:
   - Starts at a cabin on the outer surface
   - Ends at a cabin on the outer surface
   - Traverses every passageway exactly once (this is an Eulerian path problem)

This is asking for an Eulerian path in a 3D grid graph.

Key observations:
- A cabin is on the outer surface if at least one of its coordinates is 0 or at its maximum (0 to X-1, etc.)
- The number of passageways between cabins in each dimension:
  - In X direction: (X-1) × Y × Z
  - In Y direction: X × (Y-1) × Z  
  - In Z direction: X × Y × (Z-1)
  - Total passageways: (X-1)×Y×Z + X×(Y-1)×Z + X×Y×(Z-1)

For an Eulerian path to exist:
- The graph must be connected
- There must be exactly 0 or 2 vertices with odd degree

For a 3D grid, the degree of an internal vertex is 6 (connected in ±x, ±y, ±z directions).
- Corner vertices (8 total): degree 3
- Edge vertices (not corners): degree 4
- Face vertices (not on edges): degree 5
- Internal vertices: degree 6

The corners have odd degree (3), which means there are exactly 8 vertices with odd degree. This is problematic for Eulerian paths (which need 0 or 2 odd-degree vertices).

Actually, with Y=1, it's essentially a 2D problem. In a 2×1×2 grid, we might have exactly 2 odd-degree vertices.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int X, Y, Z;
    int caseNum = 1;
    
    while(cin >> X >> Y >> Z){
        // Count edges in each direction
        long long edgesX = (long long)(X - 1) * Y * Z;
        long long edgesY = (long long)X * (Y - 1) * Z;
        long long edgesZ = (long long)X * Y * (Z - 1);
        long long totalEdges = edgesX + edgesY + edgesZ;
        
        // Count odd degree vertices (corners)
        // A corner vertex has degree = number of adjacent vertices
        // For a rectangular box, corners have degree 3
        long long oddVertices = 8;
        
        // For Eulerian path, we need to duplicate (oddVertices - 2) / 2 edges
        long long extraEdges = (oddVertices - 2) / 2;
        
        long long pathLength = totalEdges + extraEdges;
        
        cout << "Case #" << caseNum << ": " << pathLength << "\n";
        caseNum++;
    }
    
    return 0;
}
```
