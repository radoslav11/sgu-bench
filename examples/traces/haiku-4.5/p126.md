Looking at this problem, I need to understand the mechanics:

1. We have two boxes with A and B balls
2. In each move, we can move balls from one box to another, but we must move exactly as many balls as the destination box already contains (effectively doubling the destination box)
3. We need to determine if we can get all balls into one box, and if so, how many moves it takes

Let me trace through the example:
- Start: A=2, B=6
- Move 1: Move 2 balls from B to A (A had 2, so we move 2). Now A=4, B=4
- Move 2: Move 4 balls from B to A (A had 4, so we move 4). Now A=8, B=0
- Result: All balls in one box in 2 moves ✓

Key observations:
1. The total number of balls A+B never changes
2. Each move doubles one box and reduces the other
3. For all balls to end up in one box, the total A+B must be a power of 2 (since we keep doubling)
4. If A+B is a power of 2, we need to check if we can reach a state where one box has A+B balls and the other has 0

Let me think about when it's possible:
- If we end up with all balls in one box, that box has A+B balls
- Working backwards, before the last move, one box had (A+B)/2 and the other had (A+B)/2
- Before that, we need to trace back further

Actually, a better approach: we can simulate the process. At each step, we try to move from the box with more balls to the box with fewer balls (doubling the smaller one). If at any point one box becomes 0, we succeeded. If we get into a cycle or can't make progress, it's impossible.

Let me implement a simulation approach:

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    long long A, B;
    cin >> A >> B;
    
    long long total = A + B;
    
    // Check if total is a power of 2
    if((total & (total - 1)) != 0){
        cout << -1 << endl;
        return 0;
    }
    
    // Simulate the moves
    int moves = 0;
    set<pair<long long, long long>> visited;
    
    while(A != 0 && B != 0){
        if(visited.count({min(A,B), max(A,B)})){
            // We're in a cycle, impossible
            cout << -1 << endl;
            return 0;
        }
        visited.insert({min(A,B), max(A,B)});
        
        // Move from the box with more balls to the one with less
        if(A < B){
            // Move from B to A (double A)
            if(B < A){
                cout << -1 << endl;
                return 0;
            }
            B -= A;
            A += A;
        } else {
            // Move from A to B (double B)
            if(A < B){
                cout << -1 << endl;
                return 0;
            }
            A -= B;
            B += B;
        }
        moves++;
        
        if(moves > 100){
            // Safety check
            cout << -1 << endl;
            return 0;
        }
    }
    
    cout << moves << endl;
    return 0;
}
```
