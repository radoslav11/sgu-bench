Looking at this problem, I need to understand what's being asked:

1. We send data to ports from K+1 to 2K
2. A port is not blocked if it has exactly 3 non-zero bits in its binary representation
3. We're told that exactly M ports in the range [K+1, 2K] have exactly 3 non-zero bits
4. We need to determine if K can be uniquely identified from M

Let me think about this:
- For a given K, I need to count how many numbers in range [K+1, 2K] have exactly 3 bits set
- If only one value of K produces exactly M such ports, answer is "YES"
- If multiple values of K produce M such ports (or no value does), answer is "NO"

The approach:
1. For each possible M value, I need to find all K values that would produce that M
2. If exactly one K produces M, answer is "YES", otherwise "NO"

But K can be very large (since M ≤ 2^31-1). However, the number of integers with exactly 3 bits set up to some value n is limited. The maximum number of 3-bit integers in range [K+1, 2K] would be around C(32, 3) ≈ 5000 or so for 32-bit numbers.

So M won't be too large. For a given M, I can search for K values that produce M ports with exactly 3 bits.

Key insight: The range [K+1, 2K] has size K. As K grows, the density of numbers with exactly 3 bits changes. For large K, roughly the number of such integers is C(log2(2K), 3) / (2K) * K ≈ C(log2(2K), 3) / 2.

Let me write code that:
1. For each M, searches for K values where exactly M numbers in [K+1, 2K] have 3 bits set
2. Counts how many such K exist
3. If count == 1, answer is YES

```cpp
#include<bits/stdc++.h>
using namespace std;

int popcount(long long x) {
    return __builtin_popcountll(x);
}

long long countThreeBits(long long start, long long end) {
    long long count = 0;
    for (long long i = start; i <= end; i++) {
        if (popcount(i) == 3) {
            count++;
        }
    }
    return count;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    while(n--) {
        long long m;
        cin >> m;
        
        if (m == 0) {
            cout << "NO\n";
            continue;
        }
        
        int validCount = 0;
        
        // K can't be too large - search up to a reasonable bound
        // For M ports with 3 bits, K should be roughly in range where this is possible
        // Maximum search: if M exists, K should be found relatively soon
        
        for (long long k = 1; k <= 100000000LL; k++) {
            long long cnt = countThreeBits(k + 1, 2 * k);
            if (cnt == m) {
                validCount++;
                if (validCount > 1) break; // Found more than one, no need to continue
            }
            if (k > m * 1000 && validCount == 0) break; // Optimization: stop if K too large
        }

            cout << "NO\n";
        }
    }
```

Actually, for practical purposes with the constraint that M ≤ 2^31-1, the search space isn't that large. But iterating through all numbers would be too slow.

Let me use a smarter approach - generate all numbers with exactly 3 bits set up to a reasonable limit, then for each candidate K, use binary search or direct calculation:

```cpp
#include<bits/stdc++.h>
using namespace std;

int popcount(long long x) {
    return __builtin_popcountll(x);
}

long long countThreeBits(long long start, long long end) {
    long long count = 0;
    // Generate all numbers with exactly 3 bits in range
    for (int i = 0; i < 32; i++) {
        for (int j = i+1; j < 32; j++) {
            for (int k = j+1; k < 32; k++) {
                long long num = (1LL << i) | (1LL << j) | (1LL << k);
                if (num >= start && num <= end) {
                    count++;
                }
            }
        }
    }
    return count;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    while(n--) {
        long long m;
        cin >> m;
        
        int validCount = 0;
        
        for (long long k = 1; k <= 50000000LL; k++) {
            if (countThreeBits(k + 1, 2 * k) == m) {
                validCount++;
                if (validCount > 1) break;
            }
        }
        
        cout << (validCount == 1 ? "YES" : "NO") << "\n";
    }
    
    return 0;
}
```
