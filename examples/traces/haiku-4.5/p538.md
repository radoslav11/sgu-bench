Looking at this problem, I need to identify which brackets are part of emoticons.

A bracket is NOT part of an emoticon if:
1. It's an opening bracket `(` with a nearest closing bracket `)` to the right, and between them there are only spaces and Latin letters
2. It's a closing bracket `)` with a nearest opening bracket `(` to the left, and between them there are only spaces and Latin letters

If a bracket doesn't satisfy these conditions, it IS part of an emoticon.

Let me trace through the first example: "Hi:) (it is me) I have bad news:-("
- Position 3: `)` - look left for nearest `(`. Position 1 is `)`, position 0 is `i`. No `(` found immediately to the left in valid range. This `)` is part of emoticon.
- Position 5: `(` - look right for nearest `)`. Position 14 is `)`. Between them: "it is me" - only letters and spaces. This is NOT part of emoticon.
- Position 14: `)` - look left for nearest `(`. Position 5 is `(`. Between them: "it is me" - only letters and spaces. This is NOT part of emoticon.
- Position 30: `(` - look right for nearest `)`. Position 31 is `(`. No `)` found. This `(` is part of emoticon.
- Position 31: `(` - look right for nearest `)`. No `)` found. This `(` is part of emoticon.

Wait, that gives 3 emoticon brackets: positions 3, 30, 31. That matches the expected output of 3.

Algorithm:
1. For each `(` at position i:
   - Find the nearest `)` to the right
   - If found and contains only letters/spaces between them, this `(` is NOT part of emoticon
   - Otherwise, it IS part of emoticon
   
2. For each `)` at position i:
   - Find the nearest `(` to the left
   - If found and contains only letters/spaces between them, this `)` is NOT part of emoticon
   - Otherwise, it IS part of emoticon

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s;
    getline(cin, s);
    
    int count = 0;
    int n = s.length();
    
    // Check each opening bracket
    for(int i = 0; i < n; i++){
        if(s[i] == '('){
            // Look for nearest closing bracket to the right
            bool is_emoticon = true;
            int j = i + 1;
            while(j < n){
                if(s[j] == ')'){
                    // Found closing bracket, check if only letters/spaces between
                    bool valid = true;
                    for(int k = i + 1; k < j; k++){
                        if(s[k] != ' ' && !isalpha(s[k])){
                            valid = false;
                            break;
                        }
                    }
                    if(valid){
                        is_emoticon = false;
                    }
                    break;
                }
                j++;
            }
            if(is_emoticon){
                count++;
            }
        }
    }
    
    // Check each closing bracket
    for(int i = 0; i < n; i++){
        if(s[i] == ')'){
            // Look for nearest opening bracket to the left
            bool is_emoticon = true;
            int j = i - 1;
            while(j >= 0){
                if(s[j] == '('){
                    // Found opening bracket, check if only letters/spaces between
                    bool valid = true;
                    for(int k = j + 1; k < i; k++){
                        if(s[k] != ' ' && !isalpha(s[k])){
                            valid = false;
                            break;
                        }
                    }
                    if(valid){
                        is_emoticon = false;
                    }
                    break;
                }
                j--;
            }
            if(is_emoticon){
                count++;
            }
        }
    }
    
    cout << count << endl;
    
    return 0;
}
```
