Looking at this problem, I need to find a number `n` such that:

`n = (sum of all positive integers less than n) / x`

The sum of all positive integers from 1 to (n-1) is: `(n-1) * n / 2`

So the equation is:
`n = ((n-1) * n / 2) / x`

Simplifying:
`n * x = (n-1) * n / 2`
`2 * n * x = (n-1) * n`
`2 * x = n - 1` (dividing both sides by n, assuming n ≠ 0)
`n = 2 * x + 1`

Let me verify with the examples:
- Example 1: x = 1, so n = 2*1 + 1 = 3
  - Sum of predecessors of 3: 1 + 2 = 3
  - 3 / 1 = 3 ✓
  
- Example 2: x = 2, so n = 2*2 + 1 = 5
  - Sum of predecessors of 5: 1 + 2 + 3 + 4 = 10
  - 10 / 2 = 5 ✓

Perfect! The formula is simply `n = 2*x + 1`.
