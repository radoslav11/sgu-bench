Looking at this problem, I need to find coordinates of triangle ABC given:
- |AB| = c (distance from A to B)
- |AC| = b (distance from A to C)
- |AM| = m (distance from A to M, where M is the midpoint of BC)

Let me set up a coordinate system:
- Place A at the origin (0, 0)
- Place B at (c, 0)
- Place C at some point (x, y)

Since M is the midpoint of BC:
- M = ((c + x)/2, y/2)

From the constraints:
1. |AC| = b means: x² + y² = b²
2. |AM| = m means: ((c + x)/2)² + (y/2)² = m²

From constraint 2:
(c + x)²/4 + y²/4 = m²
(c + x)² + y² = 4m²

Expanding:
c² + 2cx + x² + y² = 4m²

Substituting x² + y² = b²:
c² + 2cx + b² = 4m²
2cx = 4m² - c² - b²
x = (4m² - c² - b²)/(2c)

From x² + y² = b²:
y² = b² - x²
y = √(b² - x²)

For a solution to exist:
- b² - x² ≥ 0 (so y is real)
- Triangle inequality and other geometric constraints

Let me verify with the example:
- c = 5, b = 5, m = 3
- x = (4·9 - 25 - 25)/(2·5) = (36 - 50)/10 = -14/10 = -1.4
- y² = 25 - 1.96 = 23.04
- y = 4.8

So C = (-1.4, 4.8), but the expected output shows C = (4, 0)...

Let me reconsider. Looking at the output:
- A = (0, 3)
- B = (-4, 0)
- C = (4, 0)

Check:
- |AB| = √(16 + 9) = √25 = 5 ✓
- |AC| = √(16 + 9) = √25 = 5 ✓
- M = (0, 0), |AM| = 3 ✓

So I should place A at origin but allow it to be at (0, h) for some h. Let me use:
- A at (0, h)
- B at (-c/2, 0)
- C at (c/2, 0)

Then M (midpoint of BC) = (0, 0)

Check constraints:
- |AB|² = c²/4 + h² = c²  → h² = 3c²/4 (doesn't work for general case)

Let me try a general approach:
- A at origin
- B at (c, 0)
- C at (x, y) where x² + y² = b²
- M at ((c+x)/2, y/2) where ((c+x)/2)² + (y/2)² = m²

#include <bits/stdc++.h>
using namespace std;

int main(){
    double c, b, m;
    cin >> c >> b >> m;
    
    // Check triangle inequality for median
    // Using formula: 4m² = 2b² + 2c² - a² where a = |BC|
    // So a² = 2b² + 2c² - 4m²
    
    double a_squared = 2*b*b + 2*c*c - 4*m*m;
    
    if(a_squared < 0){
        cout << "Mission impossible" << endl;
        return 0;
    }
    
    double a = sqrt(a_squared);
    
    // Place A at origin, B on positive x-axis
    double Ax = 0, Ay = 0;
    double Bx = c, By = 0;
    
    // Find C such that |AC| = b and |BC| = a
    // x² + y² = b²
    // (x-c)² + y² = a²
    // x² - 2cx + c² + y² = a²
    // b² - 2cx + c² = a²
    // x = (b² + c² - a²)/(2c)
    
    double Cx = (b*b + c*c - a*a)/(2*c);
    double Cy_squared = b*b - Cx*Cx;
    
    if(Cy_squared < -1e-9){
        cout << "Mission impossible" << endl;
        return 0;
    }
    
    if(Cy_squared < 0) Cy_squared = 0;
    double Cy = sqrt(Cy_squared);
    
    cout << fixed << setprecision(5);
    cout << Ax << " " << Ay << endl;
    cout << Bx << " " << By << endl;
    cout << Cx << " " << Cy << endl;
    
    return 0;
}
