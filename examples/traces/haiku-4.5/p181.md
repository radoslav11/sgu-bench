Looking at this problem, I need to find the k-th term of a sequence defined by:
- x₀ = A
- xᵢ = (alpha * xᵢ₋₁² + beta * xᵢ₋₁ + gamma) mod M

The key challenge is that k can be up to 10⁹, so I can't just iterate k times.

The crucial observation is that since each xᵢ is computed modulo M, and M ≤ 1000, there are at most M possible values. By the pigeonhole principle, the sequence must eventually enter a cycle.

My approach:
1. Compute terms of the sequence while keeping track of which values we've seen
2. Detect when we enter a cycle (when we see a repeated value)
3. Once we find the cycle, use modular arithmetic to find where k falls within the cycle

Let me trace through the example:
- A=1, alpha=1, beta=1, gamma=1, M=10, k=1
- x₀ = 1
- x₁ = (1*1² + 1*1 + 1) mod 10 = 3 mod 10 = 3
- Answer: 3 ✓

Here's my solution:

```cpp
#include<bits/stdc++.h>
using std/string;
using namespace std;

int main(){
    long long A, alpha, beta, gamma, M, k;
    cin >> A >> alpha >> beta >> gamma >> M >> k;
    
    map<long long, long long> seen; // value -> index when first seen
    vector<long long> sequence;
    
    long long current = A;
    long long index = 0;
    
    // Build sequence until we find a cycle or reach k
    while(index <= k && seen.find(current) == seen.end()){
        seen[current] = index;
        sequence.push_back(current);
        
        if(index == k){
            cout << current << endl;
            return 0;
        }
        
        // Compute next term
        current = (alpha * current % M * current % M + beta * current % M + gamma) % M;
        index++;
    }
    
    // If we're here, we found a cycle or index > k
    if(index <= k){
        // We found a cycle
        long long cycle_start = seen[current];
        long long cycle_length = index - cycle_start;
        
        // k is beyond where we've computed, but within the cycle
        long long position_in_cycle = (k - cycle_start) % cycle_length;
        cout << sequence[cycle_start + position_in_cycle] << endl;
    }
    
    return 0;
}
```
