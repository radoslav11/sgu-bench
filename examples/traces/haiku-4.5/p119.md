Looking at this problem, I need to find all pairs (A, B) where 0 ≤ A, B < N such that:

**If A₀X + B₀Y ≡ 0 (mod N) for any integers X, Y, then AX + BY ≡ 0 (mod N) as well.**

Let me think about what this means mathematically.

The condition "A₀X + B₀Y ≡ 0 (mod N)" defines a subgroup of solutions. For any (A, B) to satisfy our requirement, whenever (X, Y) is a solution to A₀X + B₀Y ≡ 0 (mod N), we need AX + BY ≡ 0 (mod N).

This means (A, B) must be orthogonal (in the modular sense) to all solutions of A₀X + B₀Y ≡ 0 (mod N).

The key insight: Let d = gcd(A₀, B₀, N). The solutions to A₀X + B₀Y ≡ 0 (mod N) form a subspace. A pair (A, B) works if and only if for all (X, Y) satisfying A₀X + B₀Y ≡ 0 (mod N), we have AX + BY ≡ 0 (mod N).

Actually, let's think differently: (A, B) is valid if the solution set of A₀X + B₀Y ≡ 0 (mod N) is a subset of the solution set of AX + BY ≡ 0 (mod N).

The solution set of A₀X + B₀Y ≡ 0 (mod N) has size N²/gcd(A₀, B₀, N)² in a sense. 

Let me reconsider: A pair (A, B) satisfies our condition if and only if (A, B) is in the ideal generated by (A₀, B₀) modulo N. In other words, there must exist integers k₁, k₂ such that:
- A ≡ k₁A₀ + k₂B₀ (mod N)
- B ≡ k₁A₀ + k₂B₀ (mod N)

Wait, that's not quite right either.

Let me use a different approach: (A, B) is valid if it's a linear combination of (A₀, B₀) and (0, N/gcd), etc. Actually, the valid pairs form a submodule.

The simplest approach: (A, B) is a valid pair if the set of (X, Y) mod N satisfying A₀X + B₀Y ≡ 0 (mod N) is contained in the set satisfying AX + BY ≡ 0 (mod N).

After some theory, the answer is: (A, B) is valid iff (A, B) ≡ (k₁A₀ + k₂B₀, k₁A₀' + k₂B₀') (mod N) for some structure. But more directly:

**(A, B) is valid if A ≡ tA₀ (mod N) and B ≡ tB₀ (mod N) for some integer t.**

This means (A, B) is a scalar multiple of (A₀, B₀) modulo N.

Let me verify with the example: N=3, A₀=1, B₀=2
- t=0: (0, 0) ✓
- t=1: (1, 2) ✓
- t=2: (2, 4) = (2, 1) mod 3 ✓

That matches!

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N, A0, B0;
    cin >> N >> A0 >> B0;
    
    vector<pair<int, int>> valid_pairs;
    
    // For each scalar t from 0 to N-1
    for (int t = 0; t < N; t++) {
        int A = (t * A0) % N;
        int B = (t * B0) % N;
        valid_pairs.push_back({A, B});
    }
    
    // Sort pairs
    sort(valid_pairs.begin(), valid_pairs.end());
    
    // Remove duplicates
    valid_pairs.erase(unique(valid_pairs.begin(), valid_pairs.end()), valid_pairs.end());
    
    cout << valid_pairs.size() << "\n";
    for (auto p : valid_pairs) {
        cout << p.first << " " << p.second << "\n";
    }
    
    return 0;
}
```
