Looking at this problem, I need to find the largest K such that when girls stand in a circle and pass a ball by skipping K-1 girls (throwing to the K-th girl on the left), every girl gets the ball exactly once before it returns to the first girl.

Let me think about the mathematics:
- N girls are numbered 1 to N in a circle
- Starting from girl 1, we throw to girl at position (1 + K - 1) mod N = K mod N
- Then from position K mod N, we throw to (K + K - 1) mod N = 2K - 1 mod N
- Generally, after i throws, the ball is at position (i*K) mod N

For each girl to get the ball exactly once before returning to girl 1, we need the sequence of positions: 0, K mod N, 2K mod N, 3K mod N, ... to visit all positions 0 to N-1 exactly once before returning to 0.

This happens if and only if gcd(K, N) = 1. Here's why:
- The ball returns to the starting position when i*K ≡ 0 (mod N)
- The smallest positive i for this is i = N/gcd(K,N)
- For all girls to be visited, we need N/gcd(K,N) = N, which means gcd(K,N) = 1

So I need to find the largest K where 1 ≤ K ≤ N/2 and gcd(K, N) = 1.

Let me verify with the examples:
1. N = 7: Need largest K ≤ 3 where gcd(K, 7) = 1
   - K = 3: gcd(3, 7) = 1 ✓ (7 is prime, so any K < 7 works)
   - K = 2: gcd(2, 7) = 1 ✓
   - Largest is K = 3 ✓

2. N = 6: Need largest K ≤ 3 where gcd(K, 6) = 1
   - K = 3: gcd(3, 6) = 3 ✗
   - K = 2: gcd(2, 6) = 2 ✗
   - K = 1: gcd(1, 6) = 1 ✓
   - Answer is K = 1 ✓

The algorithm:
1. Start from K = N/2 and go down
2. Find the first K where gcd(K, N) = 1
3. Return that K

Wait, let me reconsider the pattern. If n is odd, then gcd(n/2, n) = 1 always. If n is even, then any even number will share factor 2 with n, so we need the largest odd number ≤ n/2, which is 1 when n/2 is even, or n/2 when n/2 is odd.

Actually for n even: n/2 is an integer. If n ≡ 2 (mod 4), then n/2 is odd, so answer is n/2. If n ≡ 0 (mod 4), then n/2 is even, answer is 1.
