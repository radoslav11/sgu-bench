Looking at this problem, I need to find the largest square submatrix where:
1. Every row is a palindrome
2. Every column is a palindrome

Let me think about the constraints for a square of size n starting at position (r, c):
- For each row to be a palindrome: grid[r+i][c+j] == grid[r+i][c+n-1-j]
- For each column to be a palindrome: grid[r+i][c+j] == grid[r+n-1-i][c+j]

Combining these constraints:
- grid[r+i][c+j] == grid[r+i][c+n-1-j] (row palindrome)
- grid[r+i][c+j] == grid[r+n-1-i][c+j] (column palindrome)

This means: grid[r+i][c+j] == grid[r+i][c+n-1-j] == grid[r+n-1-i][c+j] == grid[r+n-1-i][c+n-1-j]

So I need to check that these 4 positions (which form a rectangle) all have the same character.

Algorithm:
1. Try all possible square sizes from largest to smallest
2. For each size, try all possible top-left positions
3. For each position and size, check if it forms a valid palindrome square
4. Return the first (largest) valid square found

Let me verify with the example:
- Grid is 5x10
- Square from (1,2) to (4,5) means size 4, starting at row 0 col 1 (0-indexed)
- Rows 0-3, cols 1-4 should all be "bccb" which is indeed a palindrome
- Checking palindrome square property: each of rows 0-3 has "bccb" (palindrome), and columns should also work

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int h, w;
    cin >> h >> w;
    
    vector<string> grid(h);
    for(int i = 0; i < h; i++){
        cin >> grid[i];
    }
    
    // Try sizes from largest to smallest
    for(int size = min(h, w); size >= 1; size--){
        // Try all positions
        for(int r = 0; r + size <= h; r++){
            for(int c = 0; c + size <= w; c++){
                // Check if square from (r,c) to (r+size-1, c+size-1) is palindrome square
                bool valid = true;
                
                // Check all positions
                for(int i = 0; i < size && valid; i++){
                    for(int j = 0; j < size && valid; j++){
                        // Check if grid[r+i][c+j] matches the palindrome requirements
                        // It should match:
                        // - grid[r+i][c+size-1-j] (same row, mirrored column)
                        // - grid[r+size-1-i][c+j] (mirrored row, same column)
                        // - grid[r+size-1-i][c+size-1-j] (mirrored row, mirrored column)
                        
                        char val = grid[r+i][c+j];
                        if(grid[r+i][c+size-1-j] != val) valid = false;
                        if(grid[r+size-1-i][c+j] != val) valid = false;
                        if(grid[r+size-1-i][c+size-1-j] != val) valid = false;
                    }
                }
                
                if(valid){
                    // Found a valid square, output 1-indexed coordinates
                    cout << r+1 << " " << c+1 << " " << r+size << " " << c+size << "\n";
                    return 0;
                }
            }
        }
    }
    
    return 0;
}
```
