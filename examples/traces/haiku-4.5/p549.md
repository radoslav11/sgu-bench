Looking at this problem, I need to:

1. Find the maximum number of sets where each set has exactly k dumbbells of distinct masses
2. All sets must have the same multiset of masses
3. For a given number of sets, maximize the total cost

Key insight: If we can make `t` sets, then for each distinct mass that appears in the sets, we need at least `t` dumbbells of that mass.

Approach:
1. Group dumbbells by mass
2. For each possible number of sets `t` (from high to low):
   - Check if we can form `t` sets
   - To form `t` sets, we need to select `k` distinct masses where each mass has at least `t` dumbbells
   - For each selected mass, pick the `t` most expensive dumbbells of that mass
3. Once we find the maximum `t`, calculate the maximum cost

Let me think through the algorithm:
- Group dumbbells by mass, and for each mass, sort by cost (descending)
- Try `t` from n/k down to 0
- For a given `t`, find which masses can contribute `t` dumbbells
- If we have at least `k` such masses, we can make `t` sets
- To maximize cost for `t` sets: select the `k` masses that give maximum total cost when we take the top `t` dumbbells from each

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, k;
    cin >> n >> k;
    
    map<int, vector<int>> byMass; // mass -> list of costs (sorted descending)
    
    for(int i = 0; i < n; i++){
        int m, c;
        cin >> m >> c;
        byMass[m].push_back(c);
    }
    
    // Sort costs for each mass in descending order
    for(auto& p : byMass){
        sort(p.second.begin(), p.second.end(), greater<int>());
    }
    
    // Try to find maximum number of sets
    int maxSets = 0;
    long long maxCost = 0;
    
    // Maximum possible sets is limited by n/k
    for(int t = n/k; t >= 1; t--){
        // Check if we can make t sets
        // We need k distinct masses, each with at least t dumbbells
        
        vector<long long> costsPerMass; // cost to include t dumbbells of each viable mass
        
        for(auto& p : byMass){
            if(p.second.size() >= t){
                long long cost = 0;
                for(int i = 0; i < t; i++){
                    cost += p.second[i];
                }
                costsPerMass.push_back(cost);
            }
        }
        
        // If we have at least k masses with t dumbbells each
        if(costsPerMass.size() >= k){
            // Sort in descending order and take top k
            sort(costsPerMass.begin(), costsPerMass.end(), greater<long long>());
            
            long long totalCost = 0;
            for(int i = 0; i < k; i++){
                totalCost += costsPerMass[i];
            }
            
            maxSets = t;
            maxCost = totalCost;
            break; // Since we're going from high to low, first valid t is maximum
        }
    }
    
    cout << maxSets << " " << maxCost << "\n";
    
    return 0;
}
```
