Looking at this problem, I need to arrange numbers 0 to 2^(N+M)-1 in a 2^N × 2^M matrix such that:
1. Adjacent cells (horizontally or vertically) differ by exactly one bit
2. The matrix wraps around (toroidal topology)

This is essentially asking for a Gray code arrangement on a 2D grid with cyclic boundaries.

Let me think about the approach:
- A Gray code is a sequence where consecutive numbers differ by exactly one bit
- For a 2D matrix with cyclic boundaries, I can use a construction based on Gray codes
- For a position (i, j), I can compute the value using reflected binary code (Gray code)

The key insight is that for a 2D torus, if we use Gray codes for rows and columns separately, we can construct a valid arrangement.

For position (i, j):
- We can use the Gray code of i for the first N bits
- We can use the Gray code of j for the last M bits
- But we need to ensure the cyclic property works

Actually, a simpler approach: use the reflected Gray code directly. The idea is:
- Convert position (row, col) to a linear index using a 2D Gray code
- The value at position (i, j) should be such that adjacent positions differ by one bit

Let me use the standard approach for 2D Gray codes:
- Treat the N+M dimensional hypercube Gray code but mapped to 2D
- Use row-major ordering with Gray codes

Here's my approach:
1. For each position (i, j), compute what number should be there
2. Use Gray code to ensure adjacent cells differ by one bit
3. The Gray code for (i,j) can be computed by treating it as a (N+M)-bit number where the first N bits are the Gray code of i and last M bits are Gray code of j

Let me verify with the example:
- N=1, M=1: 2×2 matrix
- Positions: (0,0), (0,1), (1,0), (1,1)
- Gray code of 0 = 0, Gray code of 1 = 1
- So: (0,0)→0, (0,1)→2, (1,0)→1, (1,1)→3

This matches the expected output!

```cpp
#include<bits/stdc++.h>
using namespace std;

int grayCode(int n) {
    return n ^ (n >> 1);
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, M;
    cin >> N >> M;
    
    int rows = 1 << N;  // 2^N
    int cols = 1 << M;  // 2^M
    
    vector<vector<int>> matrix(rows, vector<int>(cols));
    
    // For each position (i, j), compute the value
    // We use Gray code for both dimensions
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            // Gray code of row index contributes first N bits
            int grayRow = grayCode(i);
            // Gray code of col index contributes last M bits
            int grayCol = grayCode(j);
            
            // Combine: shift grayRow left by M bits and OR with grayCol
            int value = (grayRow << M) | grayCol;
            matrix[i][j] = value;
        }
    }
    
    // Output
    for(int i = 0; i < rows; i++) {
        for(int j = 0; j < cols; j++) {
            if(j > 0) cout << " ";
            cout << matrix[i][j];
        }
        cout << "\n";
    }
    
    return 0;
}
```
