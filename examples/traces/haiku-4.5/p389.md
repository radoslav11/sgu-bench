Looking at this problem, I need to:

1. Understand that we have an n-dimensional hypercube where each vertex can be represented as an n-bit binary string
2. There are 3 kingdom capitals at positions given by 3 binary strings
3. A town at position i is neutral if its Manhattan distance to all three capitals is equal
4. Count how many such neutral towns exist

The Manhattan distance between two points in an n-dimensional space is the sum of absolute differences of their coordinates.

For a town represented by binary string `town` to be neutral:
- `distance(town, capital1) == distance(town, capital2) == distance(town, capital3)`

Let me think about this mathematically. If we denote the three capitals as c1, c2, c3, and a town as t, then:
- For each bit position i, if c1[i]=c2[i], then t[i] can be anything (contributes same distance to both)
- If c1[i]≠c2[i], then t[i] must be chosen to balance the distances

For a town to be equidistant from all 3 capitals:
- `d(t,c1) = d(t,c2) = d(t,c3)`

This means: `d(t,c1) = d(t,c2)` AND `d(t,c2) = d(t,c3)`

For each bit position independently, I need to check which values of town[i] can satisfy the constraint.

Let me think bit by bit. For position i:
- Let the bits of the three capitals at position i be a, b, c (each 0 or 1)
- For a town bit t at position i, it contributes |t-a| to d(t,c1), |t-b| to d(t,c2), |t-c| to d(t,c3)

For each bit position, I need to determine if t=0 and/or t=1 can be part of a valid solution.

The constraint is that summing over all positions: sum(|t[i]-a[i]|) = sum(|t[i]-b[i]|) = sum(|t[i]-c[i]|)

This is complex because the bit choices are interdependent. Let me use a different approach:
- For each possible town (2^n possibilities), check if distances are equal
- But n can be up to... let me check the constraints. The problem says n-bit string, and with 0.25 seconds, n should be small (probably ≤ 20)

Given the small time limit, n is likely ≤ 20. I can iterate through all 2^n towns.

```cpp
#include<bits/stdc++.h>
using namespace std;

const long long MOD = 1e9 + 7;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string c1, c2, c3;
    cin >> c1 >> c2 >> c3;
    
    int n = c1.length();
    long long count = 0;
    
    // Iterate through all possible towns (2^n possibilities)
    for(long long town = 0; town < (1LL << n); town++){
        // Calculate distances
        long long d1 = 0, d2 = 0, d3 = 0;
        
        for(int i = 0; i < n; i++){
            int bit = (town >> i) & 1;
            int a = c1[i] - '0';
            int b = c2[i] - '0';
            int c = c3[i] - '0';
            
            d1 += abs(bit - a);
            d2 += abs(bit - b);
            d3 += abs(bit - c);
        }
        
        // Check if all distances are equal
        if(d1 == d2 && d2 == d3){
            count++;
        }
    }
    
    cout << count % MOD << endl;
    
    return 0;
}
```
