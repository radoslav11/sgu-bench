Looking at this problem, I need to:

1. Take a polynomial `ax + b`
2. Calculate its k-th power: `(ax + b)^k`
3. Find the sum of all coefficients in the resulting polynomial

The key insight is that the sum of coefficients of any polynomial `P(x)` can be found by evaluating `P(1)`.

So if I have the polynomial `(ax + b)^k`, the sum of coefficients is `(a*1 + b)^k = (a + b)^k`.

Let me verify with the example:
- a = 1, b = 2, k = 2
- Polynomial: `(1*x + 2)^2 = (x + 2)^2 = x^2 + 4x + 4`
- Sum of coefficients: `1 + 4 + 4 = 9`
- Using formula: `(1 + 2)^2 = 3^2 = 9` ✓

So the solution is straightforward: compute `(a + b)^k`.

```cpp
#include<iostream>
#include<cmath>
using namespace std;

int main(){
  long long a, b, k;
  cin >> a >> b >> k;
  
  long long result = 1;
  long long base = a + b;
  
  for(int i = 0; i < k; i++){
    result *= base;
  }
  
  cout << result << endl;
  
  return 0;
}
```
