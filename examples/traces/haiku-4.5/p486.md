Looking at this problem, I need to:

1. Read two 4-digit numbers (secret and guess)
2. Count "bulls" - digits in the same position in both numbers
3. Count "cows" - digits that exist in both numbers but in different positions

Let me trace through the first example:
- Secret: 5071
- Guess: 6012
- Position 0: 5 vs 6 - no match
- Position 1: 0 vs 0 - bull! (1 bull)
- Position 2: 7 vs 1 - no match
- Position 3: 1 vs 2 - no match

Now for cows, I need to find digits that are in both numbers but not in the correct position:
- 5 in secret[0], not in guess at all - no cow
- 0 in secret[1] and guess[1] - already counted as bull, skip
- 7 in secret[2], not in guess - no cow
- 1 in secret[3], in guess at position 2 - this is a cow! (1 cow)

So the answer is 1 bull, 1 cow ✓

Algorithm:
1. Count bulls by comparing each position
2. For cows, I need to be careful not to double-count bulls
   - For each digit in the secret that's not a bull, check if it exists in the guess at a different position
   - But I need to be careful about duplicates - if a digit appears multiple times, I should only count matches that aren't already accounted for

Actually, a cleaner approach:
1. Count bulls first
2. For remaining digits (non-bull positions), create frequency maps
3. Count how many digits from secret's non-bull digits appear in guess's non-bull digits

Let me implement this:

```cpp
#include <iostream>
#include <string>
#include <map>

using namespace std;

int main() {
    string secret, guess;
    cin >> secret >> guess;
    
    int bulls = 0;
    int cows = 0;
    
    // Count bulls
    map<char, int> secret_remaining, guess_remaining;
    
    for (int i = 0; i < 4; i++) {
        if (secret[i] == guess[i]) {
            bulls++;
        } else {
            // Keep track of non-matching digits
            secret_remaining[secret[i]]++;
            guess_remaining[guess[i]]++;
        }
    }
    
    // Count cows from remaining digits
    for (auto& p : secret_remaining) {
        char digit = p.first;
        int secret_count = p.second;
        int guess_count = guess_remaining[digit];
        
        // The number of cows for this digit is the minimum of its counts
        cows += min(secret_count, guess_count);
    }
    
    cout << bulls << " " << cows << endl;
    
    return 0;
}
```
