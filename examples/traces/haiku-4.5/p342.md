Looking at this problem, I need to find the minimum number of terms to represent a number A as a sum of signed powers of B.

Let me think through the approach:

1. This is essentially representing A in a "balanced" base-B system where digits can be -1, 0, or 1 (instead of just 0 to B-1).

2. The key insight is that if we have a digit ≥ B/2 in standard base-B representation, we can use the next power with a negative coefficient. For example, if we have 7 in base 10 and B=10, we could write it as 10 - 3 instead of 7×1.

3. The algorithm:
   - Convert A to base B
   - Process digits from least significant to most significant
   - If a digit d ≥ (B+1)/2, replace it with d-B and add 1 to the next digit (this represents using the next power with negative coefficient)
   - Count non-zero digits

Let me trace through the example:
- A = 1120, B = 10
- In base 10: 1120 = 1×10³ + 1×10² + 2×10¹ + 0×10⁰
- Digits: [0, 2, 1, 1]
- Starting from position 0:
  - d=0: keep as is
  - d=2: 2 < 5, keep as is
  - d=1: 1 < 5, keep as is
  - d=1: 1 < 5, keep as is
- But wait, we can optimize: 2 in position 1 (representing 20) could be written differently
- Actually, let me reconsider: 1120 = 1000 + 100 + 20 = 1×10³ + 1×10² + 2×10¹
- We could write: 1120 = 1×10³ + 1×10² + 2×10¹ = 1×10³ + 1×10² + (1×10² - 8×10⁰) = ...
- Let me think differently: maybe we want to represent as 1×10³ + 2×10² - 8×10¹ = 1000 + 200 - 80 = 1120 (3 terms)
- Or: 1×10³ + 1×10² + 2×10¹ = 1000 + 100 + 20 (3 terms)

The algorithm should work with balanced/non-adjacent form principles.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
  string A;
  long long B;
  cin >> A >> B;
  
  // Convert A to base B
  vector<long long> digits;
  
  // Handle the number as a string, convert to base B
  long long remainder = 0;
  for(int i = 0; i < A.length(); i++){
    remainder = remainder * 10 + (A[i] - '0');
    long long digit = remainder % B;
    digits.push_back(digit);
    remainder = remainder / B;
  }
  
  // Handle remaining remainder
  while(remainder > 0){
    digits.push_back(remainder % B);
    remainder /= B;
  }
  
  // Now process digits to minimize terms
  // If digit >= (B+1)/2, we can replace with negative
  int threshold = (B + 1) / 2;
  
  for(int i = 0; i < digits.size(); i++){
    if(digits[i] >= threshold){
      digits[i] = digits[i] - B;
      if(i + 1 < digits.size()){
        digits[i + 1]++;
      } else {
        digits.push_back(1);
      }
    }
  }
  
  // Handle carries from the transformation
  for(int i = 0; i < digits.size(); i++){
    if(digits[i] >= threshold){
      digits[i] = digits[i] - B;
      if(i + 1 < digits.size()){
        digits[i + 1]++;
      } else {
        digits.push_back(1);
      }
    }
  }
  
  // Count non-zero digits
  int count = 0;
  for(auto d : digits){
    if(d != 0) count++;
  }
  
  cout << count << endl;
  
  return 0;
}
```
