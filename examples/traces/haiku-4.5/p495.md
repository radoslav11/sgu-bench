Looking at this problem, I need to find the expected number of prizes given out when M winners each randomly select from N boxes (with replacement).

Let me think about this step by step:

1. Each of the M winners picks a box uniformly at random from N boxes
2. A prize is given if the box hasn't been picked before
3. We need to find the expected number of prizes (unique boxes picked)

This is a classic "coupon collector" variant problem.

Let me use linearity of expectation. For each box i, let X_i be an indicator variable that is 1 if box i is picked at least once by the M winners, and 0 otherwise.

The expected number of prizes = E[X_1 + X_2 + ... + X_N] = E[X_1] + E[X_2] + ... + E[X_N]

By symmetry, E[X_i] is the same for all boxes.

E[X_i] = P(box i is picked at least once) = 1 - P(box i is never picked)

P(box i is never picked in M trials) = (1 - 1/N)^M = ((N-1)/N)^M

So E[X_i] = 1 - ((N-1)/N)^M

Therefore, the expected number of prizes = N * (1 - ((N-1)/N)^M)

Let me verify with the examples:
- Example 1: N=5, M=7
  Expected = 5 * (1 - (4/5)^7) = 5 * (1 - 0.2097152) = 5 * 0.7902848 = 3.951424 ✓

- Example 2: N=4, M=3
  Expected = 4 * (1 - (3/4)^3) = 4 * (1 - 27/64) = 4 * 37/64 = 148/64 = 2.3125 ✓

Perfect! The formula works.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long N, M;
    cin >> N >> M;
    
    double prob_never_picked = pow((double)(N - 1) / N, M);
    double expected_prizes = N * (1.0 - prob_never_picked);
    
    cout << fixed << setprecision(15) << expected_prizes << endl;
    
    return 0;
}
```
