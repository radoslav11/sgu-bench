Looking at this problem, I need to:

1. Read K (max lines per page) and N (number of phone records)
2. Read N phone numbers
3. Sort the phone numbers
4. Allocate pages such that:
   - First 2 pages are for directory info
   - Phone records start on page 3
   - Each page can have at most K lines
   - When a phone number with a new first digit appears, it must start on a new page
5. Count total pages needed

Let me trace through the example:
- K = 5 (max 5 lines per page)
- N = 10 phone numbers
- Numbers: 1234, 5678, 1345, 1456, 1678, 1111, 5555, 6789, 6666, 5000

After sorting: 1111, 1234, 1345, 1456, 1678, 5000, 5555, 5678, 6666, 6789

Now I need to group by first digit and allocate pages:
- First digit 1: [1111, 1234, 1345, 1456, 1678] - 5 numbers
- First digit 5: [5000, 5555, 5678] - 3 numbers  
- First digit 6: [6666, 6789] - 2 numbers

Page allocation (starting from page 3):
- Page 3: 1111, 1234, 1345, 1456, 1678 (5 lines, first digit group 1)
- Page 4: 5000, 5555, 5678 (3 lines, first digit group 5)
- Page 5: 6666, 6789 (2 lines, first digit group 6)

Total pages = 2 (header) + 3 (content) = 5 ✓

Algorithm:
1. Sort all phone numbers
2. Group consecutive numbers by their first digit
3. For each group, calculate how many pages it needs (ceiling of count/K)
4. Sum all pages needed for content, then add 2 for the header pages

```cpp
#include<bits/stdc++.h>
using std::cin;
using std::cout;
using std::string;
using std::vector;
using std::sort;

int main(){
    int K, N;
    cin >> K >> N;
    
    vector<string> phones(N);
    for(int i = 0; i < N; i++){
        cin >> phones[i];
    }
    
    // Sort phone numbers
    sort(phones.begin(), phones.end());
    
    int pages = 2; // First 2 pages for directory info
    
    int i = 0;
    while(i < N){
        char firstDigit = phones[i][0];
        int count = 0;
        
        // Count all numbers with the same first digit
        while(i < N && phones[i][0] == firstDigit){
            count++;
            i++;
        }
        
        // Calculate pages needed for this group
        int pagesForGroup = (count + K - 1) / K; // Ceiling division
        pages += pagesForGroup;
    }
    
    cout << pages << endl;
    
    return 0;
}
```

The solution:
1. Reads K and N
2. Reads all N phone numbers into a vector
3. Sorts the phone numbers lexicographically (which sorts them numerically since they're all 4-digit strings)
4. Iterates through sorted numbers, grouping by first digit
5. For each group with a new first digit, calculates how many pages are needed using ceiling division
6. Adds 2 for the header pages and outputs the total
