Looking at this problem, I need to simulate a robot moving through an N×M grid following a specific algorithm:

1. Robot starts at position (is, js)
2. At each position, it places a bomb and moves to the next building
3. The robot chooses the next building in this priority order: down (i+1,j), left (i,j-1), up (i-1,j), right (i,j+1)
4. It picks the first valid neighbor that hasn't been destroyed yet
5. If no valid neighbors exist, the robot stops
6. Output the sequence of moves

Let me trace through the example:
- Grid: 3×3, Start: (1,2)
- From (1,2): bomb at (1,2), neighbors in order: (2,2)✓, (1,1)✓, (0,2)✗, (1,3)✓ → go to (2,2) → 'D'
- From (2,2): bomb at (2,2), neighbors: (3,2)✓, (2,1)✓, (1,2)✗, (2,3)✓ → go to (3,2) → 'D'
- From (3,2): bomb at (3,2), neighbors: (4,2)✗, (3,1)✓, (2,2)✗, (3,3)✓ → go to (3,1) → 'L'
- From (3,1): bomb at (3,1), neighbors: (4,1)✗, (3,0)✗, (2,1)✓, (3,2)✗ → go to (2,1) → 'U'
- From (2,1): bomb at (2,1), neighbors: (3,1)✗, (2,0)✗, (1,1)✓, (2,2)✗ → go to (1,1) → 'U'
- From (1,1): bomb at (1,1), neighbors: (2,1)✗, (1,0)✗, (0,1)✗, (1,2)✗ → no valid neighbors, stop

Output: "DDLUU" ✓

```cpp
#include <bits/stdc++.h>
using std::cout;
using std::cin;

int main() {
    int N, M;
    cin >> N >> M;
    
    int i, j;
    cin >> i >> j;
    
    // Track destroyed buildings
    bool destroyed[11][11];
    memset(destroyed, false, sizeof(destroyed));
    
    string result = "";
    
    while (true) {
        // Mark current building as destroyed
        destroyed[i][j] = true;
        
        // Try to find next building in priority order: D, L, U, R
        int next_i = -1, next_j = -1;
        char direction = ' ';
        
        // Try down (i+1, j)
        if (i + 1 <= N && !destroyed[i + 1][j]) {
            next_i = i + 1;
            next_j = j;
            direction = 'D';
        }
        // Try left (i, j-1)
        else if (j - 1 >= 1 && !destroyed[i][j - 1]) {
            next_i = i;
            next_j = j - 1;
            direction = 'L';
        }
        // Try up (i-1, j)
        else if (i - 1 >= 1 && !destroyed[i - 1][j]) {
            next_i = i - 1;
            next_j = j;
            direction = 'U';
        }
        // Try right (i, j+1)
        else if (j + 1 <= M && !destroyed[i][j + 1]) {
            next_i = i;
            next_j = j + 1;
            direction = 'R';
        }
        
        // If no valid next building, robot stops
        if (next_i == -1) {
            break;
        }
        
        // Move to next building
        i = next_i;
        j = next_j;
        result += direction;
    }
    
    cout << result;
    
    return 0;
}
```
