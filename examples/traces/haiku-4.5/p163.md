Looking at this problem, I need to:

1. Read the number of sons
2. Read the exponent (1-3)
3. Read the mental potentials of each son (values from -3 to 3)
4. Find a subset of sons such that the sum of their potentials raised to the given exponent is maximized
5. The sum can be 0 if we choose no sons (or if all combinations give negative sums)

The key insight is that since we want to maximize the sum, and we can choose any subset (including the empty set which gives 0), we should:
- Include a son if their potential raised to the exponent is positive
- Exclude a son if their potential raised to the exponent is negative
- The contribution of each son is independent, so it's a greedy problem

Let me trace through the examples:

Example 1:
- 3 sons with potentials: 2, -1, 1
- Exponent: 3
- Powers: 2³ = 8, (-1)³ = -1, 1³ = 1
- We take sons with positive contributions: 8 + 1 = 9 ✓

Example 2:
- 1 son with potential: -1
- Exponent: 1
- Power: (-1)¹ = -1
- Since -1 is negative, we choose 0 sons, sum = 0 ✓

The algorithm is simple:
1. For each son, calculate their potential raised to the exponent
2. If the result is positive, add it to our sum
3. Output the sum (which starts at 0)

```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    int exponent;
    cin >> exponent;
    
    long long sum = 0;
    
    for (int i = 0; i < n; i++) {
        int potential;
        cin >> potential;
        
        long long power = 1;
        for (int j = 0; j < exponent; j++) {
            power *= potential;
        }
        
        if (power > 0) {
            sum += power;
        }
    }
    
    cout << sum << endl;
    
    return 0;
}
```
