Looking at this problem, I need to find the next lexicographically larger valid bracket sequence of the same length.

Let me think about the approach:

1. A valid bracket sequence has equal numbers of '(' and ')' 
2. At any prefix, the number of '(' must be >= number of ')'
3. To find the next lexicographic sequence, I should:
   - Find the rightmost position where I can change a ')' to '(' such that the sequence can still be made valid
   - After making that change, fill the rest with the lexicographically smallest valid continuation

Strategy:
1. Scan from right to left to find a position where I can replace ')' with '('
2. When I find such a position, I need to ensure:
   - There's a ')' to change to '('
   - After changing it, the remaining part can form a valid bracket sequence
3. After making the change, fill the suffix with the lexicographically smallest valid sequence

Let me think more carefully:
- I need to find the rightmost ')' that I can change to '(' while maintaining validity
- After position i where I place '(', I need to place the remaining brackets optimally
- If I have `open` unmatched '(' and need `close` more ')', the smallest sequence is all '(' followed by all ')'

Algorithm:
1. From right to left, try to find a position where we can change ')' to '('
2. At position i, if we change ')' to '(', we increase open brackets by 1
3. We need to check if with this change, we can still form a valid sequence with remaining characters
4. After making the change at position i, fill the suffix: count remaining '(' and ')', place all '(' first then all ')'

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s;
    cin >> s;
    int n = s.length();
    
    // Try to find rightmost position where we can change ')' to '('
    for(int i = n - 1; i >= 0; i--){
        if(s[i] == ')'){
            // Try changing this ')' to '('
            string temp = s.substr(0, i) + '(' + string(n - i - 1, ' ');
            
            // Check if prefix is valid
            int balance = 0;
            bool valid = true;
            for(int j = 0; j <= i; j++){
                if(temp[j] == '(') balance++;
                else balance--;
                if(balance < 0){
                    valid = false;
                    break;
                }
            }
            
            if(!valid) continue;
            
            // Count remaining '(' and ')' needed
            int remaining_len = n - i - 1;
            int open_needed = 0, close_needed = 0;
            
            // Current balance after position i
            for(int j = 0; j <= i; j++){
                if(temp[j] == '(') open_needed++;
                else open_needed--;
            }
            
            // We need to add brackets such that:
            // - Total '(' in suffix = close_needed
            // - Total ')' in suffix = open_needed (to close all opens)
            // - remaining_len = close_needed + open_needed
            
            close_needed = (remaining_len - open_needed) / 2;
            open_needed = remaining_len - close_needed;
            
            if(open_needed < 0 || close_needed < 0) continue;
            if(open_needed + close_needed != remaining_len) continue;
            
            // Fill suffix: all '(' first, then all ')'
            for(int j = 0; j < open_needed; j++){
                temp[i + 1 + j] = '(';
            }
            for(int j = 0; j < close_needed; j++){
                temp[i + 1 + open_needed + j] = ')';
            }
            
            // Verify the entire string is valid
            balance = 0;
            valid = true;
            for(int j = 0; j < n; j++){
                if(temp[j] == '(') balance++;
                else balance--;
                if(balance < 0){
                    valid = false;
                    break;
                }
            }
            
            if(valid && balance == 0){
                cout << temp << endl;
                return 0;
            }
        }
    }
    
    cout << "No solution" << endl;
    return 0;
}
```
