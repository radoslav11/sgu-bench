Looking at this problem, I need to:

1. Calculate the first K Fibonacci numbers
2. Sum them up
3. Output the sum

Let me trace through the example:
- K = 5
- Fibonacci sequence: F1=1, F2=1, F3=2, F4=3, F5=5
- Sum S = 1 + 1 + 2 + 3 + 5 = 12 ✓

The approach is straightforward:
1. Read K
2. Generate the first K Fibonacci numbers and sum them
3. Print the sum

Since K < 41, the Fibonacci numbers can get quite large, so I should use `long long` to avoid overflow.

```cpp
#include <iostream>
using namespace std;

int main() {
    int k;
    cin >> k;
    
    long long sum = 0;
    long long f1 = 1, f2 = 1;
    
    if (k >= 1) {
        sum += f1;  // F1 = 1
    }
    if (k >= 2) {
        sum += f2;  // F2 = 1
    }
    
    // Generate remaining Fibonacci numbers
    for (int i = 3; i <= k; i++) {
        long long next = f1 + f2;
        sum += next;
        f1 = f2;
        f2 = next;
    }
    
    cout << sum << endl;
    
    return 0;
}
```

The algorithm:
- Start with F1=1 and F2=1, add them to sum if K≥1 and K≥2 respectively
- For i from 3 to K, calculate the next Fibonacci number as f1+f2, add it to sum, and shift the values (f1 becomes f2, f2 becomes next)
- Output the sum
