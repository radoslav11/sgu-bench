Looking at this problem, I need to count the number of non-isomorphic colored graphs with N vertices where each edge can be one of M colors.

This is a classic application of **Burnside's lemma** (also known as the Cauchy-Frobenius lemma) combined with **Pólya enumeration**.

Key insights:
1. A complete graph with N vertices has N(N-1)/2 edges
2. Each edge can be colored in M ways
3. Two graphs are isomorphic if one can be obtained from the other by relabeling vertices
4. We need to count equivalence classes under the symmetric group S_N

The formula using Burnside's lemma is:
- Number of non-isomorphic graphs = (1/|S_N|) * Σ(over all permutations σ in S_N) of (number of colorings fixed by σ)

A coloring is fixed by a permutation σ if applying σ doesn't change the coloring. This happens when edges in the same cycle of σ have the same color.

For a permutation with cycle structure, if there are k cycles in the permutation graph (considering both vertex cycles and how they affect edges), then M^k colorings are fixed by that permutation.

The number of edge-cycles created by a permutation σ can be computed as follows:
- For vertex cycles of lengths c₁, c₂, ..., the number of edge-cycles is determined by the cycle structure

Actually, a simpler approach: for a permutation σ of vertices, the number of edge orbits (cycles) it creates equals the number of vertex cycles in σ plus the number of edge pairs that must have the same color due to σ.

Let me implement using the formula: for each permutation type (cycle type), count permutations with that cycle type and compute the number of orbits.

For a permutation with cycle type, the number of edge orbits can be calculated. If σ has cycle structure with cycles of lengths c₁, c₂, ..., cₖ, then:
- Edges within a cycle of length c contribute ⌈c/2⌉ orbits
- Edges between different cycles contribute 1 orbit per pair

After research, the formula for edge-orbits under a permutation σ is:
**orbits = (1/2) * (sum of cycle lengths + number of edges fixed within cycles)**

Let me code this using cycle index calculation:

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

ll mod_pow(ll base, ll exp, ll mod) {
    ll result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1) result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

ll mod_inverse(ll a, ll p) {
    return mod_pow(a, p - 2, p);
}

int main(){
    int N, M;
    ll P;
    cin >> N >> M >> P;
    
    // Generate all partitions of N (cycle types)
    vector<vector<int>> partitions;
    function<void(int, int, vector<int>&)> gen_partitions = [&](int n, int max_val, vector<int>& current) {
        if (n == 0) {
            partitions.push_back(current);
            return;
        }
        for (int i = min(n, max_val); i >= 1; i--) {
            current.push_back(i);
            gen_partitions(n - i, i, current);
            current.pop_back();
        }
    };
    vector<int> temp;
    gen_partitions(N, N, temp);
    
    ll answer = 0;
    
    for (auto& partition : partitions) {
        // Count permutations with this cycle type
        ll count = 1;
        map<int, int> freq;
        for (int c : partition) freq[c]++;
        
        for (auto& [len, cnt] : freq) {
            for (int i = 1; i <= cnt; i++) {
                count = (count * len) % P;
            }
            ll fact = 1;
            for (int i = 1; i <= cnt; i++) fact = (fact * i) % P;
            count = (count * mod_inverse(fact, P)) % P;
        }
        
        // Calculate number of edge orbits
        ll edge_orbits = 0;
        
        // Orbits within cycles
        for (int c : partition) {
            edge_orbits += (c * (c - 1)) / 2;
        }
        
        // Orbits between cycles
        for (int i = 0; i < partition.size(); i++) {
            for (int j = i + 1; j < partition.size(); j++) {
                edge_orbits += (ll)partition[i] * partition[j];
            }
        }
        
        ll contribution = (count * mod_pow(M, edge_orbits, P)) % P;
        answer = (answer + contribution) % P;
    }
    
    answer = (answer * mod_inverse(1, P)) % P;
    for (int i = 2; i <= N; i++) {
        answer = (answer * mod_inverse(i, P)) % P;
    }
    
    cout << answer << endl;
    return 0;
}
```

```cpp
        base = (base * base) % mod;
        exp >>= 1;
```
