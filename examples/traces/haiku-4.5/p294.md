Looking at this problem, I need to count the number of distinct circular strings of length n using alphabet {X, E}.

Two strings are considered the same if one is a circular shift of the other. This is a classic application of Burnside's lemma (also known as the Cauchy-Frobenius lemma).

**Burnside's Lemma:** The number of distinct objects under group action is:
$$\frac{1}{|G|} \sum_{g \in G} |X^g|$$

where |X^g| is the number of elements fixed by group action g.

For circular strings of length n:
- The group is the cyclic group of order n (rotations by 0, 1, 2, ..., n-1 positions)
- For a rotation by k positions, a string is fixed if it has period dividing gcd(n, k)
- If a string has period d, it's determined by its first d characters, giving 2^d fixed strings

So the formula becomes:
$$\text{Count} = \frac{1}{n} \sum_{k=0}^{n-1} 2^{\gcd(n,k)}$$

This can be rewritten using Euler's totient function:
$$\text{Count} = \frac{1}{n} \sum_{d|n} \phi(n/d) \cdot 2^d$$

Or equivalently:
$$\text{Count} = \frac{1}{n} \sum_{d|n} \phi(d) \cdot 2^{n/d}$$

Let me verify with the examples:
- n=3: divisors are {1, 3}
  - φ(1)·2³ + φ(3)·2¹ = 1·8 + 2·2 = 12
  - Result: 12/3 = 4 ✓
  
- n=4: divisors are {1, 2, 4}
  - φ(1)·2⁴ + φ(2)·2² + φ(4)·2¹ = 1·16 + 1·4 + 2·2 = 24
  - Result: 24/4 = 6 ✓

```cpp
#include <bits/stdc++.h>
using namespace std;

long long gcd(long long a, long long b) {
    while (b) {
        long long t = b;
        b = a % b;
        a = t;
    }
    return a;
}

long long phi(long long n) {
    long long result = n;
    for (long long p = 2; p * p <= n; ++p) {
        if (n % p == 0) {
            while (n % p == 0)
                n /= p;
            result -= result / p;
        }
    }
    if (n > 1)
        result -= result / n;
    return result;
}

long long power(long long base, long long exp) {
    long long result = 1;
    for (long long i = 0; i < exp; ++i) {
        result *= base;
    }
    return result;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long n;
    cin >> n;
    
    long long sum = 0;
    
    // Find all divisors of n
    for (long long d = 1; d * d <= n; ++d) {
        if (n % d == 0) {
            // d is a divisor
            sum += phi(d) * power(2, n / d);
            
            // n/d is also a divisor (if different from d)
            if (d != n / d) {
                long long d2 = n / d;
                sum += phi(d2) * power(2, n / d2);
            }
        }
    }
    
    long long result = sum / n;
    cout << result << endl;
    
    return 0;
}
```
