Looking at this problem, I need to find the minimum number of amplifiers to get from voltage 1 to voltage N.

Let me understand the operations:
- Type 1 amplifier: X → 2X - 1
- Type 2 amplifier: X → 2X + 1

Starting from voltage 1, I need to reach voltage N.

Let me think about this backwards: if I have voltage N, what could have produced it?
- If N = 2X - 1, then X = (N + 1) / 2 (must be integer)
- If N = 2X + 1, then X = (N - 1) / 2 (must be integer)

So working backwards:
- N can come from (N+1)/2 via operation 1, if (N+1) is even
- N can come from (N-1)/2 via operation 2, if (N-1) is even

This is a shortest path problem. I can use BFS working backwards from N to 1.

Let me trace through the example:
- Start at N = 5
- 5 = 2X + 1 → X = 2 (operation 2)
- 2 = 2X - 1 → X = 1.5 (not integer)
- 2 = 2X + 1 → X = 0.5 (not integer)
- Wait, let me reconsider. 2 = 2(1) - 1, so X = 1.5? No. If 2X - 1 = 2, then 2X = 3, X = 1.5 (not valid)
- If 2X + 1 = 2, then 2X = 1, X = 0.5 (not valid)

Let me re-trace the example answer: 5, with sequence "2 1"
- Start: 1
- Apply operation 2: 2(1) + 1 = 3
- Apply operation 1: 2(3) - 1 = 5 ✓

So the sequence works forward. Let me verify backward:
- Start at 5
- Reverse of op 1: if we got here via op 1, previous was (5+1)/2 = 3
- Apply backwards: 5 came from 3 via operation 1, so record "1"
- Now at 3
- Reverse of op 2: if we got here via op 2, previous was (3-1)/2 = 1
- Apply backwards: 3 came from 1 via operation 2, so record "2"
- Now at 1, done!

So the sequence backwards is "1 2", which reversed gives "2 1" ✓

I'll use BFS, working backwards from N to 1, recording the operations.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long N;
    cin >> N;
    
    if(N == 1){
        cout << 0 << "\n";
        return 0;
    }
    
    queue<pair<long long, vector<int>>> q;
    set<long long> visited;
    
    q.push({N, {}});
    visited.insert(N);
    
    while(!q.empty()){
        auto [curr, ops] = q.front();
        q.pop();
        
        if(curr == 1){
            cout << ops.size() << "\n";
            reverse(ops.begin(), ops.end());
            for(int i = 0; i < ops.size(); i++){
                if(i > 0) cout << " ";
                cout << ops[i];
            }
            cout << "\n";
            return 0;
        }
        
        // Try reverse of operation 1: curr = 2X - 1, so X = (curr + 1) / 2
        if((curr + 1) % 2 == 0){
            long long prev = (curr + 1) / 2;
            if(visited.find(prev) == visited.end()){
                visited.insert(prev);
                vector<int> newOps = ops;
                newOps.push_back(1);
                q.push({prev, newOps});
            }
        }
        
        // Try reverse of operation 2: curr = 2X + 1, so X = (curr - 1) / 2
        if((curr - 1) % 2 == 0){
            long long prev = (curr - 1) / 2;
            if(visited.find(prev) == visited.end()){
                visited.insert(prev);
                vector<int> newOps = ops;
                newOps.push_back(2);
                q.push({prev, newOps});
            }
        }
    }
    
    cout << "No solution\n";
    return 0;
}
```
