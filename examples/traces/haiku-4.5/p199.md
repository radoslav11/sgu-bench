Looking at this problem, I need to find the maximum independent set in a "hate" graph where person i hates person j if:
- (S_i ≤ S_j AND B_i ≥ B_j) OR (S_i ≥ S_j AND B_i ≤ B_j)

Key insight: Two people DON'T hate each other if:
- (S_i < S_j AND B_i < B_j) OR (S_i > S_j AND B_i > B_j)

In other words, they don't hate each other if one dominates the other in BOTH attributes (strictly).

This means if we sort by strength and look at beauties, we want people where:
- Either strength is strictly increasing AND beauty is strictly increasing, OR
- Strength is strictly increasing AND beauty is strictly decreasing

Let me think differently: if we sort by one attribute (say strength), then two people don't hate if their beauties are either both increasing or both decreasing in a certain pattern.

Actually, a simpler approach: sort by strength. Then we need to find the longest subsequence where beauties are either strictly increasing or strictly decreasing. This is because if beauties are strictly monotonic in a sorted strength sequence, no two people hate each other.

Wait, let me verify with the example:
- Person 1: S=1, B=1
- Person 2: S=1, B=2
- Person 3: S=2, B=1
- Person 4: S=2, B=2

Does person 1 hate person 4? S_1=1≤S_4=2 and B_1=1≤B_4=2 (not ≥), so NO by first condition. S_1=1≤S_4=2 (not ≥), so NO by second condition. They don't hate!

Does person 1 hate person 2? S_1=1≤S_2=1 and B_1=1≤B_2=2 (not ≥), so NO. They don't hate.

Does person 1 hate person 3? S_1=1≤S_3=2 and B_1=1≤B_3=1 (not ≥), so NO. They don't hate.

Does person 2 hate person 3? S_2=1≤S_3=2 and B_2=2≥B_3=1 - YES! They hate.
Or S_2=1≤S_3=2 (not ≥). So yes, they hate by first condition.

Does person 2 hate person 4? S_2=1≤S_4=2 and B_2=2≥B_4=2 - YES! They hate.

Does person 3 hate person 4? S_3=2≤S_4=2 and B_3=1≤B_4=2 (not ≥), so NO. They don't hate.

So answer can be {1,4} or {1,3} or {1,2,3} etc. But {1,4} has size 2, {1,3} has size 2, {1,2,3} - does 2 hate 3? Yes. So max is 2.

The strategy: sort by strength, then find longest increasing OR longest decreasing subsequence of beauties.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<tuple<long long, long long, int>> people(n);
    for(int i = 0; i < n; i++){
        long long s, b;
        cin >> s >> b;
        people[i] = {s, b, i+1};
    }
    
    sort(people.begin(), people.end());
    
    // Find longest increasing subsequence of beauties
    vector<int> lis_indices;
    vector<long long> lis_vals;
    for(int i = 0; i < n; i++){
        long long b = get<1>(people[i]);
        auto it = lower_bound(lis_vals.begin(), lis_vals.end(), b);
        if(it == lis_vals.end()){
            lis_vals.push_back(b);
            lis_indices.push_back(i);
        } else {
            *it = b;
            lis_indices[it - lis_vals.begin()] = i;
        }
    }
    
    // Find longest decreasing subsequence of beauties
    vector<int> lds_indices;
    vector<long long> lds_vals;
    for(int i = 0; i < n; i++){
        long long b = get<1>(people[i]);
        auto it = lower_bound(lds_vals.begin(), lds_vals.end(), -b);
        if(it == lds_vals.end()){
            lds_vals.push_back(-b);
            lds_indices.push_back(i);
        } else {
            *it = -b;
            lds_indices[it - lds_vals.begin()] = i;
        }
    }
    
    vector<int> result;
    if(lis_vals.size() >= lds_vals.size()){
        for(int idx : lis_indices) result.push_back(get<2>(people[idx]));
    } else {
        for(int idx : lds_indices) result.push_back(get<2>(people[idx]));
    }
    
    cout << result.size() << "\n";
    for(int x : result) cout << x << " ";
    cout << "\n";
    
    return 0;
}
```
