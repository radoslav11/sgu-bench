Looking at this problem, I need to:

1. Seat n groups of friends at tables with r seats each
2. No person from a group can sit alone (each person must have at least one other person from their group at their table)
3. Minimize the number of tables used

Key insight: For each group of size `a`, I need at least `ceil(a/r)` tables if they sit alone. But groups can share tables as long as each person has company from their own group.

Strategy:
- For each group, I need to split them into sub-groups such that each sub-group has at least 2 people
- I want to pack these sub-groups into tables of size r as efficiently as possible
- This is essentially a bin packing problem

Greedy approach:
1. For each group of size `a`, the minimum number of "people units" we need to seat is `a` (we have to seat all of them)
2. We need to ensure each group is divided into chunks of at least 2 people
3. Greedily pack: for each group, try to fit remainders into existing tables

Let me think differently:
- For each group of size `a`, if we use `k` tables, we split `a` people into `k` parts where each part is at least 2
- The minimum is achieved when we minimize total tables
- For a group of size `a`, we need at least `ceil(a/r)` tables if `a` is divisible well, but we must ensure no person sits alone

Better approach - greedy bin packing:
1. For each group, figure out how many people go to each table
2. Use a greedy approach: try to fill tables as much as possible
3. Keep track of remaining space in the current table

Let me implement a greedy solution:
- Process groups one by one
- For each group, try to place people at tables (each at least in pairs)
- Keep a list of tables with remaining capacity
- For each group of size `a`, split it optimally and try to fit into existing tables

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, r;
    cin >> n >> r;
    
    vector<int> groups(n);
    for(int i = 0; i < n; i++){
        cin >> groups[i];
    }
    
    // Use a multiset to track remaining spaces in tables
    // We'll use negative values so that larger spaces come first (for greedy packing)
    multiset<int> tables;
    
    for(int i = 0; i < n; i++){
        int remaining = groups[i];
        
        // Try to fit into existing tables first
        while(remaining > 0){
            auto it = tables.rbegin();
            
            if(it != tables.rend() && *it >= 2){
                // We can fit at least 2 people in the largest remaining space
                int space = *it;
                tables.erase(prev(tables.end()));
                
                int take = min(remaining, space);
                remaining -= take;
                space -= take;
                
                if(space > 0){
                    tables.insert(space);
                }
            } else {
                // Need a new table
                int take = min(remaining, r);
                remaining -= take;
                int space = r - take;
                
                if(space > 0){
                    tables.insert(space);
                }
            }
        }
    }
    
    cout << tables.size() << endl;
    
    return 0;
}
```

Wait, this approach has an issue. Let me reconsider.

When we try to place people from a group, we need to ensure each person has company from their group. So if we have a group of 5 people and table size 4:
- We could put 2 at one table, 3 at another
- Or 4 at one table, 1... wait, that's invalid (the last person is alone)

The greedy approach should be:
1. For each group of size `a`, try to distribute them across tables
2. When placing a group, fill existing tables with remaining capacity (at least 2 at a time)
3. Whatever is left needs new tables

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, r;
    cin >> n >> r;
    
    vector<int> groups(n);
    for(int i = 0; i < n; i++){
        cin >> groups[i];
    }
    
    multiset<int> tables;
    
    for(int i = 0; i < n; i++){
        int remaining = groups[i];
        vector<int> toInsert;
        
        while(remaining > 0){
            if(tables.empty()){
                int take = min(remaining, r);
                remaining -= take;
                toInsert.push_back(r - take);
            } else {
                auto it = prev(tables.end());
                int space = *it;
                tables.erase(it);
                
                if(space >= 2 && remaining >= 2){
                    int take = min(remaining, space);
                    remaining -= take;
                    if(space - take > 0) toInsert.push_back(space - take);
                } else if(space >= remaining && remaining >= 2){
                    int take = remaining;
                    remaining = 0;
                    if(space - take > 0) toInsert.push_back(space - take);
                } else {
                    toInsert.push_back(space);
                }
            }
        }
        
        for(int x : toInsert) tables.insert(x);
    }
    
    cout << tables.size() << endl;
    
    return 0;
}
```
