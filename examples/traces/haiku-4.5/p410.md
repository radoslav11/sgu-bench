Looking at this problem, I need to:

1. We have N planets with mistkafers and N pferds
2. Each day, either:
   - Pferds fly and take 1 mistkafer from each planet (need all planets to have at least 1)
   - Scientists clone mistkafers on one planet (double them)
3. Find minimum days to eliminate all mistkafers

The strategy should be:
- We need to reduce all planets to 0
- Each flying mission reduces each planet by 1
- If any planet has 0, we must clone somewhere first
- We want to minimize total days

Key insight: 
- If we have planets with counts [a₁, a₂, ..., aₙ], we need at least max(aᵢ) flying missions
- But if counts are unequal, some planets will reach 0 before others
- When a planet reaches 0, we need to clone it (double another planet or the same one)
- We should greedily clone the planet with the minimum count to balance them

Algorithm:
1. Simulate day by day
2. On each day, check if minimum count is 0
   - If yes, clone the planet with minimum count (double it)
   - If no, perform flying mission (subtract 1 from all)
3. Continue until all planets have 0

Let me trace through example 1:
- Start: [1, 2]
- Day 1: min=1 (not 0), but we want to check - actually min IS 1>0, so we can fly. But output says science mission first.
  
Let me reconsider: We should clone when the minimum is the bottleneck. Actually, looking at the example more carefully:
- [1, 2]: min=1, max=2. We need at least 2 flying missions.
- Day 1: Clone planet 1 → [2, 2]
- Day 2: Flying mission → [1, 1]
- Day 3: Flying mission → [0, 0]

So the strategy seems to be: we want to balance things by cloning the minimum when needed. We should perform a flying mission when all planets have the same positive count, otherwise clone the one with minimum.

Actually, simpler: keep doing flying missions when all have count > 0. When we can't (some have 0), we need to clone the minimum non-zero (or a zero). We clone the planet with the current minimum.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<long long> a(n);
    for(int i = 0; i < n; i++){
        cin >> a[i];
    }
    
    vector<string> operations;
    
    while(true){
        // Check if all are 0
        bool all_zero = true;
        for(int i = 0; i < n; i++){
            if(a[i] > 0){
                all_zero = false;
                break;
            }
        }
        if(all_zero) break;
        
        // Check if any is 0
        bool has_zero = false;
        for(int i = 0; i < n; i++){
            if(a[i] == 0){
                has_zero = true;
                break;
            }
        }
        
        if(has_zero){
            // Clone the planet with minimum
            int min_idx = 0;
            for(int i = 1; i < n; i++){
                if(a[i] < a[min_idx]){
                    min_idx = i;
                }
            }
            a[min_idx] *= 2;
            operations.push_back("science mission to the planet " + to_string(min_idx + 1));
        } else {
            // Flying mission
            for(int i = 0; i < n; i++){
                a[i]--;
            }
            operations.push_back("flying mission");
        }
    }
    
    cout << operations.size() << "\n";
    if(operations.size() <= 1000){
        for(const auto& op : operations){
            cout << op << "\n";
        }
    }
    
    return 0;
}
```
