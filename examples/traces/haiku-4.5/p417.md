Looking at this problem, I need to calculate the mass of a disc centered at (x0, y0) with radius r, where the density function is ρ(x,y) = ln(x² + y²).

The mass is given by the double integral:
M = ∫∫_D ln(x² + y²) dA

where D is the disc region.

It's easier to work in polar coordinates centered at (x0, y0). Let me use:
- x = x0 + s·cos(θ)
- y = y0 + s·sin(θ)

where s ranges from 0 to r, and θ ranges from 0 to 2π.

Then:
- x² + y² = (x0 + s·cos(θ))² + (y0 + s·sin(θ))²
- x² + y² = x0² + y0² + 2s(x0·cos(θ) + y0·sin(θ)) + s²

The Jacobian for polar coordinates gives us dA = s·ds·dθ

So:
M = ∫₀^(2π) ∫₀^r ln(x0² + y0² + 2s(x0·cos(θ) + y0·sin(θ)) + s²) · s ds dθ

Since the origin is strictly outside the disc, x0² + y0² > r², which ensures the argument of ln is always positive.

I'll use numerical integration (Gaussian quadrature or simple numerical methods) to compute this double integral.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    double x0, y0, r;
    cin >> x0 >> y0 >> r;
    
    // Numerical integration using Gaussian quadrature
    // We'll use a simple but accurate approach with adaptive or fine grid
    
    double mass = 0.0;
    int n_theta = 1000;
    int n_s = 1000;
    
    double d_theta = 2.0 * M_PI / n_theta;
    double d_s = r / n_s;
    
    for(int i = 0; i < n_theta; i++){
        double theta = (i + 0.5) * d_theta;
        double cos_theta = cos(theta);
        double sin_theta = sin(theta);
        
        for(int j = 0; j < n_s; j++){
            double s = (j + 0.5) * d_s;
            
            // Calculate x^2 + y^2
            double x = x0 + s * cos_theta;
            double y = y0 + s * sin_theta;
            double dist_sq = x * x + y * y;
            
            // Density is ln(x^2 + y^2)
            double density = log(dist_sq);
            
            // Integrate: density * s * ds * dtheta
            mass += density * s * d_s * d_theta;
        }
    }
    
    cout << fixed << setprecision(12) << mass << endl;
    
    return 0;
}
```

The solution:
1. Uses polar coordinates centered at (x0, y0)
2. Converts the integral to: ∫₀^(2π) ∫₀^r ln((x0 + s·cos(θ))² + (y0 + s·sin(θ))²) · s ds dθ
3. Performs numerical integration using a rectangular grid with midpoint rule
4. Uses 1000×1000 grid points which provides sufficient accuracy for the required precision
5. The midpoint rule is used: sample at (i+0.5)·Δ and (j+0.5)·Δ to get better accuracy
