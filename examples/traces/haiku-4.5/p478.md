Looking at this problem, I need to track:

1. Initial state: `a` boys (away), `b` girls (with teacher)
2. At each checkpoint, the teacher counts total students with him
3. Before each checkpoint, some boys can join (return) and some girls can leave
4. I need to find a valid sequence of joins/leaves that matches the observed counts

Let me denote:
- `boys_with_teacher` = number of boys currently with the teacher
- `girls_with_teacher` = number of girls currently with the teacher

Initial state: `boys_with_teacher = 0`, `girls_with_teacher = b`

For each observation:
- We need `boys_with_teacher + girls_with_teacher = observed_count`
- We can only increase `boys_with_teacher` (boys returning, up to `a` total)
- We can only decrease `girls_with_teacher` (girls leaving, down to 0 total)

Let me trace through the first example:
- Initial: 0 boys, 3 girls with teacher (total = 3)
- Observation 1: total = 2
  - Need to lose 1 person, so 1 girl leaves: boys_join=1, girls_leave=1, now 1 boy + 2 girls = 3... wait that's 3 not 2
  - Let me reconsider: if we have 1 boy join and 2 girls leave: 1 + 1 = 2 ✓
- Observation 2: total = 3
  - Currently have 1 boy, 1 girl (total 2)
  - Need total 3, so 1 more boy joins: 2 + 1 = 3 ✓

So output would be:
- Before check 1: 1 boy joins, 2 girls leave → (1, 2)
- Before check 2: 1 boy joins, 0 girls leave → (1, 0)

Actually rereading: "i-th line should contain two non-negative integers, denoting the number of schoolboys joined and the number of schoolgirls separated from the teacher respectively exactly before he recalculated students at i-th time."

So for each checkpoint, we output the boys who joined and girls who left right before that count.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    int a, b, n;
    cin >> a >> b >> n;
    
    vector<int> counts(n);
    for(int i = 0; i < n; i++){
        cin >> counts[i];
    }
    
    int boys_with_teacher = 0;
    int girls_with_teacher = b;
    
    vector<pair<int,int>> result;
    
    for(int i = 0; i < n; i++){
        int target = counts[i];
        int current_total = boys_with_teacher + girls_with_teacher;
        
        // We need to reach target
        int needed = target - current_total;
        
        // Try to find valid join/leave combination
        bool found = false;
        for(int boys_join = 0; boys_join <= a - boys_with_teacher && boys_join <= needed + girls_with_teacher; boys_join++){
            int girls_leave = boys_join - needed;
            
            if(girls_leave >= 0 && girls_leave <= girls_with_teacher){
                // Valid move
                boys_with_teacher += boys_join;
                girls_with_teacher -= girls_leave;
                result.push_back({boys_join, girls_leave});
                found = true;
                break;
            }
        }
        
        if(!found){
            cout << "ERROR\n";
            return 0;
        }
    }
    
    for(auto& p : result){
        cout << p.first << " " << p.second << "\n";
    }
    
    return 0;
}
```
