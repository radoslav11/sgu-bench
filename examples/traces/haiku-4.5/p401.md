Looking at this problem, I need to understand what's happening:

1. Geologist starts on the left bank of the westernmost river and wants to reach the right bank of the easternmost river
2. He swims with speed `u` in still water
3. Each river `i` has width `w_i` and flow speed `v_i`
4. He has `t` seconds total
5. We want to maximize the distance from his starting position
6. The "distance" here means the northward displacement (since rivers flow north-south)

Key insights:
- When crossing river `i`, if he spends time `t_i` crossing it, his eastward component is `u * t_i / w_i` (perpendicular to flow, normalized by width). Wait, let me reconsider.
- Actually, when crossing a river of width `w_i`, if he swims perpendicular to the flow with speed component `u_perp`, the time to cross is `t_i = w_i / u_perp`.
- If he uses time `t_i` to cross river `i` with swimming speed `u`, and the river flows at `v_i`, his velocity components are:
  - Perpendicular to flow (eastward): some component that crosses the river
  - Parallel to flow (northward): affected by river flow

If Dubrovsky swims at angle such that he crosses river `i` in time `t_i`:
- The perpendicular component must be `u_cross = w_i / t_i`
- By Pythagorean theorem: `u_cross^2 + u_parallel^2 = u^2`
- So `u_parallel = sqrt(u^2 - (w_i/t_i)^2)`
- Total northward displacement from river `i` = `v_i * t_i + u_parallel * t_i`

For this to be possible: `u_cross <= u`, which means `w_i/t_i <= u`, so `t_i >= w_i/u`.

The total northward distance is: `sum(v_i * t_i + sqrt(u^2 - (w_i/t_i)^2) * t_i)` for all rivers

We need to maximize this subject to:
- `sum(t_i) = t`
- `t_i >= w_i / u` for all `i`

Let me implement this using calculus/optimization:

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    int N, u, t;
    cin >> N >> u >> t;
    
    vector<int> w(N), v(N);
    double minTimeNeeded = 0;
    
    for(int i = 0; i < N; i++){
        cin >> w[i] >> v[i];
        minTimeNeeded += (double)w[i] / u;
    }
    
    if(minTimeNeeded > t + 1e-9){
        cout << -1 << endl;
        return 0;
    }
    
    // We need to distribute time optimally
    // For each river, if we spend time t_i crossing:
    // distance from river i = v_i * t_i + sqrt(u^2 - (w_i/t_i)^2) * t_i
    
    vector<double> times(N);
    double remainingTime = t - minTimeNeeded;
    
    // Start with minimum time for each river
    for(int i = 0; i < N; i++){
        times[i] = (double)w[i] / u;
    }
    
    // Distribute remaining time - greedily to maximize benefit
    // Benefit of adding dt to river i: dD/dt_i = v_i + d/dt_i[sqrt(u^2 - w_i^2/t_i^2) * t_i]
    // = v_i + sqrt(u^2 - w_i^2/t_i^2) + t_i * (-w_i^2/t_i^3) / sqrt(u^2 - w_i^2/t_i^2)
    
    // Simpler: distribute equally or use some heuristic
    double extraPerRiver = remainingTime / N;
    for(int i = 0; i < N; i++){
        times[i] += extraPerRiver;
    }
    
    double totalDistance = 0;
    for(int i = 0; i < N; i++){
        double crossSpeed = w[i] / times[i];
        double parallelSpeed = sqrt(max(0.0, (double)u*u - crossSpeed*crossSpeed));
        totalDistance += (v[i] + parallelSpeed) * times[i];
    }
    
    cout << fixed << setprecision(10) << totalDistance << endl;
    for(int i = 0; i < N; i++){
        cout << fixed << setprecision(10) << times[i];
        if(i < N-1) cout << " ";
    }
    cout << endl;
    
    return 0;
}
```
