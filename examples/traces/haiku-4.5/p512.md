Looking at this problem, I need to find pairs of "friend" points, where two points are friends if they can be contained in a rectangle (with sides parallel to axes) that doesn't contain any other points.

Let me think about this step by step:

1. Two points P1 and P2 are friends if there exists a rectangle with sides parallel to axes that:
   - Contains both P1 and P2
   - Contains no other points from the set

2. The axis-aligned rectangle containing two points has corners at (min_x, min_y) and (max_x, max_y) where these coordinates come from the two points.

3. Two points are friends if NO other point lies strictly within the rectangle bounded by them (or on its boundary, since the problem says "contains a point if the point lies within or on border").

Wait, let me reconsider. If a rectangle contains P1 and P2 on its boundary, then any third point inside or on the boundary would violate the condition. So P1 and P2 are friends if the minimal axis-aligned rectangle with P1 and P2 on its boundary contains no other points.

Algorithm:
- For each pair of points (i, j):
  - Find the bounding rectangle with P1 at one corner and P2 at the opposite corner
  - Check if any other point lies inside or on the boundary of this rectangle
  - If no other point is found, increment the friend count

Let me verify with the example:
- Points: (0,0), (0,2), (2,0), (2,2), (1,1)
- (0,0) and (0,2): rect bounds x:[0,0], y:[0,2]. Points (1,1) has x=1 which is not in [0,0], so it's outside. No other points. → friends
- (0,0) and (2,0): rect bounds x:[0,2], y:[0,0]. Point (1,1) has y=1 which is not in [0,0]. → friends
- (0,0) and (2,2): rect bounds x:[0,2], y:[0,2]. Point (1,1) is inside! → not friends
- (0,0) and (1,1): rect bounds x:[0,1], y:[0,1]. Check others - none are in this rect → friends
- (0,2) and (2,0): rect bounds x:[0,2], y:[0,2]. Point (1,1) is inside! → not friends
- (0,2) and (2,2): rect bounds x:[0,2], y:[2,2]. Point (1,1) has y=1 not in [2,2] → friends
- (0,2) and (1,1): rect bounds x:[0,1], y:[1,2]. Check others - (0,0) has y=0 not in [1,2], (2,0) has x=2 not in [0,1], (2,2) has x=2 not in [0,1] → friends
- (2,0) and (2,2): rect bounds x:[2,2], y:[0,2]. Point (1,1) has x=1 not in [2,2] → friends
- (2,0) and (1,1): rect bounds x:[1,2], y:[0,1]. Check others - none in this rect → friends
- (2,2) and (1,1): rect bounds x:[1,2], y:[1,2]. Check others - none in this rect → friends

That gives us: (0,0)-(0,2), (0,0)-(2,0), (0,0)-(1,1), (0,2)-(2,2), (0,2)-(1,1), (2,0)-(2,2), (2,0)-(1,1), (2,2)-(1,1) = 8 pairs. Perfect!

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<pair<long long, long long>> points(n);
    for(int i = 0; i < n; i++){
        cin >> points[i].first >> points[i].second;
    }
    
    int friendCount = 0;
    
    // Check all pairs
    for(int i = 0; i < n; i++){
        for(int j = i + 1; j < n; j++){
            long long x1 = points[i].first;
            long long y1 = points[i].second;
            long long x2 = points[j].first;
            long long y2 = points[j].second;
            
            long long minX = min(x1, x2);
            long long maxX = max(x1, x2);
            long long minY = min(y1, y2);
            long long maxY = max(y1, y2);
            
            // Check if any other point is in this rectangle
            bool hasForeignPoint = false;
            for(int k = 0; k < n; k++){
                if(k == i || k == j) continue;
                
                long long xk = points[k].first;
                long long yk = points[k].second;
                
                if(xk >= minX && xk <= maxX && yk >= minY && yk <= maxY){
                    hasForeignPoint = true;
                    break;
                }
            }
            
            if(!hasForeignPoint){
                friendCount++;
            }
        }
    }
    
    cout << friendCount << endl;
    
    return 0;
}
```
