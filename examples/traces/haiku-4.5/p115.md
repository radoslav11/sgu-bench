Looking at this problem, I need to:

1. Check if a given date (day N, month M) is valid in 2001
2. If valid, find what day of the week it is (1=Monday, 7=Sunday)
3. If invalid, output "Impossible"

Key facts:
- The year is 2001
- 2001 is not a leap year (not divisible by 4... wait, 2001/4 = 500.25, so not a leap year)
- I need to know the starting day of 2001 to calculate other days
- January 1, 2001 was a Monday

My approach:
1. Validate the date (check days in each month)
2. Calculate the number of days from January 1, 2001 to the given date
3. Use modulo 7 to find the day of the week

Let me verify: January 1, 2001 was a Monday (day 1).

Days in each month of 2001:
- January: 31
- February: 28 (2001 is not a leap year)
- March: 31
- April: 30
- May: 31
- June: 30
- July: 31
- August: 31
- September: 30
- October: 31
- November: 30
- December: 31

For the sample: October 21, 2001
- Days from Jan 1 to Oct 21 = 31+28+31+30+31+30+31+31+30+21 = 294 days
- But we need to count from Jan 1 (day 1 is Monday)
- So Oct 21 is 293 days after Jan 1 (not including Jan 1 itself)
- 293 % 7 = 293 - 41*7 = 293 - 287 = 6
- So it's 6 days after Monday = Sunday = day 7 ✓

Let me recalculate more carefully:
- Jan 1 = day 0 (from start), which is Monday (1)
- Number of days elapsed = total_days - 1
- (0 + (total_days - 1)) % 7 gives us the offset from Monday

Actually simpler: if Jan 1 is day 1 (Monday), then:
- Days from Jan 1 to date = 1 + days_elapsed
- (days_elapsed) % 7 tells us offset from Jan 1's day
- day_of_week = 1 + (days_elapsed % 7), but we need to handle the modulo carefully

Let me think differently:
- Total days from Jan 1 to the given date (inclusive of both)
- (total_days - 1) % 7 gives offset, then add 1 for Monday-based indexing

For Oct 21: days = 31+28+31+30+31+30+31+31+30+21 = 294
- (294 - 1) % 7 = 293 % 7 = 6, so day = 6 + 1 = 7 ✓

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
  int daysInMonth[] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
  
  int n, m;
  cin >> n >> m;
  
  // Validate month
  if(m < 1 || m > 12){
    cout << "Impossible" << endl;
    return 0;
  }
  
  // Validate day
  if(n < 1 || n > daysInMonth[m]){
    cout << "Impossible" << endl;
    return 0;
  }
  
  // Calculate total days from Jan 1
  int totalDays = 0;
  for(int i = 1; i < m; i++){
    totalDays += daysInMonth[i];
  }
  totalDays += n;
  
  // Calculate day of week (Jan 1 is Monday = 1)
  int dayOfWeek = ((totalDays - 1) % 7) + 1;
  
  cout << dayOfWeek << endl;
  
  return 0;
}
```
