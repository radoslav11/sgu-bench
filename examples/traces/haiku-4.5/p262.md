Looking at this problem, I need to:

1. Read K symbols, each represented as an N×M grid of pixels
2. Find the minimum set of pixels such that each symbol has a unique "signature" when looking only at those pixels
3. Output the count and the pixel positions

This is essentially a set cover variant where I need to find the smallest subset of pixel positions that can distinguish between all pairs of symbols.

My approach:
- Try all possible subsets of pixels (there are at most 10×10 = 100 pixels, but we need an efficient solution)
- For each subset, check if it can distinguish all K symbols
- Two symbols are distinguished if they differ in at least one pixel in the subset
- Find the subset with minimum size

Since we have at most 100 pixels and K ≤ 6 symbols, I can use bitmask enumeration over pixels and check each one.

Let me think about the checking: for a given set of pixels, I need to verify that no two symbols have the same signature (same values in those pixels).

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, M, K;
    cin >> N >> M >> K;
    
    vector<string> symbols[K];
    
    // Read all symbols
    for(int i = 0; i < K; i++){
        for(int j = 0; j < N; j++){
            string row;
            cin >> row;
            symbols[i].push_back(row);
        }
    }
    
    int totalPixels = N * M;
    int minPixels = totalPixels + 1;
    long long bestMask = 0;
    
    // Try all possible subsets of pixels
    for(long long mask = 1; mask < (1LL << totalPixels); mask++){
        // Check if this mask can distinguish all symbols
        set<vector<int>> signatures;
        bool valid = true;
        
        for(int s = 0; s < K; s++){
            vector<int> signature;
            for(int p = 0; p < totalPixels; p++){
                if(mask & (1LL << p)){
                    int row = p / M;
                    int col = p % M;
                    signature.push_back(symbols[s][row][col] - '0');
                }
            }
            if(signatures.count(signature)){
                valid = false;
                break;
            }
            signatures.insert(signature);
        }
        
        if(valid){
            int pixelCount = __builtin_popcountll(mask);
            if(pixelCount < minPixels){
                minPixels = pixelCount;
                bestMask = mask;
            }
        }
    }
    
    cout << minPixels << "\n";
    
    // Output the grid
    for(int row = 0; row < N; row++){
        for(int col = 0; col < M; col++){
            int pixelIndex = row * M + col;
            if(bestMask & (1LL << pixelIndex)){
                cout << "1";
            } else {
                cout << "0";
            }
        }
        cout << "\n";
    }
    
    return 0;
}
```

The algorithm:
1. Read all K symbols as N×M grids
2. Iterate through all possible subsets of pixels using bitmask (from 1 to 2^(N*M)-1)
3. For each subset, create a signature for each symbol (the sequence of pixel values in the chosen pixels)
4. Check if all K signatures are unique - if yes, this subset can distinguish all symbols
5. Track the smallest valid subset
6. Output the size and the grid representation of the best subset

Time complexity: O(2^(N*M) × K × N×M) which should be acceptable given N,M ≤ 10 and K ≤ 6.
