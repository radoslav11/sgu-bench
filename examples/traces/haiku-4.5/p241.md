Looking at this problem, I need to:

1. Count ways to place K non-attacking rooks on the union of two chessboards
2. Two rooks attack each other if they're in the same row or column
3. The boards are NxN and MxM, separated by horizontal distance W and vertical distance H

Key insight: Since rooks attack along rows and columns, I need to count how many rows and columns are available, considering overlaps.

Let me think about the geometry:
- Board 1: NxN (left-bottom at origin, covers columns 0 to N-1, rows 0 to N-1)
- Board 2: MxM (left-bottom at (W, H), covers columns W to W+M-1, rows H to H+M-1)

The union of two boards can have:
- Distinct columns: some may overlap
- Distinct rows: some may overlap

For K non-attacking rooks, I need to:
1. Choose K columns from all available columns
2. Choose K rows from all available rows  
3. Arrange K rooks in the selected rows/columns such that they're all on the union of boards

The number of distinct columns is at most N + M (if no overlap) but could be less.
The number of distinct rows is at most N + M (if no overlap) but could be less.

For overlap calculation:
- Column overlap: if W < N, there are min(N-W, M) overlapping columns
- Row overlap: if H < N, there are min(N-H, M) overlapping rows

Total distinct columns = N + M - (overlap count)
Total distinct rows = N + M - (overlap count)

Once we have C distinct columns and R distinct rows:
- We need to choose K columns from C: C(C, K)
- We need to choose K rows from R: C(R, K)
- We need to place K rooks in a K×K grid: K!

So answer = C(C,K) × C(R,K) × K!

But we need to be careful: the selected rows and columns must have at least one cell in the union of boards. This is complex when boards don't completely overlap.

Let me reconsider: If the two boards don't overlap in a specific row or column, we can still place rooks. The key is that we can place rooks anywhere on the union of the two boards.

Actually, simpler approach: 
- Count total distinct rows: if H + M > N, then N + M - max(0, H + M - N - N) = ...
  Actually: distinct_rows = max(N, H + M) if boards overlap, else N + M
  
Let me recalculate:
- Rows occupied: [0, N) by board 1, [H, H+M) by board 2
- Distinct rows: from min(0,H) to max(N-1, H+M-1)
- If no overlap (H >= N): total_rows = N + M
- If complete overlap (H+M <= N): total_rows = N
- If partial: total_rows = N + M - overlap, where overlap = max(0, min(N, H+M) - max(0, H))

#include<bits/stdc++.h>
using namespace std;

long long fact[45];
long long comb[45][45];

void precompute() {
    for(int i = 0; i <= 40; i++) {
        fact[i] = 1;
        for(int j = 1; j <= i; j++) fact[i] *= j;
    }
    for(int i = 0; i <= 40; i++) {
        for(int j = 0; j <= i; j++) {
            if(j == 0 || j == i) comb[i][j] = 1;
            else comb[i][j] = comb[i-1][j-1] + comb[i-1][j];
        }
    }
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    precompute();
    
    long long N, M, W, H, K;
    cin >> N >> M >> W >> H >> K;
    
    if(K > 40 || K > N + M) {
        cout << 0 << endl;
        return 0;
    }
    
    // Calculate distinct columns
    long long col_min = 0, col_max = max((long long)(N-1), W + M - 1);
    long long cols = col_max - col_min + 1;
    
    // Calculate distinct rows  
    long long row_min = 0, row_max = max((long long)(N-1), H + M - 1);
    long long rows = row_max - row_min + 1;
    
    if(K > cols || K > rows) {
        cout << 0 << endl;
        return 0;
    }
    
    cols = min(cols, (long long)40);
    rows = min(rows, (long long)40);
    
    long long result = comb[cols][K] * comb[rows][K] % 1000000000;
    result = result * fact[K] % 1000000000;
    
    cout << result << endl;
    
    return 0;
}
