Looking at this problem, I need to find the fixed monthly payment `x` for an annuity payment scheme.

Let me break down the problem:
1. Initial debt: `s`
2. Number of months: `m`
3. Interest rate: `p` percent per month
4. Fixed monthly payment: `x` (what we need to find)

Each month:
- Interest payment `a_i` = (remaining debt) × p / 100
- Principal payment `b_i` = x - a_i
- New remaining debt = old debt - b_i

Let me trace through the example:
- s=100, m=2, p=50
- Month 1: a₁ = 100 × 50/100 = 50, b₁ = x - 50, remaining = 100 - (x-50) = 150 - x
- Month 2: a₂ = (150-x) × 50/100 = 75 - 0.5x, b₂ = x - (75 - 0.5x) = 1.5x - 75
- After month 2: remaining = (150-x) - (1.5x-75) = 225 - 2.5x

For complete payoff: 225 - 2.5x = 0, so x = 90 ✓

I can derive a formula. Let r = p/100.

After month i, the remaining debt follows a pattern. If we denote the debt after month i as d_i:
- d_0 = s
- d_i = d_{i-1}(1+r) - x

This is a geometric sequence. We can solve it:
- d_i = s(1+r)^i - x((1+r)^i - 1)/r

For complete payoff at month m: d_m = 0
- s(1+r)^m - x((1+r)^m - 1)/r = 0
- s(1+r)^m = x((1+r)^m - 1)/r
- x = s·r·(1+r)^m / ((1+r)^m - 1)

Special case: when p = 0 (r = 0), we have x = s/m (simple division)

Let me verify with the example:
- s=100, m=2, p=50, so r=0.5
- (1+r)^m = 1.5^2 = 2.25
- x = 100 × 0.5 × 2.25 / (2.25 - 1) = 100 × 0.5 × 2.25 / 1.25 = 112.5 / 1.25 = 90 ✓

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    double s, m, p;
    cin >> s >> m >> p;
    
    double r = p / 100.0;
    double x;
    
    if (r == 0) {
        x = s / m;
    } else {
        double factor = pow(1 + r, m);
        x = s * r * factor / (factor - 1);
    }
    
    cout << fixed << setprecision(5) << x << endl;
    
    return 0;
}
```
