Looking at this problem, I need to:

1. Schedule chess games where:
   - Game 1: Two players play
   - Game i (i>1): Winner of game i-1 plays someone else
   - Each player plays a certain number of games

2. Given the number of games each player played, I need to output a valid schedule.

Let me think about the approach:
- The total number of games G = sum of all games / 2 (since each game involves 2 players)
- Actually, looking at the example: games are [2,4,1,5], sum = 12, and output has 6 games. So G = sum/2.
- I need to greedily schedule games where:
  - The winner of the previous game plays next
  - I pick an opponent who still needs to play games
  - After the last game, someone is the final winner

Key insight: 
- The player who won the last game has 1 more "appearance" than losses
- All other players have equal wins and losses
- If we sum up all games, the total = 2 * number_of_games
- We need to schedule so the win/loss counts work out

Strategy:
1. Start with any two players
2. Keep track of remaining games each player needs to play
3. For each subsequent game, the winner of the previous game plays someone who still needs games
4. Use a greedy approach: pick opponents smartly

Let me implement a solution using backtracking/greedy approach:

```cpp
#include<bits/stdc++.h>
using std::vector;
using std::pair;

int n;
vector<int> games;
vector<pair<int,int>> schedule;
vector<int> remaining;

bool solve(int lastWinner) {
    // Check if we're done
    bool allZero = true;
    for(int i = 0; i < n; i++) {
        if(remaining[i] > 0) {
            allZero = false;
            break;
        }
    }
    if(allZero) return true;
    
    // Try pairing lastWinner with each other player who still has games
    for(int i = 0; i < n; i++) {
        if(i != lastWinner && remaining[i] > 0) {
            // Try lastWinner wins
            remaining[lastWinner]--;
            remaining[i]--;
            schedule.push_back({lastWinner + 1, i + 1});
            
            if(solve(lastWinner)) return true;
            
            schedule.pop_back();
            remaining[lastWinner]++;
            remaining[i]++;
            
            // Try i wins
            remaining[lastWinner]--;
            remaining[i]--;
            schedule.push_back({i + 1, lastWinner + 1});
            
            if(solve(i)) return true;
            
            schedule.pop_back();
            remaining[lastWinner]++;
            remaining[i]++;
        }
    }
    
    return false;
}

int main(){
    std::cin >> n;
    games.resize(n);
    remaining.resize(n);
    
    int total = 0;
    for(int i = 0; i < n; i++) {
        std::cin >> games[i];
        remaining[i] = games[i];
        total += games[i];
    }
    
    // Try starting with each pair
    for(int i = 0; i < n && schedule.empty(); i++) {
        for(int j = i + 1; j < n && schedule.empty(); j++) {
            remaining = games;
            schedule.clear();
            
            // Try i wins first game
            remaining[i]--;
            remaining[j]--;
            schedule.push_back({i + 1, j + 1});
            if(solve(i)) break;
            schedule.pop_back();
            
            // Try j wins first game
            remaining[i]--;
            remaining[j]--;
            schedule.push_back({j + 1, i + 1});
            if(solve(j)) break;
            schedule.pop_back();
            
            schedule.clear();
        }
    }
    
    std::cout << schedule.size() << std::endl;
    for(auto& p : schedule) {
        std::cout << p.first << " " << p.second << std::endl;
    }
    
    return 0;
}
```
