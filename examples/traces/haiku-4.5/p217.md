Looking at this problem, I need to find the volume of intersection of two cylinders with radii R1 and R2 whose axes are perpendicular and intersect.

Let me set up a coordinate system:
- Cylinder 1: axis along the z-axis, equation x² + y² ≤ R1²
- Cylinder 2: axis along the x-axis, equation y² + z² ≤ R2²

To find the intersection volume, I'll integrate. The intersection is the region where both inequalities hold.

For a point to be in both cylinders:
- x² + y² ≤ R1²
- y² + z² ≤ R2²

I can compute this by integration. Let me think about this differently using symmetry.

By symmetry, I can compute the volume in one octant and multiply by 8.

In the region where x ≥ 0, y ≥ 0, z ≥ 0:
- From cylinder 1: x² + y² ≤ R1², so for fixed y, x ranges from 0 to √(R1² - y²)
- From cylinder 2: y² + z² ≤ R2², so for fixed y, z ranges from 0 to √(R2² - y²)

For a fixed y where 0 ≤ y ≤ min(R1, R2):
- x ranges from 0 to √(R1² - y²)
- z ranges from 0 to √(R2² - y²)

So the volume in one octant is:
V₁ = ∫₀^min(R1,R2) √(R1² - y²) · √(R2² - y²) dy

By symmetry, the total volume is:
V = 8 · ∫₀^min(R1,R2) √(R1² - y²) · √(R2² - y²) dy

Let me verify with R1 = R2 = 1:
V = 8 · ∫₀¹ √(1 - y²) · √(1 - y²) dy
V = 8 · ∫₀¹ (1 - y²) dy
V = 8 · [y - y³/3]₀¹
V = 8 · (1 - 1/3)
V = 8 · 2/3
V = 16/3 ≈ 5.3333 ✓

This matches the expected output!

For the general case, I need to compute:
V = 8 · ∫₀^min(R1,R2) √(R1² - y²) · √(R2² - y²) dy

I'll use numerical integration (Simpson's rule or similar) for accuracy.

#include <iostream>
#include <cmath>
#include <iomanip>
using namespace std;

int main(){
    double R1, R2;
    cin >> R1 >> R2;
    
    double minR = min(R1, R2);
    
    // Numerical integration using Simpson's rule
    int n = 100000; // number of intervals
    double h = minR / n;
    
    double sum = 0.0;
    
    // Simpson's rule
    for(int i = 0; i <= n; i++){
        double y = i * h;
        double f = sqrt(R1*R1 - y*y) * sqrt(R2*R2 - y*y);
        
        if(i == 0 || i == n){
            sum += f;
        } else if(i % 2 == 1){
            sum += 4.0 * f;
        } else {
            sum += 2.0 * f;
        }
    }
    
    double integral = sum * h / 3.0;
    double volume = 8.0 * integral;
    
    cout << fixed << setprecision(4) << volume << endl;
    
    return 0;
}
