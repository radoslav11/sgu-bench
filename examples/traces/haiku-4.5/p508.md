Looking at this problem, I need to:

1. Find a range [a, b] such that the probability of k black balls being in that range is at least p%
2. Minimize b - a
3. If there are ties, choose the smallest a

The key insight is to use Bayes' theorem. Given that we drew l1 black and l2 white balls:
- Prior: Each number of black balls k (from 0 to n) has probability 1/(n+1)
- Likelihood: P(drawing l1 black, l2 white | k black balls) = C(k, l1) * C(n-k, l2) / C(n, l)
- Posterior: P(k black balls | observation) ∝ C(k, l1) * C(n-k, l2)

For each possible k, I can calculate the posterior probability. Then I need to find the range [a, b] with minimal width that contains probability at least p.

Strategy:
1. Calculate posterior probability for each k (which must be at least max(l1, l2) and at most n - (l - l1) where l = l1 + l2)
2. Normalize these probabilities
3. For each starting position a, extend b until we reach probability p
4. Track the best [a, b]

```cpp
#include <bits/stdc++.h>
using namespace std;

long long binomial(int n, int k) {
    if (k < 0 || k > n) return 0;
    if (k == 0 || k == n) return 1;
    
    long long result = 1;
    for (int i = 0; i < k; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, l1, l2;
    double p;
    cin >> n >> l1 >> l2 >> p;
    
    int l = l1 + l2;
    p = p / 100.0;
    
    // Calculate posterior probabilities for each k
    vector<double> prob(n + 1, 0);
    double total = 0;
    
    for (int k = 0; k <= n; k++) {
        // k must be able to provide l1 black balls
        // and n-k must be able to provide l2 white balls
        if (k < l1 || n - k < l2) {
            prob[k] = 0;
        } else {
            prob[k] = (double)binomial(k, l1) * binomial(n - k, l2);
            total += prob[k];
        }
    }
    
    // Normalize
    for (int k = 0; k <= n; k++) {
        prob[k] /= total;
    }
    
    // Find best range [a, b]
    int best_a = 0, best_b = n;
    int best_width = n;
    
    for (int a = 0; a <= n; a++) {
        double cumulative = 0;
        for (int b = a; b <= n; b++) {
            cumulative += prob[b];
            if (cumulative >= p - 1e-9) {
                int width = b - a;
                if (width < best_width) {
                    best_width = width;
                    best_a = a;
                    best_b = b;
                }
                break;
            }
        }
    }
    
    cout << best_a << " " << best_b << endl;
    
    return 0;
}
```

The algorithm:
1. For each possible number of black balls k, calculate the likelihood of observing l1 black and l2 white balls using the hypergeometric distribution
2. Normalize to get posterior probabilities
3. For each potential starting position a, find the minimum b such that the cumulative probability from a to b is at least p
4. Keep track of the range with minimum width (and smallest a in case of ties, which is automatic since we iterate a from 0)
