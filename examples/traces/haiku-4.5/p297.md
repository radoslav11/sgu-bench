Looking at this problem, I need to understand what's happening:

1. N friends visit M pubs
2. At each pub, they split the bill equally, with remainder paid by some friends (each paying 1 extra ruble)
3. After all M pubs, we need to find how many friends paid more than others overall

Let me trace through the examples:

**Example 1:** N=3, M=2
- Pub 1: Bill=200. 200÷3 = 66 remainder 2. So 2 friends pay 67, 1 friend pays 66
- Pub 2: Bill=100. 100÷3 = 33 remainder 1. So 1 friend pays 34, 2 friends pay 33

We need to track the total differences. At each pub with remainder r, exactly r friends pay the extra ruble.

Over M pubs, the total remainder is the sum of all (Si mod N).

If we sum all remainders: we need this total distributed as evenly as possible among N friends. If total remainder is R, then R mod N friends will pay an extra ruble across all pubs combined.

Wait, let me reconsider. Each friend should pay approximately (total_sum / N). 

Total sum = 200 + 100 = 300
300 ÷ 3 = 100 exactly, so all pay 100. Answer: 0 ✓

**Example 2:** N=7, M=3
- Pub 1: Bill=1. 1÷7 = 0 remainder 1. One friend pays 1, others pay 0
- Pub 2: Bill=4. 4÷7 = 0 remainder 4. Four friends pay 1, others pay 0
- Pub 3: Bill=5. 5÷7 = 0 remainder 5. Five friends pay 1, others pay 0

Total sum = 1 + 4 + 5 = 10
10 ÷ 7 = 1 remainder 3
So 3 friends pay 2 rubles total, and 4 friends pay 1 ruble total.
The 3 friends who pay more (2 vs 1) is the answer: 3 ✓

The key insight: 
- Calculate total bill S = sum of all Si
- Each friend should pay approximately S/N
- Specifically: (S ÷ N) is the base amount, and (S mod N) friends pay one extra ruble
- So X = S mod N

Let me verify:
- Example 1: (200+100) mod 3 = 300 mod 3 = 0 ✓
- Example 2: (1+4+5) mod 7 = 10 mod 7 = 3 ✓

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    long long totalBill = 0;
    for (int i = 0; i < M; i++) {
        int bill;
        cin >> bill;
        totalBill += bill;
    }
    
    int X = totalBill % N;
    cout << X << endl;
    
    return 0;
}
```
