Looking at this problem, I need to find the maximum number of zones that N lines can divide an infinite sheet of paper into.

Let me think about this step by step:

- With 0 lines: 1 zone (the whole sheet)
- With 1 line: 2 zones (the line splits the sheet in 2)
- With 2 lines: Maximum 4 zones (when lines intersect at one point)
- With 3 lines: Maximum 7 zones (each new line intersects all previous lines at distinct points)

Let me derive the pattern:
- When we add the nth line to an arrangement of (n-1) lines in maximum configuration:
  - The new line should intersect all (n-1) previous lines at distinct points
  - These (n-1) intersection points divide the new line into n segments (including the two infinite ends)
  - Each segment splits an existing zone into 2, so we add n new zones

So the recurrence is:
- Z(0) = 1
- Z(n) = Z(n-1) + n

This gives us:
- Z(0) = 1
- Z(1) = 1 + 1 = 2
- Z(2) = 2 + 2 = 4
- Z(3) = 4 + 3 = 7
- Z(4) = 7 + 4 = 11
- Z(n) = 1 + 1 + 2 + 3 + ... + n = 1 + n(n+1)/2

Let me verify with the examples:
- Z(0) = 1 + 0(1)/2 = 1 ✓
- Z(1) = 1 + 1(2)/2 = 1 + 1 = 2 ✓

The formula is: **Z(n) = 1 + n(n+1)/2** = **(n² + n + 2)/2**

```cpp
#include <iostream>
using namespace std;

int main() {
    long long n;
    cin >> n;
    
    long long zones = 1 + n * (n + 1) / 2;
    cout << zones << endl;
    
    return 0;
}
```
