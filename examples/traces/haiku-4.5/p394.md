Looking at this problem, I need to:

1. Read N pizza huts with their positions (x, y) and delivery speed w (distance they can cover in 10 minutes)
2. For each pizza hut A, check if at least K OTHER pizza huts can reach it in 10 minutes
3. If yes, hut A can be closed
4. Output the count and indices of closeable huts

A pizza boy from hut B can reach hut A in 10 minutes if the Manhattan distance between them is ≤ w_B (the speed of that pizza boy).

Let me trace through the examples:

**Example 1:** N=2, K=1
- Hut 1: (0,0), w=1
- Hut 2: (1,0), w=1
- For hut 1: Can hut 2 reach it? Distance = |1-0| + |0-0| = 1 ≤ 1 ✓ (1 hut can reach)
- For hut 2: Can hut 1 reach it? Distance = |0-1| + |0-0| = 1 ≤ 1 ✓ (1 hut can reach)
- Both have at least K=1 other huts that can reach them, so both close. Output: 2, huts 1 2 ✓

**Example 2:** N=3, K=2
- Hut 1: (0,0), w=1
- Hut 2: (1,0), w=1
- Hut 3: (10,3), w=1
- For hut 1: Hut 2 distance=1≤1✓, Hut 3 distance=13>1✗ → 1 hut can reach (need 2)
- For hut 2: Hut 1 distance=1≤1✓, Hut 3 distance=12>1✗ → 1 hut can reach (need 2)
- For hut 3: Hut 1 distance=13>1✗, Hut 2 distance=12>1✗ → 0 huts can reach
- None can be closed. Output: 0 ✓

**Example 3:** N=3, K=2
- Hut 1: (0,0), w=100
- Hut 2: (13,17), w=5
- Hut 3: (10,10), w=10
- For hut 1: Hut 2 distance=30>5✗, Hut 3 distance=20>10✗ → 0 huts
- For hut 2: Hut 1 distance=30≤100✓, Hut 3 distance=|13-10|+|17-10|=3+7=10≤10✓ → 2 huts (need 2) ✓
- For hut 3: Hut 1 distance=20≤100✓, Hut 2 distance=10≤5✗ → 1 hut
- Only hut 2 can be closed. Output: 1, hut 2 ✓

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
  ios_base::sync_with_stdio(false);
  cin.tie(nullptr);
  
  int n, k;
  cin >> n >> k;
  
  vector<long long> x(n), y(n), w(n);
  
  for(int i = 0; i < n; i++){
    cin >> x[i] >> y[i] >> w[i];
  }
  
  vector<int> closeable;
  
  // For each hut A
  for(int a = 0; a < n; a++){
    int count = 0;
    
    // Check how many other huts B can reach hut A
    for(int b = 0; b < n; b++){
      if(a == b) continue; // Skip the same hut
      
      long long dist = abs(x[b] - x[a]) + abs(y[b] - y[a]);
      
      // Can pizza boy from hut b reach hut a in 10 minutes?
      if(dist <= w[b]){
        count++;
      }
    }
    
    // If at least K other huts can reach this hut, it can be closed
    if(count >= k){
      closeable.push_back(a + 1); // 1-indexed
    }
  }
  
  cout << closeable.size() << "\n";
  if(closeable.size() > 0){
    for(int i = 0; i < closeable.size(); i++){
      if(i > 0) cout << " ";
      cout << closeable[i];
    }
    cout << "\n";
  }
  
  return 0;
}
```
