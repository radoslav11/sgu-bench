Looking at this problem, I need to understand the structure:

1. We have N computers that form a tree structure
2. Computer 1 was bought first
3. For each subsequent computer i (from 2 to N), we're told which earlier computer it connects to and the cable length
4. For each computer, we need to find the maximum distance to any other computer in the network

This is a classic tree problem where for each node, we need to find the maximum distance to any other node (the "eccentricity" of each node).

Let me trace through the example:
- N = 3
- Computer 2 connects to computer 1 with cable length 1
- Computer 3 connects to computer 1 with cable length 2

So the tree looks like:
```
    1
   / \
  2   3
 (1) (2)
```

For computer 1: max distance is max(1, 2) = 2
For computer 2: max distance is max(1, 1+2) = 3 (to computer 3)
For computer 3: max distance is max(2, 2+1) = 3 (to computer 2)

Algorithm:
1. Build an adjacency list representation of the tree
2. For each node, run a BFS/DFS to find the maximum distance to any other node

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<vector<pair<int, long long>>> adj(n + 1);
    
    // Read the tree structure
    for(int i = 2; i <= n; i++){
        int parent;
        long long length;
        cin >> parent >> length;
        adj[parent].push_back({i, length});
        adj[i].push_back({parent, length});
    }
    
    // For each computer, find maximum distance to any other computer
    for(int start = 1; start <= n; start++){
        // BFS from start node
        vector<long long> dist(n + 1, -1);
        queue<int> q;
        q.push(start);
        dist[start] = 0;
        
        long long maxDist = 0;
        
        while(!q.empty()){
            int u = q.front();
            q.pop();
            
            for(auto [v, len] : adj[u]){
                if(dist[v] == -1){
                    dist[v] = dist[u] + len;
                    maxDist = max(maxDist, dist[v]);
                    q.push(v);
                }
            }
        }
        
        cout << maxDist << "\n";
    }
    
    return 0;
}
```

The solution works as follows:

1. **Input parsing**: Read N, then for each computer 2 to N, read which computer it connects to and the cable length. Build an undirected graph (adjacency list).

2. **Finding eccentricity**: For each computer (node) as a starting point:
   - Use BFS to find distances to all other nodes
   - Track the maximum distance encountered
   - Output this maximum distance

3. **Time complexity**: O(N²) since we do BFS from each of N nodes, and each BFS is O(N)

This correctly handles the tree structure and finds the required maximum distances for each node.
