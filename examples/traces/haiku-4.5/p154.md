Looking at this problem, I need to find the minimal natural number N such that N! has exactly Q trailing zeros.

The number of trailing zeros in N! is determined by the number of times 10 divides N!, which is determined by the number of pairs of factors 2 and 5. Since there are always more factors of 2 than 5, the number of trailing zeros equals the number of times 5 divides N!.

The number of trailing zeros in N! is given by:
f(N) = ⌊N/5⌋ + ⌊N/25⌋ + ⌊N/125⌋ + ⌊N/625⌋ + ...

Key observations:
1. f(N) is a non-decreasing function
2. f(N) increases by at least 1 when N goes from 4 to 5, 9 to 10, etc.
3. f(N) increases by more than 1 when N is a multiple of higher powers of 5 (like 25, 125, etc.)
4. f(N) stays the same for consecutive values until we hit a multiple of 5

For example:
- f(4) = 0, f(5) = 1
- f(9) = 1, f(10) = 2
- f(24) = 4, f(25) = 6 (jump of 2!)

This means not all values of Q have a solution. Specifically, there's no solution if Q falls in a "gap" created by multiples of 25, 125, etc.

Strategy:
1. Use binary search to find if there exists an N with exactly Q trailing zeros
2. Check if f(N) == Q for some N
3. Since gaps can exist, I need to verify that the value is achievable

Let me think about which Q values have no solution:
- Q has no solution if f(N) jumps over Q for some N
- This happens when N is a multiple of 25 or higher powers of 5

For example: f(24) = 4, f(25) = 6, so Q=5 has no solution.

Algorithm:
1. Binary search for the minimal N where f(N) >= Q
2. Check if f(N) == Q. If yes, return N.
3. If f(N) > Q, then Q has no solution (it's in a gap).

```cpp
#include<bits/stdc++.h>
using namespace std;

long long countTrailingZeros(long long n) {
    long long count = 0;
    long long power = 5;
    while (power <= n) {
        count += n / power;
        if (power > n / 5) break;  // Prevent overflow
        power *= 5;
    }
    return count;
}

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long Q;
    cin >> Q;
    
    if (Q == 0) {
        cout << 0 << endl;
        return 0;
    }
    
    // Binary search for the minimal N where f(N) >= Q
    long long left = 0, right = 10000000000LL;  // Large upper bound
    long long result = -1;
    
    while (left <= right) {
        long long mid = left + (right - left) / 2;
        long long zeros = countTrailingZeros(mid);
        
        if (zeros == Q) {
            result = mid;
            right = mid - 1;  // Continue searching for minimal N
        } else if (zeros < Q) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    if (result == -1) {
        cout << "No solution" << endl;
    } else {
        cout << result << endl;
    }
    
    return 0;
}
```
