Looking at this problem, I need to:

1. Track N soldiers moving on a circular fence of 1000 meters
2. Each soldier has an initial position and velocity
3. Two soldiers "meet" when they're at the same point and moving in opposite directions
4. Count how many times each soldier meets another soldier in T minutes

Key insights:
- Soldiers move on a circle, so positions wrap around at 1000
- A "meeting" happens when two soldiers are at the same position with opposite velocities
- I need to find all meeting times for each pair of soldiers within [0, T] minutes

For two soldiers i and j:
- Position of soldier i at time t: `(L[i] + V[i] * t) % 1000`
- Position of soldier j at time t: `(L[j] + V[j] * t) % 1000`

They meet when:
1. Their positions are equal: `(L[i] + V[i] * t) ≡ (L[j] + V[j] * t) (mod 1000)`
2. They have opposite velocities: `V[i] * V[j] < 0` (opposite signs)

From condition 1:
`L[i] - L[j] ≡ (V[j] - V[i]) * t (mod 1000)`

Let `dL = L[i] - L[j]` and `dV = V[j] - V[i]`

We need: `dL ≡ dV * t (mod 1000)`

This is a modular equation. We need to find all t in [0, T] where this holds.

The general solution is: `t = (dL / gcd(dV, 1000)) + k * (1000 / gcd(dV, 1000))` for integer k (if dL is divisible by gcd).

Let me implement this:

```cpp
#include<bits/stdc++.h>
using namespace std;

int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}

int main(){
    int N, T;
    cin >> N >> T;
    
    vector<int> L(N), V(N);
    for(int i = 0; i < N; i++) cin >> L[i];
    for(int i = 0; i < N; i++) cin >> V[i];
    
    vector<int> count(N, 0);
    
    // Check each pair of soldiers
    for(int i = 0; i < N; i++) {
        for(int j = i + 1; j < N; j++) {
            // They meet only if moving in opposite directions
            if((V[i] > 0 && V[j] > 0) || (V[i] < 0 && V[j] < 0)) {
                continue; // Same direction, no meeting
            }
            
            int dL = ((L[i] - L[j]) % 1000 + 1000) % 1000;
            int dV = V[j] - V[i]; // This will be non-zero and we need opposite velocities
            
            int g = gcd(abs(dV), 1000);
            
            if(dL % g != 0) {
                continue; // No solution
            }
            
            // Period of meetings
            int period = 1000 / g;
            
            // Find first meeting time
            // We need dL ≡ dV * t (mod 1000)
            // t = dL / dV (mod period)
            
            for(int t = 0; t <= T * 100; t++) {
                int pos_i = (L[i] + V[i] * t) % 1000;
                if(pos_i < 0) pos_i += 1000;
                int pos_j = (L[j] + V[j] * t) % 1000;
                if(pos_j < 0) pos_j += 1000;
                
                if(pos_i == pos_j) {
                    count[i]++;
                    count[j]++;
                }
                
                if(t > T * 100) break;
            }
        }
    }
    
    for(int i = 0; i < N; i++) {
        if(i > 0) cout << " ";
        cout << count[i];
    }
    cout << "\n";
    
    return 0;
}
```
