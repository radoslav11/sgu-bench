#include <iostream>

/**
 * Problem Analysis:
 * The problem asks for the maximum number of zones (regions) into which an infinite 
 * plane (sheet of paper) can be split by N straight lines. 
 * To maximize the number of regions, each new line should intersect every previous line 
 * at a unique point, and no three lines should be concurrent (they should not all 
 * intersect at the same point).
 * 
 * Let L(n) be the maximum number of regions formed by n lines.
 * - L(0) = 1 (the entire sheet)
 * - L(1) = 2 (one line divides the sheet into two)
 * - L(2) = 4 (two intersecting lines divide the sheet into four)
 * - L(3) = 7 (a third line intersecting the existing two at two new points)
 * 
 * The recurrence relation is L(n) = L(n-1) + n.
 * This is because the n-th line will cross the n-1 existing lines at n-1 distinct points,
 * thus splitting the n-th line into n segments. Each of these n segments passes 
 * through an existing region and splits it into two, adding n new regions.
 * 
 * The closed-form formula for this sequence (known as the Lazy Caterer's sequence) is:
 * L(n) = 1 + (n * (n + 1)) / 2
 * 
 * Constraints:
 * N can be up to 65535.
 * For N = 65535, the value is 1 + (65535 * 65536) / 2 = 1 + 2147450880 = 2147450881.
 * This value fits within a 32-bit signed integer (max 2147483647) and easily within a 64-bit long long.
 * To avoid overflow during the calculation of n * (n + 1), we use 'long long'.
 */

int main() {
    // Faster I/O
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    long long n;
    // Reading the input. Using a loop if multiple test cases were present,
    // but based on the problem statement, we expect one integer N.
    if (std::cin >> n) {
        // The formula for the maximum number of regions formed by n lines is 1 + n(n+1)/2.
        // We use long long to prevent intermediate overflow when calculating n * (n + 1).
        long long max_zones = 1 + (n * (n + 1)) / 2;
        std::cout << max_zones << std::endl;
    }

    return 0;
}
