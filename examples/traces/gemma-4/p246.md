/**
 * Problem: Black & White (246)
 * 
 * Problem Statement Summary:
 * Given a necklace of 2N-1 beads, K of which are black.
 * A necklace is "beautiful" if there exists a pair of black beads such that 
 * the number of beads strictly between them in one of the two arcs is exactly N.
 * We need to find the minimal K such that every possible necklace with K black beads is beautiful.
 * 
 * Analysis:
 * Let M = 2N-1. Two beads at positions i and j (0 <= i, j < M) are beautiful if:
 * (j - i - 1) % M == N  OR  (i - j - 1) % M == N
 * 
 * This simplifies to:
 * j - i \equiv N + 1 (mod M)  OR  i - j \equiv N + 1 (mod M)
 * Since (N + 1) + (N - 2) = 2N - 1 = M, these two conditions are equivalent to:
 * j - i \equiv N + 1 (mod M)  OR  j - i \equiv N - 2 (mod M)
 * 
 * Let D = N + 1. We want the smallest K such that any subset of K indices 
 * from {0, 1, ..., M-1} contains at least one pair with distance D (mod M).
 * 
 * This is a graph problem where we want to find the size of the maximum independent 
 * set in a graph where edges connect nodes with distance D (mod M).
 * The graph is a collection of disjoint cycles. The number of cycles is g = gcd(D, M).
 * Each cycle has a length L = M / g.
 * 
 * The maximum independent set in a cycle of length L is floor(L/2).
 * Since M is odd, L must be odd. Thus, floor(L/2) = (L - 1) / 2.
 * The maximum size of a set of indices with no D-distance pair is:
 * Max size = g * (L - 1) / 2 = (g * L - g) / 2 = (M - g) / 2.
 * 
 * The minimal K for which every set of K indices contains a D-distance pair is:
 * K = (Max size) + 1 = (M - g) / 2 + 1.
 * 
 * We need g = gcd(N + 1, 2N - 1).
 * g = gcd(N + 1, 2N - 1 - 2*(N + 1)) = gcd(N + 1, -3) = gcd(N + 1, 3).
 * Therefore, g is either 1 or 3.
 * 
 * Case 1: If (N + 1) % 3 != 0, then g = 1.
 * K = (M - 1) / 2 + 1 = (2N - 1 - 1) / 2 + 1 = (2N - 2) / 2 + 1 = (N - 1) + 1 = N.
 * 
 * Case 2: If (N + 1) % 3 == 0, then g = 3.
 * K = (M - 3) / 2 + 1 = (2N - 1 - 3) / 2 + 1 = (2N - 4) / 2 + 1 = (N - 2) + 1 = N - 1.
 * 
 * Input range: 5 <= 2N-1 <= 2^31-1. This requires 64-bit integers.
 */

#include <iostream>

using namespace std;

int main() {
    // Speed up input and output
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    long long M;
    // Read 2N-1 from standard input
    if (!(cin >> M)) {
        return 0;
    }

    // M = 2N - 1 => N = (M + 1) / 2
    long long N = (M + 1) / 2;

    // Compute minimal K based on gcd(N + 1, 3)
    // If (N + 1) is divisible by 3, g = 3. Otherwise g = 1.
    if ((N + 1) % 3 == 0) {
        // K = N - 1
        cout << N - 1 << endl;
    } else {
        // K = N
        cout << N << endl;
    }

    return 0;
}
