One final check: the `total_count_T_prime` should subtract `count(S, T_prime)` where `T_prime` is `c_pref + # + c_suff`. And if `c_len < S_len`, `T_prime` is just `c_full`. My code does this.
Also, `total_len` can be very large, so I used `long long`. The $N$ is small enough that it's efficient.

Wait, one last thing: `total_count_T_prime.add(c_count)` when `c_len < S_len` is correct because `count(S, T_prime)` where `T_prime = c_full` is `c_count`. This makes the logic `sum_all_counts + count(S, V) - sum(count(S, T_prime))` correct.
