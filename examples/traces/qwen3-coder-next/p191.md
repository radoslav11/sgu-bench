          How to check if a given string s (of n stands) can be generated?

          We start with 1 empty stand (signed by the initial company, say 'A').

          We do a sequence of operations. Each operation 2 increases the length by 2, and each operation 1 keeps the length the same.

          Let k = number of operation 2, then n = 1 + 2*k.
          Also, the number of empty stands at the end is: initially 1, then each operation 2 adds 1 empty stand (because we add two stands: one filled and one empty, so net empty stands: +1), and each operation 1 removes 1 empty stand.
          So: 1 + k - (number of operation 1) = 0  => number of operation 1 = 1 + k.
          Total operations = k + (1+k) = 2k+1.
          And n = 1 + 2k -> k = (n-1)/2, so n must be odd.

          In the sample: n=3, which is odd, and k=1.

          Now, how to verify a given string?

          We can simulate the process backwards. In reverse, we look for two consecutive stands that were added by an operation 2.

          What does operation 2 add? 
             It adds two stands to the left of an empty stand: specifically, it adds 'X' (the company that did the operation) and then an empty stand signed by 'Y' (the competitor).

          In the final configuration, there are no empty stands, but in the reverse process, we know that the empty stands were filled by operation 1. So the empty stands in the intermediate steps are filled in some order.

          However, note: the empty stands that were created by operation 2 are later filled by operation 1. In the final string, we only have the letters.

```cpp
#include <iostream>
#include <string>
#include <stack>
using namespace std;

int main() {
    string start, s;
    getline(cin, start);
    getline(cin, s);

    stack<char> stk;
    for (char c : s) {
        if (!stk.empty() && stk.top() != c) {
            stk.pop();
        } else {
            stk.push(c);
        }
    }

    if (stk.size() == 1 && stk.top() == start[0]) {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }

    return 0;
}
```
