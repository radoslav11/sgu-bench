3. **Edge Processing**: For each shelf, compute the intersection points of the rays from the origin through the shelf's endpoints with the polygon boundary. This involves:

    double operator % (const Point& p) const { return x * p.x + y * p.y; }

double dist(const Point& a, const Point& b) {
    double dx = a.x - b.x;
    double dy = a.y - b.y;
    return sqrt(dx*dx + dy*dy);
}
