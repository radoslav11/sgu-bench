I need to find the sum of all elements in the matrix product A^T A, where A is the incidence matrix of an undirected graph.

Let me analyze what A^T A represents:
- A is N×M (N vertices, M edges)
- A^T is M×N
- So A^T A is M×M

The element (i,j) of A^T A is the dot product of column i and column j of A.
- Column i represents edge i: it has 1's at the two vertices incident to edge i, and 0's elsewhere.
- The dot product of column i with itself (diagonal element) is: 1^2 + 1^2 = 2 (since each edge connects exactly 2 vertices)
- The dot product of column i with column j (i ≠ j) is: the number of vertices that are incident to both edge i and edge j.

So the sum of all elements in A^T A is:
- Sum of diagonal elements: each contributes 2, so total = 2M
- Sum of off-diagonal elements: for each pair of edges (i,j), count how many common vertices they have.

But there's a better way to think about this. Let me express the sum of all elements of A^T A:

Sum of all elements of A^T A = sum_{i=1 to M} sum_{j=1 to M} (A^T A)_{ij}
= sum_{i=1 to M} sum_{j=1 to M} (A^T)_{i,*} · A_{*,j}
= sum_{i=1 to M} sum_{j=1 to M} sum_{k=1 to N} A_{k,i} · A_{k,j}
= sum_{k=1 to N} sum_{i=1 to M} sum_{j=1 to M} A_{k,i} · A_{k,j}
= sum_{k=1 to N} (sum_{i=1 to M} A_{k,i}) · (sum_{j=1 to M} A_{k,j})
= sum_{k=1 to N} (degree of vertex k)^2

Why? Because for each vertex k, sum_{i=1 to M} A_{k,i} counts how many edges are incident to vertex k, which is exactly the degree of vertex k.

So the sum of all elements of A^T A equals the sum of squares of vertex degrees.

Let me verify with the sample:
- Graph: 4 vertices, 4 edges: (1,2), (1,3), (2,3), (2,4)
- Degrees:
  - Vertex 1: connected to edges (1,2) and (1,3) → degree 2
  - Vertex 2: connected to edges (1,2), (2,3), (2,4) → degree 3
  - Vertex 3: connected to edges (1,3), (2,3) → degree 2
  - Vertex 4: connected to edge (2,4) → degree 1
- Sum of squares: 2^2 + 3^2 + 2^2 + 1^2 = 4 + 9 + 4 + 1 = 18 ✓

So the algorithm is:
1. Read N and M
2. Initialize an array of degrees (size N+1, 1-indexed)
3. For each edge (u,v), increment degree[u] and degree[v]
4. Compute sum of degree[i]^2 for i from 1 to N

```cpp
#include <iostream>
#include <vector>

int main() {
    int N, M;
    std::cin >> N >> M;
    
    std::vector<int> degree(N + 1, 0);
    
    for (int i = 0; i < M; ++i) {
        int u, v;
        std::cin >> u >> v;
        degree[u]++;
        degree[v]++;
    }
    
    long long sum = 0;
    for (int i = 1; i <= N; ++i) {
        long long d = degree[i];
        sum += d * d;
    }
    
    std::cout << sum << std::endl;
    
    return 0;
}
```
