I need to solve this problem where I calculate how many cups of tea Andrew owes based on how late he is for training.

The rules are:
- If Andrew arrives on time (P <= S), he owes 0 cups
- If he's late (P > S), but less than 5 minutes (300 seconds) late, he owes 1 cup
- If he's late for 5 minutes (300 seconds) or more but less than 15 minutes (900 seconds), he owes 2 cups
- If he's late for 15 minutes (900 seconds) or more but less than 30 minutes (1800 seconds), he owes 3 cups
- If he's late for 30 minutes (1800 seconds) or more, he owes 4 cups

Let me break down the logic:
1. Calculate delay = P - S
2. If delay <= 0, owe = 0
3. If 0 < delay < 300, owe = 1
4. If 300 <= delay < 900, owe = 2
5. If 900 <= delay < 1800, owe = 3
6. If delay >= 1800, owe = 4

Let me verify with the sample tests:
- Test 1: S=10, P=10 → delay=0 → 0 cups ✓
- Test 2: S=10, P=11 → delay=1 → 1 cup ✓
- Test 3: S=0, P=300 → delay=300 → since 300 >= 300 and < 900, owe = 2 cups ✓

Now I'll write the C++ code to implement this logic.

```cpp
#include <iostream>
using namespace std;

int main() {
    int S, P;
    cin >> S >> P;
    
    int delay = P - S;
    int cups = 0;
    
    if (delay <= 0) {
        cups = 0;
    } else if (delay < 300) {
        cups = 1;
    } else if (delay < 900) {
        cups = 2;
    } else if (delay < 1800) {
        cups = 3;
    } else {
        cups = 4;
    }
    
    cout << cups << endl;
    
    return 0;
}
```
