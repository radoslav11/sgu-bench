We are to construct a cyclic matrix of size 2^N x 2^M such that:
 - The matrix contains numbers from 0 to 2^(N+M)-1 exactly once.
 - Adjacent cells (including cyclic wrap-around on rows and columns) must differ in exactly one bit.

This is essentially asking for a Hamiltonian cycle on the hypercube graph of dimension (N+M) that is arranged in a grid with cyclic wrap-around.

Note: The hypercube graph Q_{k} (with k = N+M) is bipartite and has a Hamiltonian cycle for k>=2. For k=1, it's just two nodes connected by an edge, but the problem states N,M>0 so k>=2.

However, the constraint that the matrix is cyclic (toroidal) in both dimensions is important.

#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    int rows = 1 << N;
    int cols = 1 << M;
    
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            int col_gray = j ^ (j >> 1);
            int row_gray = i ^ (i >> 1);
            int val = (col_gray << N) | row_gray;
            if (j > 0) {
                cout << " ";
            }
            cout << val;
        }
        cout << "\n";
    }
    
    return 0;
}
