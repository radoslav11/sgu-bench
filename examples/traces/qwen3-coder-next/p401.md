To solve this problem, we need to determine the maximal distance Dubrovsky can achieve from his starting point by the time of sunset, given that he must cross all rivers in sequence. The solution involves optimizing the time spent crossing each river to maximize the Euclidean distance from the origin, considering both the maximum and minimum possible north-south displacements.

### Approach
1. **Problem Analysis**:
   - Dubrovsky starts at the left bank of the westernmost river and must reach the right bank of the easternmost river.
   - Each river has a width \( w_i \) and flow speed \( v_i \).
   - Dubrovsky swims at speed \( u \) in still water. His actual path is the vector sum of his swimming velocity and the river's flow velocity.
   - The goal is to maximize the Euclidean distance from the origin (0, 0) to his final position (X, Y), where \( X \) is the sum of all river widths (fixed) and \( Y \) is the total northward displacement (variable).

2. **Key Insight**:
   - The maximum distance is determined by the maximum absolute value of \( Y \), since \( X \) is fixed. Thus, we need to compute both the maximum possible \( Y \) (denoted \( R \)) and the minimum possible \( Y \) (denoted \( L \)), then choose the one that yields the larger distance \( \sqrt{X^2 + Y^2} \).

3. **Optimization**:
   - **Maximizing \( Y \) (R)**: For each river, the northward displacement is \( v_i \cdot t_i + \sqrt{u^2 t_i^2 - w_i^2} \). Using Lagrange multipliers, we find the optimal time \( t_i \) allocation such that the total time \( \sum t_i \) equals the available time \( t \), maximizing \( R \).
   - **Minimizing \( Y \) (L)**: Similarly, the minimum displacement is \( v_i \cdot t_i - \sqrt{u^2 t_i^2 - w_i^2} \). We use Lagrange multipliers again to find \( t_i \) that minimizes \( L \) under the same time constraint.

4. **Binary Search**:
   - For both maximizing \( R \) and minimizing \( L \), we use binary search to solve for the Lagrange multipliers \( \lambda \) and \( \mu \), respectively. This efficiently finds the optimal time allocations \( t_i \) for each river.

5. **Complexity Analysis**:
   - The algorithm processes each river \( O(\text{iterations}) \) times, where iterations are fixed (200 for binary search precision). With \( N \leq 50 \), the total operations are \( O(N \times \text{iterations}) \), which is efficient for the given constraints.

    for (int i = 0; i < N; i++) {
        X += w[i];
    }

    int max_v = *std::max_element(v.begin(), v.end());
    int min_v = *std::min_element(v.begin(), v.end());

    double low1 = static_cast<double>(max_v) + u + 1e-12;
    double high1 = 1e15;
    double lambda = 0.0;
    for (int iter = 0; iter < 200; iter++) {
        lambda = (low1 + high1) * 0.5;
        double total_time = 0.0;
        for (int i = 0; i < N; i++) {
            double diff = lambda - v[i];
            double term = std::sqrt(diff * diff - u * u);
            total_time += static_cast<double>(w[i]) * diff / (u * term);
        }

            low1 = lambda;
        } else {
            high1 = lambda;
        }
    }
    lambda = (low1 + high1) * 0.5;

    std::vector<double> t1(N);
    double R = 0.0;
    for (int i = 0; i < N; i++) {

        double y_i = v[i] * t1[i] + std::sqrt(u * u * t1[i] * t1[i] - static_cast<double>(w[i]) * w[i]);
        R += y_i;

    double high2 = static_cast<double>(min_v) - u - 1e-12;

        }
        if (total_time > t_val) {
            high2 = mu;
        } else {
            low2 = mu;
        }
    }
    mu = (low2 + high2) * 0.5;

    std::vector<double> t2(N);
    double L = 0.0;
    for (int i = 0; i < N; i++) {
        double diff = v[i] - mu;
        double term = std::sqrt(diff * diff - u * u);

    }
