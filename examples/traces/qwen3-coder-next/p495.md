To solve this problem, we need to compute the mathematical expectation of the number of prizes given out when M winners each select a box at random from N boxes, where each box initially contains a prize. The key insight is recognizing that the expectation can be derived using linearity of expectation, simplifying the problem to calculating the probability that each prize is awarded at least once.

### Approach
1. **Problem Analysis**: The problem involves calculating the expected number of prizes awarded when M winners each pick a box uniformly at random from N boxes. Each winner who picks a box containing a prize takes it, making the box empty for subsequent winners. The challenge is to compute the expected number of prizes given out efficiently.
2. **Key Insight**: The expectation can be broken down using linearity of expectation. For each prize (or equivalently each box), the probability that it is awarded at least once is \(1 - \left(\frac{N-1}{N}\right)^M\). This is because the probability that a particular box is not selected by any of the M winners is \(\left(\frac{N-1}{N}\right)^M\).
3. **Mathematical Formulation**: The expected number of prizes is \(N \times \left(1 - \left(\frac{N-1}{N}\right)^M\right)\). This formula accounts for each of the N prizes being awarded with the computed probability.
4. **Efficient Computation**: To handle large values of N and M accurately, we use logarithmic and exponential functions to compute \(\left(\frac{N-1}{N}\right)^M\) as \(\exp\left(M \times \log\left(1 - \frac{1}{N}\right)\right)\). This approach ensures numerical stability, especially for large N where direct computation might lead to precision errors.

### Solution Code
```cpp
#include <iostream>
#include <cmath>
#include <iomanip>

int main() {
    long long N, M;
    std::cin >> N >> M;

    if (M == 0) {
        std::cout << std::fixed << std::setprecision(15) << 0.0 << '\n';
    } else {
        double x = 1.0 / static_cast<double>(N);
        double log_val = std::log1p(-x);
        double term = std::exp(M * log_val);
        double result = static_cast<double>(N) * (1.0 - term);
        std::cout << std::fixed << std::setprecision(15) << result << '\n';
    }

    return 0;
}
```

4. **Expectation Calculation**: The expected number of prizes awarded is \(N \times (1 - \text{term})\), where term is the probability that a specific prize is not awarded.
