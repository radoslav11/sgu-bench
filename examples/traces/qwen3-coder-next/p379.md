To solve this problem, we need to determine the maximum number of people that can be delivered to the parking floor by an elevator within a given time limit. The elevator starts at the parking floor (floor 0) and can carry at most `C` people per trip. Each floor from 1 to `N` has a certain number of people wanting to descend to the parking. The time taken to move up or down one floor is `P` seconds.

### Approach
1. **Problem Analysis**: The key observation is that the time required for a trip depends on the highest floor visited during that trip. Specifically, a trip that visits floors up to floor `k` takes `2 * k * P` seconds (going up to `k` and returning to floor 0). The goal is to maximize the number of people delivered within `T` seconds.

2. **Insight**: To minimize the total time for delivering `x` people, we should prioritize delivering people from the lowest floors first. This is because lower floors require less time per trip, allowing more people to be delivered within the time limit.

3. **Binary Search**: We use binary search to determine the maximum number of people `x` that can be delivered within `T` seconds. For each candidate `x`, we compute the minimal time required:
   - **Prefix Sum Array**: Compute the prefix sums of the people on each floor to quickly determine how many people are available from floor 1 to floor `i`.
   - **Time Calculation**: For each floor `i` (from 1 to `N`), calculate the number of trips needed to deliver the remaining people (those from floor `i` onwards) as `ceil((x - prefix[i-1]) / C)`. The total time is `2 * P` multiplied by the sum of these trip counts.

4. **Feasibility Check**: For a given `x`, if the computed time is within `T` seconds, then `x` is feasible, and we try a larger value; otherwise, we try a smaller value.

#include <iostream>
#include <vector>
using namespace std;

bool check(long long x, const vector<long long>& B, long long C, long long P, long long T) {
    if (x == 0) 
        return true;
    long long time_needed = 0;
    int N = B.size() - 1;
    for (int i = 1; i <= N; i++) {
        long long rem = x - B[i-1];
        if (rem <= 0) 
            break;
        long long terms_i = (rem + C - 1) / C;
        time_needed += 2LL * P * terms_i;
        if (time_needed > T) 
            break;
    }
    return time_needed <= T;
}

int main() {
    long long N, C, P, T_val;
    cin >> N >> C >> P >> T_val;
    vector<long long> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }

    vector<long long> B(N+1);
    B[0] = 0;
    for (int i = 1; i <= N; i++) {
        B[i] = B[i-1] + A[i-1];
    }

    long long total_people = B[N];
    long long lo = 0, hi = total_people;

        } else {

        }

    return 0;

4. **Output**: The maximum feasible number of people delivered within `T_val` seconds is printed.

This approach efficiently narrows down the solution space using binary search and leverages prefix sums to quickly compute the required time for any candidate solution, ensuring optimal performance even for large input sizes.
