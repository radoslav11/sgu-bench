Yes! Because any number is congruent to the sum of its digits modulo 3.

So digit_sum(i) ≡ i (mod 3).

Therefore, D(k) = sum_{i=1}^{k} digit_sum(i) ≡ sum_{i=1}^{k} i = k*(k+1)/2 (mod 3).

So D(k) ≡ k*(k+1)/2 (mod 3).

We need to find when k*(k+1)/2 ≡ 0 (mod 3).

Since we're working modulo 3, and 2 has an inverse (2*2 ≡ 1 mod 3), we can multiply both sides by 2:
k*(k+1)/2 ≡ 0 (mod 3) ⇔ k*(k+1) ≡ 0 (mod 3).

So we need k*(k+1) ≡ 0 (mod 3).

Since k and k+1 are consecutive integers, one of them is divisible by 2, but for divisibility by 3:
- Either k ≡ 0 (mod 3), then k is divisible by 3
- Or k+1 ≡ 0 (mod 3), i.e., k ≡ 2 (mod 3)
- Or neither, then k ≡ 1 (mod 3), and neither k nor k+1 is divisible by 3.

So k*(k+1) ≡ 0 (mod 3) if and only if k ≢ 1 (mod 3).

Therefore, D(k) is divisible by 3 if and only if k ≢ 1 (mod 3).
