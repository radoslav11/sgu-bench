To solve this problem, we need to determine the set of positive integers that divide every possible number formed by mapping distinct letters in a given string to distinct digits (without leading zeros). The key insight is that the greatest common divisor (GCD) of all such numbers is equal to the GCD of the coefficients of each distinct letter in the string when expanded as a polynomial in base 10.

### Approach
1. **Coefficient Calculation**: For each string, compute the coefficient for each distinct letter. The coefficient of a letter is the sum of \(10^{\text{position}}\) for each occurrence of the letter in the string, where positions are counted from right to left (starting at 0 for the least significant digit).
2. **GCD Calculation**: The GCD of all coefficients of the distinct letters gives the largest number that divides every possible number formed by valid digit mappings. This GCD is derived because any linear combination of the coefficients (with distinct digits) will be divisible by this GCD.
3. **Divisor Enumeration**: For each test case, enumerate all divisors of the computed GCD, sort them in increasing order, and output the result.

### Solution Code
```cpp
#include <iostream>
#include <vector>
#include <map>
#include <algorithm>
using namespace std;

long long gcd(long long a, long long b) {
    if (b == 0) 
        return a;
    return gcd(b, a % b);
}

int main() {
    int n;
    cin >> n;
    for (int caseIdx = 1; caseIdx <= n; caseIdx++) {
        string s;
        cin >> s;
        int len = s.size();

        map<char, long long> coeffMap;
        long long p = 1;
        for (int i = len - 1; i >= 0; i--) {
            coeffMap[s[i]] += p;
            p *= 10;
        }

        long long g = 0;
        for (auto &kv : coeffMap) {
            g = gcd(g, kv.second);
        }

        vector<long long> divisors;
        if (g == 0) {
            // In case of empty string (though problem states length >= 1)
            divisors.push_back(0);
        } else {
            for (long long i = 1; i * i <= g; i++) {
                if (g % i == 0) {
                    divisors.push_back(i);
                    if (i != g / i) {
                        divisors.push_back(g / i);
                    }
                }
            }
            sort(divisors.begin(), divisors.end());
        }

        cout << "Case " << caseIdx << ":";
        for (long long d : divisors) {
            cout << " " << d;
        }
        cout << endl;
    }
    return 0;
}
```

### Explanation
1. **Reading Input**: The number of test cases \(n\) is read first. For each test case, the string \(s\) is processed.
2. **Coefficient Calculation**: For each character in the string, starting from the least significant digit (rightmost), the coefficient for the character is incremented by \(10^{\text{position}}\), where position starts at 0 for the rightmost character.
3. **GCD Calculation**: The GCD of all coefficients of distinct letters is computed. This GCD represents the largest number that divides every possible number formed by valid digit mappings of the string.
4. **Divisor Enumeration**: All divisors of the computed GCD are found by checking each integer up to the square root of the GCD. Each divisor and its complement (if different) are added to a list, which is then sorted in ascending order.
5. **Output**: The results for each test case are formatted to show the case number followed by the sorted list of divisors.
