  How to run max-flow and get the cut set?

      We'll use Dinic's algorithm, and after running, we do a BFS from s in the residual graph to find all reachable vertices. Then the cut is: 
          S = { vertices reachable from s in residual graph }
          T = V \ S
      Then the cut set is all edges (u, v) in the original graph such that u in S and v in T.

      But note: the graph is undirected, so we have to consider both directions? In max-flow for undirected graph, we replace each undirected edge with two directed edges.
