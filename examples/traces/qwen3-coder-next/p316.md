I need to simulate a tank battle based on the given log of shots.

Key points:
- N tanks (2-6), M shots (1-4000)
- Each tank starts with 100 HP and 0 score
- When tank A shoots tank B:
  - Tank B loses 8 HP
  - If tank B had > 0 HP before the shot, tank A gets 3 points
- Only active tanks (HP > 0) can shoot
- At the end, active tanks get bonus points = floor(HP / 2)

Wait, I need to be careful about the order:
- For each shot in the log, check if the shooter is still active (HP > 0). If not, this shot shouldn't happen per the rules, but the problem says "you can expect to find only shots fired by the 'active' tanks in the log", so shooter is always active when the shot occurs.
- Apply the hit: target loses 8 HP
- If target had > 0 HP before the shot, shooter gets 3 points
- At the end, add bonus points to active tanks (HP > 0)

Let me trace through the example:
- 3 tanks, 3 shots: [2,1], [1,2], [2,1]

Initial state: HP = [100,100,100], score = [0,0,0]

Shot 1: tank 2 shoots tank 1
- Tank 1 had 100 > 0, so tank 2 gets 3 points
- Tank 1 HP: 100 - 8 = 92
- State: HP = [92,100,100], score = [0,3,0]

Shot 2: tank 1 shoots tank 2
- Tank 2 had 100 > 0, so tank 1 gets 3 points
- Tank 2 HP: 100 - 8 = 92
- State: HP = [92,92,100], score = [3,3,0]

Shot 3: tank 2 shoots tank 1
- Tank 1 had 92 > 0, so tank 2 gets 3 points
- Tank 1 HP: 92 - 8 = 84
- State: HP = [84,92,100], score = [3,6,0]

End bonus:
- Tank 1: HP=84 > 0, gets 84/2 = 42 bonus points → score = 3 + 42 = 45
- Tank 2: HP=92 > 0, gets 92/2 = 46 bonus points → score = 6 + 46 = 52
- Tank 3: HP=100 > 0, gets 100/2 = 50 bonus points → score = 0 + 50 = 50

Output matches: 
84 45
92 52
100 50

So the logic is correct.

Implementation:
- Use arrays for HP and score (1-indexed, so size N+1)
- Read N, M
- For each of M shots, read shooter and target
- Check shooter is active (guaranteed, but still good practice)
- target HP -= 8
- If target HP before shot was > 0, shooter score += 3
- After all shots, for each tank i from 1 to N:
  - if HP[i] > 0, score[i] += HP[i] / 2 (integer division)
- Output HP[i] and score[i] for i=1..N

Note: HP[i] / 2 in C++ with integer division gives floor division, which is correct.

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    vector<int> hp(N + 1, 100);
    vector<int> score(N + 1, 0);
    
    for (int i = 0; i < M; i++) {
        int shooter, target;
        cin >> shooter >> target;
        
        // Check if target has HP before reducing
        if (hp[target] > 0) {
            score[shooter] += 3;
        }
        hp[target] -= 8;
    }
    
    // Add bonus points for active tanks
    for (int i = 1; i <= N; i++) {
        if (hp[i] > 0) {
            score[i] += hp[i] / 2;
        }
    }
    
    // Output results for each tank
    for (int i = 1; i <= N; i++) {
        cout << hp[i] << " " << score[i] << "\n";
    }
    
    return 0;
}
```
