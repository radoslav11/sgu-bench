I need to solve the problem of counting the number of ways to place k rooks on an n×n chessboard such that no two rooks attack each other.

Key insight: Rooks attack along rows and columns, so for no two rooks to attack each other, we must have at most one rook per row and at most one rook per column.

This means:
- We can place at most n rooks on an n×n board (one per row and one per column)
- For k rooks, we need to choose k rows out of n, k columns out of n, and then place the rooks in a way that each rook occupies a unique row and column (i.e., a permutation of k columns for the k rows)

The formula is:
- If k > n, then it's impossible (0 ways)
- If k ≤ n, then we choose k rows from n (C(n,k)), choose k columns from n (C(n,k)), and then arrange the k rooks in these k rows and k columns such that no two share a row or column, which is k! (permutations)

So the answer is: C(n,k) × C(n,k) × k! = (C(n,k))^2 × k!

Alternatively, we can think of it as:
- First rook: n × n choices
- Second rook: (n-1) × (n-1) choices (excluding the row and column of first rook)
- ...
- k-th rook: (n-k+1) × (n-k+1) choices

But we need to divide by k! because the rooks are indistinguishable in terms of placement order.

So: (n × n × (n-1) × (n-1) × ... × (n-k+1) × (n-k+1)) / k!

This equals: (n! / (n-k)!)^2 / k! = (C(n,k) × k!)^2 / k! = C(n,k)^2 × k!

Let me verify with the sample: n=4, k=4
C(4,4) = 1, so answer = 1^2 × 4! = 24 ✓

For k=0: there's exactly 1 way (place no rooks)

Let me implement this using precomputed factorials and combinations.

Since n ≤ 10, I can compute everything directly without worrying about overflow.

```cpp
#include <iostream>
#include <vector>
using namespace std;

long long factorial(int n) {
    long long result = 1;
    for (int i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}

long long comb(int n, int k) {
    if (k > n || k < 0) return 0;
    if (k == 0 || k == n) return 1;
    return factorial(n) / (factorial(k) * factorial(n - k));
}

int main() {
    int n, k;
    cin >> n >> k;
    
    if (k > n) {
        cout << 0 << endl;
    } else {
        long long c = comb(n, k);
        long long result = c * c * factorial(k);
        cout << result << endl;
    }
    
    return 0;
}
```
