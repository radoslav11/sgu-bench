          (0,0): 0 + p + q + r = b00
          (0,1): 0 + p - q - r = b01
          (0,2): 0 + p + q + r = b02   [because (-1)^2=1, (-1)^(0+2)=1]
          (0,3): 0 + p - q - r = b03
          (1,0): 0 - p + q - r = b10
          (1,1): 2 - p - q + r = b11

      Similarly, (0,1) and (0,3): p-q-r = b01 = b03.

      What is the correct homogeneous solution for a grid with fixed boundaries? 

      The recurrence: 
          D[i][j] + D[i][j+1] + D[i+1][j] + D[i+1][j+1] = 0

      can be rewritten as:
          D[i][j+1] - D[i+1][j+1] = - (D[i+1][j] - D[i][j])

      Let E[i][j] = D[i][j] - D[i+1][j]
      then the equation becomes: E[i][j] = E[i][j+1]

      so E[i][j] = c_i (independent of j)

      then D[i][j] - D[i+1][j] = c_i

      => D[i][j] = D[0][j] + sum_{k=0}^{i-1} c_k

      Similarly, from the recurrence in i, we might get that D[i][j] = a_i + b_j.

      => a_i + a_{i+1} + b_j + b_{j+1} = 0 for all i
