          string dfs(string s, int count, int L, int n, set<string>& curSet) {
             if (s.size() == L) {
                 if (count == n) {
                     return s;
                 }
                 return "";
             }

             for (char c = 'a'; c <= 'z'; c++) {
                 // Compute new_count and the list of new substrings
                 vector<string> newSubs;
                 // Generate all new substrings: for i from 0 to s.size()
                 for (int i = 0; i <= (int)s.size(); i++) {
                     string candidate = s.substr(i) + c;
                     if (curSet.find(candidate) == curSet.end()) {
                         newSubs.push_back(candidate);
                     }
                 }
                 int add = newSubs.size();
                 int new_count = count + add;
                 // Prune 1: if new_count > n, skip
                 if (new_count > n) {
                     continue;
                 }

                 // Prune 2: if we are not at the last character, check if we can possibly reach n
                 int k = L - s.size() - 1;
                 if (k > 0) {
                     // maximum additional: k*(2*(s.size()) + 1 + k) / 2
                     // Note: k*(2*s.size()+1+k) might be odd, but integer division is okay because it's the sum of an arithmetic series.
                     long long max_add = (long long)k * (2*(int)s.size() + 1 + k) / 2;
                     if (new_count + max_add < n) {
                         continue;
                     }
                 }

                 // Add newSubs to curSet
                 for (string& sub : newSubs) {
                     curSet.insert(sub);
                 }

                 string res;
                 if (k == 0) {
                     // next is last char
                     if (new_count == n) {
                         res = s + c;
                     } else {
                         res = "";
                     }
                 } else {
                     res = dfs(s+c, new_count, L, n, curSet);
                 }

                 // Backtrack: remove newSubs
                 for (string& sub : newSubs) {
                     curSet.erase(sub);
                 }

                 if (res != "") {
                     return res;
                 }
             }
             return "";
          }

To solve this problem, we need to find the shortest string consisting of lowercase English letters that has exactly `n` distinct substrings. Among all such strings of minimal length, we should choose the lexicographically smallest one.

### Approach
1. **Problem Analysis**: The key observation is that the number of distinct substrings in a string of length `L` ranges from `L` (for a string with all identical characters) to `L*(L+1)/2` (for a string with all distinct characters). Given `n`, we need the smallest `L` such that there exists a string of length `L` with exactly `n` distinct substrings. For this `L`, we then find the lexicographically smallest string achieving `n` distinct substrings.

2. **Algorithm Selection**: 
   - **Iterate Over Possible Lengths**: Start with `L = 1` and incrementally check each `L` until we find the smallest `L` where a string of length `L` can have exactly `n` distinct substrings.
   - **Depth-First Search (DFS) with Pruning**: For each candidate `L`, use DFS to generate strings in lexicographical order. During DFS:
     - **Prune Early**: If the current count of distinct substrings exceeds `n`, skip further extension.
     - **Upper Bound Check**: Calculate the maximum possible distinct substrings achievable with the remaining characters. If even the maximum plus the current count is less than `n`, prune the branch.
   - **Efficient Counting**: For each candidate string extension, compute new distinct substrings introduced by appending a character. This involves checking all suffixes ending at the new character against existing substrings.

3. **Complexity Analysis**: The minimal `L` for `n ≤ 300` is at most 24 (since `24*25/2 = 300`). The DFS with pruning ensures that only feasible paths are explored, making the solution efficient for the given constraints.

### Solution Code
```cpp
#include <iostream>
#include <set>
#include <string>
#include <vector>

using namespace std;

string dfs(string s, int count, int L, int n, set<string>& curSet) {
    if (static_cast<int>(s.size()) == L) {
        if (count == n) {
            return s;
        }
        return "";
    }

    for (char c = 'a'; c <= 'z'; c++) {
        vector<string> newSubs;
        for (int i = 0; i <= static_cast<int>(s.size()); i++) {
            string candidate = s.substr(i) + c;
            if (curSet.find(candidate) == curSet.end()) {
                newSubs.push_back(candidate);
            }
        }
        int add = newSubs.size();
        int new_count = count + add;
        if (new_count > n) {
            continue;
        }

        int k = L - static_cast<int>(s.size()) - 1;
        if (k > 0) {
            long long max_add = static_cast<long long>(k) * (2 * static_cast<int>(s.size()) + 1 + k) / 2;
            if (new_count + max_add < n) {
                continue;
            }
        }

        for (string& sub : newSubs) {
            curSet.insert(sub);
        }

        string res;
        if (k == 0) {
            if (new_count == n) {
                res = s + c;
            } else {
                res = "";
            }
        } else {
            res = dfs(s + c, new_count, L, n, curSet);
        }

        for (string& sub : newSubs) {
            curSet.erase(sub);
        }

        if (res != "") {
            return res;
        }
    }
    return "";
}

int main() {
    int n;
    cin >> n;

    for (int L = 1; ; L++) {
        set<string> curSet;
        string res = dfs("", 0, L, n, curSet);
        if (res != "") {
            cout << res << endl;
            return 0;
        }
    }
}
```

### Explanation
1. **Initialization**: The main function reads `n` and iterates over possible string lengths `L` starting from 1.
2. **DFS with Pruning**: For each `L`, the `dfs` function constructs strings character by character:
   - **Base Case**: When the string length reaches `L`, it checks if the count of distinct substrings equals `n`.
   - **Recursive Case**: For each candidate character (`'a'` to `'z'`), it:
     - Computes new distinct substrings introduced by appending the character.
     - Prunes branches where the count exceeds `n` or where the remaining characters cannot achieve `n` even in the best case.
     - Recursively explores valid extensions while backtracking to maintain state.
3. **Efficiency**: The pruning strategies ensure that only promising paths are explored, leveraging the problem constraints (`n ≤ 300`) to keep the solution feasible within time limits. The lexicographical order of character generation ensures the first valid string found is the smallest. 

This approach efficiently narrows down the search space using mathematical bounds and backtracking, ensuring optimal performance for the given problem constraints.
