Looking at this problem, I need to find a Hamiltonian cycle in a graph where:
- Each person is a vertex
- Friendship is an undirected edge
- Person 1 starts and ends the cycle
- Each person is visited exactly once (except for person 1 at start/end)

The key constraint is that each person has at least ⌈(N+1)/2⌉ friends. This is similar to Dirac's theorem for Hamiltonian cycles, which states that if each vertex has degree ≥ N/2, then the graph is Hamiltonian. Here we have ⌈(N+1)/2⌉ ≥ N/2, so a Hamiltonian cycle should always exist.

However, I still need to actually find the cycle. Since N ≤ 1000, I can use backtracking or DFS to find the Hamiltonian cycle starting from vertex 1.

The approach:
1. Build an adjacency list representation of the graph
2. Use DFS/backtracking to find a Hamiltonian cycle starting from node 1
3. Since the problem guarantees (by the degree condition) that a solution exists, I can try to find any valid cycle

Given the constraints (N ≤ 1000), but worst-case backtracking would be O(N!), I need a smarter approach. However, since the graph is dense (each node has at least ⌈(N+1)/2⌉ neighbors), a greedy DFS should find a solution quickly.

Let me implement backtracking with pruning:
- Start from node 1
- Try to extend the path by visiting unvisited neighbors
- If I reach a path of length N and the last node connects back to 1, we found a solution
- Use backtracking to try all possibilities

But worst-case could still be slow. Since the graph is dense due to the degree condition, I can use a more efficient approach: try greedy extension first (always extend with an unvisited neighbor), and if stuck, backtrack.

Actually, given N ≤ 1000 but the degree condition (at least ~N/2 friends), the graph is quite dense, so backtracking should work well in practice.

Let me implement DFS with backtracking:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int N;
    cin >> N;
    
    vector<vector<int>> graph(N + 1);
    vector<bool> visited(N + 1, false);
    vector<int> path;
    
    // Read adjacency list
    for (int i = 1; i <= N; i++) {
        int friend_node;
        while (cin >> friend_node) {
            graph[i].push_back(friend_node);
        }
    }
    
    // DFS to find Hamiltonian cycle
    path.push_back(1);
    visited[1] = true;
    
    function<bool()> dfs = [&]() -> bool {
        if (path.size() == N) {
            // Check if last node connects back to 1
            for (int neighbor : graph[path.back()]) {
                if (neighbor == 1) {
                    path.push_back(1);
                    return true;
                }
            }
            return false;
        }
        
        int current = path.back();
        for (int next : graph[current]) {
            if (!visited[next]) {
                visited[next] = true;
                path.push_back(next);
                
                if (dfs()) {
                    return true;
                }
                
                path.pop_back();
                visited[next] = false;
            }
        }
        
        return false;
    };
    
    if (dfs()) {
        for (int i = 0; i < path.size(); i++) {
            if (i > 0) cout << " ";
            cout << path[i];
        }
        cout << endl;
    } else {
        cout << "No solution" << endl;
    }
    
    return 0;
}
```
