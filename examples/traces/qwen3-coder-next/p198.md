To solve this problem, we need to determine whether a circular ship (Kolobok) can escape from an archipelago composed of circular island parts. The ship can move along any trajectory but must not intersect any island (though touching is allowed). The solution involves checking if the ship's center can reach infinity without entering the interior of any expanded island circles (where each island's radius is increased by the ship's radius).

### Approach
1. **Expand Island Circles**: For each island circle, increase its radius by the ship's radius. This transforms the problem into checking if the ship's center (a point) can move to infinity without entering the interior of any of these expanded circles.
2. **Check Initial Position**: Verify if the ship's starting position is inside any expanded circle. If it is, the ship is already trapped, so the answer is "NO".
3. **Angular Intervals Calculation**: For each expanded circle, compute the angular interval (from the ship's starting position) that the circle covers. This interval represents the directions in which a ray from the starting position would intersect the circle.
4. **Merge Intervals**: Merge overlapping or contiguous angular intervals to determine the total coverage around the starting position.
5. **Check for Escape Path**: If the merged intervals do not cover the full circle (0 to 2π), there exists a direction where the ship can move to infinity without intersecting any island, so the answer is "YES". Otherwise, the ship is trapped in a bounded region, and the answer is "NO".

        double delta = asin(r / d);
        double a = base - delta;
        double b = base + delta;

        a = fmod(a, 2*PI);
        if (a < 0) a += 2*PI;
        b = fmod(b, 2*PI);
        if (b < 0) b += 2*PI;

        if (a <= b) {
            intervals.push_back(make_pair(a, b));
        } else {
            intervals.push_back(make_pair(a, 2*PI));
            intervals.push_back(make_pair(0, b));
        }
    }

    sort(intervals.begin(), intervals.end());

    double current_end = 0.0;
