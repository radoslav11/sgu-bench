## Problem Understanding

1. We need to manage ice-cream inventory with different prices
2. Two types of operations:
   - `ARRIVE n c`: Add n pieces of ice-cream, each costing c
   - `BUY n t`: A student wants to buy n pieces with t money
3. For BUY operations:
   - We sell the n cheapest pieces we have (if possible)
   - Student gets happy only if total cost of n cheapest pieces ≤ t
4. We need to output HAPPY or UNHAPPY for each BUY query

## Algorithm Design

1. We need to efficiently:
   - Insert ice-cream batches (with price and quantity)
   - Extract the cheapest ice-cream pieces one by one (or in batches)
   - Calculate the sum of the n cheapest pieces

2. Data structure choices:
   - We can use a min-heap (priority queue) to store ice-cream by price
   - But we also need to handle quantities efficiently
   - Better approach: use a priority queue of pairs (price, quantity)
   - For each BUY query, we might need to take from multiple batches

3. Implementation approach:
   - Use `priority_queue<pair<long long, long long>, vector<pair<long long, long long>>, greater<>>` (min-heap)
   - Each element: (price, quantity)
   - For BUY n t:
     * Try to take n cheapest pieces by popping from the heap
     * Calculate the total cost
     * If total cost ≤ t: update the inventory (put back remaining pieces) and output HAPPY
     * Otherwise: put back all taken pieces and output UNHAPPY

4. Edge cases:
   - Not enough ice-cream total
   - Need to handle the case where we might partially take from a batch

## Complexity Analysis

- Each ARRIVE: O(log k) where k is number of different price batches
- Each BUY: In worst case O(k log k) where k is number of different batches
- Since total queries ≤ 10^5, and each batch can be split at most once per BUY query, overall complexity should be acceptable

#include <iostream>
#include <queue>
#include <vector>
#include <string>
#include <sstream>

using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    // Min-heap: (price, quantity)
    priority_queue<pair<long long, long long>, 
                   vector<pair<long long, long long>>, 
                   greater<pair<long long, long long>>> pq;
    
    string line;
    while (getline(cin, line)) {
        if (line.empty()) continue;
        
        istringstream iss(line);
        string op;
        iss >> op;
        
        if (op == "ARRIVE") {
            long long n, c;
            iss >> n >> c;
            pq.emplace(c, n);
        } else if (op == "BUY") {
            long long n, t;
            iss >> n >> t;
            
            // First, check if we have enough pieces and if total cost <= t
            long long pieces_left = n;
            long long total_cost = 0;
            bool can_buy = true;
            
            // Create a copy of the queue to check
            vector<pair<long long, long long>> temp;
            while (pieces_left > 0 && !pq.empty()) {
                auto [price, quantity] = pq.top();
                pq.pop();
                
                long long take = min(pieces_left, quantity);
                if (total_cost + take * price > t) {
                    can_buy = false;
                    temp.emplace_back(price, quantity);
                    while (!pq.empty()) {
                        temp.push_back(pq.top());
                        pq.pop();
                    }
                    break;
                }
                
                total_cost += take * price;
                pieces_left -= take;
                
                if (quantity > take) {
                    temp.emplace_back(price, quantity - take);
                }
            }
            
            if (pieces_left > 0) {
                can_buy = false;
                while (!pq.empty()) {
                    temp.push_back(pq.top());
                    pq.pop();
                }
            }
            
            // Restore the original state
            for (auto& batch : temp) {
                pq.push(batch);
            }
            temp.clear();
            
            if (can_buy && total_cost <= t) {
                // Actually remove the pieces
                pieces_left = n;
                total_cost = 0;
                while (pieces_left > 0) {
                    auto [price, quantity] = pq.top();
                    pq.pop();
                    
                    long long take = min(pieces_left, quantity);
                    pieces_left -= take;
                    
                    if (quantity > take) {
                        pq.emplace(price, quantity - take);
                    }
                }
                cout << "HAPPY\n";
            } else {
                cout << "UNHAPPY\n";
            }
        }
    }
    
    return 0;
}
