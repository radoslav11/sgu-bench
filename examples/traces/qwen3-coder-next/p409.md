### Approach
1. **Special Cases Handling**:
   - **K = 0**: All cells are red ('.'), so output a grid of all dots.
   - **K = N²**: All cells are blue ('*'), so output a grid of all stars.
   - **K = 1**: Use a permutation-based approach where each block (i, j) has a blue cell at position (i*N + j, j*N + i).
   - **K = N² - 1**: Complement the solution for K = 1 by flipping all cells (blue becomes red and vice versa).
   - **K = N**: For each block (i, j), place blue cells at (x, (x + i + j) mod N) for all x in [0, N-1].
   - **K = N² - N**: Complement the solution for K = N.

2. **General Case (Other K values)**:
   - For each block (i, j), place blue cells at (i*N + x, j*N + y) where y = (x + i + j) mod N and y < K. This ensures each block, row, and column meets the K blue cells requirement.
