We are to find an integer n such that:
  n = (sum of its positive integer predecessors) / x

Note: The positive integer predecessors of n are 1, 2, 3, ..., n-1.
The sum of the first (n-1) positive integers is: S = (n-1)*n/2.

So we have:
  n = S / x   =>   n = [ (n-1)*n/2 ] / x

Assuming n != 0 (which it isn't because n must be positive and at least 2 to have predecessors),
we can divide both sides by n:
  1 = (n-1)/(2x)   =>   n-1 = 2x   =>   n = 2x + 1

But wait: let's check with the examples.

Example 1: x = 1 -> n = 2*1+1 = 3.
  Predecessors of 3: 1,2 -> sum = 3. Then 3 / 1 = 3 -> which equals n. Correct.

Example 2: x = 2 -> n = 2*2+1 = 5.
  Predecessors of 5: 1,2,3,4 -> sum = 10. Then 10 / 2 = 5 -> which equals n. Correct.

However, note that the problem says "an integer number, which is x times less than the sum". 
This means: sum = n * x.

So the solution is n = 2x + 1.

We have: (n-1)*n/2 = n * x
=> n(n-1) = 2*n*x
=> n^2 - n = 2nx
=> n^2 - n - 2nx = 0
=> n^2 - (2x+1)n = 0
=> n(n - (2x+1)) = 0
